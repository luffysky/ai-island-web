### 2. UI 元件庫（2026 主流選擇）

```text
🟢 shadcn-vue（最熱、2025-2026 主流）
   - 從 React 的 shadcn/ui port 過來
   - copy-paste 元件、不是 npm 包（程式碼歸你）
   - Tailwind 為底、超彈性
   - 適合：個人 / 新創、想自訂

🟢 Element Plus（中國老牌、企業愛用）
   - 阿里出品、後台 / 表單最強
   - 大公司 / 中國市場標配
   - 適合：後台系統、enterprise

🟢 Naive UI（TS first、設計現代）
   - 完全 TypeScript、型別滿分
   - 設計簡潔現代
   - 適合：新專案、TS 重度使用

🟢 PrimeVue（巨大、enterprise）
   - 80+ 元件、最完整
   - 跨框架版本（React、Angular 也有）
   - 適合：超大型 enterprise

🟡 Vuetify（Material design）
   - Google Material 風格
   - 適合：Android-like UI、不愛 Tailwind

🟡 Quasar（含 mobile build）
   - 一套 code 出 Web + Android + iOS + Electron
   - 適合：跨平台需求

🟡 Ant Design Vue（阿里出）
   - Ant Design 風格、中國後台常見
   - 適合：跟 Ant Design React 配對

🟡 Tailwind UI（付費、最美）
   - Tailwind 官方付費元件庫（HTML、要自己包 Vue）
   - 適合：商業 SaaS、想設計美
```

**選擇建議**：
- 想學最新 → **shadcn-vue**
- 做後台系統 → **Element Plus** 或 **Naive UI**
- 大企業專案 → **PrimeVue**
- 跨平台 → **Quasar**

  
### 3. 動畫 / 過場

```text
🎬 GSAP（GreenSock Animation Platform）
   - 業界最強動畫庫、付費 / 商業需要授權
   - Apple 等級的動畫
   - 適合：Landing page、行銷網站

🎬 Motion One
   - Web Animations API 的優雅 wrapper
   - 輕量、開源、效能好
   - 適合：大部分 App

🎬 AutoAnimate（FormKit）
   - 一行加在 element、列表新增 / 刪除自動有動畫
   - 零配置、瞬間有質感
   - 適合：列表 / Todo / 任何 v-for

🎬 Vue Transition（內建）
   - <Transition> / <TransitionGroup>
   - 元件進場 / 退場用 CSS class 控制
   - 適合：簡單切換動畫

🎬 Lottie + lottie-web
   - 設計師用 After Effects 做動畫、Web 直接播
   - 適合：複雜 illustration 動畫
```

### 10. 5 個學習常犯錯

```text
❌ 1. 只看教程不寫 code
   → 看 100 小時不如自己寫 10 小時
   → 修：看 20 分鐘必須關掉、自己打一遍

❌ 2. 想學完所有東西才開始做 project
   → 永遠學不完、永遠不敢開始
   → 修：基礎課（前 5 課）走完就動手、邊做邊查

❌ 3. project 做半年都不上線
   → 一直「再改一點點」、永遠 99%
   → 修：強制 2 個月內必上線、再爛都先發

❌ 4. 不寫 GitHub README
   → 面試官 / 客戶看不懂你 project 在幹嘛
   → 修：README 寫：截圖 + 功能 + 技術 stack + Demo 連結

❌ 5. 不參加社群
   → 一個人卡 bug 卡 3 天
   → 修：加 Vue.js 台灣 Discord、卡關 5 分鐘就問
```
