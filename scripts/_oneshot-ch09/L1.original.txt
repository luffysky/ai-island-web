Vue 由前 Google 工程師**尤雨溪**（Evan You）2014 年發布。他從 AngularJS 學到 declarative rendering，從 React 學到 component 化，提煉成「**容易上手、漸進式採用**」的框架。

  

React 像**樂高散裝**，給你積木自己拼，自由但要選很多。Vue 像**樂高套裝**，幫你配好了路由（Vue Router）、狀態（Pinia）、CLI，按指引走就行。

  

  
### Vue 3 的革命（2020+）

  
- **Composition API**：取代 Options API，更接近 React Hooks（但更好用）
- **Proxy-based Reactivity**：取代 Object.defineProperty，效能跟靈活性大躍進
- **TypeScript 一等公民**：原生支援，型別推導好
- **Vite**：尤雨溪自己做的 build tool，秒速啟動
- **更小 bundle**：tree-shaking 友善

### 2026 Vue 生態

  
- **核心**：Vue 3.5+
- **路由**：Vue Router 4
- **狀態**：Pinia（官方推薦，取代 Vuex）
- **全端**：Nuxt 3+
- **UI**：Element Plus、Naive UI、Vuetify、Quasar、PrimeVue
- **build**：Vite（預設）
