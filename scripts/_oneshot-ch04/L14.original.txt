### 串流回應

```js
// 處理大型回應（檔案下載、AI 串流）
const r = await fetch("/api/ai-stream");
const reader = r.body.getReader();
const decoder = new TextDecoder();

while (true) {
  const { value, done } = await reader.read();
  if (done) break;
  const chunk = decoder.decode(value);
  console.log(chunk); // AI 回應一塊塊出來
}
```
