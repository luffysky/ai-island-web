**Web Components — 自訂 HTML 元素**。把一堆 HTML + CSS + JS 打包成「一個自訂標籤」、別人寫 `<my-button>` 就能用、像 React 元件但**不依賴任何框架**。

> 💡 **2026 為什麼重要**：React / Vue 元件只能在自己生態用、Web Component **跨框架共用**——你公司同時用 React 跟 Vue、Web Component 是唯一能共用的 UI library 方式。Shopify / GitHub / YouTube 都大量用。

  
### 1. Web Components 四大支柱

```text
🎨 Custom Elements      — 自訂標籤（<my-button>）
🌳 Shadow DOM           — 樣式 / 結構封閉、不被外面 CSS 影響
📦 HTML Templates       — <template> + <slot>
📥 ES Modules           — JS 載入機制
```

  
### 2. 最簡單的 Custom Element

```text
<!DOCTYPE html>
<html>
<head><meta charset="UTF-8"><title>My Web Component</title></head>
<body>
  <my-greeting name="林董"></my-greeting>

  <script>
    class MyGreeting extends HTMLElement {
      connectedCallback() {
        const name = this.getAttribute('name') ?? '路人';
        this.innerHTML = `<h2>哈囉、${name}！</h2>`;
      }
    }
    customElements.define('my-greeting', MyGreeting);
  </script>
</body>
</html>
```

**規則：**

```text
✅ 自訂標籤名「必須含 -」
   ✓ my-button / app-header
   ✗ mybutton（被當未知標籤）

✅ 必繼承 HTMLElement

✅ 用 customElements.define('名字', class) 註冊
```

  
### 3. 生命週期 4 callback

```text
class MyButton extends HTMLElement {
  constructor() { super(); /* 元素一建立 */ }

  connectedCallback() {
    /* 加進 DOM 時跑、最常用 */
  }

  disconnectedCallback() {
    /* 從 DOM 拿掉時跑、清 timer / listener */
  }

  static get observedAttributes() { return ['count']; }

  attributeChangedCallback(name, oldVal, newVal) {
    /* 屬性改變時跑 */
  }
}
```

  
### 4. Shadow DOM — 樣式封閉

**問題**：寫 `<my-button>` 內樣式會被外面 CSS 蓋（或反過來污染）。

**解法**：Shadow DOM。

```text
class MyCard extends HTMLElement {
  connectedCallback() {
    const shadow = this.attachShadow({ mode: 'open' });
    shadow.innerHTML = `
      <style>
        .card { padding: 16px; border: 1px solid #ddd; border-radius: 8px; }
        h3 { color: #10b981; margin: 0; }
      </style>
      <div class="card">
        <h3><slot name="title">預設標題</slot></h3>
        <p><slot>預設內容</slot></p>
      </div>
    `;
  }
}
customElements.define('my-card', MyCard);
```

**用法：**

```text
<my-card>
  <span slot="title">AI 島</span>
  從 0 到 1 學寫程式
</my-card>
```

**`<slot>`** = 讓使用者塞自訂內容進來、像 React 的 `{children}`。

  
### 5. 屬性 + 屬性反映（attribute reflection）

```text
class Counter extends HTMLElement {
  static get observedAttributes() { return ['count']; }

  connectedCallback() {
    this.render();
    this.addEventListener('click', () => {
      this.count = (Number(this.count) || 0) + 1;
    });
  }

  attributeChangedCallback() { this.render(); }

  get count() { return this.getAttribute('count'); }
  set count(v) { this.setAttribute('count', v); }

  render() {
    this.innerHTML = `<button>點 ${this.count ?? 0} 次</button>`;
  }
}
customElements.define('my-counter', Counter);
```

**用法：**

```text
<my-counter count="0"></my-counter>
```

點一下計數 +1、屬性自動更新、attributeChangedCallback 跑 render。

  
### 6. 完整實戰：可重用的 Toast 通知元件

📄 **寫進程式檔**：`toast.html`

```text
<!DOCTYPE html>
<html>
<head><meta charset="UTF-8"><title>Toast Component</title></head>
<body>
  <button id="trigger">顯示成功 toast</button>

  <script>
    class AppToast extends HTMLElement {
      connectedCallback() {
        const type = this.getAttribute('type') ?? 'info';
        const colors = { success: '#10b981', error: '#ef4444', info: '#3b82f6' };
        const shadow = this.attachShadow({ mode: 'open' });
        shadow.innerHTML = `
          <style>
            :host {
              position: fixed; top: 20px; right: 20px;
              padding: 12px 16px;
              background: ${colors[type]};
              color: white;
              border-radius: 6px;
              animation: slide-in 0.3s ease-out;
              z-index: 9999;
            }
            @keyframes slide-in {
              from { transform: translateX(100%); }
              to { transform: translateX(0); }
            }
          </style>
          <slot>通知</slot>
        `;
        // 3 秒後自動消失
        setTimeout(() => this.remove(), 3000);
      }
    }
    customElements.define('app-toast', AppToast);

    document.getElementById('trigger').addEventListener('click', () => {
      const toast = document.createElement('app-toast');
      toast.setAttribute('type', 'success');
      toast.textContent = '✅ 儲存成功';
      document.body.appendChild(toast);
    });
  </script>
</body>
</html>
```

**`:host`** = Shadow DOM 內指「這個 component 本身」的選擇器、外面看不到、樣式不會污染。

  
### 7. 跟 React / Vue 怎麼配？

```text
✅ React 直接用 Web Component
   <my-card title="嗨"></my-card>
   → React 18+ 完整支援

✅ Vue 直接用 Web Component
   <my-card :title="title"></my-card>
   → Vue 3 完整支援

✅ 純 HTML 也能用
   → 這就是 Web Component 的最大優勢
```

**生態系：**

```text
🛠️ Lit — Google 出品、Web Component 的 React（更好用）
🛠️ Stencil — Ionic 出品、編譯成框架無關元件
🛠️ Shoelace — 預製 UI library、可直接用
```

  
### 8. 5 大常見坑

```text
❌ 1. 標籤名沒有「-」
   → 例 <mybutton> 不行
   → 修：永遠用 kebab-case（my-button）

❌ 2. constructor 內操作 attribute
   → 元素還沒加進 DOM、attribute 可能還沒設
   → 修：移到 connectedCallback

❌ 3. Shadow DOM 內 form 元素被截斷
   → 表單 submit 不會帶到外面的 <form>
   → 修：用 ElementInternals + formAssociated

❌ 4. observedAttributes 沒寫
   → 屬性改了 attributeChangedCallback 不會跑
   → 修：static get observedAttributes() { return [...]; }

❌ 5. SSR 不支援 Shadow DOM
   → Next.js / Nuxt SSR 看不到 shadow 內容
   → 修：用 Declarative Shadow DOM（2026 已支援）
```
