**JSON-LD 結構化資料**——給 Google + AI 引擎（ChatGPT / Perplexity / Bing Copilot）看的「結構化說明書」。讓你的網頁有機會出現在「**富片段（Rich Snippet）**」：評分星星、價格、FAQ 直接在搜尋結果顯示。

> 💡 **2026 為什麼超重要**：傳統 SEO 已經卷不動了、但 GEO（Generative Engine Optimization、給 AI 引擎優化）才剛開始。**AI 引擎抓網頁、第一個看的就是 JSON-LD**——寫得好你的內容會被 ChatGPT / Perplexity 引用、不寫的人完全沒戲。

  
### 1. 什麼是 JSON-LD？

```text
JSON-LD = JSON for Linked Data
→ 用 JSON 格式描述「這頁是什麼」
→ 放在 <script type="application/ld+json"> 內
→ 機器讀、人看不到
→ Google / AI 引擎優先吃這個
```

**vs Microdata / RDFa：**

```text
Microdata（舊）：<div itemscope itemtype="...">
RDFa（舊）：<div property="...">
JSON-LD（新）：<script type="application/ld+json">{...}</script>

→ Google 官方推薦 JSON-LD（其他兩個是歷史包袱）
→ JSON-LD 最大優點：跟 HTML 解耦、不汙染標籤
```

  
### 2. 最基本的 JSON-LD（文章頁）

放在 `<head>` 或 `<body>` 內都行：

```text
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "為什麼 React 比 Vue 紅？",
  "author": {
    "@type": "Person",
    "name": "Luffy Lin"
  },
  "datePublished": "2026-05-28",
  "dateModified": "2026-05-28",
  "image": "https://ai-island-web.snowrealm.pet/blog/react-vs-vue.png"
}
</script>
```

**5 大欄位：**

```text
@context  必填、永遠是 "https://schema.org"
@type     這頁是什麼類型（Article / Product / FAQPage / ...）
其他欄位  依 @type 而定

→ 完整 schema 看 https://schema.org/
```

  
### 3. 4 種最有用的 @type

  
### 3-1. `Article` — 部落格 / 新聞

```text
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Next.js 14 vs Remix 全面比較",
  "image": ["https://example.com/cover.jpg"],
  "datePublished": "2026-05-28T08:00:00+08:00",
  "dateModified": "2026-05-28T08:00:00+08:00",
  "author": [{
    "@type": "Person",
    "name": "Luffy Lin",
    "url": "https://ai-island-web.snowrealm.pet/u/luffysky"
  }],
  "publisher": {
    "@type": "Organization",
    "name": "AI 島",
    "logo": {
      "@type": "ImageObject",
      "url": "https://ai-island-web.snowrealm.pet/logo.png"
    }
  }
}
</script>
```

**搜尋結果效果**：作者名 + 發布日 + 來源 logo 顯示。

  
### 3-2. `Product` — 商品 / 服務

```text
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "AI 島 Pro 年費",
  "image": "https://example.com/pro.png",
  "description": "75 章全開 + 無限 AI 對話 + 優先支援",
  "brand": { "@type": "Brand", "name": "AI 島" },
  "offers": {
    "@type": "Offer",
    "url": "https://ai-island-web.snowrealm.pet/pricing",
    "priceCurrency": "TWD",
    "price": "2990",
    "availability": "https://schema.org/InStock"
  },
  "aggregateRating": {
    "@type": "AggregateRating",
    "ratingValue": "4.8",
    "reviewCount": "247"
  }
}
```

**搜尋結果效果**：⭐⭐⭐⭐⭐ 4.8（247） · NT$2,990 直接顯示。**點擊率比沒寫高 35%**。

  
### 3-3. `FAQPage` — FAQ 頁

```text
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "AI 島可以免費試用嗎？",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "可以、註冊就能用免費版前 10 章。Pro 版 7 天試用。"
    }
  }, {
    "@type": "Question",
    "name": "支援哪些付款方式？",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "信用卡、綠界、LINE Pay。"
    }
  }]
}
```

**搜尋結果效果**：問答直接展開在 Google 結果頁、占整頁 1/3 高度。

  
### 3-4. `BreadcrumbList` — 麵包屑

```text
{
  "@context": "https://schema.org",
  "@type": "BreadcrumbList",
  "itemListElement": [{
    "@type": "ListItem",
    "position": 1,
    "name": "首頁",
    "item": "https://ai-island-web.snowrealm.pet/"
  }, {
    "@type": "ListItem",
    "position": 2,
    "name": "章節",
    "item": "https://ai-island-web.snowrealm.pet/chapters"
  }, {
    "@type": "ListItem",
    "position": 3,
    "name": "Ch01 HTML 完整"
  }]
}
```

**搜尋結果效果**：「首頁 > 章節 > Ch01」這條路徑取代 URL 顯示。

  
### 4. 完整範例：課程頁

📄 **寫進程式檔**：`chapter-01.html`

```text
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>HTML 完整 25 課 — AI 島 Ch01</title>

  <!-- JSON-LD：課程 + 麵包屑 -->
  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "Course",
    "name": "HTML 完整",
    "description": "25 課從 HTML 基礎到 SEO + 無障礙、新手友善",
    "provider": {
      "@type": "Organization",
      "name": "AI 島",
      "url": "https://ai-island-web.snowrealm.pet"
    },
    "offers": {
      "@type": "Offer",
      "category": "education",
      "priceCurrency": "TWD",
      "price": "0"
    },
    "hasCourseInstance": {
      "@type": "CourseInstance",
      "courseMode": "online",
      "courseWorkload": "PT25H"
    }
  }
  </script>

  <script type="application/ld+json">
  {
    "@context": "https://schema.org",
    "@type": "BreadcrumbList",
    "itemListElement": [
      { "@type": "ListItem", "position": 1, "name": "首頁", "item": "https://ai-island-web.snowrealm.pet/" },
      { "@type": "ListItem", "position": 2, "name": "章節", "item": "https://ai-island-web.snowrealm.pet/chapters" },
      { "@type": "ListItem", "position": 3, "name": "Ch01 HTML 完整" }
    ]
  }
  </script>
</head>
<body>
  <h1>Ch01 HTML 完整 25 課</h1>
  <!-- 內容 -->
</body>
</html>
```

> 💡 **一頁可以放多個 JSON-LD `<script>`**：商品頁可以放 `Product` + `BreadcrumbList` + `FAQPage` 三個。Google 全部讀。


### 6. AI 引擎（GEO）特別吃這幾個

```text
2026 AI 引擎優先讀：
✅ Article（文章）
✅ FAQPage（FAQ）— ChatGPT / Perplexity 引用最多
✅ HowTo（步驟教學）
✅ Recipe（食譜）
✅ Product + Review（商品評論）

→ 你的內容是上述類型、補 JSON-LD = 被 AI 引用機率大幅提高
```

> 💡 **GEO 趨勢**：Google 流量逐漸轉到 AI 答案、SEO 流量天花板可預見。**早寫 JSON-LD 早被 AI 引擎引用、未來 1-2 年是黃金窗口**。

  
### 7. 5 大常見坑

```text
❌ 1. JSON 語法錯（多逗號 / 缺引號）
   → Google 直接忽略整個 script
   → 修：用 validator.schema.org 驗證每次改完

❌ 2. @type 名稱拼錯
   → 例 "Articles" / "FAQ"（正確：Article / FAQPage）
   → 修：複製貼上 schema.org 官方範例

❌ 3. 圖片 URL 用相對路徑
   → 必須 https:// 完整 URL
   → 修：永遠絕對路徑

❌ 4. 日期格式錯
   → 必須 ISO 8601（2026-05-28T08:00:00+08:00）
   → 修：JS 用 .toISOString()、PHP 用 date('c')

❌ 5. 寫了但網頁上沒這內容
   → 例 JSON-LD 寫 5 顆星、頁面上沒評分 → Google 會手動懲罰
   → 修：JSON-LD 內容必須跟頁面實際內容一致
```
