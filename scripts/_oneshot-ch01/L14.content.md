**SVG（Scalable Vector Graphics）**——用「數學公式」畫的圖、放多大都不糊、用 CSS 能改顏色、檔案還超小。**2026 logo / icon / 圖表主流全是 SVG**。

> 💡 **SVG vs PNG/JPG 一張對比**：同一個 logo、SVG 1KB / PNG 50KB / JPG 80KB。SVG 放大 100 倍也清晰、PNG 放大就糊。**logo / icon 絕對用 SVG**。

  
### 1. SVG 基本長相

```text
<svg width="100" height="100" viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" fill="#10b981" />
</svg>
```

**5 個必懂屬性：**

```text
🎨 width / height — 顯示尺寸
📐 viewBox        — 內部座標系統（重要！）
   viewBox="0 0 100 100" = 左上(0,0)、右下(100,100)
   不寫死 width/height 就用 viewBox 縮放
🖌️ fill           — 填色（預設黑色）
🖋️ stroke         — 描邊色
📏 stroke-width   — 描邊粗細
```

> 💡 **`viewBox` 的價值**：viewBox 設「邏輯座標」、width/height 設「實際顯示」。一個 SVG 用 `viewBox="0 0 24 24"` 內部用 24x24 設計、外面 width 可隨便調。

  
### 2. 基本形狀

```text
<!-- 矩形 -->
<rect x="10" y="10" width="80" height="40" rx="8" fill="#10b981" />

<!-- 圓形 -->
<circle cx="50" cy="50" r="30" fill="red" />

<!-- 橢圓 -->
<ellipse cx="50" cy="50" rx="40" ry="20" fill="blue" />

<!-- 線 -->
<line x1="10" y1="10" x2="90" y2="90" stroke="black" stroke-width="2" />

<!-- 多邊形 -->
<polygon points="50,5 90,80 10,80" fill="orange" />

<!-- 自由路徑（最強大）-->
<path d="M 10 10 L 90 10 L 50 90 Z" fill="purple" />

<!-- 文字 -->
<text x="10" y="50" font-size="20">Hello</text>
```

**`<path>` 的 `d` 屬性：**

```text
M x y    Move to（移到）
L x y    Line to（畫線到）
H x      Horizontal line
V y      Vertical line
C ...    Cubic Bezier（曲線）
A ...    Arc（弧）
Z        Close path（封閉）
```

  
### 3. CSS 控制 SVG（icon 的祕密）

```text
<style>
  .icon {
    width: 24px;
    height: 24px;
    fill: currentColor;   /* 繼承文字色 */
    transition: fill 0.2s;
  }
  .icon:hover { fill: #ef4444; }
</style>

<button>
  收藏
  <svg class="icon" viewBox="0 0 24 24">
    <path d="M12 21l-1.45-1.32C5.4 15.36 2 12.28 2 8.5 2 5.42 4.42 3 7.5 3c1.74 0 3.41.81 4.5 2.09C13.09 3.81 14.76 3 16.5 3 19.58 3 22 5.42 22 8.5c0 3.78-3.4 6.86-8.55 11.18L12 21z" />
  </svg>
</button>
```

**祕訣**：`fill="currentColor"` → SVG 自動跟文字同色、配 `color: red` 切換顏色。

  
### 4. 從外面載入 SVG（4 種方法）

**方法 1：`<img>`（最簡單、不能改色）**

```text
<img src="icon.svg" alt="" width="24">
```

**方法 2：inline SVG（最彈性、可改色 + 動畫）**

```text
<svg viewBox="0 0 24 24"><path d="..." /></svg>
```

**方法 3：CSS background**

```text
.icon { background: url('icon.svg') no-repeat; }
```

**方法 4：`<use>` symbol（SVG sprite、多 icon 一檔）**

```text
<!-- 一次定義所有 icon -->
<svg style="display:none">
  <symbol id="i-heart" viewBox="0 0 24 24">
    <path d="..." />
  </symbol>
  <symbol id="i-star" viewBox="0 0 24 24">
    <path d="..." />
  </symbol>
</svg>

<!-- 用 -->
<svg width="24" height="24"><use href="#i-heart" /></svg>
<svg width="24" height="24"><use href="#i-star" /></svg>
```

> 💡 **大量 icon 用 sprite**：100 個 icon 用 sprite 1 個 HTTP 請求、用 `<img>` 100 個請求。

  
### 5. 動畫（SMIL + CSS + JS 三種）

**CSS 動畫（最常用）：**

```text
<svg viewBox="0 0 100 100">
  <style>
    .spinner {
      transform-origin: 50% 50%;
      animation: rotate 1s linear infinite;
    }
    @keyframes rotate {
      to { transform: rotate(360deg); }
    }
  </style>
  <circle class="spinner" cx="50" cy="50" r="40" fill="none" stroke="#10b981" stroke-width="6" stroke-dasharray="60 30" />
</svg>
```

**SMIL（SVG 內建）：**

```text
<circle cx="50" cy="50" r="20" fill="red">
  <animate attributeName="r" values="20;40;20" dur="2s" repeatCount="indefinite" />
</circle>
```

  
### 6. 實戰：客製化 loading spinner

📄 **寫進程式檔**：`spinner.html`

```text
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>Custom Spinner</title>
  <style>
    body { display: flex; justify-content: center; align-items: center; min-height: 100vh; background: #0a0a0a; }
    .loader {
      width: 80px; height: 80px;
      animation: spin 1s linear infinite;
    }
    @keyframes spin { to { transform: rotate(360deg); } }
    .loader circle {
      fill: none;
      stroke: #10b981;
      stroke-width: 6;
      stroke-linecap: round;
      stroke-dasharray: 200;
      stroke-dashoffset: 50;
    }
  </style>
</head>
<body>
  <svg class="loader" viewBox="0 0 80 80">
    <circle cx="40" cy="40" r="30" />
  </svg>
</body>
</html>
```

### 8. 5 大常見坑

```text
❌ 1. 沒寫 viewBox
   → SVG 不能 scale、寫死大小就鎖死
   → 修：永遠寫 viewBox="0 0 W H"

❌ 2. 用 width/height 寫死、又想響應式
   → 寫 viewBox 後 width 可設 100% 自適應
   → 修：<svg viewBox="..." width="100%" height="auto">

❌ 3. fill 寫死顏色、想 hover 改色
   → fill="red" 就不能透過 CSS 改
   → 修：fill="currentColor" + 用 color 控制

❌ 4. SVG 內 <style> 用 :hover 但是 <img> 載入
   → <img> 是黑盒、CSS 無效
   → 修：改用 inline SVG

❌ 5. SVG 檔案沒最佳化、超肥
   → Illustrator 匯出的 SVG 一堆冗餘
   → 修：用 SVGO（https://jakearchibald.github.io/svgomg/）壓
```

  
### 🎯 重點回顧

- **SVG = 向量圖、放大不糊、檔案小、CSS 可控**
- **`viewBox` 是內部座標、`width/height` 是顯示尺寸**
- **`fill="currentColor"` 讓 SVG 跟文字同色**
- **大量 icon 用 `<symbol>` + `<use>` sprite**
- **icon 庫**：Lucide / Heroicons / Phosphor / Font Awesome
