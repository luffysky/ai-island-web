**嵌入媒體進階：iframe / embed / video / audio / object**——L6 教過基本、這課深入「**第三方嵌入**」實戰：YouTube / Google Maps / Spotify / Twitter / Typeform 怎麼嵌得安全 + 快 + 響應式。

> 💡 **iframe 是把雙面刃**：方便（一行嵌 YouTube）、但風險高（cross-site scripting / 效能 / 隱私）。**寫對 sandbox + lazy + loading="lazy" + size + title 才安全**。

  
### 1. iframe 基本 + 安全屬性

```text
<iframe
  src="https://www.youtube.com/embed/VIDEO_ID"
  width="560"
  height="315"
  title="YouTube — 影片說明"
  loading="lazy"
  allow="autoplay; encrypted-media; picture-in-picture"
  allowfullscreen
  referrerpolicy="strict-origin-when-cross-origin"
  sandbox="allow-scripts allow-same-origin allow-presentation"
></iframe>
```

**8 個重要屬性：**

```text
🌐 src           → 嵌入內容 URL（必填）
📏 width/height  → 尺寸
📝 title         → a11y 必加（螢幕閱讀器念）
😴 loading="lazy" → 滾到才載（效能必加）

🎬 allow         → 開放權限
   autoplay / encrypted-media / picture-in-picture / fullscreen / camera / microphone

🛡️ sandbox       → 安全沙箱（白名單能力）
   allow-scripts        → 允許 JS
   allow-same-origin    → 允許同源 cookie / storage
   allow-forms          → 允許 form submit
   allow-popups         → 允許 window.open
   allow-presentation   → 允許全螢幕

🔍 allowfullscreen → 可全螢幕（影片必加）

🔗 referrerpolicy → 控制 Referer header
   strict-origin-when-cross-origin（推薦預設）
```

> ⚠️ **`sandbox=""` 空字串是「最嚴」模式**——iframe 內什麼都不能做。逐項白名單加回來。

  
### 2. YouTube 嵌入 + 響應式

**陽春版**：

```text
<iframe
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube"
  allowfullscreen
  loading="lazy"
></iframe>
```

**進階響應式（16:9）：**

```text
<div class="video-wrapper">
  <iframe
    src="https://www.youtube.com/embed/VIDEO_ID"
    title="影片標題"
    loading="lazy"
    allow="autoplay; encrypted-media; picture-in-picture"
    allowfullscreen
  ></iframe>
</div>

<style>
  .video-wrapper {
    position: relative;
    aspect-ratio: 16 / 9;     /* 現代寫法、推薦 */
    width: 100%;
  }
  .video-wrapper iframe {
    position: absolute;
    inset: 0;
    width: 100%;
    height: 100%;
    border: 0;
  }
</style>
```

**舊瀏覽器 fallback（padding-bottom hack）：**

```text
.video-wrapper {
  position: relative;
  padding-bottom: 56.25%;    /* 9/16 = 56.25% */
  height: 0;
}
```

**`youtube-nocookie.com` — 隱私版：**

```text
<iframe src="https://www.youtube-nocookie.com/embed/VIDEO_ID" ...>
```

不會塞 cookie 直到使用者點 play。**GDPR / 隱私政策友善**。

  
### 3. YouTube 進階：lite-youtube-embed（效能）

陽春嵌入 YouTube 一個 iframe ~ 500KB JS + ~200KB CSS。如果頁面有 5 個影片 = 3.5MB。

**解法**：先載「縮圖 + 假播放鍵」、使用者點才換成真 iframe。

```text
<!-- 用 lite-youtube-embed 套件 -->
<lite-youtube videoid="VIDEO_ID" playlabel="播放"></lite-youtube>

<!-- 自己寫的簡化版 -->
<div class="youtube-facade" data-id="VIDEO_ID">
  <img src="https://i.ytimg.com/vi/VIDEO_ID/hqdefault.jpg" alt="影片縮圖">
  <button aria-label="播放">▶</button>
</div>

<script>
  document.querySelectorAll('.youtube-facade').forEach(el => {
    el.addEventListener('click', () => {
      const id = el.dataset.id;
      el.innerHTML = `<iframe
        src="https://www.youtube.com/embed/${id}?autoplay=1"
        allow="autoplay; encrypted-media"
        allowfullscreen
        loading="eager"
        title="YouTube"
      ></iframe>`;
    });
  });
</script>
```

**效能差異**：首屏 3.5MB → 50KB。**Lighthouse 從 50 直接跳到 90+**。

  
### 4. Google Maps 嵌入

```text
<!-- 1. 簡單版 -->
<iframe
  src="https://www.google.com/maps/embed?pb=!1m18!..."
  width="600"
  height="450"
  title="店家位置"
  loading="lazy"
  referrerpolicy="no-referrer-when-downgrade"
  allowfullscreen
></iframe>

<!-- 2. Google Maps Embed API (要 API key) -->
<iframe
  src="https://www.google.com/maps/embed/v1/place?key=API_KEY&q=台北市+信義區+信義路+5+段+7+號"
  width="600"
  height="450"
  loading="lazy"
  allowfullscreen
></iframe>
```

> 💡 **Google Maps Embed API 免費額度**：每月 1000 次免費、夠用、要 API key 但比較穩。

  
### 5. 第三方平台嵌入

**Spotify 音樂：**

```text
<iframe
  src="https://open.spotify.com/embed/track/TRACK_ID"
  width="100%"
  height="152"
  title="Spotify"
  loading="lazy"
  allow="autoplay; clipboard-write; encrypted-media; picture-in-picture"

```

**Twitter / X 貼文：**

```text
<blockquote class="twitter-tweet">
  <a href="https://twitter.com/USERNAME/status/STATUS_ID"></a>
</blockquote>
<script async src="https://platform.twitter.com/widgets.js"></script>
```

**Typeform 表單：**

```text
<div data-tf-widget="FORM_ID" style="height:500px"></div>
<script src="//embed.typeform.com/next/embed.js"></script>
```

**CodePen：**

```text
<iframe
  src="https://codepen.io/USERNAME/embed/PEN_ID"
  title="CodePen"
  loading="lazy"
  allowfullscreen
></iframe>
```

  
### 6. `<video>` 自託管完整版

📄 **寫進程式檔**：`self-hosted-video.html`

```text
<video
  controls
  width="800"
  height="450"
  poster="/video/cover.jpg"
  preload="metadata"
  playsinline
  loading="lazy"
>
  <!-- 多 source（瀏覽器選支援的）-->
  <source src="/video/intro.av1.mp4" type="video/mp4; codecs=av01.0.05M.08">
  <source src="/video/intro.h264.mp4" type="video/mp4; codecs=avc1.42E01E">
  <source src="/video/intro.webm" type="video/webm">

  <!-- 字幕 -->
  <track kind="captions" src="/video/zh.vtt" srclang="zh" label="繁中" default>
  <track kind="captions" src="/video/en.vtt" srclang="en" label="English">

  <!-- fallback -->
  <p>瀏覽器不支援、<a href="/video/intro.mp4">直接下載</a>。</p>
</video>
```

**`preload` 3 種值：**

```text
none        什麼都不預載
metadata    預載元資料（時長 / 第一幀）← 推薦
auto        全部預載（影片大會炸頻寬）
```

  
### 7. 完整實戰：blog 內嵌 3 種媒體

```text
<article>
  <h1>什麼是 RAG</h1>

  <!-- 1. YouTube 影片（facade 版）-->
  <div class="youtube-facade" data-id="VIDEO_ID" tabindex="0" role="button" aria-label="播放介紹影片">
    <img src="https://i.ytimg.com/vi/VIDEO_ID/maxresdefault.jpg" alt="影片縮圖">
    <span class="play-btn">▶</span>
  </div>

  <p>正文...</p>

  <!-- 2. CodePen demo -->
  <iframe
    src="https://codepen.io/USERNAME/embed/PEN_ID?default-tab=result"
    title="RAG 流程示意 demo"
    height="400"
    loading="lazy"
  ></iframe>

  <p>更多正文...</p>

  <!-- 3. Spotify podcast 集數 -->
  <iframe
    src="https://open.spotify.com/embed/episode/EPISODE_ID"
    width="100%"
    height="232"
    title="Podcast 訪談"
    loading="lazy"
    allow="encrypted-media"
  ></iframe>
</article>
```

  
### 8. 5 大常見坑

```text
❌ 1. iframe 沒寫 title
   → 螢幕閱讀器念「iframe」、盲人不知道在幹嘛
   → 修：永遠加 title="描述"

❌ 2. iframe 沒 loading="lazy"
   → 一進頁面就載 YouTube 3MB JS、首屏 LCP 慘
   → 修：永遠加 loading="lazy"（首屏除外）

❌ 3. <video> 沒 controls
   → 使用者沒法控制播放、變呆畫面
   → 修：除非自製 player、不然永遠加 controls

❌ 4. 不寫 sandbox
   → 嵌入的網站可以塞 cookie / 跑 script / 釣魚
   → 修：陌生來源加 sandbox + 白名單

❌ 5. autoplay 沒配 muted
   → Chrome / Safari 直接擋
   → 修：autoplay muted loop playsinline 一起寫
```

  
### 🎯 重點回顧

- **iframe 五要素**：title / loading / sandbox / allow / referrerpolicy
- **YouTube facade pattern**：用縮圖 + 假播放鍵替代 iframe、效能跳升
- **`aspect-ratio: 16/9`** 是最現代的響應式 iframe 寫法
- **Google Maps / Spotify / Twitter / Typeform 都有官方 embed**
- **`<video>` 自託管要多 source 多 codec、配 `<track>` 字幕**
