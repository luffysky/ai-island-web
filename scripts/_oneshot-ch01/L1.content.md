**HTML（HyperText Markup Language、超文本標記語言）** = 寫網頁的「結構」。1991 年 Tim Berners-Lee 發明、原本是為了讓物理學家分享論文。**30 年後、全世界的網頁都是它寫的**。

> 💡 **這課的價值**：HTML 不是「**畫好看**」（那是 CSS）、是「**告訴瀏覽器這是什麼**」——標題、段落、列表、連結。

  
### HTML 的兩個身份

```text
1. 給人看（透過瀏覽器渲染）
   → 使用者看到的網頁畫面

2. 給機器看（透過 parser / crawler / AI）
   → Google 抓內容做 SEO
   → AI（ChatGPT / Perplexity）摘要 / 引用
   → 螢幕閱讀器（screen reader）讀給盲人聽
```

> 💡 **重點**：很多人只顧第一個身份、結果機器看不懂 → SEO 爛 / 無障礙差 / AI 摘要錯。**會寫好 HTML 的人、網頁能被找到、能被讀懂、能被 AI 引用**。這比視覺設計值錢得多。

  
### HTML 標籤（Tag）長什麼樣

```text
=== 基本語法 ===

<開始標籤>內容</結束標籤>

範例：
<h1>大標題</h1>
<p>一段文字</p>
<a href="https://example.com">連結</a>

→ < > 中間是「標籤名」
→ 大部分標籤都「成對」（開 + 關）
→ 少數「自閉合」如 <img /> / <br /> / <hr />
```

  
### 完整 HTML 範本

📄 **寫進程式檔**：`index.html`

```text
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <title>我的第一頁</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>這是我的第一個網頁</p>
</body>
</html>
```

> 💡 **這個結構**：(1) `<!DOCTYPE html>` 宣告「我是 HTML5」 (2) `<html>` 整個網頁的根 (3) `<head>` 給機器看的設定 (4) `<body>` 給人看的內容。

  
### 5 個 head 必有

```text
🔧 <!DOCTYPE html>
   → 告訴瀏覽器是 HTML5
   → 永遠第一行、不寫舊瀏覽器會用奇怪模式

🌐 <html lang="zh-Hant">
   → 語言（影響 SEO / 無障礙 / 自動翻譯）
   → 繁體 zh-Hant、簡體 zh-Hans、英文 en

📝 <meta charset="UTF-8">
   → 編碼（沒寫中文會變亂碼）
   → 永遠 UTF-8、其他別用

📱 <meta name="viewport" content="width=device-width, initial-scale=1.0">
   → 手機要這個才會正確縮放
   → 沒寫手機會看到桌面版縮小

🏷️ <title>分頁標題</title>
   → 分頁標題、Google 搜尋結果標題、社群分享預覽
   → SEO 最重要的標籤之一
```

  
### 2026 SEO + AI 加分項

```text
✨ <meta name="description" content="...">
   → Google 搜尋結果的描述文字
   → AI 抓內容會看

✨ <link rel="canonical" href="...">
   → 主要 URL、避免重複內容扣分

✨ Open Graph tags
   → og:title / og:description / og:image
   → Facebook / LINE / Discord 分享預覽

✨ JSON-LD 結構化資料
   → Google 富片段（rich snippet）
   → AI 引擎（ChatGPT / Perplexity）愛
```

> 💡 **L9 / L20 詳教 SEO meta、L10 / L21 詳教 JSON-LD**——這課先知道有就好。

### HTML 常見「自閉合」標籤

```text
=== 不用結束標籤 ===

<br>          換行
<hr>          水平線
<img src="">  圖片
<input>       表單欄位
<meta>        meta 設定
<link>        連結（CSS / icon）

→ HTML5 也可寫成 <br /> / <img />
→ React JSX 強制要 /
```

  
### 標籤屬性（Attributes）

```text
<a href="https://google.com" target="_blank">
   ↑     ↑                       ↑
  標籤  屬性 1                  屬性 2

href="..." → 連結網址
target="_blank" → 在新分頁打開
```

  
**常見屬性：**

```text
href      連結網址（給 <a>）
src       圖片 / 影片來源（給 <img> / <video>）
alt       圖片 alt 文字（無障礙必加）
class     CSS class
id        唯一識別
style     行內 CSS（不推薦）
data-*    自訂資料屬性
```

  
### HTML / CSS / JavaScript 三角關係

```text
🏗️ HTML
   → 結構（這是標題、這是段落）

🎨 CSS
   → 樣式（標題紅色、字大 30px）

⚡ JavaScript
   → 互動（按按鈕做事）

三者合作：
  HTML 蓋骨架 → CSS 化妝 → JS 動起來
```

  
### 為什麼 HTML 仍然重要（2026）

```text
✅ 所有網頁底層都是 HTML
   → React / Vue / Next.js 最終 render 成 HTML

✅ SEO 完全靠 HTML
   → JS 渲染的內容 Google 看不太清楚

✅ AI 引擎依賴 HTML 語意
   → Perplexity / ChatGPT 看 HTML 結構

✅ 無障礙必須懂 HTML
   → 螢幕閱讀器讀 HTML

✅ Email 模板永遠寫 HTML
   → 行銷 / 通知信
```

  
### 學 HTML 的 3 個階段

```text
Stage 1：基本結構（L1-L11）
  → 知道 <h1> / <p> / <a> / <img> / <ul>
  → 能寫一個個人簡介頁

Stage 2：語意化 + 無障礙（L12-L18）
  → 用 <header> / <main> / <article> / <section>
  → 加 aria attributes
  → SEO 友善

Stage 3：進階 + 整合（L19-L25）
  → 表單驗證 / 多媒體 / Web Components
  → JSON-LD 結構化資料
  → 寫業界級 portfolio
```
