**連結（`<a>`）跟媒體（`<img>` / `<video>` / `<audio>` / `<iframe>`）**——這是網頁跟外面世界連在一起的關鍵。**沒有連結、整個 Web 就只是一堆孤島**。

> 💡 **這課的價值**：連結是 Web 的本質（HyperText 的 "Hyper" 就是「連結」）、媒體是讓網頁活起來的東西。寫對這 5 個標籤、SEO + 無障礙 + 行動裝置體驗一次到位。

  
### 1. `<a>` — 超連結

```text
<a href="https://example.com">點我去 example.com</a>
<a href="about.html">點我去 about 頁</a>
<a href="#top">回到頂部</a>
<a href="mailto:me@example.com">寄信給我</a>
<a href="tel:+886912345678">打電話給我</a>
```

**3 種 href 寫法：**

```text
1. 絕對路徑（含 https://）
   → 連到外站
   → href="https://google.com"

2. 相對路徑（不含 https://）
   → 連到自己網站別頁
   → href="about.html" / href="/blog/post-1"

3. 錨點（#）
   → 跳到同頁某段
   → href="#section-2"
   → 配 <h2 id="section-2"> 用
```

> 💡 **`/` 開頭 vs 不開頭差很多**：`/about.html` 是「網站根目錄」、`about.html` 是「當前資料夾」。新手最常踩這個坑。

  
### 2. `<a>` 的 4 個必懂屬性

```text
🔗 href — 連去哪
   → 必填、沒寫就不是連結了

🆕 target="_blank" — 開新分頁
   → 不寫預設「同分頁取代」
   → 外站連結建議加、不要把使用者帶離你的網站

🛡️ rel="noopener noreferrer" — 安全 + 隱私
   → 配 target="_blank" 一定要加
   → 不加會有資安漏洞（tabnabbing）+ 給對方看到 referrer

📥 download — 下載而不是開啟
   → href="report.pdf" download="月報.pdf"
   → 強制下載、可改檔名
```

**完整外部連結寫法（業界標準）：**

```text
<a href="https://example.com" target="_blank" rel="noopener noreferrer">
  外站連結
</a>
```

> ⚠️ **`target="_blank"` 沒配 `rel="noopener noreferrer"` = 資安漏洞**：對方網頁可以用 `window.opener` 改回你這頁的 URL（釣魚）。Chrome 後來預設加 noopener、但其他瀏覽器不一定、永遠手動加。

  
### 3. `<img>` — 圖片

```text
<img src="cat.jpg" alt="一隻睡覺的橘貓" width="400" height="300">
```

**4 個必懂屬性：**

```text
🖼️ src — 圖片路徑（必填）

📝 alt — 替代文字（必填！）
   → 圖片載入失敗時顯示
   → 螢幕閱讀器念給盲人聽
   → SEO 也吃這個（Google 圖片搜尋）
   → 純裝飾用空字串 alt=""

📐 width / height — 寬高
   → 強烈建議加（避免 layout shift / CLS）
   → 寫實際像素數字、不用單位
```

> 💡 **alt 寫不出來就是設計爛**：好的 alt 描述「圖在說什麼」、不是「這張圖叫什麼」。例：✅ `alt="一隻橘貓在地毯上睡覺"` ❌ `alt="cat.jpg"` ❌ `alt="圖片"`。

  
### 4. `<img>` 進階：響應式 + lazy load

```text
<!-- Lazy load — 滾到才載、首頁 5 倍快 -->
<img src="hero.jpg" alt="..." loading="lazy">

<!-- 響應式 — 手機載小圖、桌面載大圖（省流量）-->
<img
  src="hero-800.jpg"
  srcset="hero-400.jpg 400w, hero-800.jpg 800w, hero-1600.jpg 1600w"
  sizes="(max-width: 600px) 100vw, 800px"
  alt="..."
>

<!-- 多格式 — WebP 不支援退回 JPG -->
<picture>
  <source srcset="hero.webp" type="image/webp">
  <img src="hero.jpg" alt="...">
</picture>
```

**2026 圖片標準：**

```text
✅ 永遠加 alt（即使是空字串）
✅ 永遠加 width / height（防 CLS）
✅ 首屏以外 loading="lazy"
✅ 用 WebP / AVIF（比 JPG 小 30-50%）
✅ Hero 圖手機桌面分開（省流量）
```

> 💡 **Next.js / Nuxt 的 `<Image>` 元件**：自動幫你做完上面所有事——響應式、lazy、format 轉換、CLS 防護。Ch10 詳教。

  
### 5. `<video>` — 影片

```text
<video controls width="640" poster="cover.jpg">
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
  <p>瀏覽器不支援、<a href="video.mp4">直接下載</a>。</p>
</video>
```

**常用屬性：**

```text
🎬 controls — 顯示播放控制條（必加、不然使用者沒法控制）
🖼️ poster — 封面圖（影片載入前 / 暫停時顯示）
🔁 loop — 播完自動重播
🔇 muted — 預設靜音（配 autoplay 用、不然 Chrome 會擋）
▶️ autoplay — 自動播放（要配 muted）
🖥️ playsinline — iOS 上不要全螢幕（嵌入式播放器必加）
```

**手機自動播影片（IG / TikTok 那種）：**

```text
<video autoplay muted loop playsinline poster="cover.jpg">
  <source src="hero.mp4" type="video/mp4">
</video>
```

> ⚠️ **autoplay 一定要 muted**：Chrome / Safari / Firefox 都會擋「有聲音的自動播」、影響 UX 也省流量。要有聲音、使用者點才有。

  
### 6. `<audio>` — 音訊

```text
<audio controls>
  <source src="song.mp3" type="audio/mpeg">
  <source src="song.ogg" type="audio/ogg">
</audio>
```

跟 `<video>` 很像、屬性也一樣（controls / loop / autoplay / muted）。

  
### 7. `<iframe>` — 嵌入別人網站

```text
<!-- YouTube 影片 -->
<iframe
  src="https://www.youtube.com/embed/VIDEO_ID"
  width="560" height="315"
  title="YouTube video"
  allow="autoplay; encrypted-media; picture-in-picture"
  allowfullscreen
  loading="lazy"
></iframe>

<!-- Google Maps -->
<iframe
  src="https://www.google.com/maps/embed?pb=..."
  width="600" height="450"
  loading="lazy"
></iframe>
```

> 💡 **iframe 一定加 `title`**：螢幕閱讀器靠這個告訴盲人「這是 YouTube 影片」。沒寫的話無障礙審查直接扣分。

### 5 大常見坑

```text
❌ 1. <img> 沒寫 alt
   → SEO 扣分、盲人完全看不到圖
   → 修：永遠寫 alt（裝飾圖也要 alt=""）

❌ 2. target="_blank" 沒配 rel="noopener noreferrer"
   → 資安漏洞 + referrer 洩漏
   → 修：永遠成對寫

❌ 3. 圖片沒寫 width / height
   → CLS（Layout Shift）爆表、Lighthouse 直接扣
   → 修：永遠寫實際像素

❌ 4. video autoplay 沒配 muted
   → 瀏覽器擋、自動播失效
   → 修：autoplay muted loop playsinline 一起寫

❌ 5. iframe 沒寫 title
   → 螢幕閱讀器無從識別、無障礙審查不過
   → 修：永遠加 title="..."
```

  
### 🎯 重點回顧

- **`<a>` 是 Web 的本質**——hyperlink 把所有頁面連起來
- **`<img>` 永遠寫 alt + width + height**——SEO / 無障礙 / 不晃畫面
- **外站連結用 `target="_blank" rel="noopener noreferrer"`**——資安基本盤
- **影片用 mp4 + webm 兩格式**——相容性最廣
- **iframe 永遠寫 title**——無障礙
