**文字標籤完整版**——HTML 內**寫文字**的所有標籤：標題 / 段落 / 強調 / 引用 / 程式碼 / 換行。**每個都有不同語意**、別亂用。

> 💡 **核心心法**：HTML 標籤代表「**這段文字是什麼角色**」、不是「**長什麼樣**」（長相是 CSS 的事）。

  
### 標題：h1-h6

```text
<h1>最大標題</h1>      <!-- 一頁只能一個 -->
<h2>次標題</h2>        <!-- 章節 -->
<h3>小標題</h3>        <!-- 子章節 -->
<h4>更小</h4>
<h5>很小</h5>
<h6>最小</h6>
```

  
**SEO 3 大鐵律：**

```text
✅ 每頁只一個 <h1>
   → 搜尋引擎看這個判斷主題
   → 多個 h1 = 主題不明、扣分

✅ 標題層級不要跳
   → h2 後不要直接 h4
   → 螢幕閱讀器會混亂

✅ 不要為了字大用 h1
   → 字大用 CSS 調 font-size
   → 別濫用 heading
```

  
### 段落 + 換行

```text
<p>第一段。</p>
<p>第二段。</p>

<!-- 換行（同一段內換行）-->
<p>第一行<br>第二行</p>

<!-- 水平線（分隔不同主題）-->
<hr>
```

  
**`<br>` vs `<p>` 區別：**

```text
<p>單一段落、自帶上下空間</p>

<p>
  第一行<br>
  第二行
</p>
→ 同段落內換行、沒額外空間

❌ 別這樣濫用 <br>：
線 1<br>
線 2<br>
線 3<br>

✅ 應該用 <p>：
<p>線 1</p>
<p>線 2</p>
<p>線 3</p>
```

  
### 文字強調 4 兄弟

| 標籤 | 語意 | 長相 | 何時用 |
|---|---|---|---|
| `<strong>` | **重要** | 粗體 | 重要 / 警告 |
| `<em>` | **強調** | 斜體 | 強調語氣 |
| `<mark>` | **螢光標記** | 黃底 | 重點 / 搜尋結果 |
| `<small>` | **細字** | 字小 | 註解 / 法律條款 |

```text
<p>
  我們的咖啡 <strong>免運費</strong>、
  <em>限今天</em>、
  <mark>已售出 1000 杯</mark>、
  <small>運費政策詳見官網</small>。
</p>
```

> 💡 **不要用 `<b>` 跟 `<i>`**（雖然合法）——它們**只有視覺意義、沒有語意**。要粗就用 `<strong>`、要斜用 `<em>`、SEO + 螢幕閱讀器才會強調。

  
### 引用

```text
<!-- 區塊引用 -->
<blockquote cite="https://example.com">
  <p>「引用文字、來自別人」</p>
</blockquote>

<!-- 行內引用 -->
<p>他說 <q cite="https://example.com">這是直接引述</q>。</p>

→ <q> 瀏覽器自動加引號
```

  
### 程式碼相關標籤

  
**1. `<code>`（行內程式碼）：**

```text
<p>用 <code>const x = 5</code> 宣告變數。</p>
```

  
**2. `<pre>`（保留格式）：**

```text
<pre><code>function hi() {
  console.log("hi");
}</code></pre>

→ <pre> 保留空白 + 換行
→ <code> 表示「這是程式碼」
→ 兩個一起用 = 多行程式碼塊
```

  
**3. `<kbd>`（鍵盤輸入）：**

```text
<p>按 <kbd>Ctrl</kbd> + <kbd>S</kbd> 儲存。</p>

→ 表示「鍵盤按鍵」
→ 教學常用
```

  
**4. `<samp>`（程式輸出）：**

```text
<p>會看到 <samp>Hello World</samp>。</p>

→ 表示「程式輸出範例」
```

  
**5. `<var>`（變數）：**

```text
<p>變數 <var>x</var> 等於 <var>y</var> + 5。</p>

→ 數學或程式變數
→ 預設斜體
```

  
### 縮寫 + 定義

```text
<!-- 縮寫 -->
<p><abbr title="HyperText Markup Language">HTML</abbr> 是寫網頁的語言。</p>

→ 滑鼠移上去顯示「HyperText Markup Language」

<!-- 定義 -->
<p><dfn>HTML</dfn> 是寫網頁的語言。</p>

→ 表示「這是術語的定義」
```

  
### 上下標

```text
<p>水的化學式：H<sub>2</sub>O</p>
<p>2 的 10 次方：2<sup>10</sup> = 1024</p>

→ <sub> 下標（化學式）
→ <sup> 上標（次方 / 註腳）
```

  
### 刪除 + 插入（追蹤改動）

```text
<p>原價 <del>$100</del> <ins>$80</ins>。</p>

→ <del> 刪除（畫橫線）
→ <ins> 新增（畫底線）
→ 用於促銷 / 修訂版本
```

  
### 完整文字標籤範本

```text
<article>
  <h1>2026 AI 工具大比拼</h1>
  
  <p>
    本文比較 <strong>3 個主流 AI 工具</strong>、
    包含 <abbr title="GitHub Copilot">Copilot</abbr>、
    <abbr title="Cursor IDE">Cursor</abbr>、
    跟 <abbr title="Claude Code">Code</abbr>。
  </p>
  
  <h2>程式碼範例</h2>
  
  <p>用 <code>npm install</code> 裝套件：</p>
  
  <pre><code>npm install react
npm install typescript</code></pre>
  
  <h2>數學公式</h2>
  
  <p>圓面積：π × r<sup>2</sup></p>
  
  <h2>引用</h2>
  
  <blockquote cite="https://2026.dev">
    <p>「AI 不會取代工程師、但會用 AI 的工程師會取代不用的。」</p>
  </blockquote>
  
  <h2>快捷鍵</h2>
  
  <p>按 <kbd>Cmd</kbd> + <kbd>K</kbd> 開 Cursor chat。</p>
  
  <h2>促銷</h2>
  
  <p>
    原價 <del>$20/月</del> 限時 <ins>$10/月</ins>、
    <mark>3 天內報名</mark>。
  </p>
  
  <p>
    <small>※ 折扣限新用戶、不可與其他活動併用。</small>
  </p>
</article>
```

  
### 4 大常見錯誤

```text
1. ❌ 用 <h1> 當大字
   <h1 style="font-size: 100px;">超大字</h1>
   → SEO 災難、Google 認為主題是「超大字」
   ✅ 用 <p> + CSS font-size

2. ❌ <br><br><br> 取代段落
   → 沒語意
   ✅ 用 <p>

3. ❌ <b> / <i> 表示重要
   → 只有視覺、沒語意
   ✅ <strong> / <em>

4. ❌ 程式碼用 <p>
   <p>npm install</p>
   → 沒等寬字、會亂排
   ✅ <code> 包行內、<pre><code> 包多行
```

  
### 語意 vs 樣式對照

| 你想要 | 語意標籤 | CSS（樣式）|
|---|---|---|
| 強調重要 | `<strong>` | font-weight: bold |
| 強調語氣 | `<em>` | font-style: italic |
| 標題 | `<h1>-<h6>` | font-size 大 |
| 程式碼 | `<code>` | font-family: monospace |
| 引用 | `<blockquote>` | margin / border-left |

> 💡 **規則**：先選對標籤（語意）、再用 CSS 改長相。別反過來。

  
### 5 大寫文字標籤心法

```text
1. ⭐ 先選對標籤、再 CSS 改樣式
   → 標籤決定語意、CSS 決定長相

2. ⭐ heading 嚴守 h1-h6 順序
   → 不跳級

3. ⭐ 強調用 <strong> / <em>
   → 不要用 <b> / <i>

4. ⭐ 程式碼用 <code> + <pre>
   → 不要用 <p>

5. ⭐ 一頁只能一個 <h1>
   → SEO 主題明確
```

  
    **☕ 用人話講**
    
文字標籤 5 件事：(1) **標題 h1-h6** 別跳級、一頁一個 h1 (2) **段落 `<p>`** + **`<br>`** 換行（別濫用） (3) **強調 4 兄弟**：strong / em / mark / small（**有語意**、別用 b / i） (4) **程式碼** `<code>` 行內 + `<pre><code>` 多行 (5) **引用 `<blockquote>` / `<q>`**、縮寫 `<abbr>`、刪除 `<del>` 都有專屬語意。**心法**：標籤選對 → SEO + 無障礙加分。
