**meta 進階：robots / hreflang / verification / 進階 OG / Theme color**——L9 教過 SEO 基本 meta（title / description / OG / Twitter Card）、這課補完「進階場景」：多語言、爬蟲控制、平台驗證、不同 OG type。

> 💡 **這課的價值**：L9 是 90% 網站夠用的基本盤、這課讓你處理「多國語言網站 / 部落格 / 商品頁 / 新聞站」這類需要進階 SEO 的場景。

  
### 1. `<meta name="robots">` — 爬蟲控制

```text
<!-- 預設（不寫等於這個）-->
<meta name="robots" content="index, follow">

<!-- 不要被搜尋引擎收錄 -->
<meta name="robots" content="noindex">

<!-- 不要追蹤頁面內連結 -->
<meta name="robots" content="nofollow">

<!-- 兩個都不要 -->
<meta name="robots" content="noindex, nofollow">

<!-- 不要顯示快取版本 -->
<meta name="robots" content="noarchive">

<!-- 不要顯示 description（罕用）-->
<meta name="robots" content="nosnippet">

<!-- 只給 Google -->
<meta name="googlebot" content="noindex">

<!-- 只給 Bing -->
<meta name="bingbot" content="noindex">
```

**典型用法：**

```text
🔒 後台 / 管理頁
   <meta name="robots" content="noindex, nofollow">

🚧 暫時下線的頁
   <meta name="robots" content="noindex">

📄 重複內容（用 canonical + 不需要 noindex）
   <link rel="canonical" href="https://example.com/original">

📰 新聞頁、希望快取保留
   不寫 / 預設 即可
```

> ⚠️ **`noindex` vs `robots.txt`**：robots.txt 是「請求」、有些爬蟲會無視；`noindex` 是「強制」、Google 一定遵守。**敏感頁兩個都加**。

  
### 2. `hreflang` — 多語言 SEO 必加

**場景**：你網站有中文 + 英文版、Google 該知道「給台灣人看中文版、給美國人看英文版」。

```text
<!-- 在每個語言版本的 <head> 都寫所有版本 -->
<link rel="alternate" hreflang="zh-Hant" href="https://example.com/tw/">
<link rel="alternate" hreflang="zh-Hans" href="https://example.com/cn/">
<link rel="alternate" hreflang="en" href="https://example.com/en/">
<link rel="alternate" hreflang="ja" href="https://example.com/jp/">

<!-- 「找不到語言」的預設 -->
<link rel="alternate" hreflang="x-default" href="https://example.com/">
```

**3 個語言代碼規則：**

```text
ISO 639-1 語言代碼
   zh / en / ja / ko / es / fr / de ...

+ ISO 3166-1 國家代碼
   zh-TW (台灣繁中)
   zh-CN (中國簡中)
   zh-HK (香港繁中)
   en-US (美式英文)
   en-GB (英式英文)
   pt-BR (巴西葡萄牙文)
   pt-PT (葡萄牙葡萄牙文)

或 ISO 15924 文字代碼
   zh-Hant (繁中、不分國家)
   zh-Hans (簡中、不分國家)
```

  
### 3. Google / Bing / Pinterest 驗證

```text
<!-- Google Search Console -->
<meta name="google-site-verification" content="abc123...">

<!-- Bing Webmaster Tools -->
<meta name="msvalidate.01" content="abc123...">

<!-- Pinterest -->
<meta name="p:domain_verify" content="abc123...">

<!-- Yandex（俄羅斯）-->
<meta name="yandex-verification" content="abc123...">

<!-- Facebook（部分功能用）-->
<meta property="fb:app_id" content="123456">
```

**取得方式**：到各平台「驗證網站」流程拿、複製 meta tag 貼到 `<head>`。

  
### 4. Theme color + 行動裝置體驗

```text
<!-- 手機 URL bar / Android 上方狀態列顏色 -->
<meta name="theme-color" content="#10b981">

<!-- 配 dark mode -->
<meta name="theme-color" media="(prefers-color-scheme: light)" content="#10b981">
<meta name="theme-color" media="(prefers-color-scheme: dark)" content="#0a0a0a">

<!-- iOS Safari status bar 風格（PWA 用）-->
<meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">

<!-- iOS 加到主畫面後標題 -->
<meta name="apple-mobile-web-app-title" content="AI 島">

<!-- iOS 全螢幕（PWA）-->
<meta name="apple-mobile-web-app-capable" content="yes">

<!-- 行動裝置 disable 數字自動偵測（電話 / 日期）-->
<meta name="format-detection" content="telephone=no, date=no">
```

  
### 5. OG type 變化（依內容類型）

**`og:type` 5 大常用值：**

```text
website     一般網站 / landing page（預設）
article     部落格文章 / 新聞
product     商品頁
profile     個人檔案頁
video.movie  影片
music.song   音樂
book        書籍
```

**`og:type="article"` 的附加 meta：**

```text
<meta property="og:type" content="article">
<meta property="article:published_time" content="2026-05-28T08:00:00+08:00">
<meta property="article:modified_time" content="2026-05-28T10:00:00+08:00">
<meta property="article:author" content="https://example.com/u/luffy">
<meta property="article:section" content="技術">
<meta property="article:tag" content="React">
<meta property="article:tag" content="Next.js">
```

**`og:type="product"` 的附加 meta：**

```text
<meta property="og:type" content="product">
<meta property="product:price:amount" content="2990">
<meta property="product:price:currency" content="TWD">
<meta property="product:availability" content="in stock">
```

### 7. 5 大常見坑

```text
❌ 1. hreflang 只在一個語言版本寫
   → 必須每個版本都列所有語言
   → 修：複製貼上 hreflang 區塊到所有語言版本

❌ 2. hreflang 用錯代碼
   → 例 "zh-tw" 應該 "zh-TW"（注意大小寫 / 用 hyphen）
   → 修：用 zh-Hant / zh-TW 等標準代碼

❌ 3. 後台沒 noindex
   → /admin / 後台被 Google 收錄、有資安風險
   → 修：noindex, nofollow + robots.txt 雙重保險

❌ 4. theme-color 只寫一個顏色
   → dark mode 下手機 URL bar 是亮色、不協調
   → 修：寫兩個（含 media query）

❌ 5. 同一張 og:image 全站用
   → 每頁分享預覽長一樣、辨識度差
   → 修：每頁 / 每文章獨立 OG 圖（可用動態生成）
```

  
### 🎯 重點回顧

- **`robots`** 控制爬蟲、後台必 noindex
- **`hreflang`** 多語言 SEO 必加、每版本都列
- **`theme-color`** 配 dark mode 改善行動體驗
- **`og:type`** 依內容選對（article / product / website）
- **平台驗證 meta** 開 Google Search Console / Bing 必要
