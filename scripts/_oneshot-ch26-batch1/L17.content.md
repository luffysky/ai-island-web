### 第一個 ML 模型：房價預測（線性回歸）

最簡單的 ML：根據「坪數」預測「房價」、線性回歸（找一條直線擬合）。

```text
import numpy as np
from sklearn.linear_model import LinearRegression

# === 1. 準備資料（X = 特徵、y = 答案）===
X = np.array([[20], [25], [30], [35], [40], [50]])    # 6 間房的坪數
y = np.array([800, 1000, 1200, 1400, 1600, 2000])    # 對應房價（萬元）
# ⚠️ X 一定是 2D（每列一筆資料、即使只有 1 個特徵也要 [[20]] 不是 [20]）

# === 2. 建模（建一個 LinearRegression 物件）===
model = LinearRegression()

# === 3. 訓練（用 .fit() 餵資料、model 自己學）===
model.fit(X, y)

# === 4. 預測（給新資料、要 .predict() 回答）===
new_house = np.array([[27]])              # 一間 27 坪的房子
predicted = model.predict(new_house)
print(f"預測：{predicted[0]:.0f} 萬")     # 預測：1080 萬

# === 5. 看模型學了什麼 ===
print(f"斜率：{model.coef_[0]}")          # 40（每多 1 坪 +40 萬）
print(f"截距：{model.intercept_}")        # 0
# 等於學會「房價 = 40 × 坪數 + 0」這條線
```

  
### scikit-learn 統一 API（學一次、所有 model 通用）

sklearn 最強的設計：**所有 model 都是同一套 API**。換演算法只需要改 import + model 那行、其他完全一樣。

```text
# === 通用 4 步驟 ===

# 1. 建立 model
model = SomeAlgorithm()

# 2. 訓練（給 X, y）
model.fit(X_train, y_train)

# 3. 預測（給 X_test、要 y_pred）
predictions = model.predict(X_test)

# 4. 評估（給 X_test, y_test、回分數）
score = model.score(X_test, y_test)
```

→ 學會這 4 步、換 100 種演算法都通用、不用重學

  
### 分類範例：iris 鳶尾花（ML 界的「Hello World」）

iris 鳶尾花資料集 = 150 筆、3 種花（setosa / versicolor / virginica）、每筆 4 個特徵（萼片長寬、花瓣長寬）。

```text
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, classification_report

# === 1. 載入資料（sklearn 內建這個經典資料集）===
iris = load_iris()
X, y = iris.data, iris.target              # X = 特徵、y = 0/1/2 (三種花)

# === 2. 切 train / test（80% 訓練、20% 測試）===
X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.2,                          # 測試集 20%
    random_state=42                         # 固定亂數種子、結果可重現
)
# ⚠️ random_state=42 是 ML 慣例（《銀河便車指南》梗）、其實任何數字都行

# === 3. 建模 + 訓練 ===
model = RandomForestClassifier(n_estimators=100)
# n_estimators=100 = 用 100 棵決策樹投票
model.fit(X_train, y_train)

# === 4. 預測 ===
y_pred = model.predict(X_test)

# === 5. 評估 ===
print(f"準確率：{accuracy_score(y_test, y_pred):.2%}")
# 例如：96.67%

print(classification_report(y_test, y_pred, target_names=iris.target_names))
# 顯示每個類別的 precision / recall / f1-score（L19 詳教）
```

### 實戰：客戶流失預測（完整 ML pipeline）

模擬真實案例：用客戶資料預測「會不會流失（churn）」。

```text
import pandas as pd
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.preprocessing import StandardScaler
from sklearn.ensemble import GradientBoostingClassifier
from sklearn.metrics import classification_report

# === 1. 載入資料 ===
df = pd.read_csv("customers.csv")

# === 2. 特徵工程（feature engineering、L18 詳教）===
df["avg_monthly"] = df["total_charge"] / df["tenure_months"]   # 衍生新特徵
df = pd.get_dummies(df, columns=["plan_type", "country"])      # 類別變數轉 one-hot

# === 3. 分 X 跟 y ===
X = df.drop(columns=["churned"])           # 除了答案外都是特徵
y = df["churned"]                          # 答案：0 = 沒流失、1 = 流失

# === 4. 標準化（Scaling、讓所有特徵在同尺度）===
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)
# StandardScaler：減平均、除標準差、讓資料以 0 為中心、標準差 1

# === 5. 切 train / test（注意 stratify=y、保證 train/test 流失比例一致）===
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, stratify=y
)

# === 6. 訓練 ===
model = GradientBoostingClassifier(n_estimators=200)
model.fit(X_train, y_train)

# === 7. 評估 ===
y_pred = model.predict(X_test)
print(classification_report(y_test, y_pred))

# === 8. 交叉驗證（CV、cross-validation、L20 詳教）===
scores = cross_val_score(model, X_scaled, y, cv=5)
print(f"CV：{scores.mean():.3f} ± {scores.std():.3f}")
# 把資料切 5 份、輪流當 test、跑 5 次取平均、結果更可信

# === 9. 看哪個特徵最重要（feature importance）===
importance = pd.DataFrame({
    "feature": X.columns,
    "importance": model.feature_importances_
}).sort_values("importance", ascending=False)
print(importance.head(10))
# 列出最重要的 10 個特徵、回頭可以加強這幾個
```

  
### 儲存 model + 部署到 API

訓練好的 model 要存檔、不然每次都要重訓練。

```text
import joblib

# === 存 ===
joblib.dump(model, "model.pkl")            # model 存成檔案
joblib.dump(scaler, "scaler.pkl")          # scaler 也要存（預測時要用同一個）

# === 載 ===
model = joblib.load("model.pkl")
```

  
**部署到 FastAPI（L25 詳教 FastAPI）：**

```text
from fastapi import FastAPI

app = FastAPI()

@app.post("/predict")
def predict(features: dict):
    # 1. 轉成 model 認得的格式
    X = scaler.transform([list(features.values())])
    
    # 2. 預測機率（predict_proba 回每個類別的機率）
    pred = model.predict_proba(X)[0, 1]    # [0, 1] = 第 1 筆、類別 1 的機率
    
    return {"churn_probability": pred}
```
