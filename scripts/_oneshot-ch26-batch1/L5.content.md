流程控制 = 「程式怎麼分支、怎麼重複」。Python 最大特色：**用「縮排」（indentation）取代其他語言的 `{}` 大括號** —— 4 個空白縮排代表「這幾行是同一個區塊」。

> ⚠️ **縮排很嚴格**——不能混 Tab 跟 Space、不然會 IndentationError。VS Code 預設用 Space、保持就好。

  
### if / elif / else（如果 / 否則如果 / 否則）

**elif** = else if 的縮寫（其他語言要寫兩字、Python 一個字搞定）。

```text
age = 20

if age >= 18:
    print("成人")
elif age >= 12:
    print("青少年")
else:
    print("兒童")
```

  
**邏輯運算 + 比較：**

```text
# 結合多個條件用 and / or / not（不是 && || ! ）
if age >= 18 and has_license:           # 「並且」
    print("能開車")

if is_admin or is_owner:                # 「或」
    print("有權限")

if not is_active:                       # 「相反」
    print("不活躍")
```

> 💡 **注意：判斷相等用 `==`（兩個等號）、不是 `=`（一個等號）**。`=` 是「指派」（把右邊塞給左邊）、`==` 才是「比較」。

  
**三元運算（ternary、一行寫完 if-else）：**

```text
# Python 三元：「值 if 條件 else 另一個值」
status = "成人" if age >= 18 else "未成年"

# 對比其他語言（JS / Java / C）：
# const status = age >= 18 ? "成人" : "未成年"
# Python 沒有 ? : 語法、用「if-else 表達式」表達同樣意思
```

  
**Match-case（模式匹配、Python 3.10+ 才有）：**

**match-case** 像加強版的 switch、可以「按結構配對」、不只比值。

```text
def describe(point):
    match point:
        case (0, 0):
            return "原點"
        case (x, 0):
            return f"在 X 軸、x={x}"
        case (0, y):
            return f"在 Y 軸、y={y}"
        case (x, y):
            return f"在 ({x}, {y})"
        case _:                        # _ = 預設（其他情況）
            return "不是 2D 點"

describe((3, 0))    # "在 X 軸、x=3"
describe((1, 2))    # "在 (1, 2)"
```

  
### for 迴圈（重複跑某段、跑完一個 iterable 的每個元素）

**iterable（可迭代物件）** = 「可以一個一個拿出來的東西」——list / tuple / set / dict / 字串 / range 都是。

```text
# === 跑 list ===
for fruit in ["蘋果", "香蕉", "橘子"]:
    print(fruit)

# === 跑 range（產生數字序列、最常用）===
for i in range(5):           # 0, 1, 2, 3, 4  ⚠️ 不含 5！
    print(i)

for i in range(2, 8):        # 2, 3, 4, 5, 6, 7
    print(i)

for i in range(0, 10, 2):    # 0, 2, 4, 6, 8（step = 2）
    print(i)

# === 跑字串（一個字一個字）===
for char in "Python":
    print(char)              # P / y / t / h / o / n
```

  
**進階：搭配內建函式跑迴圈**

```text
# === enumerate：同時要 index 又要值 ===
for i, fruit in enumerate(["蘋果", "香蕉"]):
    print(f"{i}: {fruit}")
# 0: 蘋果
# 1: 香蕉

# enumerate 想從 1 開始
for i, fruit in enumerate(["蘋果", "香蕉"], start=1):
    print(f"第 {i} 個：{fruit}")

# === zip：同時跑兩個 list（並排迭代）===
names = ["Alice", "Bob", "Charlie"]
ages = [30, 25, 35]
for name, age in zip(names, ages):
    print(f"{name}: {age}")

# === reversed：反過來跑 ===
for fruit in reversed(["蘋果", "香蕉"]):
    print(fruit)

# === sorted：排序後跑 ===
for num in sorted([3, 1, 4, 1, 5]):
    print(num)

# === 跑 dict（很常用）===
user = {"name": "Alice", "age": 30}

for key in user:                # 預設跑 key
    print(key)

for key, value in user.items():  # 同時要 key + value
    print(f"{key}: {value}")
```

  
**break + continue（控制迴圈流程）：**

```text
for i in range(10):
    if i == 3:
        continue   # continue = 跳過這次、繼續下一個
    if i == 7:
        break      # break = 直接跳出整個迴圈
    print(i)
# 印出：0, 1, 2, 4, 5, 6（沒 3、沒 7 之後的）
```

  
**for-else（Python 特別語法、其他語言沒有）：**

「`else` 在 `for` 後面」= 「迴圈**沒被 break 中斷**才會跑 else」。

```text
for x in [1, 2, 3]:
    if x == 5:
        break
else:
    print("沒找到 5")       # 沒 break 才印這
# 印出：沒找到 5

# 應用：找東西
for user in users:
    if user.name == "Alice":
        print("找到了")
        break
else:
    print("找不到 Alice")    # 全部跑完沒 break = 沒找到
```

  
### while 迴圈（條件為真就一直跑）

```text
count = 0
while count < 5:
    print(count)
    count += 1
# 印 0, 1, 2, 3, 4

# 也可以配 break
n = 10
while n > 0:
    n -= 1
    if n == 5:
        break
else:
    print("自然結束、沒被 break")   # 這次有 break、不會印

# === Python 沒有 do-while ===
# 用 while True + break 模擬「至少跑一次再判斷」
while True:
    do_stuff()
    if condition:
        break
```

  
### Comprehension（生成式）：4 種容器都能用

之前 L4 教過 list comprehension。其實 4 種容器都有生成式語法、寫法很像。

```text
# === List comprehension（[ ... ]）===
squares = [x ** 2 for x in range(10)]

# === Set comprehension（{ ... }、自動去重）===
unique_squares = {x ** 2 for x in [-1, 0, 1, 2, -2]}
# {0, 1, 4}（-1²=1、1²=1 重複會去掉）

# === Dict comprehension（{key: value ...}）===
square_dict = {x: x ** 2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# === Generator expression（( ... )、lazy 計算）===
gen = (x ** 2 for x in range(10))
# 不立刻算！要用 next(gen) 或 for 跑才算
# 適合：超大資料、省記憶體（不會把全部結果一次算出來）
```

  
**進階寫法（加條件、巢狀、條件表達式）：**

```text
# 加 if 條件
evens = [x for x in range(20) if x % 2 == 0]
positives = {x for x in [-1, -2, 1, 2, 3] if x > 0}

# 多層（巢狀）
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flat = [x for row in matrix for x in row]
# [1, 2, 3, 4, 5, 6, 7, 8, 9]（攤平）

# 三元式 + 生成式（每個元素都做分支）
labels = ["odd" if x % 2 else "even" for x in range(5)]
# ['even', 'odd', 'even', 'odd', 'even']
```

→ Comprehension 是 **pythonic（地道 Python 風格）** 寫法
→ 比 `for` 迴圈 + `append` 簡潔
→ 比 `map()` / `filter()` 自然

  
### 常見 pattern（一行解決常見任務）

```text
# === 1. 加總 ===
total = sum([1, 2, 3, 4, 5])         # 15（不用寫 for 迴圈）

# === 2. 找最大 / 最小 ===
max([3, 1, 4, 1, 5])                  # 5
min([3, 1, 4, 1, 5])                  # 1

# 自訂排序規則（用 key 參數）
users = [{"name": "Alice", "age": 30}, {"name": "Bob", "age": 25}]
oldest = max(users, key=lambda u: u["age"])   # 找年紀最大的

# === 3. 算符合條件的數量 ===
count = len([x for x in nums if x > 10])      # 大於 10 的有幾個

# 或用 Counter（前面 L4 教過）
from collections import Counter
counts = Counter(words)

# === 4. 全部符合 / 至少一個符合（用 all / any）===
all_active = all(u.is_active for u in users)  # True if 全部 active
any_admin = any(u.is_admin for u in users)    # True if 至少一個 admin

# === 5. 找第一個符合的（用 next + generator）===
first_admin = next((u for u in users if u.is_admin), None)
# 找到就回傳那個 user、找不到回 None（給預設值）

# === 6. 平面化巢狀 list ===
nested = [[1, 2], [3, 4], [5, 6]]
flat = [x for sub in nested for x in sub]
# [1, 2, 3, 4, 5, 6]
```

  
    **☕ 用人話講**
    
Python 流程控制重點：(1) 用 **縮排** 不用 `{}`、4 個 Space (2) **`if elif else`** 不是 `else if` (3) **`for x in iterable`**、不是 C 那種 `for(i=0; i<n; i++)` (4) **list comprehension** 是 pythonic、寫熟才像 Python 工程師：`[x * 2 for x in nums]` (5) **`range()` 不含尾**——`range(5)` 是 0~4、不到 5。**新手最大陷阱**：縮排混 Tab 跟 Space 會錯、設編輯器只用 Space（VS Code 預設）。
