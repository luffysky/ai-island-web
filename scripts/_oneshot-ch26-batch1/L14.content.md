### DataFrame：Pandas 的核心物件

**DataFrame** = 「**有列名也有欄名的二維表**」、長得就像 Excel 一張工作表。

```text
import pandas as pd                     # 慣例暱稱：pd

# === 從 dict 建（每個 key 是一個欄位）===
df = pd.DataFrame({
    "name": ["Alice", "Bob", "Charlie"],
    "age": [30, 25, 35],
    "city": ["Taipei", "Tokyo", "Seoul"]
})
print(df)
#       name  age    city
# 0    Alice   30  Taipei
# 1      Bob   25   Tokyo
# 2  Charlie   35   Seoul
```

  
**從各種檔案 / 資料源讀進來：**

```text
df = pd.read_csv("users.csv")                          # 讀 CSV
df = pd.read_excel("users.xlsx")                       # 讀 Excel
df = pd.read_json("users.json")                        # 讀 JSON
df = pd.read_sql("SELECT * FROM users", con=engine)    # 直接從 DB 讀
df = pd.read_parquet("data.parquet")                   # 讀 Parquet（大資料常用）
```

  
**看資料（每個 data scientist 拿到資料的第一步）：**

```text
df.head()              # 看前 5 列（最常用）
df.head(10)            # 看前 10 列
df.tail()              # 看最後 5 列
df.sample(5)           # 隨機抽 5 列
df.shape               # (列數, 欄數) tuple
df.columns             # 所有欄位名
df.dtypes              # 每個欄位的型別
df.describe()          # 數值欄位的統計總覽（mean / std / min / max / quartile）
df.info()              # 整體資訊（型別 / 缺失值 / 記憶體）
df.isnull().sum()      # 每個欄位有多少缺失值（NaN）
```

  
### 選資料（最常用的操作）

```text
# === 選欄位 ===
df["name"]                              # 1 欄、回 Series（一維）
df[["name", "age"]]                     # 多欄、回 DataFrame（二維）

# === 選 row（用位置 index）===
df.iloc[0]                              # 第 1 列（integer location）
df.iloc[0:3]                            # 前 3 列
df.iloc[:, 0]                           # 所有列、第 0 欄

# === 選 row（用 label index）===
df.loc[0]                               # label 是 0 的那列
df.loc[df["age"] > 30, "name"]          # 條件 + 指定欄位

# === 篩選（最重要的能力）===
df[df["age"] > 30]                                              # 年紀 > 30 的
df[df["city"] == "Taipei"]                                      # 在台北的
df[(df["age"] > 25) & (df["age"] < 40)]                         # 25-40 的（要用 &、不能 and）
df[df["city"].isin(["Taipei", "Tokyo"])]                        # 在這幾個城市
df.query("age > 30 and city == 'Taipei'")                       # SQL 風格（可讀性更好）
```

> ⚠️ **Pandas 篩選用 `&` / `|`、不能用 `and` / `or`**！而且每個條件要加括號。

  
### 修改 + 新增欄位

```text
# === 加新欄位（直接賦值）===
df["country"] = ["TW", "JP", "KR"]                              # 給整欄
df["age_in_5"] = df["age"] + 5                                  # 用其他欄算

# === 用 apply 套用函式 ===
df["category"] = df["age"].apply(
    lambda x: "young" if x < 30 else "senior"
)

# === assign（不改原 df、回傳新的、適合 method chain）===
df_new = df.assign(
    age_doubled=df["age"] * 2,
    full_label=df["name"] + "_" + df["city"]
)

# === 改某個 cell ===
df.loc[0, "age"] = 31                   # 第 1 列的 age 改 31

# === rename 欄位 ===
df = df.rename(columns={"name": "full_name", "city": "location"})

# === drop 欄位 / row ===
df = df.drop(columns=["country"])       # 刪欄位
df = df.drop(index=[0, 2])              # 刪第 1、3 列
```

  
### 聚合（groupby + agg、Pandas 最強功能）

`groupby` = SQL 的 GROUP BY、按某欄分組、然後算統計。

```text
# === 範例資料：訂單 ===
orders = pd.DataFrame({
    "product": ["A", "B", "A", "C", "B", "A"],
    "amount": [100, 200, 150, 300, 250, 120],
    "qty":    [1, 2, 1, 3, 2, 1],
})

# === 按 product 分組、算 amount 的總和 ===
orders.groupby("product")["amount"].sum()
# product
# A    370
# B    450
# C    300

# === 多個聚合一次算 ===
orders.groupby("product").agg(
    total_amount=("amount", "sum"),
    avg_amount=("amount", "mean"),
    order_count=("amount", "count"),
)
#          total_amount  avg_amount  order_count
# product
# A                 370   123.333333            3
# B                 450   225.000000            2
# C                 300   300.000000            1
```

  
**Pivot（樞紐分析、把長表變寬表）：**

```text
sales = pd.DataFrame({
    "month": ["Jan", "Jan", "Feb", "Feb"],
    "product": ["A", "B", "A", "B"],
    "sales": [100, 200, 150, 250],
})

sales.pivot(index="month", columns="product", values="sales")
# product    A    B
# month
# Feb      150  250
# Jan      100  200
```

  
### 處理缺失值（NaN、Not a Number）

```text
# === 看哪裡缺 ===
df.isnull().sum()                       # 每欄有幾個 NaN

# === 刪除有 NaN 的 row ===
df_clean = df.dropna()                  # 任何一欄缺都刪
df_clean = df.dropna(subset=["age"])    # 只看 age 欄是否缺

# === 填補 NaN ===
df["age"] = df["age"].fillna(0)                       # 填 0
df["age"] = df["age"].fillna(df["age"].mean())        # 填平均
df["city"] = df["city"].fillna("Unknown")             # 填字串
df = df.ffill()                                       # 用前一列的值填（forward fill）
df = df.bfill()                                       # 用下一列的值填（backward fill）
```

  
### 合併（merge / concat、像 SQL 的 JOIN）

```text
# === merge（按 key 欄位合併、像 SQL JOIN）===
users = pd.DataFrame({"id": [1, 2, 3], "name": ["Alice", "Bob", "Charlie"]})
orders = pd.DataFrame({"user_id": [1, 1, 2], "amount": [100, 200, 150]})

result = users.merge(orders, left_on="id", right_on="user_id", how="inner")
# how = "inner" / "left" / "right" / "outer"

# === concat（垂直 / 水平拼接）===
df_combined = pd.concat([df1, df2])                   # 垂直疊（增列）
df_combined = pd.concat([df1, df2], axis=1)           # 水平接（增欄）
```

  
### 排序 + 寫出檔案

```text
# === 排序 ===
df.sort_values("age")                                 # 按 age 升冪
df.sort_values("age", ascending=False)                # 降冪
df.sort_values(["city", "age"])                       # 多欄排序

# === 寫出 ===
df.to_csv("output.csv", index=False)                  # 寫 CSV、不要寫多餘 index
df.to_excel("output.xlsx", index=False)               # 寫 Excel
df.to_json("output.json", orient="records")           # 寫 JSON
df.to_parquet("output.parquet")                       # 寫 Parquet（大資料用）
```
