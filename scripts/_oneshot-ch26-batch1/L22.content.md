### Tensor（張量、PyTorch 的核心資料結構）

**Tensor** = 「**會記得梯度、能在 GPU 跑的多維陣列**」——基本上像 NumPy ndarray（L13）、但多了 2 個特性：
1. **可以在 GPU 跑**（NumPy 只能 CPU）
2. **可以追蹤梯度**（自動微分用、見下方）

```text
import torch

# === 從 list 建 ===
x = torch.tensor([1, 2, 3, 4])
print(x.shape)              # torch.Size([4])
print(x.dtype)              # torch.int64

# === 從 NumPy 轉（互通）===
import numpy as np
x = torch.from_numpy(np.array([1, 2, 3]))

# === 特殊建（同 NumPy）===
torch.zeros(3, 4)           # 3x4 全 0
torch.ones(2, 3)            # 2x3 全 1
torch.randn(3, 3)           # 標準常態分佈
```

  
**運算（跟 NumPy 幾乎一樣的 API）：**

```text
a = torch.tensor([1, 2, 3])
b = torch.tensor([4, 5, 6])

a + b                       # 元素加：[5 7 9]
a @ b                       # 矩陣乘 / dot product：32
a.sum()                     # 6
a.mean()                    # 2.0

# === reshape ===
x.view(6, 4)                # 改形狀（要連續記憶體）
x.reshape(-1, 4)            # 同上、-1 自動算
```

  
**把 Tensor 搬到 GPU（重點！沒搬 GPU = 沒加速）：**

```text
# === 自動選 device（GPU 優先、沒有就 CPU）===
device = torch.device("cuda" if torch.cuda.is_available() else "cpu")

# === Mac M1 / M2 / M3 用 MPS（Metal Performance Shaders）===
device = torch.device("mps" if torch.backends.mps.is_available() else "cpu")

# === 把 tensor 搬上去 ===
x = x.to(device)
```

  
### Autograd（自動微分、PyTorch 最強特性）

**autograd** = 「**自動算梯度**」—— PyTorch 會自己記錄你做了哪些運算、然後一行 `.backward()` 反向算出每個變數的梯度。

```text
# === 標記「我要追蹤這個變數的梯度」===
x = torch.tensor([2.0], requires_grad=True)

# === 做運算（PyTorch 自動建「計算圖」）===
y = x ** 2 + 3 * x + 1

# === 一行算梯度 ===
y.backward()
print(x.grad)               # tensor([7.])
# 為什麼是 7？dy/dx = 2x + 3、x=2 時 = 7
```

> 💡 **神經網路就靠這個更新 weight**——每個 weight 都 `requires_grad=True`、跑 `loss.backward()` 自動算出「每個 weight 該往哪調」。

  
### 完整範例：第一個 NN 分類 iris

把 L17 的 iris 分類用 PyTorch 重寫一次、看 DL 跟傳統 ML 流程差別。

**準備資料（跟 L17 一樣 sklearn 抓）：**

```text
import torch
import torch.nn as nn                       # nn = neural network 模組
import torch.optim as optim                 # optimizer 模組
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

# 1. 載入 + 標準化（L18 教過）
iris = load_iris()
X = StandardScaler().fit_transform(iris.data)
X_train, X_test, y_train, y_test = train_test_split(
    X, iris.target, test_size=0.2, random_state=42
)

# 2. 轉成 PyTorch Tensor
X_train = torch.FloatTensor(X_train)        # 特徵用 float（NN 要連續數值）
X_test = torch.FloatTensor(X_test)
y_train = torch.LongTensor(y_train)         # label 用 long（整數類別 id）
y_test = torch.LongTensor(y_test)
```

  
**定義 Model（繼承 `nn.Module`）：**

```text
class IrisNet(nn.Module):
    def __init__(self):
        super().__init__()                  # 必呼叫父類別 init
        # 定義「層」（這裡 3 層全連接）
        self.fc1 = nn.Linear(4, 16)         # 輸入 4 特徵 → 隱藏 16 neuron
        self.fc2 = nn.Linear(16, 8)         # 隱藏 16 → 隱藏 8
        self.fc3 = nn.Linear(8, 3)          # 隱藏 8 → 輸出 3 類別
        self.relu = nn.ReLU()               # activation function
    
    def forward(self, x):                   # 前向傳播（PyTorch 自動呼叫）
        x = self.relu(self.fc1(x))          # 第 1 層 + ReLU
        x = self.relu(self.fc2(x))          # 第 2 層 + ReLU
        x = self.fc3(x)                     # 第 3 層（output 不加 activation、後面 loss 處理）
        return x

model = IrisNet()
```

  
**Loss + Optimizer：**

```text
# === Loss：CrossEntropy（多分類用）===
criterion = nn.CrossEntropyLoss()

# === Optimizer：Adam（L21 提過、新手預設）===
optimizer = optim.Adam(model.parameters(), lr=0.01)
# model.parameters() = 所有 weight、optimizer 會更新這些
```

  
**訓練 loop（PyTorch 5 步驟、背起來）：**

```text
for epoch in range(100):                    # 跑 100 個 epoch
    # === 1. Forward（model 預測）===
    outputs = model(X_train)
    
    # === 2. 算 Loss ===
    loss = criterion(outputs, y_train)
    
    # === 3. 清空 gradient（不清會累加、踩坑）===
    optimizer.zero_grad()
    
    # === 4. Backward（算 gradient）===
    loss.backward()
    
    # === 5. Update weights ===
    optimizer.step()
    
    if (epoch + 1) % 10 == 0:
        print(f"Epoch {epoch+1}, Loss: {loss.item():.4f}")
```

> 💡 **「zero_grad → forward → loss → backward → step」 是 PyTorch 訓練的「五字訣」**、所有 NN 都這樣寫、背起來。

  
**評估（記得 `model.eval()` + `torch.no_grad()`）：**

```text
model.eval()                                # 切到「評估模式」（關掉 dropout / BN 訓練行為）
with torch.no_grad():                       # 不算梯度（省記憶體 + 快）
    outputs = model(X_test)
    _, predicted = torch.max(outputs, 1)    # 取每筆最大機率的類別
    accuracy = (predicted == y_test).float().mean()
    print(f"Test Accuracy: {accuracy:.2%}")
```

  
### DataLoader（處理大資料、自動分 batch）

大資料一次塞進 model 會 OOM、要分 batch（一次餵一小批）。PyTorch 內建 `Dataset` + `DataLoader` 處理。

```text
from torch.utils.data import Dataset, DataLoader

# === 1. 定義自己的 Dataset（繼承 Dataset）===
class IrisDataset(Dataset):
    def __init__(self, X, y):
        self.X = torch.FloatTensor(X)
        self.y = torch.LongTensor(y)
    
    def __len__(self):                      # 必實作：總共幾筆資料
        return len(self.X)
    
    def __getitem__(self, idx):             # 必實作：拿第 idx 筆
        return self.X[idx], self.y[idx]

# === 2. 建 DataLoader（自動分 batch + shuffle）===
train_dataset = IrisDataset(X_train, y_train)
train_loader = DataLoader(
    train_dataset,
    batch_size=32,                          # 一次 32 筆
    shuffle=True                            # 每個 epoch 打亂順序
)
```

  
**用 DataLoader 訓練：**

```text
for epoch in range(100):
    for batch_X, batch_y in train_loader:   # 每個 batch 跑五字訣
        outputs = model(batch_X)
        loss = criterion(outputs, batch_y)
        
        optimizer.zero_grad()
        loss.backward()
        optimizer.step()
```

  
### 存 + 載 model

```text
# === 存 state_dict（推薦、只存 weight、檔案小）===
torch.save(model.state_dict(), "model.pth")

# === 載 ===
model = IrisNet()                           # 先建一個一樣結構的 model
model.load_state_dict(torch.load("model.pth"))
model.eval()                                # 載完切評估模式
```

> 💡 **為什麼推薦存 state_dict 而不存整個 model？** 存整個 model 會綁定當下的 class 定義、之後重構 class 就讀不回來。存 state_dict 只存「數字」、更可靠。
