Python 內建 4 個容器（裝資料的盒子）：**list / tuple / set / dict**——組合起來解 90% 資料處理問題。

> 💡 **這課全部範例都可以在 REPL 試**（在終端機打 `python` 進去）。容器是 Python 最核心的東西、一定要邊試邊學、不要只看。

  
### List（清單、可變、有順序）

**list** = 「一排有順序、可以隨時增刪改的東西」。最常用、最萬用。

📄 在 REPL 試：

```text
# 建立清單（用方括號 []）
fruits = ["蘋果", "香蕉", "橘子"]
empty = []                                  # 空清單
mixed = [1, "hello", True, [1, 2]]          # 什麼都能裝、甚至 list 套 list

# === 取單個（用 index、從 0 開始）===
fruits[0]              # "蘋果"
fruits[-1]             # "橘子"（負數 = 從後往前）

# === 切片（slice、取連續一段）===
fruits[0:2]            # ["蘋果", "香蕉"]
fruits[:2]             # 同上、開頭可以省略
fruits[1:]             # ["香蕉", "橘子"]、結尾省略 = 到最後
fruits[::-1]           # ["橘子", "香蕉", "蘋果"]、反轉整個清單

# === 修改（list 是 mutable（可變）的、可以直接改）===
fruits[0] = "草莓"
print(fruits)          # ["草莓", "香蕉", "橘子"]
```

  
**新增 / 刪除（最常用的操作）：**

```text
# === 新增 ===
fruits.append("葡萄")              # 加到最後面（append = 附加）
fruits.insert(0, "西瓜")           # 插到指定位置（這裡是位置 0）
fruits.extend(["檸檬", "桃子"])    # 一次加多個（extend = 延伸）
# 或用 +=
fruits += ["檸檬", "桃子"]         # 同 extend

# === 刪除 ===
fruits.remove("香蕉")              # 按「值」刪（刪第一個遇到的）
fruits.pop()                       # 刪最後 1 個、回傳被刪的（pop = 彈出）
fruits.pop(0)                      # 刪指定位置
del fruits[0]                      # 同 pop(0)、但不回傳值
fruits.clear()                     # 全清空
```

  
**查詢 / 排序：**

```text
# === 查 ===
"蘋果" in fruits                   # True（in 運算符、超快又直觀）
fruits.index("橘子")               # 2（找位置）
fruits.count("蘋果")               # 1（算出現幾次）
len(fruits)                        # 3（長度）

# === 排序 ===
nums = [3, 1, 4, 1, 5, 9, 2, 6]

nums.sort()                        # 「原地」排序、會直接改 nums
print(nums)                        # [1, 1, 2, 3, 4, 5, 6, 9]

nums.sort(reverse=True)            # 降冪（大到小）

# 不想動原本 → 用 sorted()、回傳「新的」list
sorted_nums = sorted(nums)         # 新 list、原 nums 不變

# 自訂排序規則（用 key 參數 + lambda 小函式）
users = [{"name": "Alice", "age": 30}, {"name": "Bob", "age": 25}]
users.sort(key=lambda u: u["age"])  # 依年齡排（lambda 是「臨時小函式」、L6 會詳教）

# === 反轉 ===
nums.reverse()             # 原地反轉、改 nums
reversed_nums = nums[::-1] # 新的反轉清單、原 nums 不變

# === 串接 / 重複 ===
nums + [10, 11]            # 串成新 list
[0] * 5                    # [0, 0, 0, 0, 0]（重複 5 次）
```

  
### List Comprehension（生成式、Python 特色語法）

**comprehension（生成式）** = 「用一行精簡語法、快速產生 list」。Python 寫 code 最 **idiomatic（地道、行話最常用）** 的寫法之一。

```text
# === 經典場景：建一個 0 到 9 的平方清單 ===

# 傳統寫法（4 行）
squares = []
for x in range(10):
    squares.append(x ** 2)

# Comprehension（1 行）
squares = [x ** 2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 念法：「對於 range(10) 裡的每個 x、算 x 的 2 次方、組成 list」
```

  
**進階寫法（加條件、巢狀）：**

```text
# === 加條件 if（只要符合條件的）===
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]（只留偶數）

# 變形 + 條件
squared_evens = [x ** 2 for x in range(10) if x % 2 == 0]
# [0, 4, 16, 36, 64]（偶數的平方）

# === 巢狀（nested、外圈套內圈）===
matrix = [[i * j for j in range(3)] for i in range(3)]
# [[0, 0, 0], [0, 1, 2], [0, 2, 4]]
# 念法：「對 i 從 0 到 2、內層對 j 從 0 到 2、產生 i*j」

# === 平面化（flatten、把 list 套 list 攤平成一條）===
matrix = [[1, 2, 3], [4, 5, 6], [7, 8, 9]]
flat = [x for row in matrix for x in row]
# [1, 2, 3, 4, 5, 6, 7, 8, 9]

# === 字串處理 ===
names = ["alice", "bob", "charlie"]
upper_names = [n.upper() for n in names]
# ['ALICE', 'BOB', 'CHARLIE']
```

→ List comprehension 是 Python 的「正字標記」、寫 Python 一定要熟練
→ 比 `map()` / `filter()` 更 **pythonic（地道 Python 風格）**

  
### Tuple（元組、不可變、有順序）

**tuple** = 「一組固定不能改的東西」。長得像 list、但**建立後不能加 / 不能刪 / 不能改**。

```text
# 建立（用括號 () 或乾脆不用、靠逗號就懂）
point = (3, 4)
point = 3, 4              # 一樣！Python 看到逗號就知道是 tuple
empty = ()                # 空 tuple
single = (5,)             # ⚠️ 一個元素要加逗號、不然會被當成「括號內的數字」
single = 5,               # 也行

# 索引、slice 寫法跟 list 一樣
point[0]                  # 3
point[-1]                 # 4

# ⚠️ Tuple 是 immutable（不可變）
point[0] = 99             # ❌ TypeError: 'tuple' object does not support item assignment
```

  
**為什麼用 tuple？4 個情境：**

```text
# 1. 表示「一組相關但不會改」的資料
person = ("Alice", 30, "Engineer")

# 然後可以「解包（unpack）」成多個變數
name, age, job = person   # Python 特色語法
# 等於 name = person[0]; age = person[1]; job = person[2]

# 2. 函式回傳多個值（其實就是回傳一個 tuple）
def divide(a, b):
    return a // b, a % b      # 回 (商數, 餘數)
q, r = divide(10, 3)          # q=3, r=1

# 3. 當 dict 的 key（list 不能、tuple 可以！因為不變）
locations = {
    (25.04, 121.56): "Taipei",
    (35.69, 139.69): "Tokyo",
}

# 4. 性能（比 list 略快一點、因為不變、Python 可以最佳化）
```

  
**進階：Named Tuple（命名元組、像迷你 class）：**

```text
# 用 collections.namedtuple
from collections import namedtuple

Point = namedtuple("Point", ["x", "y"])
p = Point(3, 4)
print(p.x, p.y)            # 3 4（可以用 .欄位名 取、不用記 index）

# 2024+ 主流：用 typing.NamedTuple（有 type hint 更清楚）
from typing import NamedTuple

class Point(NamedTuple):
    x: int
    y: int

p = Point(3, 4)
print(p.x)                 # 3
```

  
### Set（集合、不重複、沒順序）

**set** = 「一堆東西、不允許重複、沒有順序」。最大用途：**去重**。

```text
# 建立（用大括號 {}、但要小心：空 set 不能用 {} 那是 dict）
fruits = {"蘋果", "香蕉", "橘子"}
empty = set()                     # ✅ 空 set 必須用 set()
empty_wrong = {}                  # ❌ 這是空 dict、不是空 set

# 自動去重（最常用！）
nums = [1, 2, 3, 1, 2, 3]
unique = set(nums)                # {1, 2, 3}（重複的自動消失）

# 新增 / 刪除
fruits.add("葡萄")
fruits.remove("香蕉")             # 不存在會 raise（拋出）KeyError
fruits.discard("xxx")             # 不存在「不會」報錯、適合不確定時用
```

  
**集合運算（數學課那種、Python 直接內建）：**

```text
a = {1, 2, 3, 4}
b = {3, 4, 5, 6}

a | b           # 聯集（union）         {1, 2, 3, 4, 5, 6}
a & b           # 交集（intersection） {3, 4}
a - b           # 差集（difference）   {1, 2}（在 a、不在 b）
a ^ b           # 對稱差（symmetric diff）{1, 2, 5, 6}（只在其中一邊）

# 子集 / 父集（subset / superset）
{1, 2}.issubset({1, 2, 3})         # True（{1,2} 是 {1,2,3} 的子集）
{1, 2, 3}.issuperset({1, 2})       # True（反過來）
```

  
**Set 的 3 大用途：**

```text
# 1. 去重
items = ["a", "b", "a", "c", "b"]
unique = list(set(items))         # ['a', 'b', 'c']（順序可能變、因為 set 無序）

# 2. 快速查找（in 是 O(1)、超快——比 list 的 O(n) 快很多）
# O(1) = 不管 set 多大、查一次的時間都一樣
banned_words = {"spam", "abuse", "scam"}
if word in banned_words:          # 不管 banned_words 有 10 個還是 1 萬個都一樣快
    block()

# 3. 算差異（誰加入、誰離開）
old_users = {"alice", "bob"}
new_users = {"bob", "charlie"}
left = old_users - new_users      # {"alice"}（離開的）
joined = new_users - old_users    # {"charlie"}（新加入的）

# === Set comprehension（跟 list 一樣可以生成式）===
unique_lengths = {len(word) for word in ["hi", "hello", "world"]}
# {2, 5}（去重後）
```

  
### Dict（字典、鍵值對、最強大的容器）

**dict** = 「用『鍵（key）→ 值（value）』成對存資料」。像查字典：給一個詞（key）、找定義（value）。

```text
# 建立（用大括號 {}、key:value 一對）
user = {
    "name": "Alice",
    "age": 30,
    "email": "alice@x.com"
}

empty = {}                    # 空 dict
```

  
**取值（有 2 種方式、推薦用 `.get()`）：**

```text
user["name"]              # "Alice"
user["phone"]             # ❌ 沒這個 key 會 raise KeyError

# === 用 .get()（沒 key 不會錯、回傳 None）===
user.get("name")          # "Alice"
user.get("phone")         # None（沒就回 None）
user.get("phone", "N/A")  # "N/A"（可以給「預設值」、找不到就用這個）

# 推薦：不確定 key 在不在、用 .get()
```

  
**新增 / 修改 / 刪除：**

```text
# 新增 / 修改（語法一樣、key 存在就改、不存在就加）
user["age"] = 31
user["phone"] = "0912-..."

# 刪除
del user["phone"]
user.pop("phone")             # 也行、回傳被刪的 value
user.pop("xxx", None)         # 沒 key 不會報錯（給預設值 None）

# 檢查 key 在不在
"name" in user                # True
"phone" in user               # False
```

  
**迭代（for 跑過所有元素）：**

```text
# 預設跑「key」
for key in user:
    print(key)
# name / age / email

# 跑 value
for value in user.values():
    print(value)

# 同時跑 key + value（最常用！）
for key, value in user.items():
    print(f"{key}: {value}")
# name: Alice
# age: 30
# email: alice@x.com

# 長度
len(user)                     # 3（多少對 key-value）
```

  
**合併 dict：**

```text
# 方法 1：update（原地改）
user.update({"city": "Taipei"})

# 方法 2：spread 展開（Python 3.5+、不改原本、產生新 dict）
combined = {**user, "city": "Taipei"}

# 方法 3：| 運算符（Python 3.9+、最新最簡潔）
combined = user | {"city": "Taipei"}
```

  
**Dict Comprehension（同樣有生成式語法）：**

```text
# 建一個「數字→平方」對照
squares = {x: x ** 2 for x in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# 反轉 key-value（把字典翻過來）
inverted = {v: k for k, v in user.items()}
```

  
**進階：`collections` 模組裡的兩個神器**

```text
# === defaultdict（給每個 key 一個預設值、省去「先檢查再加」）===
from collections import defaultdict

counts = defaultdict(int)         # 預設值是 int()、也就是 0
for word in ["a", "b", "a", "c", "a"]:
    counts[word] += 1             # 即使 word 還不在 dict 裡也不會錯（自動建 0）
# defaultdict(int, {'a': 3, 'b': 1, 'c': 1})

# === Counter（直接幫你算次數、最常用）===
from collections import Counter

words = ["a", "b", "a", "c", "a"]
c = Counter(words)                # Counter({'a': 3, 'b': 1, 'c': 1})
c.most_common(2)                  # [('a', 3), ('b', 1)]（出現最多的前 2 名）
```

  
### 4 個容器一覽表（學完背這個就夠）

| 容器 | 有順序 | 可改 | 允許重複 | 主要用途 |
|---|---|---|---|---|
| **List** | ✓ | ✓ | ✓ | 順序資料、最常用 |
| **Tuple** | ✓ | ✗ | ✓ | 固定組合、函式多回傳值 |
| **Set** | ✗ | ✓ | ✗ | 去重、快速查找 |
| **Dict** | ✓ (3.7+) | ✓ | key 不重複 | 鍵值對、最強大 |

  
    **🚀 進階技巧**
    
Python 容器選擇心法：(1) **要順序 + 可改** → list (2) **要當 dict key / 不會變** → tuple (3) **要去重 / 快速 `in` 檢查** → set (4) **用名字找東西** → dict。**2026 加分**：`collections.Counter`（算次數）、`collections.defaultdict`（給預設值）、`typing.NamedTuple`（替代純 tuple、加 type hint 更清楚）。學完這 4 個 + comprehension = 寫 90% Python code 順手。
