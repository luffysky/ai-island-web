
- `uv add fastapi` —— 裝 FastAPI 框架本身
- `uvicorn` —— ASGI server（負責「實際跑」FastAPI 程式、見下方 ASGI 段）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
from fastapi import FastAPI

app = FastAPI()                              # 建立一個 FastAPI app 實例

@app.get("/")                                # @app.get = 註冊 GET /
def hello():
    return {"message": "Hello, World"}       # 自動轉 JSON

@app.get("/users/{user_id}")                 # {user_id} = 路徑參數
def get_user(user_id: int):                  # type hint int = 自動驗證 + 轉型
    return {"user_id": user_id, "name": "Alice"}
```

**這行做什麼？逐個解釋：**

### 路徑參數 / 查詢參數 / Body 3 種傳值方式

```text
from fastapi import FastAPI
from pydantic import BaseModel

app = FastAPI()

# === 1. 路徑參數（path parameter、URL 內的變數）===
@app.get("/users/{user_id}")
def get_user(user_id: int):                  # type hint = 自動驗證 + 轉型
    return {"user_id": user_id}
# 用：GET /users/42

# === 2. 查詢參數（query parameter、URL 後面 ? 開始）===
@app.get("/search")
def search(q: str, limit: int = 10, offset: int = 0):
    return {"q": q, "limit": limit, "offset": offset}
# 用：GET /search?q=python&limit=20

# === 3. 路徑 + 查詢混用 ===
@app.get("/users/{user_id}/orders")
def get_orders(user_id: int, status: str = "all"):
    return {"user_id": user_id, "status": status}
# 用：GET /users/42/orders?status=pending
```

  
**Body（POST / PUT 用、JSON 格式、Pydantic 自動驗證）：**

```text
class UserCreate(BaseModel):
    name: str
    age: int
    email: str

@app.post("/users")
def create_user(user: UserCreate):           # FastAPI 自動把 request body 解析成 UserCreate
    return {"created": user}
```

  
**自動驗證（這是 FastAPI 最大優勢）：**

```text
=== 收到合法 body ===
{"name": "Alice", "age": 30, "email": "..."}
→ ✓ 200 OK

=== 缺欄位 ===
{"name": "Alice"}
→ ❌ 422 Unprocessable Entity
   FastAPI 自動回詳細錯誤：「age field required」

=== 型別錯 ===
{"name": "Alice", "age": "thirty"}
→ ❌ 422
   「age must be int」
```

  
### ASGI vs WSGI（Python 後端兩種標準）

```text
=== WSGI（Web Server Gateway Interface、傳統）===
- Django / Flask 用
- 同步、1 request 1 thread
- 高並發痛苦（thread 多 = 記憶體爆）
- 不支援 WebSocket / 即時通訊

=== ASGI（Asynchronous Server Gateway Interface、modern）===
- FastAPI / Starlette / Sanic 用
- async 原生（一個 thread 處理上千個 connection）
- 支援 WebSocket / Server-Sent Events / HTTP/2
- 高並發強（爬蟲 / 即時應用必選）
```

  
**2026 主流 ASGI server：**

| Server | 特色 |
|---|---|
| **Uvicorn** | 最快、純 ASGI、FastAPI 預設 |
| **Hypercorn** | 支援 HTTP/2 / HTTP/3 |
| **Daphne** | Django Channels 用 |

  
**Production 跑法：**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（production 跑、4 個 worker process）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uvicorn main:app --host 0.0.0.0 --port 8000 --workers 4
```

**這行做什麼？逐個解釋：**
- `--host 0.0.0.0` —— 監聽**所有網路介面**（不只 localhost）、外部能連
- `--port 8000` —— 監聽 port 8000
- `--workers 4` —— 起 4 個 worker process 並行處理（一般設 = CPU 核心數）
- **沒有 `--reload`** —— production 不要熱重啟、會慢

> ⚠️ **真實 production 不會直接這樣跑**——前面通常有 nginx / Caddy 當 reverse proxy + SSL、再轉給 Uvicorn。L32 部署課會教完整方案。

  
### 路由組織（大型 app 用 APIRouter 拆檔）

當 API 變多、main.py 會爆炸。用 `APIRouter` 模組化。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 **寫進程式檔 routers/users.py**（一個 user 相關 API 的模組）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
from fastapi import APIRouter

router = APIRouter(
    prefix="/users",                          # 所有路由自動加 /users 前綴
    tags=["users"]                            # 自動文件分類
)

@router.get("/")
def list_users():
    return []

@router.get("/{user_id}")
def get_user(user_id: int):
    return {"user_id": user_id}

@router.post("/")
def create_user(...):
    ...
```

  
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 **寫進程式檔 main.py**（把 router 接進 app）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
from fastapi import FastAPI
from routers import users, products

app = FastAPI()
app.include_router(users.router)              # 把 user router 接進來
app.include_router(products.router)
```

  
### HTTP method + status code（HTTP 基本功）

```text
from fastapi import HTTPException, status

# === GET：讀取（200 OK 預設）===
@app.get("/users")
def list_users():
    return []

# === POST：新增（201 Created 慣例）===
@app.post("/users", status_code=201)
def create_user(user: UserCreate):
    return {"id": 1, **user.dict()}

# === PATCH：部分修改 ===
@app.patch("/users/{user_id}")
def update_user(user_id: int, user: UserUpdate):
    ...

# === DELETE：刪除（204 No Content 慣例、無回傳）===
@app.delete("/users/{user_id}", status_code=204)
def delete_user(user_id: int):
    ...
```

  
**自訂錯誤（找不到 = 404、不能用 status 200 回 error 訊息！）：**

```text
@app.get("/users/{user_id}")
def get_user(user_id: int):
    user = db.find(user_id)
    if not user:
        raise HTTPException(
            status_code=404,
            detail=f"User {user_id} not found"
        )
    return user
```

> 💡 **HTTP status code 速記**（L75 完整教）：
> - **2xx 成功**：200 OK / 201 Created / 204 No Content
> - **4xx 客戶端錯**：400 Bad Request / 401 未登入 / 403 沒權限 / 404 找不到 / 422 驗證失敗
> - **5xx 伺服器錯**：500 Internal Error / 502 Bad Gateway / 503 Service Unavailable
