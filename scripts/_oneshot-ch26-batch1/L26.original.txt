### 基本用法

```text
from pydantic import BaseModel
from typing import Optional

class Product(BaseModel):
    name: str
    price: float
    stock: int = 0                  # 預設值
    description: Optional[str] = None
    tags: list[str] = []

# 建立
p = Product(name="iPhone", price=999.99, stock=10)

# 從 dict
data = {"name": "iPhone", "price": "999.99"}
p = Product(**data)
# Pydantic 會自動轉 string "999.99" → float 999.99

# 自動驗證
Product(name="iPhone", price="not a number")
# ValidationError
```

  
### 進階型別 + 驗證

```text
from pydantic import BaseModel, EmailStr, HttpUrl, conint, constr
from datetime import datetime

class User(BaseModel):
    # 範圍限制
    age: conint(ge=0, le=120)           # 0-120
    
    # 字串限制
    username: constr(min_length=3, max_length=20)
    
    # Email 格式
    email: EmailStr
    
    # URL 格式
    website: HttpUrl
    
    # 時間
    created_at: datetime
    
    # 列表
    tags: list[str] = []
    
    # 巢狀
    address: "Address"

class Address(BaseModel):
    street: str
    city: str
    country: str

# 自動驗證 email、url、字串長度、整數範圍...
```

# Discriminated Union（多型）
from typing import Literal
from pydantic import Field

class Cat(BaseModel):
    pet_type: Literal["cat"]
    meow: str

class Dog(BaseModel):
    pet_type: Literal["dog"]
    bark: str

class PetOwner(BaseModel):
    pet: Cat | Dog = Field(discriminator="pet_type")
