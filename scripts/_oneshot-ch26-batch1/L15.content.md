### Matplotlib：基礎繪圖（所有 Python 圖的祖師爺）

**慣例 import 方式**：`import matplotlib.pyplot as plt`（暱稱 `plt`、業界統一）。

```text
import matplotlib.pyplot as plt

# === 折線圖（line chart）===
plt.plot([1, 2, 3, 4], [10, 20, 25, 30])
plt.title("Sales")                      # 圖標題
plt.xlabel("Month")                     # X 軸標籤
plt.ylabel("Revenue")                   # Y 軸標籤
plt.show()                              # 顯示（Jupyter 內可省略）
```

  
**多條線 + 圖例（legend）：**

```text
plt.plot(months, product_a, label="A", marker="o")   # marker="o" 每點畫圓
plt.plot(months, product_b, label="B", marker="s")   # marker="s" 每點畫方
plt.legend()                            # 顯示圖例（用 label 的值）
plt.show()
```

  
**其他常用圖型：**

```text
# === 長條圖（bar chart）===
plt.bar(["A", "B", "C"], [10, 24, 36])

# === 直方圖（histogram、看數值分布）===
plt.hist(np.random.randn(1000), bins=30)
# bins=30 = 把資料切成 30 個「桶」

# === 散佈圖（scatter plot、看 X-Y 關係）===
plt.scatter(x, y)

# === 圓餅圖（pie chart）===
plt.pie([30, 25, 20, 15], labels=["A", "B", "C", "D"])
```

  
**子圖（subplots、一張畫布放多個圖）：**

```text
fig, axes = plt.subplots(2, 2, figsize=(10, 8))      # 2x2 排版、總大小 10x8 吋
axes[0, 0].plot([1, 2, 3], [1, 4, 9])                # 左上：折線
axes[0, 1].bar(["a", "b"], [1, 2])                   # 右上：長條
plt.tight_layout()                       # 自動調間距、避免重疊
plt.savefig("chart.png", dpi=300)        # 存檔（dpi=300 印刷品質）
```

  
### Seaborn：統計圖、預設美觀

**Seaborn** 是「**蓋在 Matplotlib 上的高階套件**」——同樣功能寫起來更簡潔、預設配色就漂亮、特別適合**統計探索**（看資料分布、相關性、組間比較）。

```text
import seaborn as sns                   # 慣例暱稱：sns

# === 直接吃 Pandas DataFrame、不用先處理 ===
sns.scatterplot(data=df, x="age", y="salary", hue="department")
# hue="department" = 按 department 上不同顏色

sns.histplot(data=df, x="salary", bins=30)          # 直方圖
sns.boxplot(data=df, x="department", y="salary")    # 箱形圖（看分布 + 中位數 + 四分位）
```

  
**熱力圖（heatmap、看相關性矩陣）：**

```text
correlation = df.corr()                              # Pandas 算所有數值欄的相關性
sns.heatmap(
    correlation,
    annot=True,                                       # 在每格顯示數字
    cmap="coolwarm"                                   # 顏色配置：冷暖色（負紅、正藍）
)
```

  
**Pair plot（一次畫所有 2 欄組合、超快探索）：**

```text
sns.pairplot(df, hue="category")
# 自動把所有數值欄兩兩配對、畫散佈圖矩陣
# 對角線自動畫直方圖
# 超適合「第一次拿到資料、看哪兩個欄位相關」
```

  
**風格設定：**

```text
sns.set_style("whitegrid")              # 白底 + 灰格線（推薦）
sns.set_palette("husl")                 # 換調色盤（colorblind / pastel / husl / Set2...）
```

→ **Seaborn = 統計探索神器**、Data Science 第一選擇

  
### Plotly：互動式 + Web 友善

**Plotly** 強在「**互動性**」—— 滑鼠 hover 看數值、可 zoom / pan、適合做 dashboard / 嵌入網頁。

```text
import plotly.express as px             # 慣例：px（plotly express、高階簡潔 API）

# === 互動式散佈圖（hover 顯示資料、可 zoom）===
fig = px.scatter(
    df, x="age", y="salary",
    color="department",                 # 按部門上色
    hover_data=["name", "city"]         # 滑鼠 hover 額外顯示這些欄位
)
fig.show()                              # 跳出互動圖

# === 折線（時間序列必用）===
px.line(df, x="date", y="sales", color="product")

# === 3D 散佈圖 ===
px.scatter_3d(df, x="x", y="y", z="z", color="category")

# === 動畫（時間軸自動跑、超酷）===
px.scatter(
    df, x="gdp", y="life_exp",
    animation_frame="year",             # 按年份做動畫
    size="population",
    color="continent"
)

# === 寫成 HTML 嵌入網站 ===
fig.write_html("dashboard.html")
```

→ **Plotly = Web / Dashboard 場景的標配**
→ Dash 框架（Plotly 出的）= Python 版 Tableau

  
### 怎麼選？場景對照表

| 場景 | 推薦工具 |
|---|---|
| 學術論文、靜態圖、印刷 | **Matplotlib**（精細控制 + savefig PNG） |
| 統計探索分析（EDA） | **Seaborn**（一行畫好看的圖） |
| 互動 dashboard、嵌入網頁 | **Plotly**（hover / zoom） |
| Jupyter Notebook 日常 | 3 個都行（看當下需求） |
| 快速看 DataFrame 一眼 | **`df.plot()` 內建**（Pandas 自帶） |

  
### Pandas 內建快速畫圖（最方便、不用 import）

Pandas 內建 `.plot()` 直接畫、底層就是 matplotlib。

```text
df["sales"].plot()                      # 一行畫折線
df["sales"].plot.bar()                  # 長條
df["sales"].plot.hist(bins=30)          # 直方圖
df.plot.scatter(x="age", y="salary")    # 散佈圖

# === 加參數 ===
df.plot(
    figsize=(10, 6),                    # 圖大小（寬 x 高、單位英吋）
    title="Sales",                      # 標題
    grid=True                           # 顯示格線
)
plt.show()
```
