Python 4 個基本型態—學熟就掌握 80% 程式都在處理的東西。

> 💡 **這課全部範例都可以在 REPL 試**（在終端機打 `python` 進去）—— 不需要寫成 .py 檔。打一行、立刻看結果、最適合學型態。

  
### int（整數、integer）

**int** 就是「沒有小數的數字」、例如 25、0、-10。Python 的 int 有個特色：**沒有上限**（其他語言會 overflow（溢位、數字超過上限會跑出怪結果）、Python 不會）。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 寫進程式檔 / 或在 REPL 試

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
age = 25
big_num = 1_000_000        # 底線分位、讓人類好讀（電腦會忽略底線）
negative = -10
zero = 0

# Python 整數沒上限
huge = 10 ** 100           # 10 的 100 次方（一個 1 後面接 100 個 0）
print(huge)
# 印出整串巨大數字、不會出錯

# 對照：Java / Go 會 overflow（溢位、數字大到撐爆變怪結果）、Python 不會
```

  
**運算符號：**

```text
a, b = 10, 3
print(a + b)     # 13   加
print(a - b)     # 7    減
print(a * b)     # 30   乘
print(a / b)     # 3.333333... （除法、結果永遠是 float 小數）
print(a // b)    # 3    （整數除、只保留商數）
print(a % b)     # 1    （取餘數）
print(a ** b)    # 1000 （次方、a 的 b 次方）
```

  
**進制（不同寫法表達同一個數）：**

```text
binary  = 0b1010      # 二進位（0b 開頭）   = 10
octal   = 0o12        # 八進位（0o 開頭）   = 10
hex_num = 0xa         # 十六進位（0x 開頭） = 10
```

  
**型別轉換：**

```text
int("42")            # 42  （從字串轉成整數）
int("42", 16)        # 66  （把 "42" 當成 16 進位字串解析）
int(3.9)             # 3   （浮點轉整數、直接砍掉小數、不是四捨五入！）
int(True)            # 1   （Python 裡 True = 1）
int(False)           # 0   （False = 0）
```

> 💡 **Python 3 只有 int、沒有 Java 那種 long type**——所有整數都用 int 就好。

  
### float（浮點數、有小數的數字）

**float** = 帶小數點的數字、例如 3.14、1.5e10、-0.001。

```text
pi = 3.14159
big = 1.5e10        # 科學記號、= 1.5 × 10 的 10 次方 = 15000000000
small = 1.5e-5      # = 1.5 × 10 的負 5 次方 = 0.000015

# int 跟 float 混算 → 結果自動變 float
print(10 + 1.5)     # 11.5（變成 float）
```

  
**⚠️ 浮點陷阱（所有語言都有、不只 Python）：**

```text
print(0.1 + 0.2)          # 0.30000000000000004  ← 不是 0.3！
print(0.1 + 0.2 == 0.3)   # False！
```

> **為什麼？** 電腦用二進位存小數、有些十進位小數（像 0.1）在二進位是「無限循環」、會被截斷、所以有微小誤差。**比較浮點數不能直接用 `==`**。

  
**對策有 3 種：**

```text
# 對策 1：math.isclose（檢查兩個值是不是「夠接近」）
import math
math.isclose(0.1 + 0.2, 0.3)   # True

# 對策 2：用 Decimal（精確小數、給「錢」相關計算用）
from decimal import Decimal
a = Decimal("0.1")
b = Decimal("0.2")
print(a + b)        # 0.3（精確、沒誤差）

# 對策 3：自己 round（四捨五入到指定位數）
round(0.1 + 0.2, 2)   # 0.3
```

  
**特殊值：inf 跟 nan**

- **inf** = infinity（無限大）
- **nan** = Not a Number（不是數字、例如除以 0 之類的結果）

```text
inf = float('inf')      # 無限大
neg_inf = float('-inf') # 負無限大
nan = float('nan')      # Not a Number

print(inf > 1000000)    # True（無限大比任何數都大）
print(nan == nan)       # False! ← nan 跟誰都不相等（包括自己）

import math
math.isnan(nan)         # True（要用 math.isnan 才能正確判斷）
```

  
**float 轉換：**

```text
float("3.14")           # 3.14
float("inf")            # inf
float(3)                # 3.0
```

  
### str（字串、文字）

**str**（string）= 一段文字。

```text
# 字串有 3 種寫法
s1 = "Hello"
s2 = 'Hello'             # 單引號、雙引號都可以、隨你喜歡
s3 = """多行
字串"""                   # 三個引號 = 可以跨多行
```

  
**特殊字元要 escape（跳脫、用反斜線 \ 標記）：**

```text
quote   = "She said \"hi\""    # \" = 引號本身、不算字串結尾
newline = "Line 1\nLine 2"     # \n = 換行
tab     = "col1\tcol2"         # \t = Tab 鍵
```

  
**Raw string（原始字串、不解釋 escape）：**

```text
# 字串前加 r、反斜線就「不會被當 escape」、直接是反斜線本身
path = r"C:\Users\Alice\Documents"
# 上面這個 \U 不會被當成 Unicode escape、就是普通反斜線
# 適合寫 Windows 檔案路徑、或正規表示式
```

  
**Unicode（萬國碼）——Python 字串可以含任何語言的字、含 emoji：**

```text
greeting = "你好"            # 中文沒問題
emoji = "👋 hello"           # emoji 也可以
```

  
**字串連接（拼起來）：**

```text
name = "Alice"

# 用 + 號連接
greeting = "Hello, " + name              # "Hello, Alice"

# 用 f-string（2026 主流寫法、引號前加 f）
greeting = f"Hello, {name}"              # "Hello, Alice"

# 用 .format() 方法（老式、現在很少用）
greeting = "Hello, {}".format(name)
```

  
**重複 + 查長度：**

```text
print("abc" * 3)         # "abcabcabc"（字串可以乘、就是重複）

print(len("Hello"))      # 5
print(len("你好"))        # 2（看字數、不是 byte 數）
```

  
**索引（index）+ 切片（slice）—— 從字串中取出一部分：**

> 💡 **重要：Python 索引從 0 開始**——第 1 個字是 index 0、不是 1。

```text
s = "Python"

# 單個字（用 [index]）
s[0]    # 'P'   ← 第 1 個字
s[1]    # 'y'   ← 第 2 個字
s[-1]   # 'n'   ← 從後面數第 1 個（負數是從右往左）
s[-2]   # 'o'   ← 從後面數第 2 個

# Slice（切片、取連續一段、用 [起點:終點]）
s[0:3]   # 'Pyt'   ← 從 index 0 取到 3（不含 3、所以是 0、1、2）
s[3:]    # 'hon'   ← 從 index 3 取到結尾
s[:3]    # 'Pyt'   ← 從開頭取到 index 3（不含）
s[::2]   # 'Pto'   ← 每隔 1 個取（step = 2）
s[::-1]  # 'nohtyP' ← step = -1 = 反轉整個字串
```

  
**⚠️ 字串是 immutable（不可變）—— 不能改某一位：**

```text
s = "Python"
s[0] = 'J'        # ❌ 會錯：TypeError: 'str' object does not support item assignment
                  # （TypeError = 型別錯誤、string 不支援這種操作）

# 想要改 → 建一個「新字串」取代
s = "J" + s[1:]   # 'Jython'（新字串、原來那個沒變）
```

  
### 字串常用方法（method）

**method** = 一個物件「自己會做的事」、用「物件.方法()」語法呼叫。例如字串有 `s.upper()` 把自己變大寫。

📄 在 REPL 試這些最快：

```text
s = "  Hello World  "

# === 大小寫 ===
s.upper()         # '  HELLO WORLD  '   （全變大寫）
s.lower()         # '  hello world  '   （全變小寫）
s.title()         # '  Hello World  '   （每個單字首字母大寫）
s.capitalize()    # '  hello world  '   （整段只有第一個字大寫）

# === 去掉首尾空白 ===
s.strip()         # 'Hello World'       （兩邊空白都去）
s.lstrip()        # 'Hello World  '     （只去左邊、left strip）
s.rstrip()        # '  Hello World'     （只去右邊、right strip）

# === 找位置 ===
s.find("World")   # 8   （找到、回傳位置 index）
s.find("Python")  # -1  （找不到、回傳 -1、不會錯誤）
s.index("World")  # 8   （同 find、但找不到會 raise（拋出）錯誤）

# === 判斷有沒有包含 ===
"Hello" in s              # True   （in 運算符、查包含）
"Python" in s             # False
s.startswith("  H")       # True   （是不是以這個開頭）
s.endswith("d  ")         # True   （是不是以這個結尾）

# === 替換 ===
s.replace("World", "Python")  # '  Hello Python  '

# === 切割（split） + 拼接（join） ===
"a,b,c".split(",")             # ['a', 'b', 'c']     （切成 list）
"line 1\nline 2".splitlines()  # ['line 1', 'line 2'] （按行切）

",".join(["a", "b", "c"])      # 'a,b,c'             （用 "," 把 list 拼成字串）
" ".join(["Hello", "World"])   # 'Hello World'

# === 判斷字串性質 ===
"abc".isalpha()    # True  （全部都是字母嗎）
"123".isdigit()    # True  （全部都是數字嗎）
"abc123".isalnum() # True  （全部都是字母或數字嗎、alphanumeric）
"   ".isspace()    # True  （全部都是空白字元嗎）

# === 格式化（對齊） ===
"{:>10}".format("hi")   # '        hi'     （右對齊、總寬 10）
"{:<10}".format("hi")   # 'hi        '     （左對齊）
"{:^10}".format("hi")   # '    hi    '     （置中）
"{:*<10}".format("hi")  # 'hi********'     （左對齊、用 * 補滿）
```

  
### bool（布林值、真假值）

**bool** 只有兩個值：**True** 跟 **False**。**注意：首字大寫**（Python 大小寫敏感、`true` / `false` 都會錯）。

```text
is_student = True
is_admin = False

# 邏輯運算
print(True and False)   # False（and = 同時為真才真）
print(True or False)    # True  （or = 任一為真就真）
print(not True)         # False （not = 反轉）

# Python 裡哪些值會被當「假」？
# False、None、0、0.0、""（空字串）、[]（空清單）、{}（空字典）、set()
# 其他都是「真」

# 應用：if 判斷
name = ""
if name:                 # 空字串 = False、不會進這
    print("有名字")
else:
    print("沒名字")        # 印這個
```

  
### 比較運算 + 連鎖比較

```text
# 基本比較
5 > 3     # True
5 == 5    # True   ← 兩個等號才是「比較」、一個等號是「指派」
5 != 6    # True   （!= 是不等於）
5 >= 5    # True   （大於或等於）
5 <= 4    # False

# Python 特有：連鎖比較（chain comparison）—— 其他語言沒有
age = 25
if 18 <= age < 65:       # 念作「age 大於等於 18 且小於 65」
    print("成年人")
# 不用寫成 age >= 18 and age < 65
```

  
### 型別查詢 + 轉換總表

```text
# 查當前型別
type(5)         # <class 'int'>
type(3.14)      # <class 'float'>
type("hi")      # <class 'str'>
type(True)      # <class 'bool'>

# 轉換（不能轉的會 raise ValueError）
int("42")       # 42
float("3.14")   # 3.14
str(123)        # "123"
bool(1)         # True
bool(0)         # False
bool("")        # False
bool("hi")      # True

# input() 永遠回傳字串、要算數一定先轉
age_str = input("幾歲？")         # 例如使用者打 "25"
age = int(age_str)               # 25（int）
print(age + 1)                   # 26
```

  
    **☕ 用人話講**
    
4 個基本型態學熟 = Python 入門 50%。**新手最常踩**：(1) `input()` 拿到的永遠是 **str**——算數一定要 `int()` 或 `float()` 轉 (2) **浮點數**比較不能直接 `==`、用 `math.isclose()` (3) **字串 immutable**——不能 `s[0] = 'x'`、要建新字串 (4) Python 的 `True / False / None` **首字一定大寫**（小寫會錯）。**f-string** 是 2026 寫 Python 必用：`f"hi {name}, you're {age}"`。
