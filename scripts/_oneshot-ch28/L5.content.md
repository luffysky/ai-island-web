**反爬機制 = 「**網站防止你爬的招數**」**——學會破解才能爬到資料。這課列 7 大反爬技術 + 對應解法、從簡單到極難。

> ⚠️ **這課重點是「了解」**——不是教你「破解所有反爬」。反爬越強通常代表網站越不想被爬（FB / IG）、強破可能違法。**評估值不值得、再決定**。

  
### 7 大反爬技術 + 解法

**1. User-Agent 偵測（最常見、最簡單）**

**問題**：網站看你 UA、是 `python-requests/2.x` → 直接 block。

**解法**：

```text
# === 用真實瀏覽器 UA ===
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64) AppleWebKit/537.36'
}

# === 進階：用 fake-useragent 每次隨機 ===
```

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add fake-useragent
```

**這行做什麼？** 裝 `fake-useragent`、含上千個真實瀏覽器 UA、可以每次隨機切換、避免被認出。

```text
from fake_useragent import UserAgent
ua = UserAgent()
headers = {'User-Agent': ua.random}
```

  
**2. IP 限流（最普遍）**

**問題**：同一 IP 1 分鐘超過 60 次 → 封 IP 1 小時 / 24 小時。

**解法 1：sleep 控速**

```text
import time, random

for url in urls:
    res = requests.get(url)
    time.sleep(random.uniform(1, 3))        # 隨機 1-3 秒（看起來像人）
```

**解法 2：用 IP 池（proxy）**

```text
proxies = {
    'http': 'http://proxy.example.com:8080',
    'https': 'http://proxy.example.com:8080',
}
res = requests.get(url, proxies=proxies)
# 多個 proxy 輪流用、L6 詳教
```

  
**3. Referer 檢查**

**問題**：網站看你從哪個頁面點過來、`Referer` header 不對 → 拒絕。

**解法**：

```text
headers = {
    'User-Agent': '...',
    'Referer': 'https://example.com/'       # 偽裝從首頁過來
}
```

  
**4. Cookie / Session 要求**

**問題**：沒帶 cookie 直接拒絕、要先逛過首頁拿 cookie 才能進內頁。

**解法**：用 `Session` 自動處理 cookie（L2 教過）

```text
session = requests.Session()
session.get('https://example.com')          # 第一次訪問拿 cookie
session.get('https://example.com/api/data') # 自動帶 cookie
```

  
**5. JavaScript Challenge / Cloudflare（中等難度）**

**問題**：對方塞一段 JS、要你跑完才放行 —— **Cloudflare Bot Protection** 最常見。

**解法 1：用 Playwright（會跑 JS）**

```text
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto(url)                          # 自動跑完 Cloudflare 挑戰
    # ... 抓資料
```

**解法 2：用 cloudscraper（專門破 Cloudflare 的套件）**

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add cloudscraper
```

**這行做什麼？** 裝 `cloudscraper`、專門破 Cloudflare JS Challenge 的套件、比 Playwright 輕量。

```text
import cloudscraper
scraper = cloudscraper.create_scraper()
res = scraper.get('https://cloudflare-protected.com')
```

  
**6. 圖形驗證碼 / reCAPTCHA（高難度）**

**問題**：跳出「**請點選所有交通號誌**」之類的圖形驗證碼。

**3 種對策：**

```text
1. 避開
   - 換 IP / Cookie / 改頻率、避免觸發驗證碼
   - 最划算、永遠先試這個

2. 用第三方解碼服務
   - 2Captcha / Anti-Captcha：人工解、$2 / 1000 題
   - DeathByCaptcha：類似
   - 適合：規模大、有預算

3. 用 AI（OCR / GPT-4V）
   - 簡單文字驗證碼 → Tesseract OCR
   - 圖形驗證碼 → GPT-4V / Claude vision
   - 通用方案、彈性大
```

> ⚠️ **reCAPTCHA v3 幾乎無解**——Google 用 ML 評分、機器分數低就擋。要過只能用真實瀏覽器 + 高評分 IP。

  
**7. 設備指紋（Canvas / WebGL fingerprint、2025+ 高階）**

**問題**：用瀏覽器特徵（Canvas 渲染 / WebGL / 字型清單 / 螢幕解析度）建 fingerprint、Headless 瀏覽器都有特徵。

**解法**：用真實瀏覽器（Playwright）+ 反指紋套件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add playwright-stealth
```

**這行做什麼？** 裝 `playwright-stealth`、修改 Playwright 啟動的瀏覽器、隱藏「**這是自動化**」的特徵、繞過 fingerprint 偵測。

```text
from playwright.sync_api import sync_playwright
from playwright_stealth import stealth_sync

with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    stealth_sync(page)                       # 套用反指紋
    page.goto('https://protected-site.com')
```

  
### 反爬難度排行（看你要爬什麼）

| 等級 | 範例 | 難度 |
|---|---|---|
| **🟢 簡單** | 政府網站 / 學術網站 / 部落格 / Wiki | 沒反爬、用 requests 就行 |
| **🟡 中等** | 蝦皮 / Momo / 大部分電商 | IP 限流、sleep + proxy 解 |
| **🟠 困難** | Amazon / Google | 多層偵測、Playwright + stealth |
| **🔴 極難** | Instagram / TikTok / LinkedIn | 設備指紋 + ML、官方 API 才實際 |
| **🚫 幾乎不可能** | Cloudflare Turnstile + reCAPTCHA Enterprise | 投降、用官方 API |

  
### 反反爬黃金組合（爬有保護的網站、推薦配置）

**「Playwright + Stealth + Proxy + Random UA + Sleep」5 件套**：

```text
from playwright.sync_api import sync_playwright
from playwright_stealth import stealth_sync
import random, time

def scrape_protected(url):
    with sync_playwright() as p:
        browser = p.chromium.launch(
            headless=True,
            proxy={'server': 'http://proxy.io:8080'}    # 1. 用 proxy
        )
        context = browser.new_context(
            user_agent='Mozilla/5.0 ...',                # 2. 真實 UA
            viewport={'width': 1920, 'height': 1080},    # 3. 真實解析度
            locale='zh-TW',
        )
        page = context.new_page()
        stealth_sync(page)                               # 4. 反指紋
        
        page.goto(url)
        time.sleep(random.uniform(2, 5))                 # 5. 模擬人類停頓
        
        # === 隨機滾動 ===
        page.evaluate('window.scrollBy(0, 500)')
        time.sleep(random.uniform(1, 2))
        
        data = page.inner_html('body')
        browser.close()
        return data
```

  
    **☕ 用人話講**
    
反爬 7 大技術：(1) **UA 偵測** → 用真實 UA / fake-useragent (2) **IP 限流** → sleep + proxy (3) **Referer 檢查** → 加 Referer header (4) **Cookie 要求** → 用 Session (5) **Cloudflare JS Challenge** → Playwright 或 cloudscraper (6) **驗證碼** → 避開 / 第三方解 / AI (7) **設備指紋** → playwright-stealth。**心法**：**先試簡單方案**、不夠再加複雜的、不要一開始就上 Playwright + Stealth（殺雞用牛刀、慢 10 倍）。

  

  
    
      ⚠️
      法律警告
    
    強破反爬可能違法：(1) **破解技術防護**（爬蟲對抗 Cloudflare / reCAPTCHA）可能違反「**電腦犯罪條例**」、刑法 358-360 條 (2) **大規模爬導致網站癱瘓** = 妨害電腦使用罪 (3) **爬付費內容 / 登入後內容** = 妨害著作權 + 破壞防護措施。**法律灰色地帶**：抓「網站根本沒設防護」的公開資料較安全、但**最好寄信問** **the website owner**：「我要爬 X 資料、用於 Y 目的、可以嗎？」、得到 OK 才動手。
