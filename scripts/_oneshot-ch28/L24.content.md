**爬蟲接案 + 變現**——爬蟲是 indie 最賺的技能之一。**4 種變現模式 + 6 大熱門領域 + 報價心法**。

> 💡 **核心心法**：不要報「**爬蟲設定費**」、要報「**節省成本 + ROI**」。客戶不在乎你寫多少 code、在乎「**幫他賺多少 / 省多少**」。

  
### 4 種變現模式

**1. 一次性接案**

```text
- 單價：NT$ 5,000-30,000
- 工時：1-3 天完成
- 客戶：中小企業、indie 賣家
- 案件範例：
  「幫我抓蝦皮競品價格、做 Excel 報表」
  「抓政府公開資料、轉成 dashboard」
  「抓股票資料、做技術指標計算」
```

  
**2. 持續監控（月費、被動收入主軸）⭐**

```text
- 單價：NT$ 1,000-5,000 / 月 / 客戶
- 5-10 個客戶 = 月入 NT$ 10,000-50,000 被動
- 案件範例：
  「每天爬對手價格、6am 推 LINE」
  「監控你的 GA / FB Ads 數據異常」
  「每週抓求職網新缺、寄 email summary」
```

> 💡 **持續監控是最划算的模式**——做一次、收一輩子、客戶不換你就一直賺。

  
**3. SaaS 工具（門檻最高、收益上限最高）**

```text
- 月費：NT$ 100-500 / 用戶
- 100 用戶 = 月入 NT$ 10,000-50,000
- 1000 用戶 = 月入 NT$ 100,000-500,000
- 範例：比價網、求職通知、加密貨幣 alert
```

  
**4. 顧問 / 教學**

```text
- 1 對 1 顧問：NT$ 1,500-3,000 / hr
- 公司內訓：NT$ 30,000-100,000 / 場
- 線上課程：NT$ 1,000-3,000 / 學員、可規模化
```

  
### 6 大熱門領域（最賺的接案類型）

**1. 電商比價 ⭐⭐⭐**

```text
- 目標：蝦皮 / PChome / Momo / Yahoo / 蝦皮商城
- 客戶：賣家、品牌、消費者
- 單價：NT$ 5,000-50,000 + 月費
- 為什麼賺：賣家天天要看對手價、需求穩定
```

  
**2. 房地產 ⭐⭐⭐**

```text
- 目標：591 / 樂屋 / 信義 / 永慶
- 客戶：投資人、仲介、買房自住
- 案件：「捷運站 1km 內降價物件 alert」
```

  
**3. 股票 / 金融 ⭐⭐**

```text
- 目標：MoneyDJ / 雅虎股市 / 證交所
- 客戶：量化交易、個人投資人
- 案件：「漲跌幅 + 新聞情緒分析」
```

  
**4. 求職（104 / yourator）⭐⭐**

```text
- 客戶：求職者、HR
- 案件：「我關心的職位有新缺通知」「公司缺人多 + 薪資範圍分析」
```

  
**5. 競品 / 行銷監控 ⭐⭐⭐**

```text
- 目標：對手網站 / 社群
- 客戶：行銷、品牌、新創
- 案件：「對手部落格新文章自動 summary」「IG / FB 廣告監控」
```

  
**6. 政府公開資料 ⭐**

```text
- 目標：data.gov.tw、各部會
- 客戶：NGO、學術、新聞媒體
- 案件：「環保署空氣品質 + 政府預算視覺化」
- 優勢：合法、有社會意義、容易做新聞
```

  
### 報價心法

**1. 時間估算（粗略）：**

| 複雜度 | 工時 | 單價 |
|---|---|---|
| **簡單**（單頁靜態） | 4-8 hr | NT$ 5,000-15,000 |
| **中等**（多頁 + 反爬） | 16-40 hr | NT$ 15,000-50,000 |
| **複雜**（SPA + Captcha + DB） | 40-100 hr | NT$ 50,000-200,000 |
| **時薪參考** | — | NT$ 800-3,000/hr |

  
**2. Value-based（推薦、收費更高）**

**不報時薪、報「**客戶價值**」**：

```text
❌ 「我寫這個要 20 小時、時薪 1500、所以 30000」
✅ 「省你 1 個人月人力成本（NT$ 40k）+ 每天自動跑、
   我收 NT$ 50k + 月費 NT$ 3k（含維護 / 改版 / 失敗通知）」
```

  
**3. 月費維護**（最重要、被動收入）

```text
- 爬蟲會壞（網站改版）、要持續維護
- 月費 NT$ 3,000-15,000、客戶覺得便宜（vs 雇人）
- 你 1 個客戶賺 NT$ 36k-180k / 年
- 10 個客戶 = 月入 NT$ 30k-150k 被動
```

  
### 完整接案案例：蝦皮比價

**客戶**：3C 賣家、痛點「**對手價格天天變、要手動查超累**」

  
**架構：**

```text
[Cron 6am 每天跑]
        ↓
[爬蝦皮 / PChome / Momo 各 100 個對手商品]
        ↓
[PostgreSQL 存今日價格 + 跟昨日比]
        ↓
[Telegram Bot 推報告：哪些降價、降多少、連結]
```

  
**報價：**

```text
Setup（一次性）：NT$ 30,000
- 含：爬蟲開發 + 部署 + 教學
- 工時：3 天

月費維護：NT$ 5,000
- 含：網站改版修 bug + 失敗通知 + Sentry 監控
- 加新對手商品 NT$ 500/個

→ 年收入：30k + 5k * 12 = NT$ 90k / 客戶
→ 5 個客戶 = NT$ 450k / 年（兼差）
→ 10 個客戶 = NT$ 900k / 年（接近全職）
```

  
**客戶感受**：

```text
「以前我每天花 1 小時手動查價、月成本 NT$ 50k 人力」
「現在 NT$ 5k/月、自動跑 + 通知」
「省 NT$ 45k/月、CP 值爆表」

→ 雙贏、客戶開心付月費
```

  
### 接案心法 5 條

```text
1. ✅ 不報「爬蟲設定費」、報「節省成本 + ROI」
2. ✅ 月費維護 = 被動收入主軸（被動收入大於一次性收入）
3. ✅ 跑 10 個客戶 = 月入 NT$ 30k+ 被動
4. ✅ 用 ScraperAPI / Sentry / Streamlit 包成 deliverable、客戶覺得專業
5. ✅ 合約寫清楚「客戶承擔法律責任」、L23 教過
```

  
### 4 大反 pattern（不要做）

```text
1. ❌ 接「**抓個資**」案件（違法、賠不起）
2. ❌ 接「**衝擊對方網站**」案件（妨害電腦使用罪）
3. ❌ 不簽合約（口頭答應、客戶反悔你沒保護）
4. ❌ 報「**時薪**」不報「**價值**」（收入封頂）
```

```text

5. 朋友推薦：「**有需要爬蟲的朋友介紹給我**」
```
