```text

```

**這行做什麼？逐個解釋：**
- `playwright codegen` —— Playwright 的「**錄製模式**」

**怎麼用：**
1. 跑完跳出瀏覽器視窗 + Code 視窗

> 💡 **這是 Playwright 最強的功能**——不用手寫 selector、看你怎麼點、自動生 code。複雜表單自動化用這個快 10 倍。

  
### Trace Viewer（debug 神器、看每個 step 的截圖 / DOM）

```text
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch()
    context = browser.new_context()
    
    # === 開錄 trace ===
    context.tracing.start(screenshots=True, snapshots=True, sources=True)
    
    page = context.new_page()
    page.goto('https://example.com')
    page.click('button.submit')
    
    # === 停錄 + 存 trace.zip ===
    context.tracing.stop(path='trace.zip')
    browser.close()
```

### Async 並發優化（同時跑 50 個爬蟲、單機極限）

```text
import asyncio
from playwright.async_api import async_playwright

async def scrape_page(browser, url):
    """每個 URL 跑一個 page、共用同一個 browser"""
    page = await browser.new_page()
    await page.goto(url)
    data = await page.locator('h1').text_content()
    await page.close()
    return data

async def main():
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        
        urls = [f'https://example.com/page/{i}' for i in range(1, 51)]
        
        # === 同時跑 50 個（共用同一個 browser、超省記憶體）===
        results = await asyncio.gather(*[scrape_page(browser, u) for u in urls])
        
        await browser.close()
        return results

asyncio.run(main())
```

  
**用 `Semaphore` 限制並發數（避免被網站封）：**

```text
async def main():
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        
        # === 同時最多 10 個（不要 50 個一次打、會被封）===
        semaphore = asyncio.Semaphore(10)
        
        async def limited_scrape(url):
            async with semaphore:
                return await scrape_page(browser, url)
        
        urls = [f'https://example.com/page/{i}' for i in range(1, 101)]
        results = await asyncio.gather(*[limited_scrape(u) for u in urls])
        
        await browser.close()
        return results
```

  
### 多 Context 隔離（每個 user 一個 context）

**Browser Context** = 「**獨立的瀏覽器 session**」——cookie / localStorage / cache 分開。一個 browser 可以開多個 context。

```text
async with async_playwright() as p:
    browser = await p.chromium.launch()
    
    # === Context 1：user A 登入 ===
    context_a = await browser.new_context()
    page_a = await context_a.new_page()
    await page_a.goto('https://example.com/login')
    # ... 登入 user A ...
    
    # === Context 2：user B 登入 ===
    context_b = await browser.new_context()
    page_b = await context_b.new_page()
    await page_b.goto('https://example.com/login')
    # ... 登入 user B ...
    
    # 兩個 context cookie 完全隔離、互不影響
```

**用途**：同時測試多個 user / 多語系 / 多 device viewport。

  
### 攔截 API（L3 教過、這裡進階）

**進階 1：修改 response（mock 用）：**

```text
async def handle_route(route):
    """攔截特定 API、回傳假資料"""
    if '/api/products' in route.request.url:
        await route.fulfill(
            status=200,
            content_type='application/json',
            body='{"products": [{"id": 1, "name": "Mock"}]}'
        )
    else:
        await route.continue_()                # 其他 API 正常打

await page.route('**/*', handle_route)
```

**用途**：開發 / 測試時 mock 後端 API、不依賴真實 server。

  
**進階 2：抓所有 GraphQL request：**

```text
graphql_data = []

async def handle_response(response):
    if 'graphql' in response.url:
        try:
            graphql_data.append(await response.json())
        except:
            pass

page.on('response', handle_response)
await page.goto('https://shop.example.com')
# 自動收集所有 GraphQL 回應
```

  
### 反偵測強化（playwright-stealth + 自訂 UA + viewport）

```text
from playwright.async_api import async_playwright
from playwright_stealth import stealth_async

async def main():
    async with async_playwright() as p:
        browser = await p.chromium.launch(
            headless=True,
            args=[
                '--disable-blink-features=AutomationControlled',  # 隱藏 webdriver flag
            ]
        )
        context = await browser.new_context(
            user_agent='Mozilla/5.0 ...',
            viewport={'width': 1920, 'height': 1080},
            locale='zh-TW',
            timezone_id='Asia/Taipei',
        )
        page = await context.new_page()
        await stealth_async(page)              # 反指紋
        
        await page.goto('https://example.com')
```
