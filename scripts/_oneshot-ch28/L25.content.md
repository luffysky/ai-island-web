### 5 大心法（資深 vs 新手差別）

```text
1. ✅ 爬蟲是工具、不是終點
   學爬蟲是為了解決問題、不是收集 GitHub stars

2. ✅ 結合領域知識（金融 / 電商 / 房地產）才值錢
   只會技術 = 工具人、懂業務 + 技術 = 顧問

3. ✅ 法律風險評估 > 技術
   爬錯網站、賺 1 萬賠 100 萬、CP 值不對

4. ✅ 永遠先試官方 API
   API 慢、貴、限制多、但 100% 合法 + 穩定

5. ✅ 持續輸出（blog / GitHub）
   每月 1 篇技術文 + 1 個 GitHub 專案 = 12 個月後客戶自己上門
```
