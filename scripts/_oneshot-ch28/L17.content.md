**爬 API 直接拿 JSON = 「**不用解 HTML、最穩最快**」**——L3 / L7 提過 API 攔截、這課完整教「**怎麼找 API + 怎麼用**」。

> 💡 **業界共識**：能爬 API 就不要爬 HTML——快 10 倍、結構乾淨、不會因 UI 改版壞掉。**99% 現代網站都有內部 API**。

  
### 怎麼找網站的內部 API（F12 標準流程）

```text
1. F12 開 Chrome DevTools
2. 切到 Network 分頁
3. 篩選 Fetch/XHR（過濾掉圖片 / CSS / JS）
4. 重新整理頁面（F5）
5. 看哪些 request 回傳 JSON
6. 找含你要的資料的 endpoint
7. 右鍵 → Copy → Copy as cURL
8. 翻成 Python（github.com/curlconverter/curlconverter 一鍵轉）
```

**怎麼快速判斷是 API：**
- URL 含 `/api/` / `/v1/` / `/v2/` / `.json`
- Response Headers 的 Content-Type 是 `application/json`
- Response 是 JSON 格式（不是 HTML）

  
### 範例：爬蝦皮商品 API

```text
import requests

# === F12 看到蝦皮搜尋 API ===
headers = {
    'User-Agent': 'Mozilla/5.0 (Macintosh) Chrome/131.0',
    'Referer': 'https://shopee.tw/',         # ⚠️ 蝦皮會檢查 Referer、必加
}

res = requests.get(
    'https://shopee.tw/api/v4/search/search_items',
    params={
        'keyword': 'iPhone',
        'limit': 60,
        'newest': 0,                          # offset、0 = 第 1 頁
    },
    headers=headers
)

data = res.json()
for item in data['items']:
    print(
        item['item_basic']['name'],
        item['item_basic']['price'] / 100_000  # 蝦皮 price 是「乘 10 萬」的整數
    )
```

  
### 爬 API 4 大優勢（vs 爬 HTML）

```text
✅ 1. 資料結構化（dict / list、不用解 HTML）
✅ 2. 不會因 UI 改版壞（前端改一萬次、API 通常不動）
✅ 3. 速度快 10 倍（不用渲染、不用解析）
✅ 4. 分頁明確（通常 ?page=X 或 ?offset=X）

❌ 缺點：
- 對方可能要 API key / authentication
- 對方可能加 rate limit（每秒 X 次）
- 反爬可能特別針對 API（如 token 校驗）
```

  
### GraphQL API（2026 新興、要會）

**GraphQL** vs 傳統 REST 差別：
- REST: 多個 endpoint（`/api/users` / `/api/products`）
- **GraphQL: 一個 endpoint、用 query 描述要什麼欄位**

```text
import requests

res = requests.post(
    'https://api.example.com/graphql',
    json={
        'query': '''
            query GetProducts($limit: Int, $offset: Int) {
                products(limit: $limit, offset: $offset) {
                    id
                    name
                    price
                    reviews(limit: 5) {
                        rating
                        text
                    }
                }
            }
        ''',
        'variables': {'limit': 20, 'offset': 0}
    },
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)

data = res.json()
for product in data['data']['products']:
    print(product['name'], product['price'])
```

> 💡 **GitHub / Shopify / Linear / Notion API 都是 GraphQL**——學一次受用所有現代 SaaS。

  
### 分頁處理（最常見的 API 模式）

**3 種主流分頁方式：**

```text
# === 方式 1：page 分頁 ===
for page in range(1, 11):
    res = requests.get(api, params={'page': page})
    items = res.json()['items']
    if not items:
        break

# === 方式 2：offset 分頁 ===
offset = 0
while True:
    res = requests.get(api, params={'limit': 100, 'offset': offset})
    items = res.json()['items']
    if not items:
        break
    offset += 100

# === 方式 3：cursor 分頁（最 modern）===
cursor = None
while True:
    params = {'limit': 100}
    if cursor:
        params['cursor'] = cursor
    res = requests.get(api, params=params).json()
    items = res['items']
    if not items:
        break
    cursor = res.get('next_cursor')
    if not cursor:
        break
```

  
### 處理 Rate Limit（429 Too Many Requests）

```text
import time

def fetch_with_rate_limit(url, params, max_retries=3):
    for attempt in range(max_retries):
        res = requests.get(url, params=params)
        
        if res.status_code == 429:            # Rate limited
            # 看 Retry-After header（標準）
            wait = int(res.headers.get('Retry-After', 60))
            print(f'被限流、等 {wait} 秒')
            time.sleep(wait)
            continue
        
        res.raise_for_status()
        return res.json()
    
    raise Exception('Rate limit 太多次')
```

  
### 處理需要 API key 的服務

**很多公開 API 要先註冊拿 key：**

```text
# === 1. 從環境變數讀 key（不要寫死 code、L8 教過）===
import os
API_KEY = os.environ['EXAMPLE_API_KEY']

# === 2. 帶 key 發 request（依 API 要求位置）===
# 方式 A：放 header（最常見）
headers = {'Authorization': f'Bearer {API_KEY}'}
# 或：headers = {'X-API-Key': API_KEY}

# 方式 B：放 query parameter
params = {'api_key': API_KEY}

res = requests.get(url, headers=headers)
```

### 包裝成自己的 client（業界做法）

```text
class ShopeeClient:
    BASE = 'https://shopee.tw/api/v4'
    
    def __init__(self):
        self.session = requests.Session()
        self.session.headers.update({
            'User-Agent': 'Mozilla/5.0 ...',
            'Referer': 'https://shopee.tw/',
        })
    
    def search(self, keyword, page=0, limit=60):
        return self.session.get(
            f'{self.BASE}/search/search_items',
            params={'keyword': keyword, 'newest': page * limit, 'limit': limit}
        ).json()
    
    def product(self, item_id, shop_id):
        return self.session.get(
            f'{self.BASE}/item/get',
            params={'itemid': item_id, 'shopid': shop_id}
        ).json()

# === 用 ===
client = ShopeeClient()
results = client.search('iPhone', page=0)
```
