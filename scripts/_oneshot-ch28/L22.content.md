### PDF（用 pdfplumber、最強）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add pdfplumber
```

**這行做什麼？** 裝 `pdfplumber`、最強的 Python PDF 處理套件、能精準抽文字 + 表格 + 圖片。

  
```text
import pdfplumber

with pdfplumber.open('file.pdf') as pdf:
    for page in pdf.pages:
        # === 抽文字 ===
        text = page.extract_text()
        
        # === 抽表格（超強、自動辨識 row/col）===
        tables = page.extract_tables()
        for table in tables:
            for row in table:
                print(row)                    # ['Name', 'Price', 'Stock']
        
        # === 抽圖片 ===
        for img in page.images:
            print(img['x0'], img['y0'], img['width'], img['height'])
        
        # === 抽超連結 ===
        for link in page.hyperlinks:
            print(link['uri'])
```

  
**範例：抽政府公告 PDF 的表格資料**

```text
import pdfplumber
import pandas as pd

with pdfplumber.open('government_report.pdf') as pdf:
    all_data = []
    for page in pdf.pages:
        tables = page.extract_tables()
        for table in tables:
            df = pd.DataFrame(table[1:], columns=table[0])
            all_data.append(df)

result = pd.concat(all_data, ignore_index=True)
result.to_csv('extracted.csv', index=False, encoding='utf-8-sig')
```

> 💡 **vs PyPDF2 / PyMuPDF**：pdfplumber **表格識別** 最準、PyMuPDF 速度快但文字位置抓不準、新手用 pdfplumber。

  
### Excel（pandas 直接讀）

```text
import pandas as pd

# === 讀單一 sheet ===

    print(f'{name}: {len(df)} rows')

# === 跳過前 N 列（很多政府 Excel 有 header note）===
df = pd.read_excel('file.xlsx', skiprows=3)

# === 指定欄位型別（防 Pandas 亂猜）===
df = pd.read_excel('file.xlsx', dtype={'id': str, 'price': float})
```

### OCR（圖片轉文字、發票 / 收據用）

**3 種方式、品質不同**：

  
**方式 1：Tesseract（傳統 OCR、免費、品質普通）**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 Tesseract Python binding）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add pytesseract pillow
```

**這 2 個套件做什麼？**
- `pytesseract` —— Tesseract OCR 引擎的 Python wrapper
- `pillow` —— Python 圖片處理（讀 / 寫圖檔）

> ⚠️ **還要另外安裝 Tesseract binary**（系統層）：
> - Mac：`brew install tesseract tesseract-lang`
> - Ubuntu：`apt install tesseract-ocr tesseract-ocr-chi-tra`
> - Windows：去 github.com/UB-Mannheim/tesseract 下載

  
```text
import pytesseract
from PIL import Image

# === 英文 ===
text = pytesseract.image_to_string(Image.open('invoice.png'))

# === 繁體中文 + 英文 ===
text = pytesseract.image_to_string(
    Image.open('invoice.png'),
    lang='chi_tra+eng'
)

# === 進階：含位置資訊（給座標）===
data = pytesseract.image_to_data(Image.open('invoice.png'), output_type='dict')
```

> ⚠️ **Tesseract 品質**：印刷字 OK、手寫 / 歪斜 / 模糊就爛。

  
**方式 2：Claude Vision / GPT-4V（最強、付費）**

**真正強的 OCR**：用 LLM 視覺、不只認字、還理解結構（發票上「金額在哪、品項在哪」）。

```text
import anthropic
import base64

client = anthropic.Anthropic()

with open('invoice.png', 'rb') as f:
    img = base64.b64encode(f.read()).decode()

response = client.messages.create(
    model='claude-opus-4-7',
    max_tokens=1024,
    messages=[{
        'role': 'user',
        'content': [
            {
                'type': 'image',
                'source': {
                    'type': 'base64',
                    'media_type': 'image/png',
                    'data': img
                }
            },
            {
                'type': 'text',
                'text': '抽出發票上：商家、日期、總金額、品項清單（JSON 格式回）'
            }
        ]
    }]
)

print(response.content[0].text)
# {
#   "merchant": "全家便利商店",
#   "date": "2026-05-27",
#   "total": 165,
#   "items": [
#     {"name": "茶葉蛋", "qty": 1, "price": 12},
#     ...
#   ]
# }
```

> 💡 **這超強**——不只 OCR、還結構化輸出 JSON、發票 / 收據 / 名片自動化必用。**成本約 $0.01-0.05/張**。

  
**方式 3：Cloud Vision API（Google / Azure、適合大量）**

```text
# Google Cloud Vision API
from google.cloud import vision

client = vision.ImageAnnotatorClient()
with open('invoice.png', 'rb') as f:
    image = vision.Image(content=f.read())

response = client.text_detection(image=image)
text = response.text_annotations[0].description
# 比 Tesseract 準、比 Claude 便宜（量大）
```

  
### 音訊（YouTube 字幕 / Podcast）

**Whisper（OpenAI 出、最強開源語音轉文字）：**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add openai-whisper
```

**這行做什麼？** 裝 OpenAI Whisper、最強開源語音識別、繁中日韓 99 種語言都支援。

  
```text
import whisper

model = whisper.load_model('base')             # 模型大小：tiny/base/small/medium/large
result = model.transcribe('audio.mp3')
print(result['text'])

# === 含時間戳（做字幕用）===
for segment in result['segments']:
    print(f"[{segment['start']:.1f}s] {segment['text']}")
```

> 💡 **Whisper 速度 vs 準度**：tiny 快但爛、medium 平衡、large 最準但慢。**繁中用 large 才準**。

### 真實應用案例

```text
1. 政府公開資料（PDF）
   → pdfplumber 抽表格 → Pandas 分析 → 發報告

2. 上市公司財報抓取
   → 抓 PDF / Excel 財報 → 計算財務指標 → 量化選股

3. 收據自動化（個人 / 公司報帳）
   → 手機拍照 → Claude Vision 抽資料 → 進 Excel

4. YouTube 字幕分析
   → yt-dlp 抓字幕 → NLP 找關鍵字 → 內容研究

5. Podcast 內容索引
   → Whisper 轉文字 → 建 search index → 找特定話題
```

  
    **☕ 用人話講**
    
非 HTML 爬蟲 5 件套：(1) **pdfplumber** 抽 PDF 文字 + 表格 (2) **pandas.read_excel** 讀 Excel (3) **pytesseract** 簡單 OCR（免費） (4) **Claude Vision** 強 OCR + 結構化（付費但超強） (5) **whisper** 音訊轉文字。**接案差異化**：99% 爬蟲工程師只會 HTML、會 PDF / Vision / 音訊 = 接案 CP 值翻倍。**重點**：先學 pdfplumber（最常用）+ Claude Vision（最有 wow factor）。
