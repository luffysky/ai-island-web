### 爬蟲 5 大應用


- 💰 **價格比較**：蝦皮 vs Momo vs PChome、找最便宜
- 📰 **新聞聚合**：抓各大網站新聞做摘要
- 📊 **市場研究**：競品價格 / 評論 / 銷量
- 🤖 **AI 訓練資料**：抓網路文章 / 圖片訓練模型
- 📈 **量化策略**：抓股價 / 新聞做交易訊號


### 合法 vs 非法


- ✅ **合法**：抓公開資料、遵守 robots.txt、合理頻率、不破解付費牆
- ⚠️ **灰色**：抓社群網站（FB / IG）— ToS 禁止但實際大家在做
- ❌ **違法**：破解登入、繞過反爬被告、商用付費資料、轉售別人資料


### 台灣相關案例


2020 台灣有人爬蝦皮商品被告、最後判決爬「公開資料」不構成電腦犯罪。**但**—大量爬導致對方伺服器當機可能涉刑法。


### 道德守則


- 看 robots.txt（網站告訴你哪能爬）
- 每次請求 sleep 1-2 秒、不要打爆對方
- User-Agent 寫真的（不要假裝瀏覽器騙人）
- 有 API 用 API、爬蟲是最後手段
- 抓到的資料不要商用 / 轉售（除非有授權）


### 爬蟲 3 大難度


- 🟢 **Easy**：靜態 HTML、無反爬（Wiki / 政府網站）
- 🟡 **Medium**：JS 渲染、Cookie / 簡單反爬（蝦皮 / Momo）
- 🔴 **Hard**：Cloudflare / 圖形驗證 / IP 限制（FB / Instagram）
