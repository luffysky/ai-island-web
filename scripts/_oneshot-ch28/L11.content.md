import time, random

def human_sleep(min_s=1, max_s=3):
    """隨機等 1-3 秒、模擬人類速度"""
    time.sleep(random.uniform(min_s, max_s))

for url in urls:

### IP 池輪詢（L6 詳教、這裡速查）

```text
import random
import requests

PROXIES = [
    'http://user:pass@proxy1.com:8080',
    'http://user:pass@proxy2.com:8080',
    'http://user:pass@proxy3.com:8080',
]

def get_with_proxy(url):
    proxy = random.choice(PROXIES)
    return requests.get(
        url,
        proxies={'http': proxy, 'https': proxy},
        timeout=10
    )
```

### curl_cffi（神器、繞 Cloudflare TLS fingerprint）

**curl_cffi** = 用 C 寫的 curl Python binding、**可以「偽裝成真實瀏覽器的 TLS fingerprint」**——這是 2024+ Cloudflare 偵測的關鍵。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 curl_cffi）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add curl_cffi
```

**這行做什麼？** 裝 `curl_cffi`、繞 Cloudflare TLS fingerprint 偵測的神器、是 2024-2026 最強的反反爬工具之一。

```text
from curl_cffi import requests as cf_requests

# === impersonate 偽裝成 Chrome（含 TLS fingerprint）===
res = cf_requests.get(
    'https://cloudflare-protected.com',
    impersonate='chrome'                       # 也可 'safari' / 'firefox'
)
print(res.text)
# 連 Cloudflare Bot Protection 都騙過、超猛
```

> 💡 **curl_cffi 比 cloudscraper 更強**——cloudscraper 只破 JS challenge、curl_cffi 連 TLS 層都偽裝、Cloudflare 完全認不出。

    sitekey='6Lc-...',                         # 從目標網站 HTML 抓

### 反爬決策樹（遇到不同情況用什麼武器）

```text
網站擋你 / 回 403 怎辦？

第 1 步：F12 看 Network、確認你被擋
└── 沒擋、是你 selector 寫錯 → 改 selector

第 2 步：加完整 Headers + 真實 UA
├── 過 → 結束
└── 還是 403 → 第 3 步

第 3 步：用 Session + Cookie（先逛首頁再進內頁）
├── 過 → 結束
└── 還是 403 → 第 4 步

第 4 步：sleep + jitter（不要太快）
├── 過 → 結束
└── 還是被擋 → 第 5 步

第 5 步：用 curl_cffi（impersonate Chrome、繞 TLS fingerprint）
├── 過 → 結束（中等難度網站 80% 在這裡過）
└── 還是 403 → 第 6 步

第 6 步：上 Playwright + stealth + proxy
├── 過 → 結束（高難度網站、如 Cloudflare）
└── 還是被擋 → 第 7 步

第 7 步：遇到驗證碼 / reCAPTCHA → 用 2Captcha 解
├── 過 → 結束（很燒錢、量大才划算）
└── 還是過不去 → 投降、用官方 API

第 8 步：投降、用官方 API
- 大部分網站有官方 API（YouTube / Twitter / Reddit）
- 雖然要審批 / 付費、但合法 + 穩定
```

  
### 法律（L1 + L5 已強調、再次重申）

```text
✅ 安全做：
  - 抓「公開」資料
  - 尊重 robots.txt
  - 加 delay 不打爆對方
  - 用真實 UA 寫上聯絡 email

❌ 風險高：
  - 抓「個資」（個資法、刑事責任）
  - 「衝擊」對方伺服器（妨害電腦使用罪、刑法 360 條）
  - 「破解」付費內容 / 登入後內容
  - 商用爬到的內容（侵權）
```
