### 常見反爬技術


#### 1. User-Agent 偵測


看你的 UA、是 Python 預設 → 直接 block。

```python
# 解法：用真實瀏覽器 UA
headers = {
    'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64) AppleWebKit/537.36'
}

# 進階：用 fake-useragent 套件、每次隨機
from fake_useragent import UserAgent
ua = UserAgent()
headers = {'User-Agent': ua.random}
```


#### 2. IP 限流（最常見）


同一 IP 1 分鐘超過 60 次 → 封 IP 1 小時。

```python
# 解法 1：sleep 控速
import time, random
for url in urls:
    res = requests.get(url)
    time.sleep(random.uniform(1, 3))  # 隨機 1-3 秒

# 解法 2：用 IP 池
proxies = {
    'http': 'http://proxy.example.com:8080',
    'https': 'http://proxy.example.com:8080',
}
res = requests.get(url, proxies=proxies)
```


#### 3. Referer 檢查


看你從哪個頁面點過來、檢查 referer header。

```python
headers = {
    'User-Agent': '...',
    'Referer': 'https://example.com/'  # 偽裝
}
```


#### 4. Cookie / Session 要求


沒帶 cookie 直接拒。

```python
# 用 Session 自動處理
session = requests.Session()
session.get('https://example.com')  # 拿 cookie
session.get('https://example.com/api/data')  # 帶上去
```


#### 5. JavaScript Challenge / Cloudflare


對方塞一段 JS、必須跑完才放行—Cloudflare Bot Protection。

```python
# 解法 1：用 Playwright（自動跑 JS）
from playwright.sync_api import sync_playwright
with sync_playwright() as p:
    browser = p.chromium.launch()
    page = browser.new_page()
    page.goto(url)
    # 自動跑完 Cloudflare 挑戰

# 解法 2：用 undetected-chromedriver / cloudscraper
import cloudscraper
scraper = cloudscraper.create_scraper()
res = scraper.get('https://cloudflare-protected.com')
```


#### 6. 圖形驗證碼 / reCAPTCHA


最難破解。3 種對策：


- 避開（換 IP / Cookie 不觸發）
- 用第三方解碼服務（2Captcha / Anti-Captcha、$2 / 1000 題）
- 用 AI（OCR / GPT-4V）— 簡單驗證碼可破


#### 7. 設備指紋（Canvas / WebGL fingerprint）


2025+ 高階反爬—用瀏覽器特徵建立 fingerprint。解法：用真實瀏覽器（Playwright）+ 反指紋套件（playwright-stealth）。


### 反爬難度排行


- 🟢 簡單：政府網站 / 學術網站 / 部落格
- 🟡 中等：蝦皮 / Momo / 大部分電商（IP 限流）
- 🟠 困難：Amazon / Google（多層偵測）
- 🔴 極難：Instagram / TikTok / LinkedIn（設備指紋 + ML）
- 🚫 幾乎不可能：Cloudflare Turnstile + reCAPTCHA Enterprise


### 反反爬黃金組合

```python
# Playwright + Stealth + Proxy + Random UA + Sleep
from playwright.sync_api import sync_playwright
from playwright_stealth import stealth_sync
import random, time

def scrape_protected(url):
    with sync_playwright() as p:
        browser = p.chromium.launch(
            headless=True,
            proxy={'server': 'http://proxy.io:8080'}  # 用 proxy
        )
        context = browser.new_context(
            user_agent='Mozilla/5.0 ...',
            viewport={'width': 1920, 'height': 1080},
            locale='zh-TW',
        )
        page = context.new_page()
        stealth_sync(page)  # 反指紋

        page.goto(url)
        time.sleep(random.uniform(2, 5))  # 模擬人類

        # 隨機滾動
        page.evaluate('window.scrollBy(0, 500)')
        time.sleep(random.uniform(1, 2))

        data = page.inner_html('body')
        browser.close()
        return data
```
