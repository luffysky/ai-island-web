### 完整 Spider（含 custom_settings 局部設定）

```text
import scrapy

class ProductsSpider(scrapy.Spider):
    name = 'products'
    start_urls = ['https://example.com/products?page=1']
    
    # === 局部 settings（只影響這個 spider、不動全域）===
    custom_settings = {
        'DOWNLOAD_DELAY': 1,
        'CONCURRENT_REQUESTS': 8,
        'AUTOTHROTTLE_ENABLED': True,
    }
    
    def parse(self, response):
        for card in response.css('div.product-card'):
            yield {
                'name': card.css('.name::text').get().strip(),
                'price': card.css('.price::text').get(),
                # === response.urljoin 自動處理相對 / 絕對路徑 ===
                'link': response.urljoin(card.css('a::attr(href)').get()),
            }
        
        # === 自動跟「下一頁」===
        next_page = response.css('a.next::attr(href)').get()
        if next_page:
            yield response.follow(next_page, self.parse)
```

  
**跑：**

```text
scrapy crawl products -O products.json
# -O 大寫 = 覆蓋；-o 小寫 = 追加
```

  
### CrawlSpider（自動跟所有匹配 pattern 的連結、超強）

**`CrawlSpider`** = 比 `Spider` 高階、用 **Rule** 定義「**遇到符合 pattern 的連結就跟、跟到的頁面用 X callback 處理**」。

```text
from scrapy.spiders import CrawlSpider, Rule
from scrapy.linkextractors import LinkExtractor

class SiteSpider(CrawlSpider):
    name = 'site'
    start_urls = ['https://example.com']
    
    rules = (
        # === 規則 1：找 /products/123 之類的連結、跑 parse_product ===
        Rule(
            LinkExtractor(allow=r'/products/\d+'),
            callback='parse_product'
        ),
        # === 規則 2：找 /category/ 連結、繼續往下爬（follow=True）===
        Rule(
            LinkExtractor(allow=r'/category/'),
            follow=True
        ),
    )
    
    def parse_product(self, response):
        yield {
            'name': response.css('h1::text').get(),
            'price': response.css('.price::text').get(),
        }
# 自動爬遍整個網站、不用自己寫 follow 邏輯
```

  
### Pipeline 進階（多個 pipeline 串接）

```text
# pipelines.py

# === 1. 清洗 pipeline ===
class CleanPipeline:
    def process_item(self, item, spider):
        # 價格 '$1,299' → 1299
        item['price'] = float(item['price'].replace('$', '').replace(',', ''))
        # 去頭尾空白
        item['name'] = item['name'].strip()
        return item

# === 2. 去重 pipeline ===
class DedupPipeline:
    def __init__(self):
        self.seen = set()
    
    def process_item(self, item, spider):
        if item['link'] in self.seen:
            raise scrapy.exceptions.DropItem('重複')
        self.seen.add(item['link'])
        return item

# === 3. 寫 MongoDB pipeline ===
from pymongo import MongoClient
class MongoPipeline:
    def open_spider(self, spider):
        self.client = MongoClient('mongodb://localhost:27017')
    
    def process_item(self, item, spider):
        self.client.db.products.insert_one(dict(item))
        return item
```

  
**settings.py 啟用（數字 = 優先序、越小越前）：**

```text
ITEM_PIPELINES = {
    'myspider.pipelines.CleanPipeline': 100,      # 第 1 個跑
    'myspider.pipelines.DedupPipeline': 200,      # 第 2 個跑
    'myspider.pipelines.MongoPipeline': 300,      # 第 3 個跑（最後寫 DB）
}
```

  
### Middleware（攔截每個請求 / 回應、加 header / proxy）

**Downloader Middleware** = 「**請求被送出前 + 回應收到後**」可以攔截處理。

```text
# middlewares.py
import random

class RotateUserAgentMiddleware:
    USER_AGENTS = [
        'Mozilla/5.0 (Windows NT 10.0) Chrome/120.0.0.0',
        'Mozilla/5.0 (Macintosh) Chrome/120.0.0.0',
        'Mozilla/5.0 (X11; Linux) Firefox/120.0',
    ]
    
    def process_request(self, request, spider):
        request.headers['User-Agent'] = random.choice(self.USER_AGENTS)
        return None

class ProxyMiddleware:
    def process_request(self, request, spider):
        request.meta['proxy'] = 'http://proxy.io:8080'
        return None
```

**settings.py 啟用：**

```text
DOWNLOADER_MIDDLEWARES = {
    'myspider.middlewares.RotateUserAgentMiddleware': 500,
    'myspider.middlewares.ProxyMiddleware': 750,
}
```

  
### Item Loader（資料清洗模板化）

**Item Loader** 把「**清洗邏輯**」跟 Spider 解耦、複雜專案必用。

```text
# items.py
import scrapy
from itemloaders.processors import TakeFirst, MapCompose

def strip_currency(value):
    return float(value.replace('$', '').replace(',', ''))

class ProductItem(scrapy.Item):
    name = scrapy.Field(
        input_processor=MapCompose(str.strip),
        output_processor=TakeFirst()
    )
    price = scrapy.Field(
        input_processor=MapCompose(strip_currency),
        output_processor=TakeFirst()
    )

# spider 內
from itemloaders import ItemLoader

def parse(self, response):
    for card in response.css('div.product-card'):
        loader = ItemLoader(item=ProductItem(), selector=card)
        loader.add_css('name', '.name::text')
        loader.add_css('price', '.price::text')
        yield loader.load_item()
```

### Scrapy 心法 5 條

```text
1. ✅ 用 custom_settings 局部調整（不要全域改）
2. ✅ Pipeline 多層串（清洗 → 去重 → 寫 DB）
3. ✅ Middleware 處理 UA / proxy / cookie 輪換
4. ✅ CrawlSpider + LinkExtractor 自動爬全站
5. ✅ Item Loader 解耦 spider 跟清洗邏輯
```

  
    **☕ 用人話講**
    
Scrapy 進階 5 大武器：(1) **CrawlSpider** 自動跟連結 (2) **Pipeline** 多層處理 (3) **Middleware** 攔截請求 (4) **Item Loader** 模板化清洗 (5) **Scrapy Cloud** 雲端部署。**怎麼學**：先把 L4 基本打通、找一個真實電商練手（如 books.toscrape.com）、依序加 Pipeline → Middleware → CrawlSpider、變成完整 production 級爬蟲。**心法**：別過度工程、簡單需求用 requests + BS4 就好、Scrapy 是給「**大型專案 / 持續維護**」用的。
