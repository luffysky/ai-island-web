**爬蟲（web scraping）= 「**用程式自動抓網頁資料**」**。沒有 API 時的 plan B、業務 / 競品分析 / AI 訓練資料 / 接案需求超多。

> ⚠️ **這課重點不是技術、是法律 + 道德**——爬錯有民事 / 刑事責任。L2 之後才教程式碼。

  
### 爬蟲 5 大應用場景（為什麼有人付錢給你爬）

```text
💰 價格比較：蝦皮 vs Momo vs PChome、找最便宜
📰 新聞聚合：抓各大網站新聞做摘要
📊 市場研究：競品價格 / 評論 / 銷量
🤖 AI 訓練資料：抓網路文章 / 圖片訓練模型（ChatGPT 也是這樣訓的）
📈 量化策略：抓股價 / 新聞做交易訊號
```

  
### ⚠️ 合法 vs 灰色 vs 違法（**這段一定要看完**）

| 類型 | 範例 | 結論 |
|---|---|---|
| **✅ 合法** | 抓公開資料、遵守 robots.txt、合理頻率、不破解付費牆 | 安全做 |
| **⚠️ 灰色** | 抓社群網站（FB / IG）—— ToS 禁止但實際大家在做 | 自己評估風險 |
| **❌ 違法** | 破解登入 / 繞過反爬被告 / 商用付費資料 / 轉售別人資料 | **不要做、會被告** |

  
### 台灣相關案例（真實判例）

**2020 蝦皮商品爬蟲案**：
- 有人爬蝦皮商品資料、被告「妨害電腦使用罪」
- 最終判決：**爬「公開資料」不構成電腦犯罪**
- **但**——大量爬導致對方伺服器當機 / 業務受損、可能涉刑法 360 條（無故取得他人電腦資料罪）

**白話結論**：
- 爬公開資料 = 合法（但要看 ToS）
- 高頻率打爆對方 = **可能違法**
- 破解付費牆 / 登入後資料 = **絕對違法**

  
### 爬蟲 5 大道德守則（職業道德）

```text
1. 看 robots.txt
   - 網站 https://example.com/robots.txt
   - 列出「哪些 URL 禁止爬」、要尊重

2. 每次請求 sleep 1-2 秒
   - 不要打爆對方
   - 用 time.sleep(1) 或 asyncio.sleep(1)

3. User-Agent 寫真實的（不要假裝瀏覽器騙人）
   - 慣例：寫聯絡 email、讓對方能聯絡你
   - 例：'MyBot/1.0 (contact: you@example.com)'

4. 有 API 用 API、爬蟲是最後手段
   - API 通常有 rate limit + 認證、合法
   - 爬蟲是「**沒 API 才不得已**」

5. 抓到的資料不要商用 / 轉售
   - 除非有明確授權
   - 學術 / 個人研究通常 OK
```

  
### 爬蟲 3 大難度等級（看你要爬什麼）

| 等級 | 範例 | 用什麼工具 |
|---|---|---|
| **🟢 Easy** | 靜態 HTML、無反爬（Wiki / 政府網站 / PTT） | requests + BeautifulSoup（L2） |
| **🟡 Medium** | JS 渲染、Cookie / 簡單反爬（蝦皮 / Momo / Dcard） | Playwright（L3）/ Scrapy（L4） |
| **🔴 Hard** | Cloudflare / 圖形驗證 / IP 限制（FB / IG / LinkedIn） | 多 IP 池（L6）+ undetected-chromedriver |

> 💡 **新手起手**：先學 Easy 等級的 requests + BeautifulSoup（L2）、把握住 80% 場景。Medium / Hard 等真的有需求再學、不然容易學完用不到。

  
### Python 為什麼是爬蟲首選（vs Node.js / Go）

| 語言 | 強項 | 弱項 |
|---|---|---|
| **Python** ⭐ | 生態最完整、AI 整合好、最多教學 | 速度慢（不影響爬蟲場景） |
| **Node.js** | async 原生、Playwright JS 版超強 | 庫沒 Python 多（ch29 詳教） |
| **Go** | 並發超強、效能極致 | 學習曲線陡（ch30 詳教） |

**結論**：99% 場景用 **Python**、需要超大規模才考慮 Go。

**這行做什麼？逐個解釋：**
- `requests` —— 同步 HTTP 客戶端、最常用（L2）
- `beautifulsoup4` —— HTML 解析（L2）
- `lxml` —— BeautifulSoup 的快速 parser
- `httpx` —— async HTTP 客戶端（L14）
- `playwright` —— JS 渲染瀏覽器自動化（L3）
