**這行做什麼？逐個解釋：**
- `requests` —— 同步 HTTP 客戶端、最直觀（Kenneth Reitz 寫的、API 設計典範）
- `beautifulsoup4` —— HTML 解析器（import 時叫 `bs4`、不是 `beautifulsoup4`）
- `lxml` —— BeautifulSoup 的後端 parser、比預設 `html.parser` 快 5 倍

  
**第一個爬蟲（3 步驟）：**

```text
import requests
from bs4 import BeautifulSoup

# === 1. 抓網頁 ===
url = 'https://example.com'
res = requests.get(url)
print(res.status_code)              # 200 = OK、404 = 找不到、403 = 被禁
print(res.text[:500])               # 看前 500 字

# === 2. 解析 HTML ===
soup = BeautifulSoup(res.text, 'lxml')

# === 3. 找資料（用 CSS selector、跟前端一致）===
title = soup.select_one('h1').text                  # 取第一個 h1
links = [a['href'] for a in soup.select('a')]      # 取所有連結
prices = [p.text for p in soup.select('.price')]   # 取所有 class="price" 的文字
```

  
### Requests 常用功能

**1. 加 User-Agent（假裝是瀏覽器、防 403）：**

```text
headers = {
    'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)'
}
res = requests.get(url, headers=headers)
```

> 💡 **很多網站擋預設 `python-requests/x.x`** UA、看到就 403。加上瀏覽器 UA 通常就能爬。

  
**2. Query 參數（URL 後 ?）：**

```text
params = {'q': 'python', 'page': 2}
res = requests.get('https://example.com/search', params=params)
# 自動產出 https://example.com/search?q=python&page=2
```

  
**3. POST 請求（送表單、登入用）：**

```text
data = {'username': 'alice', 'password': 'secret'}
res = requests.post('https://example.com/login', data=data)
```

  
**4. Cookie / Session（持久登入狀態）：**

```text
session = requests.Session()                 # 自動管 cookie
session.post('/login', data=data)            # 登入、拿到 cookie
res = session.get('/profile')                # 自動帶 cookie
```

  
**5. Timeout（**必設**、不然會 hang 住）：**

```text
res = requests.get(url, timeout=10)          # 10 秒沒回應就 raise
```

  
**6. 錯誤處理（**永遠 try-except**）：**

```text
try:
    res = requests.get(url, timeout=10)
    res.raise_for_status()                   # 4xx / 5xx 會 raise HTTPError
except requests.RequestException as e:
    print(f'錯誤：{e}')
```

> ⚠️ **不寫 timeout 是新手坑**——爬蟲跑一半卡住、process 永遠不結束、要強制 kill。**永遠寫 timeout**。

  
### BeautifulSoup 找資料 3 種方式

**1. CSS Selector（推薦、跟前端一致）：**

```text
soup.select('div.product .title')             # 多個結果
soup.select_one('#main > h1')                 # 一個結果
soup.select('a[href^="/product/"]')           # 屬性開頭比對
soup.select('div.product:nth-child(2)')      # 偽類選擇
```

  
**2. find / find_all（傳統方法、可讀性高）：**

```text
soup.find('h1')                               # 一個
soup.find_all('div', class_='product')        # 多個（class 要加底線）
soup.find('a', attrs={'data-id': '123'})     # 自訂屬性
```

  
**3. 取資料：**

```text
el.text                                       # 文字內容（含子元素）
el.get_text(strip=True)                       # 去除頭尾空白
el['href']                                    # 屬性值
el.attrs                                      # 全部屬性（dict）
el.name                                       # tag 名稱（'a' / 'div'）
```

> 💡 **怎麼找 CSS selector？** Chrome F12 → 右鍵元素 → Copy → Copy selector。會給你完整路徑、但通常太長、自己簡化。

  
### 實戰：爬 PTT 八卦版標題

**情境**：抓最新 PTT 八卦版前 20 則標題 + 推文數 + 作者。

```text
import requests
from bs4 import BeautifulSoup

url = 'https://www.ptt.cc/bbs/Gossiping/index.html'
cookies = {'over18': '1'}                    # ⚠️ PTT 八卦版要這個 cookie 過 18 歲確認
headers = {'User-Agent': 'Mozilla/5.0'}

res = requests.get(url, headers=headers, cookies=cookies)
soup = BeautifulSoup(res.text, 'lxml')

for post in soup.select('.r-ent'):           # 每篇文章是一個 .r-ent
    title_el = post.select_one('.title a')
    if title_el:                              # 有些是「(本文已被刪除)」沒 a
        title = title_el.text.strip()
        link = 'https://www.ptt.cc' + title_el['href']
        push = post.select_one('.nrec').text  # 推文數（推爆顯示「爆」）
        author = post.select_one('.author').text
        print(f'{push} {title} by {author}')
```

> 💡 **為什麼用 `if title_el`?** 防 NoneType error——已刪除文章沒 `.title a`、select_one 回 None、後面 .text 會 AttributeError。

  
### 儲存到 CSV / DB（搭 Pandas）

```text
import pandas as pd

results = []
for post in soup.select('.r-ent'):
    title_el = post.select_one('.title a')
    if not title_el:
        continue
    results.append({
        'title': title_el.text.strip(),
        'author': post.select_one('.author').text,
        'push': post.select_one('.nrec').text,
        'link': 'https://www.ptt.cc' + title_el['href'],
    })

df = pd.DataFrame(results)
df.to_csv('ptt_titles.csv', index=False, encoding='utf-8-sig')
# ⚠️ utf-8-sig（不是 utf-8）= Excel 直接開不亂碼
```

  
### 加 sleep + retry（業界做法）

```text
import time
import random

for url in url_list:
    try:
        res = requests.get(url, headers=headers, timeout=10)
        res.raise_for_status()
        # ... 處理 ...
    except requests.RequestException as e:
        print(f'失敗 {url}: {e}')
        time.sleep(5)                         # 失敗等久一點
        continue
    
    # === 隨機 sleep 1-3 秒（不打爆對方、看起來像人）===
    time.sleep(random.uniform(1, 3))
```
