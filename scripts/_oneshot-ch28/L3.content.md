**Playwright = JS 渲染網站殺手**——Requests 抓不到的「**JS 才載入的資料**」（SPA / React / Vue 網站）、用這套解決。**蝦皮 / Momo / Dcard 必用**。

> 💡 **何時跳 Playwright？** Requests 抓網頁 view source 沒看到資料、但瀏覽器 F12 Network 看得到資料 → 那是 JS 渲染 → 用 Playwright。

  
### 安裝 + 第一個 Playwright 爬蟲

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 Playwright + 下載 Chromium）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add playwright
playwright install chromium
```

**這 2 行做什麼？逐個解釋：**
- `uv add playwright` —— 裝 Python 套件
- `playwright install chromium` —— **下載瀏覽器 binary**（~150MB）、Playwright 控制這個瀏覽器跑

> 💡 也可以 `playwright install`（不指定）裝 Chromium + Firefox + WebKit 三個、~500MB。新手只裝 chromium 夠。

  
**基本使用：**

```text
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    # === 啟動瀏覽器 ===
    browser = p.chromium.launch(headless=True)
    # headless=True：背景跑（production）
    # headless=False：開出瀏覽器視窗、看 debug 用
    
    page = browser.new_page()
    page.goto('https://example.com')
    
    # === 等待元素出現（JS 載入需要時間）===
    page.wait_for_selector('.product')
    
    # === 取資料（用 locator API、跟 BS4 select 類似）===
    title = page.locator('h1').inner_text()
    products = page.locator('.product').all()
    
    for prod in products:
        name = prod.locator('.name').inner_text()
        price = prod.locator('.price').inner_text()
        print(name, price)
    
    browser.close()
```

  
### 互動：點擊 / 輸入 / 捲動

**Playwright 強項 = 跟真實瀏覽器一樣操作**：

```text
# === 點按鈕 ===
page.click('button.load-more')

# === 填表單 ===
page.fill('input[name="email"]', 'a@b.com')
page.fill('input[name="password"]', 'secret')
page.click('button[type="submit"]')

# === 等待跳轉到新頁面 ===
page.wait_for_url('**/dashboard')

# === 捲動到頁尾（觸發 lazy load）===
page.evaluate('window.scrollTo(0, document.body.scrollHeight)')

# === 等 N 毫秒（不建議、太死板）===
page.wait_for_timeout(2000)              # 2 秒

# === 等網路 idle（最穩、所有 request 完成才繼續）===
page.wait_for_load_state('networkidle')

# === 截圖（debug 神器）===
page.screenshot(path='shot.png', full_page=True)
```

  
### 實戰：無限捲動抓所有商品

**情境**：蝦皮 / 抖音類「**滾到底自動載入更多**」的頁面、抓所有商品。

```text
from playwright.sync_api import sync_playwright

def scrape_infinite_scroll(url, max_scrolls=20):
    """無限捲動抓資料、最多滾 max_scrolls 次"""
    with sync_playwright() as p:
        browser = p.chromium.launch(headless=True)
        page = browser.new_page()
        page.goto(url)
        
        previous_height = 0
        for _ in range(max_scrolls):
            # === 1. 滾到底 ===
            page.evaluate('window.scrollTo(0, document.body.scrollHeight)')
            # === 2. 等新內容載入 ===
            page.wait_for_timeout(1500)
            # === 3. 檢查高度有沒有變（沒變 = 沒新內容、結束）===
            current_height = page.evaluate('document.body.scrollHeight')
            if current_height == previous_height:
                break
            previous_height = current_height
        
        # === 抓所有商品 ===
        products = page.locator('.product-card').all()
        results = []
        for prod in products:
            results.append({
                'name': prod.locator('.name').inner_text(),
                'price': prod.locator('.price').inner_text(),
            })
        
        browser.close()
        return results
```

  
### 處理彈窗 / Cookie 同意

```text
# === 自動處理 alert / confirm dialog ===
page.on('dialog', lambda d: d.accept())

# === 關閉 Cookie 同意彈窗（很多歐美網站 GDPR 必跳）===
try:
    page.click('button:has-text("同意")', timeout=3000)
except:
    pass                                      # 沒彈窗也不出錯
```

> 💡 **`button:has-text("同意")`** 是 Playwright 強大的 selector——直接用「**按鈕內含的文字**」找、超直覺。

  
### 截攔 API 回應（超強技巧、比解析 HTML 快 10 倍）

**很多 SPA 後端打 JSON API、Playwright 可以攔截、直接拿乾淨 JSON、不用解 HTML！**

```text
api_data = []

def handle_response(response):
    # 過濾：只攔 /api/products 的回應
    if '/api/products' in response.url:
        api_data.append(response.json())

page.on('response', handle_response)
page.goto('https://shop.example.com')

# api_data 自動填入後端 JSON
# 比 HTML parse 快很多 + 結構乾淨
```

> 💡 **這招是爬蟲高手的秘訣**——大部分現代網站都是 SPA、前端打 JSON API、攔截後端比解前端 HTML 簡單 10 倍。

  
### Async 版本（並行 10 個爬蟲、超快）

```text
import asyncio
from playwright.async_api import async_playwright

async def scrape(url):
    async with async_playwright() as p:
        browser = await p.chromium.launch()
        page = await browser.new_page()
        await page.goto(url)
        text = await page.inner_text('h1')
        await browser.close()
        return text

# === 同時跑 10 個 URL ===
async def main():
    urls = [f'https://example.com/page/{i}' for i in range(10)]
    results = await asyncio.gather(*[scrape(u) for u in urls])
    print(results)

asyncio.run(main())
```

  
### Playwright vs Selenium 對比（為什麼選 Playwright）

| 項目 | **Playwright**（推薦） | Selenium（傳統） |
|---|---|---|
| **發布** | 2020（Microsoft 出） | 2004 |
| **速度** | 快 2-3 倍 | 中 |
| **API 設計** | 現代、async-first | 老派 |
| **自動 wait** | 內建（不用 sleep） | 要自己寫 |
| **多瀏覽器** | Chromium / Firefox / WebKit | 全部 |
| **截攔 API** | ✓ 原生支援 | 需 extension |
| **新專案建議** | ⭐ Playwright | 老 codebase 維護才用 |

  
    **☕ 用人話講**
    
Playwright = **2026 爬蟲首選**（取代 Selenium）。**5 件事背起來**：(1) `page.goto()` 開網頁 (2) `page.locator()` 取資料（CSS selector） (3) `page.wait_for_selector()` 等元素出現 (4) **`page.on('response', ...)` 攔 API**——這招超強 (5) **headless=False** debug 看畫面。**怎麼練習**：找一個 JS 渲染的網站（蝦皮 / Dcard）、用 Playwright 抓資料、體驗「**真實瀏覽器自動化**」的威力。
