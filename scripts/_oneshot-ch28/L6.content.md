**IP 池 / Proxy = 「**用多個 IP 輪流發 request、避免單一 IP 被封**」**——商業爬蟲核心、L5 反爬的進階解法。

> 💡 **何時需要 proxy？** 同一 IP 高頻率被擋（IP 限流）、或要爬有「地理限制」的內容（如美國 IP 才看得到的 Netflix 美國版 catalog）。**小規模爬蟲不用、別過度工程**。

  
### Proxy 4 大類型（依品質 / 價格分）

| 類型 | 特色 | 價格 | 適合 |
|---|---|---|---|
| **免費 Proxy** | 不穩 / 慢 / 可能被監聽 | $0 | **不建議、別用！** |
| **Datacenter Proxy** | 機房 IP、便宜、易識破 | $1-5/月/IP | 學習 / 小規模 |
| **Residential Proxy** ⭐ | 真實住宅 IP、難識別 | $10-15/GB | 中型商業 |
| **Mobile Proxy** | 4G IP、最難封 | $30-100/月 | 高難度（FB / IG） |

> ⚠️ **免費 proxy 千萬別用**——常常被駭客架的、會記錄你的請求 + 偷密碼。

### 用法（Requests）

```text
proxies = {
    'http': 'http://user:pass@proxy.smartproxy.com:7000',
    'https': 'http://user:pass@proxy.smartproxy.com:7000',
}
res = requests.get('https://example.com', proxies=proxies)
print(res.json()['origin'])                  # 看到 proxy 的 IP（不是你電腦的）
```

> 💡 **驗證 proxy 有效**：去 `https://httpbin.org/ip` 看回的 IP、跟你電腦真實 IP（去 google「what is my ip」）對比、不同表示 proxy 生效。

  
### IP 池輪詢（多個 proxy 輪流用、業界做法）

```text
import random
import requests

PROXY_POOL = [
    'http://proxy1.io:8080',
    'http://proxy2.io:8080',
    'http://proxy3.io:8080',
]

def get_with_proxy(url):
    """隨機選 proxy、失敗自動 retry 3 次"""
    for _ in range(3):
        proxy = random.choice(PROXY_POOL)
        try:
            res = requests.get(
                url,
                proxies={'https': proxy},
                timeout=10
            )
            if res.status_code == 200:
                return res
        except:
            continue                          # 這個 proxy 壞、試下一個
    return None
```

  
### 自己抓免費 Proxy 清單（小規模測試用、不建議 production）

```text
import requests
from bs4 import BeautifulSoup

def get_free_proxies():
    """從 free-proxy-list.net 抓免費 HTTPS proxy"""
    res = requests.get('https://www.free-proxy-list.net')
    soup = BeautifulSoup(res.text, 'lxml')
    proxies = []
    for row in soup.select('table tbody tr'):
        cols = row.select('td')
        if cols[6].text == 'yes':            # 第 7 欄是「支援 https?」
            proxies.append(f'http://{cols[0].text}:{cols[1].text}')
    return proxies

# === ⚠️ 免費 proxy 通常 10 個有 1 個能用、不穩 ===
```
