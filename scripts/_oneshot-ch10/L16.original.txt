### 檔案結構

```text
nuxt-app/
├── nuxt.config.ts          ← 全局設定
├── app.vue                  ← 根元件
├── pages/                   ← 自動檔案路由
│   ├── index.vue            → /
│   ├── about.vue            → /about
│   └── blog/
│       ├── index.vue        → /blog
│       └── [slug].vue       → /blog/:slug
├── layouts/                 ← 共用 layout
│   ├── default.vue
│   └── admin.vue
├── components/              ← 自動 import（不用手寫 import）
│   └── MyButton.vue
├── composables/             ← 自動 import
│   └── useCounter.ts
├── server/
│   ├── api/
│   │   └── users.get.ts     → GET /api/users
│   ├── middleware/
│   └── plugins/
├── middleware/              ← 路由 middleware
│   └── auth.ts
├── plugins/                 ← Nuxt 啟動時跑
├── public/                  ← 靜態檔
├── assets/                  ← 處理過的資源
└── stores/                  ← Pinia store（自動匯入）
```

  
### 自動 import：Nuxt 殺手鐧

  
放在 `components/`、`composables/`、`utils/` 的東西**不用 import 就能用**：

```vue

// 不用寫 import！
const count = ref(0);          // 自動 import vue
const route = useRoute();        // 自動 import 路由
const { data } = await useFetch("/api/users");  // Nuxt 內建 composable
const counter = useCounter(); // 你自己寫的 composable

    

```
