### 選擇

  
- **NextAuth.js / Auth.js**：最紅、支援所有 OAuth、open source
- **Lucia Auth**：輕量、自由度高、要自己寫 UI
- **Clerk**：付費但 UX 最好、整合最快
- **Supabase Auth**：跟 Supabase DB 一起用方便
- **自己寫**：完全控制（學習用）

### JWT vs Session

  
|  | JWT | Session |
| --- | --- | --- |
| 儲存 | cookie / localStorage | server DB + cookie ID |
| 失效 | 到期才失效 | 可即時撤銷 |
| 擴展性 | 無狀態，好 scale | 要 sticky session 或共享 store |
| 建議 | 簡單應用、跨域 | 需要立刻登出全裝置 |

  
**🚨 資安警報**

JWT 不要放**敏感資訊**（payload 是 base64，不是加密）。token 一律存 `httpOnly cookie`，**不要存 localStorage**（XSS 一中就完）。
