**時序資料**（time series）= 「**按時間順序排的資料**」——股價 / 銷售 / 流量 / 氣溫 / 感測器數據都是。Pandas 對時序處理有專門優化、學會這套受用 5 年。

> 💡 **時序資料 3 個關鍵動作**：(1) 把時間欄變 `datetime` (2) 設為 index (3) 用 `resample` 重採樣聚合。**這 3 件事不會、處理時序就累。**

  
### 1. 把欄位變 datetime 型別 + 設為 index

**為什麼要設 index？** Pandas 對「時間 index」有特殊優化、可以用 `df['2026-01']` 這種超直覺切片。

```text
import pandas as pd

# === 1. 字串時間轉 datetime ===
df['date'] = pd.to_datetime(df['date'])
# 自動處理多種格式：
# "2026-01-15"、"2026/01/15"、"Jan 15, 2026"、"20260115" 都行

# === 2. 設為 index ===
df.set_index('date', inplace=True)
# 之後 df 的 index 就是 date 欄、可以用時間直接切片
```

  
### 2. 時間範圍切片（超直覺）

```text
df['2026-01']                                # 整個 1 月所有資料
df['2026']                                   # 整年
df['2026-01-01':'2026-03-31']                # Q1（含頭含尾）
df.loc['2026-01-15':'2026-02-15']            # 中間 1 個月

# === Python 標準 datetime 也行 ===
from datetime import datetime
df[df.index > datetime(2026, 1, 1)]          # > 2026 元旦
```

  
### 3. Resample（重採樣、改變時間粒度）

**Resample** = 「**把細的時間單位變粗、順便算統計**」。例如：每秒資料變每分鐘平均、每日資料變每月總和。

```text
# === 簡寫：D=日 / W=週 / M=月 / Q=季 / Y=年 / H=時 / T=分 / S=秒 ===

df['sales'].resample('D').sum()              # 每日總和
df['sales'].resample('W').mean()             # 每週平均
df['sales'].resample('M').sum()              # 每月總和（最常用）
df['sales'].resample('Q').sum()              # 每季
df['sales'].resample('Y').sum()              # 每年

# === 多種聚合一次算 ===
df.resample('M').agg({
    'sales': 'sum',
    'visitors': 'mean',
    'orders': 'count'
})
```

> 💡 **Resample vs GroupBy** —— `groupby` 適合「**離散類別**」、`resample` 專門給「**時間粒度**」優化過、寫起來更直覺。

  
### 4. Rolling Window（滾動視窗、移動平均、技術指標必用）

**Rolling** = 「**過去 N 個時間點的滑動平均 / 統計**」——股票均線、感測器去雜訊、銷售趨勢都用這個。

```text
# === 7 日均線（最常見的「日均線」）===
df['ma7'] = df['price'].rolling(7).mean()

# === 30 日均線 ===
df['ma30'] = df['price'].rolling(30).mean()

# === 30 日標準差（量化波動度）===
df['std30'] = df['price'].rolling(30).std()

# === 50 日最大值（技術分析的「壓力線」）===
df['high50'] = df['price'].rolling(50).max()
```

  
**畫均線圖（搭 matplotlib、L5 教過）：**

```text
import matplotlib.pyplot as plt
plt.figure(figsize=(12, 6))
plt.plot(df.index, df['price'], label='原價', alpha=0.5)
plt.plot(df.index, df['ma7'], label='7 日均線')
plt.plot(df.index, df['ma30'], label='30 日均線')
plt.legend()
plt.show()
# 經典股票圖：原價 + 短中長期均線
```

  
### 5. 變化率（pct_change、報酬率必用）

```text
# === 日報酬率（今天比昨天漲跌多少 %）===
df['pct_change'] = df['price'].pct_change()
# 例：[0.02, -0.01, 0.03]  → 漲 2%、跌 1%、漲 3%

# === 累積報酬（從第一天到今天賺多少倍）===
df['cumulative'] = (1 + df['pct_change']).cumprod()
# 例：1.02 → 1.0098 → 1.0401  = 累積漲 4.01%

# === 計算年化波動度（金融常用）===
annual_vol = df['pct_change'].std() * (252 ** 0.5)
# 252 = 一年交易日、開根號 = 從日波動轉年波動
```

  
### 6. 時間欄位提取（衍生特徵、ML 常用）

**用 `index.dt.xxx` 拿時間屬性**（記得 index 已是 datetime）：

```text
df['year'] = df.index.year                    # 年份
df['month'] = df.index.month                  # 月份（1-12）
df['day'] = df.index.day                      # 日期（1-31）
df['weekday'] = df.index.day_name()           # 'Monday', 'Tuesday'...
df['is_weekend'] = df.index.dayofweek >= 5    # 週末 = True
df['quarter'] = df.index.quarter              # 季（1-4）
df['hour'] = df.index.hour                    # 小時（時序到秒等級用）

# === 國定假日 / 特殊日（手動標）===
holidays = ['2026-01-01', '2026-02-10', '2026-02-11']
df['is_holiday'] = df.index.isin(pd.to_datetime(holidays))
```

  
### 7. Shift（位移、跟前 / 後值比較）

```text
# === 拿「昨天的價格」當新欄位 ===
df['prev_price'] = df['price'].shift(1)       # 整欄往下移 1 列
df['change'] = df['price'] - df['prev_price'] # 算「漲跌差」

# === 拿「明天的價格」（預測用）===
df['next_price'] = df['price'].shift(-1)      # 往上移 1 列
```

  
### 實戰：股票簡易回測

**情境**：「**5 日均線突破 20 日均線就買進、跌破就賣出**」黃金交叉策略、用台積電股價回測。

```text
import yfinance as yf                         # 金融資料免費 API

# === 1. 抓台積電 1 年資料 ===
df = yf.download('2330.TW', start='2025-01-01', end='2026-05-27')
df.index = pd.to_datetime(df.index)

# === 2. 算均線 ===
df['ma5'] = df['Close'].rolling(5).mean()
df['ma20'] = df['Close'].rolling(20).mean()

# === 3. 訊號（1=持有、0=現金）===
df['signal'] = (df['ma5'] > df['ma20']).astype(int)

# === 4. 算策略報酬率 ===
df['daily_return'] = df['Close'].pct_change()
df['strategy_return'] = df['signal'].shift(1) * df['daily_return']
# shift(1) = 昨天訊號決定今天進場

# === 5. 累積績效 ===
df['cumulative'] = (1 + df['strategy_return']).cumprod()
print(f"策略總報酬：{(df['cumulative'].iloc[-1] - 1) * 100:.2f}%")
```
