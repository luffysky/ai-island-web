**matplotlib + seaborn = Python 視覺化兩件套**。matplotlib 是底層、seaborn 是「**統計圖好看版**」、實戰常一起用。

> 💡 **如果你看過 ch26 L15 視覺化**——這課是「**資料分析角度的速查**」、強調**中文設定**（很多新手卡在這）跟**實戰 8 種圖**速查。

  
### 基本設定：中文 + 美化（新手最常卡的地方）

**⚠️ 預設 matplotlib 中文會變方框（豆腐字）！** 一定要設字型。

```text
import matplotlib.pyplot as plt
import seaborn as sns

# === Mac / Linux 字型 ===
plt.rcParams['font.family'] = ['PingFang TC', 'Noto Sans CJK TC']

# === Windows 字型 ===
plt.rcParams['font.family'] = ['Microsoft JhengHei']

# === 共通設定：負號正常顯示（不會變方框）===
plt.rcParams['axes.unicode_minus'] = False

# === seaborn 全域風格（推薦設定）===
sns.set_theme(
    style='whitegrid',                        # 白底 + 灰格線
    palette='Set2'                            # 配色：colorblind / pastel / Set2 / husl
)
```

> 💡 **不知道你的系統有什麼字型？** 在 Jupyter 跑：`!fc-list :lang=zh-tw`（Mac / Linux）或 Windows 控制台「字型」看清單。

  
### 8 種主流圖表（資料分析最常用、按使用頻率排序）

**1. 折線圖（line chart）—— 時序資料必用：**

```text
plt.figure(figsize=(10, 6))                  # 設圖大小（寬 10 高 6 吋）
plt.plot(df['date'], df['sales'], marker='o')
plt.title('銷售趨勢')
plt.xlabel('日期')
plt.ylabel('銷售額')
plt.show()
```

  
**2. 長條圖（bar chart）—— 分類比較：**

```text
sns.barplot(
    data=df,
    x='category',                            # X 軸：類別
    y='sales',                               # Y 軸：值
    hue='region'                             # 按 region 上不同顏色
)
```

  
**3. 散點圖（scatter plot）—— 看兩個變數關係：**

```text
sns.scatterplot(
    data=df,
    x='age', y='salary',
    hue='gender',                            # 按性別上色
    size='experience'                         # 按年資調點大小（bubble chart）
)
```

  
**4. 直方圖（histogram）—— 看單一變數分布：**

```text
sns.histplot(
    df['age'],
    bins=30,                                 # 切成 30 個區間（桶）
    kde=True                                 # 加上 KDE 密度曲線
)
```

  
**5. 盒鬚圖（box plot）—— 看分布 + 找離群值：**

```text
sns.boxplot(data=df, x='department', y='salary')
# 一張圖看：中位數 / 四分位 / 離群值 / 各組對比
```

  
**6. 熱力圖（heatmap）—— 看相關性矩陣：**

```text
sns.heatmap(
    df.corr(),                               # Pandas 算所有數值欄相關性
    annot=True,                              # 每格顯示數字
    cmap='coolwarm',                         # 冷暖色（負紅、正藍）
    center=0                                 # 0 為中心
)
```

  
**7. 圓餅圖（pie chart）—— 看比例（用 Pandas 內建快）：**

```text
df['category'].value_counts().plot(
    kind='pie',
    autopct='%1.1f%%'                        # 顯示百分比、保留 1 位小數
)
```

  
**8. Pair Plot（多變量關係矩陣）—— EDA 神器：**

```text
sns.pairplot(df, hue='species')
# 自動：所有數值欄兩兩做散點、對角線畫直方圖
# 第一次拿到資料用這個一秒看完所有關係
```

  
### 子圖（subplots、一張圖多 chart、報告用）

```text
fig, axes = plt.subplots(2, 2, figsize=(12, 8))  # 2x2 排版、總大小 12x8 吋

# === 左上：折線 ===
axes[0, 0].plot(x, y1)
axes[0, 0].set_title('銷售趨勢')

# === 右上：長條 ===
axes[0, 1].bar(cats, vals)
axes[0, 1].set_title('類別比較')

# === 左下：seaborn 盒鬚（用 ax 參數指定軸）===
sns.boxplot(data=df, ax=axes[1, 0])

# === 右下：seaborn 熱力圖 ===
sns.heatmap(corr_mat, ax=axes[1, 1])

plt.tight_layout()                            # 自動調間距、避免重疊
plt.show()
```

  
### 儲存圖片（給簡報 / 報告用）

```text
plt.savefig('chart.png', dpi=300, bbox_inches='tight')

```

  
### Plotly：互動式圖表（之後 L11 詳教）

**為什麼用 Plotly？** 可以**滑鼠 hover 看數值** / **zoom / pan / 嵌入網頁** —— matplotlib 是「靜態圖」、Plotly 是「會動的圖」。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 Plotly）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add plotly
```

**這行做什麼？** 裝 Plotly 套件、含 `plotly.express`（高階簡潔 API）+ `plotly.graph_objects`（底層精細控制）。

```text
import plotly.express as px

fig = px.scatter(
    df, x='age', y='salary',
    color='dept',                            # 按部門上色
    hover_data=['name'],                     # 滑鼠 hover 額外顯示
    size='experience'                         # bubble chart
)
fig.show()                                    # 跳出互動圖

# === 寫成 HTML 嵌入網頁（做 dashboard 用）===
fig.write_html('chart.html')
```

  
### 速查：什麼資料用什麼圖

| 想看什麼 | 用什麼圖 |
|---|---|
| 一個變數的**分布** | 直方圖 / 密度圖 |
| 兩個變數的**關係** | 散點圖 |
| 隨時間**變化** | 折線圖 |
| 各類別**比較** | 長條圖 |
| **比例 / 占比** | 圓餅圖 / 堆疊長條 |
| **離群值 + 分布** | 盒鬚圖 |
| **多變數相關** | 熱力圖 / Pair plot |
| **互動探索 / Web** | Plotly |

matplotlib + seaborn 兩件套：(1) **matplotlib** 底層、能畫一切但 API 較囉嗦 (2) **seaborn** 包在 matplotlib 外、API 簡潔 + 預設超美。**新手 3 個必背**：(1) 設**中文字型**（不然顯示豆腐字） (2) `sns.pairplot()` 一秒看完所有變數關係 (3) `sns.heatmap(df.corr())` 看相關性。**做 dashboard / 給網頁用就跳 Plotly**（L11 詳教）。
