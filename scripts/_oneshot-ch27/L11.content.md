**Plotly = 2026 互動式圖表的主流選擇**——hover 看數值 / zoom / 嵌網頁 / 做 dashboard 必用。matplotlib 是「靜態圖」、Plotly 是「會動的圖」。

> 💡 **何時跳 Plotly？** (1) 做 web dashboard (2) 給客戶互動看 (3) 大資料（用 WebGL 撐百萬點不卡）。**論文 / 印刷品還是用 matplotlib**。

  
### Plotly vs matplotlib 對照

| 特性 | matplotlib | Plotly |
|---|---|---|
| **靜態 vs 互動** | 靜態 PNG / PDF | 互動 HTML（hover / zoom / pan） |
| **適合場景** | 論文、報告、簡報 | Dashboard、Web 嵌入 |
| **大資料** | 一萬點開始卡 | WebGL 撐百萬點 |
| **學習曲線** | 中（API 囉嗦） | 中（Express 簡潔、Graph Objects 細） |
| **檔案輸出** | PNG / PDF / SVG | HTML / PNG / PDF（要 kaleido） |

### Plotly Express（簡潔版、推薦新手）

**`plotly.express`** 是「**高階簡潔 API**」、一行寫完一張圖。

```text
import plotly.express as px
import pandas as pd

df = pd.read_csv("data.csv")

# === 散點圖（bubble chart）===
fig = px.scatter(
    df,
    x="gdp", y="life_expectancy",
    size="population",                       # 點大小依人口
    color="continent",                       # 顏色依大洲
    hover_name="country",                    # hover 顯示國家名
    log_x=True,                              # X 軸用 log scale（GDP 差異大）
    title="GDP vs 壽命"
)
fig.show()
```

  
**其他常用：**

```text
# === 線圖 ===
fig = px.line(df, x="year", y="sales", color="region")

# === Bar ===
fig = px.bar(df, x="category", y="amount", color="country")

# === Histogram ===
fig = px.histogram(df, x="age", nbins=30)

# === Box plot ===
fig = px.box(df, x="country", y="income")

# === Heatmap（相關矩陣）===
fig = px.imshow(df.corr(), text_auto=True)

# === 地圖（散點地圖）===
fig = px.scatter_geo(
    df,
    locations="iso_alpha",                   # 國家 ISO 代碼（TWN / USA）
    size="population",
    color="continent"
)
```

  
### Plotly Graph Objects（精細控制、進階）

當 Express 不夠用、要疊多個 trace、自訂 layout、用 `graph_objects`：

```text
import plotly.graph_objects as go

fig = go.Figure()

# === 加多條線（不同年份對比）===
fig.add_trace(go.Scatter(
    x=x1, y=y1,
    name="2025",
    mode="lines+markers"
))
fig.add_trace(go.Scatter(
    x=x2, y=y2,
    name="2026",
    mode="lines+markers"
))

fig.update_layout(
    title="年度對比",
    xaxis_title="月份",
    yaxis_title="營收"
)
fig.show()
```

  
### 進階圖類型（matplotlib 做不到 / 很難做）

**Sankey 流向圖（看「從哪→到哪」、流量 / 預算分配視覺化）：**

```text
fig = go.Figure(go.Sankey(
    node={"label": ["A", "B", "C", "D"]},
    link={
        "source": [0, 1, 0, 2],              # 從哪個 node
        "target": [2, 3, 3, 3],              # 到哪個 node
        "value": [10, 20, 5, 15]             # 流量
    }
))
# 例：使用者來源（FB / Google / 直接訪問）→ 到產品頁面分佈
```

  
**Sunburst 多層圓餅（階層資料、組織結構 / 分類）：**

```text
fig = px.sunburst(
    df,
    path=["region", "country", "city"],      # 由外到內的階層
    values="population"
)
# 例：地球 → 大洲 → 國家 → 城市的人口分布
```

  
**Treemap（樹狀矩陣、產品 / 預算佔比）：**

```text
fig = px.treemap(
    df,
    path=["category", "subcategory"],
    values="amount"
)
# 比圓餅更清楚、適合多類別比較
```

  
**3D Scatter（多維度視覺化）：**

```text
fig = px.scatter_3d(
    df,
    x="x", y="y", z="z",
    color="category"
)
# 可以拖拽旋轉、看高維資料
```

  
### 輸出（4 種方式、看用途選）

```text
# === 1. 即時顯示（Jupyter 內 inline / 瀏覽器跳出）===
fig.show()

# === 2. 存 HTML（互動保留、超適合給客戶看）===
fig.write_html("chart.html")
# 客戶開瀏覽器、能 hover / zoom、超專業

# === 3. 存圖片（PNG / PDF、要先裝 kaleido）===
# 終端機跑：uv add kaleido
fig.write_image("chart.png", width=1200, height=800)

# === 4. 嵌入網頁（拿到 HTML 字串、塞進自己網站）===
html_str = fig.to_html(include_plotlyjs="cdn")
# include_plotlyjs="cdn" = Plotly JS 從 CDN 載、HTML 比較小
```

  
### Dash（Plotly 的 Dashboard 框架）

**Dash** = Plotly 出的「**Python 寫 web dashboard 框架**」、純 Python 不用 HTML / JS。

```text
import dash
from dash import dcc, html

app = dash.Dash(__name__)

app.layout = html.Div([
    html.H1("我的 Dashboard"),
    dcc.Graph(figure=fig1),                  # 嵌入 Plotly 圖
    dcc.Graph(figure=fig2),
])

app.run_server(debug=True)
# http://localhost:8050
```

  
### Dash vs Streamlit 對比（你選哪個？）

| 框架 | 特色 | 適合 |
|---|---|---|
| **Streamlit**（L12 詳教） | 最簡單、5 分鐘上手 | 資料分析師、indie |
| **Plotly Dash** | 類 React 元件架構、彈性高 | 工程師、大型 dashboard |
| **Gradio** | 機器學習 model demo 專用 | AI / ML 展示 |

**新手選 Streamlit、進階選 Dash、AI demo 選 Gradio**。

  
    **☕ 用人話講**
    
Plotly 心法：(1) **新手用 `plotly.express`**（簡潔、一行畫圖） (2) **要疊多 trace / 細控用 `graph_objects`** (3) **存 HTML 給客戶看** —— 互動式超專業 (4) **做 dashboard** —— Streamlit（L12）比 Dash 快 5 倍。**3 個 matplotlib 做不到的圖**：Sankey 流向、Sunburst 階層、3D scatter——遇到這種需求才會發現 Plotly 必要性。**怎麼練習**：把你做過的任何 matplotlib 圖、用 px 重做一次、體驗互動性差異。
