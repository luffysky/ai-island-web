**資料清理 = 資料分析 60% 的時間**。「**Garbage in, garbage out**」、清理沒做好、後面分析全錯。這課完整 4 大主題：**缺失值 / 重複 / 異常 / 型別字串日期**。

> 💡 **心法**：「**清理前先備份**」——`df_original = df.copy()`、之後想對照原資料隨時 reset。

  
### 缺失值（Missing Data、最常遇到）

**先偵測：知道缺多少、缺哪裡。**

```text
df.isna()                                     # 每格 True / False
df.isna().sum()                               # 各欄缺多少筆
df.isna().sum() / len(df)                     # 缺失比例 %

# === 視覺化（找缺失 pattern）===
import seaborn as sns
sns.heatmap(df.isna(), cbar=False)
# 一張圖看完：哪幾欄缺、是「整列缺」還是「散布缺」
```

  
**處理策略 4 種：**

**1. 刪除：**

```text
df.dropna()                                   # 任何欄 NaN 就刪整列
df.dropna(subset=["email"])                   # 只看 email 欄 NaN
df.dropna(thresh=3)                           # 至少 3 個非 NaN 才留
```

  
**2. 填充：**

```text
df["age"].fillna(df["age"].mean())            # 平均
df["age"].fillna(df["age"].median())          # 中位數（防 outlier、推薦）
df["category"].fillna("unknown")              # 固定字串
df["score"].fillna(method="ffill")            # 用前一個（時序常用）
df["score"].fillna(method="bfill")            # 用後一個

# === 進階：按組填平均 ===
df["age"].fillna(
    df.groupby("country")["age"].transform("mean")
)
# 每個國家內、用該國平均填、比全域平均合理
```

  
**3. 插值（時序專用）：**

```text
df["temp"].interpolate(method="linear")       # 線性內插
df["temp"].interpolate(method="time")         # 時間插值（按時間軸權重）
```

  
**4. ML 預測填補（進階）：**

```text
from sklearn.impute import KNNImputer

imputer = KNNImputer(n_neighbors=5)           # 用最相似 5 筆預測
df_imputed = pd.DataFrame(
    imputer.fit_transform(df),
    columns=df.columns
)
# 適合：缺失多 + 重要欄位、簡單 fillna 不夠用
```

  
### 重複資料

```text
# === 找重複 ===
df.duplicated()                               # 整列完全相同
df.duplicated(subset=["email"])               # 只看 email 是否重複

# === 刪除 ===
df.drop_duplicates()                          # 整列重複才刪
df.drop_duplicates(subset=["email"], keep="first")  # 留第一筆
df.drop_duplicates(subset=["email"], keep="last")   # 留最後一筆
df.drop_duplicates(subset=["email"], keep=False)    # 全刪（連第一個都不留）
```

  
### 異常值（Outliers、3 種方法）

**1. IQR 方法（四分位距、最常用）：**

```text
Q1 = df["amount"].quantile(0.25)              # 25 百分位
Q3 = df["amount"].quantile(0.75)              # 75 百分位
IQR = Q3 - Q1                                 # IQR = Q3 - Q1

# === 找出異常值 ===
outliers = df[
    (df["amount"] < Q1 - 1.5 * IQR) |
    (df["amount"] > Q3 + 1.5 * IQR)
]
# 1.5*IQR 是統計界慣例（Tukey 提出）

# === 用 clip 截掉（不刪、改成邊界值）===
df["amount"] = df["amount"].clip(
    lower=Q1 - 1.5 * IQR,
    upper=Q3 + 1.5 * IQR
)
```

  
**2. Z-score 方法（標準差距離）：**

```text
from scipy import stats

df["z"] = stats.zscore(df["amount"])
outliers = df[abs(df["z"]) > 3]
# z > 3 = 偏離平均 3 個標準差（99.7% 信賴區間外）
```

  
**3. Isolation Forest（ML、處理高維資料）：**

```text
from sklearn.ensemble import IsolationForest

iso = IsolationForest(contamination=0.05)     # 假設 5% 是異常
df["is_outlier"] = iso.fit_predict(df[["amount"]])
# -1 = 異常、1 = 正常
```

  
### 資料型別（dtype 修正）

```text
df.dtypes                                     # 看每欄型別

# === 轉型 ===
df["age"] = df["age"].astype(int)             # 強制轉 int（小心 NaN）
df["date"] = pd.to_datetime(df["date"])
df["amount"] = pd.to_numeric(df["amount"], errors="coerce")
# errors="coerce" = 無法轉的變 NaN、不會 raise
```

  
### 字串清理（最常見：去空白 / 統一大小寫 / 提取）

```text
df["name"].str.strip()                        # 去頭尾空白（最常用）
df["name"].str.lower()                        # 全小寫
df["name"].str.upper()                        # 全大寫
df["name"].str.replace(" ", "_")              # 替換

# === 條件篩選（字串 method）===
df["email"].str.contains("@")                 # 含 @
df["phone"].str.match(r"^09\d{8}$")           # 正規表達式（台灣手機）

# === 提取資訊 ===
df["domain"] = df["email"].str.split("@").str[1]
# alice@gmail.com → gmail.com

df["year"] = df["date"].str.extract(r"(\d{4})")
# 從字串抽 4 位數年份
```

  
### 日期清理

```text
df["date"] = pd.to_datetime(
    df["date"],
    errors="coerce",                          # 無效日期變 NaT
    format="%Y-%m-%d"                         # 明確格式（更快、防誤判）
)

# === 找出無效日期 ===
df[df["date"].isna()]

# === 統一格式輸出 ===
df["date_str"] = df["date"].dt.strftime("%Y-%m-%d")

# === 抽出時間特徵（feature engineering、L21 詳教）===
df["year"] = df["date"].dt.year
df["month"] = df["date"].dt.month
df["dayofweek"] = df["date"].dt.dayofweek
df["is_weekend"] = df["date"].dt.dayofweek >= 5
```

### 5 大清理心法（資深 vs 新手差別）

```text
1. ❌ 不要追完美、追「夠用」
   → 清到能跑分析就行、別卡 1 週清資料

2. ✅ 清理前先備份
   → df_original = df.copy()、隨時對照

3. ✅ 紀錄每個步驟
   → 用 Markdown cell 寫「為什麼這樣清」
   → 3 個月後自己 / 同事看得懂

4. ✅ 異常值不一定要刪
   → 可能是真實 outlier（VIP 大額交易、不是錯誤）
   → 先看「為什麼異常」再決定處理

5. ✅ 缺失值的「為什麼缺」比怎麼填重要
   → MCAR（完全隨機缺）vs MAR（依其他變數缺）vs MNAR（依自己值缺）
   → 不同類型用不同策略
```
