**Pandas 5 大資料操作完整速查**：Filter / Sort / GroupBy / Merge / Pivot。L4 教過 8 大基礎、這課**深入每個操作**、給實戰常見寫法。

> 💡 **建議跟著 Pandas + SQL 對照學**——很多公司面試會問「同樣 query 用 Pandas / SQL 各怎麼寫」、會 Pandas 也順便會 SQL。

  
### DataFrame 基本回顧

```text
import pandas as pd

df = pd.read_csv("data.csv")
df.head()                                    # 前 5 列
df.tail(10)                                  # 後 10 列
df.info()                                    # 欄位 + 型別 + 缺值
df.describe()                                # 統計摘要
df.shape                                     # (列數, 欄數)
df.columns                                   # 所有欄位名
df.dtypes                                    # 每欄型別
```

  
### 1. Filter（過濾、相當於 SQL WHERE）

```text
# === 單條件 ===
adults = df[df["age"] >= 18]

# === 多條件（用 & 不是 and、每個條件加括號）===
adults_tw = df[(df["age"] >= 18) & (df["country"] == "TW")]

# === isin（在某個 list 內）===
many = df[df["name"].isin(["Alice", "Bob", "Charlie"])]

# === notna / isna（缺值處理）===
not_null = df[df["email"].notna()]

# === 字串過濾（str. 系列方法）===
df[df["name"].str.contains("王")]            # 包含「王」
df[df["email"].str.endswith("@gmail.com")]   # 結尾
df[df["phone"].str.startswith("09")]         # 開頭
df[df["text"].str.len() > 10]                # 長度
df[df["code"].str.match(r"^[A-Z]{3}\d+$")]   # 正規表達式
```

  
### 2. Sort（排序）

```text
# === 單欄降冪 ===
df.sort_values("amount", ascending=False)

# === 多欄混合排序（先按 category 升冪、同 category 內按 amount 降冪）===
df.sort_values(
    ["category", "amount"],
    ascending=[True, False]
)

# === 按 index 排（如時序資料設了 datetime index）===
df.sort_index()

# === 取前 N / 後 N ===
df.nlargest(10, "amount")                    # 金額前 10
df.nsmallest(5, "score")                     # 分數最低 5

# ⚠️ sort_values 預設不會改 df、要存回去：
df = df.sort_values("amount")
# 或加 inplace=True（但業界趨勢是不用 inplace）
```

  
### 3. GroupBy（樞紐分析核心、最重要的操作）

**`groupby` = SQL 的 GROUP BY、按某欄分組、然後聚合計算。**

```text
# === 各國銷售總額（最基本）===
df.groupby("country")["amount"].sum()
# Output 是 Series（index = country、value = sum）

# === 多欄分組 + 多種統計（最常用）===
df.groupby(["country", "category"]).agg({
    "amount": ["sum", "mean", "count"],      # 一欄算 3 種
    "user_id": "nunique"                     # nunique = 不重複數
})

# === named aggregation（推薦、欄位名清楚）===
df.groupby("country").agg(
    total_revenue=("amount", "sum"),
    avg_order=("amount", "mean"),
    customer_count=("user_id", "nunique")
)
# 結果欄位名是 total_revenue / avg_order / customer_count、超清楚
```

  
**Transform（取得 group 內排名 / 標準化、進階用法）：**

```text
# === 每個國家內、銷售排名 ===
df["rank"] = df.groupby("country")["amount"].transform(
    lambda x: x.rank(ascending=False)
)

# === 每個國家內、銷售 z-score（找各國的 outlier）===
df["amount_zscore"] = df.groupby("country")["amount"].transform(
    lambda x: (x - x.mean()) / x.std()
)
# zscore > 2 = 該國的離群銷售

# === transform vs agg 差別 ===
# agg：每組回一個數（縮短 df）
# transform：每列回對應的 group 計算結果（df 長度不變、新欄位）
```

  
### 4. Merge（合併、相當於 SQL JOIN）

```text
# === inner join（最常用、兩邊都有才留）===
result = pd.merge(orders, users, on="user_id", how="inner")

# === left join（保留左邊全部、右邊配不上 → NaN）===
result = pd.merge(orders, users, on="user_id", how="left")

# === right / outer join ===
result = pd.merge(orders, users, on="user_id", how="right")  # 反向
result = pd.merge(orders, users, on="user_id", how="outer")  # 全留

# === 欄位名不同時：left_on / right_on ===
result = pd.merge(
    orders, users,
    left_on="user_id",                       # orders 內叫 user_id
    right_on="id"                            # users 內叫 id
)

# === 多欄 join（用 list）===
result = pd.merge(a, b, on=["date", "store_id"])
```

> 💡 **效能提示**：大資料 merge 前、把 join key 設為 index 會快很多（`df.set_index("user_id")`）。

  
### 5. Pivot / Melt（樞紐表 + 反向、Excel 用 Pandas 寫）

**Pivot Table（樞紐表、長變寬）：**

```text
pivot = pd.pivot_table(
    df,
    values="amount",                         # 算什麼
    index="month",                           # 列
    columns="category",                      # 欄
    aggfunc="sum",                           # 聚合方式（sum / mean / count）
    fill_value=0                             # 缺值填什麼
)

# 結果類似 Excel pivot：
#         電子    服裝    食品
# month
# 1月     50000   30000   20000
# 2月     45000   35000   18000
```

  
**Melt（pivot 反向、寬變長、ML 常用）：**

```text
melted = pd.melt(
    df,
    id_vars=["id"],                          # 不動的欄
    value_vars=["jan", "feb", "mar"],        # 要 melt 的欄
    var_name="month",                        # 新的「欄位名」欄
    value_name="sales"                       # 新的「值」欄
)

# 原本：id | jan | feb | mar
# 變成：id | month | sales
#        1 | jan  | 100
#        1 | feb  | 200
```

> 💡 **何時用 melt？** ML 需要「**長表**」（每列一筆觀察）、但你拿到的是「**寬表**」（每月一欄）、用 melt 轉換。

  
### 其他常用操作（加碼）

**新增欄位（衍生欄）：**

```text
# === 直接賦值 ===
df["total"] = df["price"] * df["quantity"]
df["category_upper"] = df["category"].str.upper()

# === apply + lambda（適合簡單邏輯）===
df["tier"] = df["amount"].apply(
    lambda x: "high" if x > 1000 else "low"
)

# === np.where（單條件、比 apply 快很多）===
import numpy as np
df["tier"] = np.where(df["amount"] > 1000, "high", "low")

# === np.select（多條件）===
df["grade"] = np.select(
    condlist=[df["score"] >= 90, df["score"] >= 60],
    choicelist=["A", "B"],
    default="F"
)
```

  
**刪除：**

```text
df.drop("col_name", axis=1)                  # 刪欄（axis=1 是欄）
df.drop(index=[0, 5])                        # 刪指定列
df.drop_duplicates()                         # 整列重複的去掉
df.drop_duplicates(subset=["email"])         # 只看 email 欄是否重複
df.dropna(subset=["email"])                  # email 為 NaN 的整列刪
```

  
**填補缺值：**

```text
df["age"].fillna(df["age"].mean(), inplace=True)    # 數值欄填平均
df["age"].fillna(df["age"].median(), inplace=True)  # 填中位數（更穩、不被 outlier 影響）
df["category"].fillna("unknown", inplace=True)       # 字串填預設
df = df.ffill()                                      # 用前一列填（時序常用）
```

  
### 速查表：Pandas 操作 vs SQL

| Pandas | SQL |
|---|---|
| `df.head()` | `SELECT * LIMIT 5` |
| `df[df["age"] > 25]` | `WHERE age > 25` |
| `df.sort_values("age")` | `ORDER BY age` |
| `df.groupby("city")["age"].mean()` | `GROUP BY city, AVG(age)` |
| `pd.merge(a, b, on="id")` | `JOIN a, b ON a.id = b.id` |
| `df.drop_duplicates()` | `SELECT DISTINCT` |
| `df.nlargest(10, "amount")` | `ORDER BY amount DESC LIMIT 10` |
| `df["city"].value_counts()` | `GROUP BY city, COUNT(*)` |
| `pd.pivot_table(...)` | `PIVOT`（SQL Server 語法） |

  
    **☕ 用人話講**
    
Pandas 5 大操作背起來涵蓋 90% 工作：(1) **Filter** `df[條件]` (2) **Sort** `df.sort_values()` (3) **GroupBy** `df.groupby().agg()`（最重要） (4) **Merge** `pd.merge(a, b, on='id')`（像 SQL JOIN） (5) **Pivot** `pd.pivot_table()`（像 Excel 樞紐）。**進階加碼**：`transform`（group 內排名）、`np.where`（單條件）、`np.select`（多條件）。**怎麼練習**：拿真實資料、把每個操作各用一次、最後試「用同樣資料寫 SQL + Pandas 各一次」對照看。
