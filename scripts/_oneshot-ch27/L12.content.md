|---|---|
| **純 Python** | 不用碰 HTML / CSS / JS、寫 Python 像寫腳本 |
| **5 分鐘上手** | Hello World 到 dashboard 只要 30 行 |
| **內建 widget** | selectbox / slider / button 一行寫完 |
| **部署一鍵** | Streamlit Community Cloud 免費部署 |
| **接案神器** | 客戶要 dashboard、Streamlit 1 天交付 |

### 基本元件（背 10 個涵蓋 80% 需求）

**文字 / 標題：**

```text
import streamlit as st

st.title("標題")                              # 最大標題
st.header("子標題")
st.subheader("子子標題")
st.markdown("**粗體** *斜體*")                # 支援完整 Markdown
st.code("print('hello')", language="python")  # 程式碼 block
```

  
**資料展示：**

```text
st.dataframe(df)                              # 互動表格（可排序 / scroll）
st.table(df)                                  # 靜態表格
st.json({"key": "value"})                     # JSON 漂亮顯示
st.metric("營收", "NT$ 1.2M", "+12%")        # 數字 + 變化（KPI 卡）
```

  
**圖表：**

```text
st.line_chart(df)                             # Streamlit 內建快速折線
st.bar_chart(df)                              # 內建長條
st.plotly_chart(fig)                          # 嵌 Plotly 圖（L11）

```

  
**媒體：**


**選擇類：**

**Columns（左右分欄、KPI 必用）：**

**Tab（分頁、內容多時用）：**

```text

with tab1:
    st.line_chart(df)

with tab2:
    st.dataframe(df)
```

  
**Sidebar（側欄、放篩選 widget）：**

```text
with st.sidebar:
    country = st.selectbox("國家", df["country"].unique())

# 用 sidebar 的選擇來過濾資料
filtered = df[df["country"] == country]
st.dataframe(filtered)
```

  
### 進階功能

**1. Session State（跨 rerun 保留狀態）：**

> 💡 **Streamlit 特性**：使用者每次互動、整個 script 都會「**從頭跑一次**」（rerun）。所以「計數器」之類的狀態要存進 `session_state`。

```text
# === 初始化（只在第一次）===
if "count" not in st.session_state:
    st.session_state.count = 0

# === 互動 ===
if st.button("+1"):
    st.session_state.count += 1

st.write(f"count: {st.session_state.count}")
```

  
**2. Cache（避免重算、效能必用）：**

```text
@st.cache_data
def load_data():
    return pd.read_csv("big.csv")            # 假設要 5 秒

df = load_data()                              # 第一次跑 5 秒、第二次秒回
# Streamlit 自動快取結果、參數沒變不重算
```

  
**3. Form（整批送出、不要每改一個都 rerun）：**

```text
with st.form("my_form"):
    name = st.text_input("姓名")
    age = st.number_input("年齡")
    submitted = st.form_submit_button("送出")
    if submitted:
        st.write(f"Hi {name}, you are {age}")
# form 內的 widget 改值不會 rerun、按 submit 才一次處理
```

  
### 實戰範例：銷售 Dashboard（30 行寫完）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 **寫進 app.py**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
import streamlit as st
import pandas as pd
import plotly.express as px

# === 頁面設定（要放最前面）===
st.set_page_config(page_title="銷售 Dashboard", layout="wide")
st.title("📊 2026 銷售分析")

# === 載資料（快取）===
@st.cache_data
def load():
    return pd.read_csv("sales.csv")

df = load()

# === Sidebar 篩選 ===
with st.sidebar:
    countries = st.multiselect(
        "國家",
        df["country"].unique(),
        default=df["country"].unique()        # 預設全選
    )
    date_range = st.date_input(
        "日期區間",
        value=[df["date"].min(), df["date"].max()]
    )

filtered = df[df["country"].isin(countries)]

# === 3 個 KPI 卡 ===
col1, col2, col3 = st.columns(3)
col1.metric("總營收", f"NT$ {filtered['amount'].sum():,.0f}")
col2.metric("訂單數", len(filtered))
col3.metric("客單價", f"{filtered['amount'].mean():.0f}")

# === Plotly 圖 ===
fig = px.bar(
    filtered.groupby("month")["amount"].sum().reset_index(),
    x="month", y="amount",
    title="月營收"
)
st.plotly_chart(fig, use_container_width=True)        # 寬度跟容器一致

# === 表格 ===
st.dataframe(filtered)
```

→ **30 行寫完一個完整 dashboard、含篩選 / KPI / 圖 / 表**
