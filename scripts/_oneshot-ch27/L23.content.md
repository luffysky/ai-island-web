**Geo 分析（地理空間分析）= 「**讓資料跟地圖結合**」**——外送 App / 房仲 / 物流 / 政府都用。Python 4 件套：**Folium 地圖 / GeoPandas 空間 / Geopy 距離 / Plotly mapbox**。

> 💡 **何時用？** 任何「**跟位置有關**」的分析：方圓 X km / 區域統計 / 路徑規劃 / 熱力圖。

  
### 4 大典型應用場景

```text
1. 「最近的 5 家店」          → 距離計算 + 排序
2. 「方圓 1km 銷售統計」      → 空間範圍 query
3. 「各區房價熱力圖」          → Heatmap
4. 「客戶分布地圖」            → Marker 視覺化
```

  
### Folium（互動地圖、leaflet.js Python 封裝）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 Folium）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add folium
```

**這行做什麼？** 裝 Folium、底層用 leaflet.js（Web 地圖標準）、輸出互動 HTML。

  
**基本地圖：**

```text
import folium

# === 建立地圖（中心 + zoom level）===
m = folium.Map(
    location=[25.04, 121.5],                 # [緯度, 經度]
    zoom_start=12                            # 1-20、數字越大越近
)
m.save("map.html")                            # 存成 HTML、瀏覽器打開能互動
```

  
**加 marker（點 + popup）：**

```text
folium.Marker(
    [25.04, 121.5],
    popup="台北 101",                         # 點擊跳出
    tooltip="點我",                           # hover 顯示
    icon=folium.Icon(color="green")
).add_to(m)

# === 加圓圈（範圍視覺化）===
folium.Circle(
    location=[25.04, 121.5],
    radius=500,                               # 單位：公尺
    color="red",
    fill=True
).add_to(m)
```

  
### 從 DataFrame 加多個 markers（真實場景）

```text
import pandas as pd

df = pd.DataFrame({
    "lat": [25.04, 25.05, 25.03],
    "lng": [121.5, 121.51, 121.49],
    "name": ["A 店", "B 店", "C 店"],
    "sales": [100, 200, 150]
})

m = folium.Map(location=[25.04, 121.5], zoom_start=13)

for _, row in df.iterrows():
    folium.CircleMarker(
        [row["lat"], row["lng"]],
        radius=row["sales"] / 10,             # 點大小代表銷售（視覺化第三維）
        popup=f"{row['name']}: NT$ {row['sales']}",
        color="green",
        fill=True
    ).add_to(m)

m.save("stores.html")
```

  
### Heatmap（熱力圖、看分布密度）

```text
from folium.plugins import HeatMap

m = folium.Map(location=[25.04, 121.5], zoom_start=12)

# 三欄：緯度 / 經度 / 權重（用 sales）

```

  
### Marker Cluster（大量點、自動聚合）

```text
from folium.plugins import MarkerCluster
```

### GeoPandas（空間查詢、Pandas 的地理版）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 GeoPandas）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add geopandas
```

**這行做什麼？** 裝 GeoPandas、Pandas 的地理版、能做空間 JOIN / 距離計算 / 範圍 query。

  
```text
import geopandas as gpd
from shapely.geometry import Point

# === 從 DataFrame 建 GeoDataFrame ===
df["geometry"] = df.apply(
    lambda r: Point(r["lng"], r["lat"]),     # ⚠️ Point(經度, 緯度) 順序!
    axis=1
)
gdf = gpd.GeoDataFrame(
    df,
    geometry="geometry",
    crs="EPSG:4326"                          # WGS84 經緯度（最常用）
)

# === 載入 shapefile（區域邊界）===
districts = gpd.read_file("taipei_districts.shp")

# === 空間 JOIN：判斷每個店在哪一區 ===
result = gpd.sjoin(
    gdf, districts,
    how="left",
    predicate="within"                       # 點在多邊形內
)
# 之後 result 有「district_name」欄、能做「各區銷售統計」
```

  
### 距離計算（geopy）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**（裝 geopy）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add geopy
```

**這行做什麼？** 裝 geopy、Python 標準地理計算套件（距離 / geocoding）。

  
```text
from geopy.distance import distance

# === 兩點距離（公里）===
dist = distance((25.04, 121.5), (25.05, 121.51)).km
# 0.7 公里

# === 套到整個 DataFrame ===
def calc_dist(row):
    return distance((25.04, 121.5), (row["lat"], row["lng"])).km

df["dist_from_101"] = df.apply(calc_dist, axis=1)
# 每筆資料加「距 101 多少 km」欄
```

  
### 範圍內查詢（找方圓 X km 內的東西）

```text
from shapely.geometry import Point

target = Point(121.5, 25.04)                  # 經度, 緯度
gdf["dist"] = gdf.distance(target) * 111      # 約轉成 km（粗略）

nearby = gdf[gdf["dist"] < 1]                 # 1 km 內
```

> 💡 **精準距離用 geopy**（球面距離）、Shapely 的 `.distance()` 是「**平面距離**」、小範圍能用、大範圍誤差大。

  
### Geocoding（地址 ↔ 經緯度）

```text
from geopy.geocoders import Nominatim

geocoder = Nominatim(user_agent="my-app")    # 要寫 user_agent

# === 地址 → 經緯度 ===
location = geocoder.geocode("台北市信義區信義路五段 7 號")
print(location.latitude, location.longitude)
# 25.0339, 121.5645

# === 反向：經緯度 → 地址 ===
location = geocoder.reverse((25.04, 121.5))
print(location.address)
```

> ⚠️ **Nominatim 免費但有 rate limit**（1 秒 1 次）、量大用 Google Maps API（付費）。

```text
fig = px.scatter_mapbox(

fig.show()
# 比 Folium 互動更流暢、適合 dashboard
```
