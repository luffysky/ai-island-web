### 步驟 0：環境設定 + 載入套件

```text
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# === 中文字型（L5 教過、忘記會豆腐字）===
plt.rcParams['font.family'] = ['PingFang TC']    # Mac
# Windows 用 'Microsoft JhengHei'

# === seaborn 全域樣式 ===
sns.set_theme(style='whitegrid', palette='Set2')
```

  
### Step 1：讀資料

```text
# parse_dates 自動把指定欄位轉成 datetime（L6 教過）
df = pd.read_csv('sales_2026Q1.csv', parse_dates=['order_date'])

print(f'資料筆數：{len(df)}')
print(df.info())                              # 看欄位 / 型別 / 缺值
```

> 💡 **第一步永遠先 `df.info()`** —— 知道有幾欄、什麼型別、哪裡缺值、之後 cleaning 才有方向。

  
### Step 2：資料清洗（cleaning）

**清洗 4 動作：缺值 / 型別 / 異常值 / 衍生欄位**：

```text
# === 1. 處理缺值（L4 教過）===
df['discount'] = df['discount'].fillna(0)     # 折扣 NaN 視為 0
df = df.dropna(subset=['customer_id'])        # 沒 customer_id 的整列刪（必要欄位）

# === 2. 型別轉換（防止字串混進數字欄）===
df['amount'] = pd.to_numeric(df['amount'], errors='coerce')
# errors='coerce' = 無法轉的變 NaN、不會 raise

# === 3. 移除異常值（負金額不合理）===
df = df[df['amount'] > 0]

# === 4. 加衍生欄位（feature engineering、L21 詳教）===
df['month'] = df['order_date'].dt.month
df['weekday'] = df['order_date'].dt.day_name()
df['is_weekend'] = df['order_date'].dt.dayofweek >= 5

# 用 pd.cut 把連續金額切成價格層級（L4 教過）
df['price_tier'] = pd.cut(
    df['amount'],
    bins=[0, 500, 2000, 10000, np.inf],
    labels=['低', '中', '高', '豪華']
)
```

  
### Step 3：探索性分析（EDA）—— 看資料的「故事」

**整體狀況（先有大局觀）：**

```text
print(f'總營業額：NT$ {df["amount"].sum():,.0f}')
print(f'平均客單價：NT$ {df["amount"].mean():,.0f}')
print(f'訂單數：{len(df):,}')
print(f'客戶數：{df["customer_id"].nunique():,}')         # nunique = 不重複數量
```

  
**各類別表現（GroupBy + agg、L4 教過）：**

```text
category_stats = df.groupby('category').agg(
    revenue=('amount', 'sum'),               # 營收
    orders=('order_id', 'count'),            # 訂單數
    avg_value=('amount', 'mean')             # 平均客單價
).sort_values('revenue', ascending=False)
print(category_stats)
```

  
**Top 10 客戶（VIP 找出來）：**

```text
top_customers = (
    df.groupby('customer_id')['amount']
      .sum()
      .nlargest(10)                          # nlargest = 取最大的 N 個
)
print('Top 10 VIP 客戶：', top_customers)
```

> 💡 **80/20 法則**：通常 20% VIP 客戶帶來 80% 營收、用 `nlargest` 一秒找出來、行銷可重點維繫這群人。

### EDA 流程口訣（背起來）

```text
1. Read    → pd.read_csv / read_excel
2. Inspect → df.info() / describe() / head()
3. Clean   → fillna / dropna / 型別 / 異常值 / 衍生欄
4. Explore → groupby / agg / nlargest / 相關係數
5. Visualize → 直方 / 散點 / 折線 / 熱力圖
6. Report  → 結論 + 建議 + 匯出
```
