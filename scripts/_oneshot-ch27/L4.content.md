### 核心物件：DataFrame + Series

```text
import pandas as pd                          # 慣例暱稱 pd

# === DataFrame = 二維表（多列多欄）===
df = pd.DataFrame({
    'name': ['小明', '小華', '小美'],
    'age': [25, 30, 28],
    'city': ['台北', '高雄', '台中']
})

# === Series = 一維（單欄、或一列）===
ages = df['age']                             # Series（一欄）
first_row = df.iloc[0]                       # Series（一列）
```

  
### 8 大核心操作（資料分析每天用）

**1. 讀資料（多格式都支援）：**

```text
df = pd.read_csv('sales.csv', encoding='utf-8')      # CSV（最常用、中文用 utf-8）
df = pd.read_excel('data.xlsx', sheet_name='2026Q1') # Excel
df = pd.read_sql('SELECT * FROM orders', engine)     # 從 DB 撈（搭 SQLAlchemy）
df = pd.read_json('data.json')                       # JSON
df = pd.read_csv('https://example.com/data.csv')     # 直接讀 URL（爬蟲常用）
df = pd.read_parquet('data.parquet')                 # 大資料 Parquet 格式
```

  
**2. 看資料（第一步必做）：**

```text
df.head()                                    # 前 5 列（最常用）
df.tail(10)                                  # 後 10 列
df.sample(5)                                 # 隨機抽 5 列
df.shape                                     # (列數, 欄數) tuple
df.info()                                    # 欄位 + 型別 + 缺值數量
df.describe()                                # 數值欄統計摘要（mean/std/min/max/quartile）
df.dtypes                                    # 每欄型別
```

  
**3. 選取 + 過濾（最常用、要熟練）：**

```text
# === 單 / 多欄 ===
df['name']                                   # 一欄、回 Series
df[['name', 'age']]                          # 多欄、回 DataFrame

# === 條件過濾 ===
df[df['age'] > 25]                           # 年齡 > 25 的
df[(df['age'] > 25) & (df['city'] == '台北')]  # 多條件用 & 不是 and
df[df['city'].isin(['台北', '台中'])]         # 在這幾個城市
df[df['name'].str.contains('明')]            # 字串包含「明」

# === 精準選取（loc 用 label、iloc 用位置）===
df.loc[0:2, 'name']                          # 第 0-2 列的 name 欄
df.iloc[0:3, 1:3]                            # 第 0-2 列、第 1-2 欄

# === SQL 風格查詢（query、可讀性高）===
df.query('age > 25 and city == "台北"')
```

  
**4. 處理缺值（NaN、實戰必懂）：**

```text
df.isna().sum()                              # 每欄有幾個缺值
df.dropna()                                  # 任一欄缺就刪整列
df.dropna(subset=['age'])                    # 只看 age 欄是否缺
df.fillna(0)                                 # 缺值全填 0
df.fillna(df.mean())                         # 數值欄填平均
df.interpolate()                             # 內插（時序資料常用、用前後平均）
```

  
**5. GroupBy（這是 Pandas 最重要的功能、SQL 的 GROUP BY）：**

```text
# === 各城市平均年齡 ===
df.groupby('city')['age'].mean()

# === 多種聚合一次算 ===
df.groupby('city').agg({
    'age': ['mean', 'max'],                  # age 算平均 + 最大
    'salary': ['sum', 'count']               # salary 算總和 + 次數
})

# === 樞紐分析（像 Excel pivot table）===
df.pivot_table(
    values='sales',
    index='month',
    columns='product',
    aggfunc='sum'
)
# 結果：列=月份、欄=產品、值=銷售總和
```

  
**6. 合併（JOIN、橫 / 縱向）：**

```text
# === 縱向疊起來（垂直、增列）===
pd.concat([df1, df2])

# === 橫向 JOIN（像 SQL JOIN、增欄）===
pd.merge(
    orders, users,
    on='user_id',                            # 用 user_id 對應
    how='left'                               # left / inner / right / outer
)
```

  
**7. 新增欄位（衍生新特徵）：**

```text
# === 直接賦值 ===
df['is_adult'] = df['age'] >= 18             # boolean 欄

# === pd.cut 分組（連續變類別、年齡分群超常用）===
df['age_group'] = pd.cut(
    df['age'],
    bins=[0, 18, 35, 60, 100],
    labels=['兒童', '青年', '壯年', '老年']
)

# === apply（每列跑自訂函式）===
df['name_upper'] = df['name'].apply(lambda x: x.upper())
df['salary_k'] = df['salary'].apply(lambda x: f"{x/1000:.0f}K")
```

  
**8. 輸出（分析完存檔給人 / 別系統用）：**

```text
df.to_csv('output.csv', index=False)         # CSV、不要寫多餘 index 欄
df.to_excel('output.xlsx', sheet_name='Q1')  # Excel
df.to_sql('orders', engine, if_exists='append')  # 寫進 DB
df.to_json('output.json', orient='records', force_ascii=False)  # JSON（中文不 escape）
df.to_parquet('output.parquet')              # 大資料 Parquet
```

  
### 速查表：Pandas 常用方法對照 SQL

| Pandas | SQL |
|---|---|
| `df.head()` | `SELECT * LIMIT 5` |
| `df[['name', 'age']]` | `SELECT name, age` |
| `df[df['age'] > 25]` | `WHERE age > 25` |
| `df.groupby('city')['age'].mean()` | `GROUP BY city、SELECT AVG(age)` |
| `df.sort_values('age')` | `ORDER BY age` |
| `pd.merge(a, b, on='id')` | `JOIN a, b ON a.id = b.id` |
| `df.drop_duplicates()` | `SELECT DISTINCT` |
| `df['city'].value_counts()` | `GROUP BY city、SELECT count(*)` |

  
    **☕ 用人話講**
    
Pandas 是「**Excel 的程式版**」、處理 10 萬列也順。**8 大核心操作**背起來：(1) **讀資料** `pd.read_csv()` (2) **看資料** `df.head() / info() / describe()` (3) **選取** `df[條件]` (4) **缺值** `fillna() / dropna()` (5) **GroupBy** `groupby + agg` (6) **合併** `merge / concat` (7) **新增欄位** `df[新欄] = 算式` (8) **輸出** `to_csv()`。**怎麼練習**：Kaggle 抓任何 CSV 資料集、開 Jupyter、把這 8 個操作各跑一次。
