> 💡 **資料分析師的價值在「圖會說話」**——不只「畫出來」、要「畫得讓老闆秒懂 + 自己回看不困惑」。

  
### matplotlib 基礎 + 中文設定

```text
import matplotlib.pyplot as plt

# === 中文字體（解決豆腐字、L5 強調過）===
plt.rcParams["font.family"] = "Noto Sans CJK TC"     # Mac / Linux
plt.rcParams["axes.unicode_minus"] = False           # 負號正常顯示

# === 線圖（最基本）===
plt.plot(x, y)
plt.title("月銷售")
plt.xlabel("月份")
plt.ylabel("營收")
plt.savefig("plot.png", dpi=300)                     # 存檔
plt.show()
```

  
### 5 種基本圖（L5 補充版）

**Line（趨勢、時序首選）：**

```text
plt.plot(months, sales, marker="o")                  # marker="o" 每點畫圓
plt.plot(months, sales, linestyle="--", color="red") # 虛線紅色
```

**Bar（分類比較）：**

```text
plt.bar(categories, values)                          # 垂直
plt.barh(categories, values)                         # 水平（類別名太長用這個）
```

**Scatter（兩變數關係）：**

```text
plt.scatter(
    x, y,
    s=sizes,         # 點大小（依某欄）= bubble chart
    c=colors,        # 點顏色（依某欄）
    alpha=0.5        # 透明度 0-1
)
```

**Histogram（分布）：**

```text
plt.hist(data, bins=30)                              # 30 個區間
```

**Pie（少用、面積誤導性高）：**

```text
plt.pie(values, labels=labels, autopct="%.1f%%")     # %.1f%% 顯示百分比 1 位小數
```

> 💡 **業界共識：圓餅圖能不用就不用**——人眼比較「角度」很爛、用橫向 bar chart 永遠更清楚。

  
### 子圖（一張畫布多 chart）

```text
fig, axes = plt.subplots(2, 2, figsize=(12, 8))      # 2x2 排版

axes[0, 0].plot(x, y)
axes[0, 1].bar(categories, values)
axes[1, 0].scatter(x, y)
axes[1, 1].hist(data)

plt.tight_layout()                                    # 自動調間距、避免重疊
plt.show()
```

  
### seaborn 進階：8 種統計圖

```text
import seaborn as sns

# === Bar（自動算平均 + error bar）===
sns.barplot(data=df, x="category", y="amount")

# === Box plot（看分布 + 中位數 + 離群值）===
sns.boxplot(data=df, x="country", y="income")

# === Violin（box + density、看分布形狀）===
sns.violinplot(data=df, x="country", y="income")

# === Heatmap（相關矩陣、看哪兩欄相關）===
sns.heatmap(
    df.corr(),
    annot=True,                                       # 每格顯示數字
    cmap="coolwarm",                                  # 冷暖色
    fmt=".2f"                                         # 數字格式：2 位小數
)

# === Scatter + 回歸線（看趨勢）===
sns.regplot(data=df, x="price", y="rating")

# === Pair plot（多變數關係矩陣、EDA 神器）===
sns.pairplot(df[["age", "income", "spending"]])

# === Distribution（直方圖 + KDE 密度曲線）===
sns.histplot(df["age"], kde=True)
```

  
### 樣式設定（讓圖看起來更專業）

```text
# === 全域風格 ===
sns.set_theme(style="whitegrid")                     # 白底 + 灰格線
sns.set_palette("husl")                              # 換調色盤

# === 自訂顏色（用品牌色）===
colors = ["#50fa7b", "#8be9fd", "#ff79c6"]
sns.barplot(data=df, x="cat", y="val", palette=colors)
```

  
### 進階：故事化圖（讓圖會說話）

**加註解（annotation、標重要事件）：**

```text
plt.plot(months, sales)
plt.annotate(
    "iPhone 17 發表",                                # 註解文字
    xy=(7, 50000),                                   # 指向哪個點
    xytext=(5, 60000),                               # 文字位置
    arrowprops={"arrowstyle": "->"}                  # 箭頭樣式
)
# 一張圖立刻有「故事感」、老闆秒懂為什麼那個月飆升
```

  
**加水平線（基準 / 目標）：**

```text
plt.axhline(
    y=target,
    color="red",
    linestyle="--",
    label="目標"
)
plt.legend()
# 視覺一秒看出「哪幾個月有達標」
```

  
**突出重點（達標綠色 / 未達灰色）：**

```text
colors = ["#50fa7b" if v > target else "#cccccc" for v in values]
plt.bar(categories, values, color=colors)
# 讀者眼睛立刻被吸引到綠色（達標）那幾根
```

  
### 5 大視覺化工具對比

| 工具 | 適合 | 學習曲線 |
|---|---|---|
| **matplotlib** | 細節控制、客製、出版品質 | 中（API 較囉嗦） |
| **seaborn** | 統計圖、預設超美、EDA 首選 | 低 |
| **Plotly** | 互動式、嵌網頁、dashboard | 中（L11 詳教） |
| **Bokeh** | 互動式 + 大資料（百萬點不卡） | 中高 |
| **Altair** | 宣告式語法、Grammar of Graphics | 中（學術界喜歡） |

  
### 出版品質（給論文 / 簡報用、調到完美）

```text
fig, ax = plt.subplots(figsize=(8, 5), dpi=300)      # dpi=300 = 印刷品質

ax.plot(x, y, linewidth=2, color="#50fa7b")
ax.set_xlabel("Month", fontsize=12)
ax.set_ylabel("Revenue (NT$)", fontsize=12)
ax.set_title("2026 月營收", fontsize=14, fontweight="bold")
ax.grid(alpha=0.3)                                   # 淺淺格線

# === 隱藏右邊 / 上邊邊框（更乾淨）===
ax.spines["top"].set_visible(False)
ax.spines["right"].set_visible(False)

plt.savefig("publication.pdf", bbox_inches="tight")  # PDF 向量、印不糊
```

  
    **☕ 用人話講**
    
matplotlib + seaborn 兩件套涵蓋 90% 靜態圖需求。**3 個進階心法**：(1) **annotation 加註解** —— 一張圖有故事 (2) **axhline 加基準** —— 有達標 / 未達標對比 (3) **顏色突出重點** —— 達標綠色、其他灰色、讀者眼睛自動聚焦。**互動式做 dashboard** → 跳 Plotly（L11）。**論文 / 印刷** → matplotlib + PDF。**怎麼練習**：找一張你做過的圖、用這課技巧重做一次、比較前後差異。
