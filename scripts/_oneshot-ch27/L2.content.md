**Jupyter Notebook** = 資料分析師的「**主場 IDE**」。寫一段 code 立刻看結果 + 畫圖 + 寫筆記、適合「**邊探索邊思考**」的分析工作。

> 💡 **跟 .py 檔最大差別**：Jupyter 是「**cell（細胞 / 段落）為單位**」、每個 cell 獨立執行、可以隨時往回改、變數保留在記憶體。寫 .py 是「整個檔案跑一次」、Jupyter 是「一段一段跑」。

  
### 4 種使用方式（怎麼選？）

| 方式 | 特色 | 適合 |
|---|---|---|
| **Jupyter Lab**（本地、推薦） | 完整環境、多視窗 / 檔案總管 | 本機開發、長期專案 |
| **Google Colab**（雲端） | **免費 + 免費 GPU**、零安裝 | 學習 / 跑 ML / 借 GPU |
| **VS Code + Jupyter 擴充** | 在 VS Code 寫 .ipynb、整合 Git | 已用 VS Code 的人 |
| **JupyterHub**（多人共用） | 公司 / 學校共用伺服器 | 團隊 / 教學 |

### Cell 基本概念（Jupyter 核心）

**Jupyter 把程式分成「cell（細胞）」、每個 cell 獨立跑、結果立刻顯示在下方。**

```text
=== 3 種 cell 類型 ===
1. Code cell（程式碼）—— 跑 Python、結果顯示下方
2. Markdown cell（文字）—— 寫說明 / 標題 / 圖片
3. Raw cell（原始文字）—— 純文字、不解析

=== Cell 執行順序 ===
- Cell 可以「不照順序」跑（左上角 [數字] 是執行順序）
- 變數會「保留在記憶體」、上面 cell 定義的、下面 cell 看得到
- ⚠️ 注意：跳著跑容易亂、結束前最好「Restart Kernel + Run All」一次確認
```

  
### 必背快捷鍵（學會省 50% 時間）

| 快捷鍵 | 動作 |
|---|---|
| **`Shift + Enter`** | 執行當前 cell + 跳下一個 cell（最常用） |
| **`Ctrl + Enter`** | 執行當前 cell + 留原處（一直改一直跑用） |
| **`A`** | 在當前 cell **上方**插入新 cell（command mode 下） |
| **`B`** | 在當前 cell **下方**插入新 cell |
| **`DD`**（按 2 下 D） | 刪除當前 cell |
| **`M`** | 切到 Markdown cell |
| **`Y`** | 切到 Code cell |
| **`Shift + Tab`** | 看函式說明（cursor 放函式名上） |
| **`Ctrl + /`** | 註解 / 取消註解選取行 |
| **`Esc`** | 從編輯模式切到 command mode（用快捷鍵前要先 Esc） |

  
### Magic Commands（Jupyter 專屬超能力、必學）

**Magic command 是 Jupyter 特有的「**`%` 或 `%%` 開頭**」指令、做超出純 Python 的事。**

```text
# === %%time：計時這個 cell ===
%%time
result = [x**2 for x in range(1_000_000)]
# 跑完顯示：Wall time: 0.32 s

# === %whos：看當前定義的所有變數 ===
%whos
# Variable  Type    Data/Info
# x         int     42
# df        DataFrame  100 rows x 5 columns

# === !指令：跑 shell（驚嘆號開頭）===
!pip install plotly                        # 在 Jupyter 內裝套件
!ls -la                                    # 看當前資料夾檔案
!cat data.csv                              # 看檔案內容

# === %load_ext autoreload + %autoreload 2 ===
%load_ext autoreload
%autoreload 2
# 改自己寫的 .py 模組、Jupyter 自動重 import、不用重啟 kernel
# 開發時超好用

# === %matplotlib inline（畫圖內嵌、新版預設）===
%matplotlib inline                         # 圖直接顯示在 cell 下方
# 或 %matplotlib widget（互動式圖）
```

### 分析工作流（用 Jupyter 的標準流程）

```text
=== 1. 探索（exploration）===
- 載資料：df = pd.read_csv(...)
- 看前 5 列：df.head()
- 看 shape / dtype / 缺失：df.info(), df.isnull().sum()
- 統計總覽：df.describe()

=== 2. 清理（cleaning）===
- 處理缺失值（L20）
- 處理重複
- 處理異常值

=== 3. 視覺化（visualization）===
- 直方圖看分布：sns.histplot(df, x='age')
- 散佈圖看關係：sns.scatterplot(df, x='age', y='salary')
- 熱力圖看相關：sns.heatmap(df.corr())

=== 4. 假設驗證（hypothesis）===
- A/B test（L16）
- 統計檢定

=== 5. 報告（report）===
- Markdown cell 寫結論
- 匯出 PDF / HTML（File → Export Notebook）
- 給老闆 / 客戶看的「會說話」報告
```
