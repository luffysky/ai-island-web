## 中文分詞（基礎）
英文用空白分、中文要先「斷詞」。

### jieba（最熱門）
```bash
pip install jieba
```

```python
import jieba

text = "我愛吃台灣牛肉麵和滷肉飯"
words = list(jieba.cut(text))
print(words)
# ['我', '愛', '吃', '台灣', '牛肉麵', '和', '滷肉飯']

# 自訂詞典
jieba.add_word("AI 島")
jieba.del_word("不要的詞")

# 載入外部詞典
jieba.load_userdict("custom_dict.txt")

# 詞性標註
import jieba.posseg
words = jieba.posseg.cut(text)
for w in words:
    print(w.word, w.flag)
```

## TF-IDF（重要詞抽取）
```python
from sklearn.feature_extraction.text import TfidfVectorizer
import jieba

docs = [
    "我愛 Python",
    "Python 是程式語言",
    "Python 適合資料分析"
]

# 中文要先斷詞
def tokenize(text):
    return list(jieba.cut(text))

vec = TfidfVectorizer(tokenizer=tokenize)
tfidf = vec.fit_transform(docs)
print(vec.get_feature_names_out())
```

## 情感分析

### 用預訓練模型（推薦）
```bash

```

```python
from transformers import pipeline

# 中文情感模型
classifier = pipeline(
    "sentiment-analysis",
    model="uer/roberta-base-finetuned-jd-binary-chinese"
)

results = classifier([
    "這家店超好吃！",
    "服務態度很差、再也不來"
])
# [{label: positive, score: 0.99}, {label: negative, score: 0.95}]
```

## 主題建模（LDA）
```python
from sklearn.decomposition import LatentDirichletAllocation

# tfidf → LDA
lda = LatentDirichletAllocation(n_components=5, random_state=42)
lda.fit(tfidf)

# 看每個主題的關鍵詞
for i, topic in enumerate(lda.components_):
    top_words = [vec.get_feature_names_out()[j] for j in topic.argsort()[-10:]]
    print(f"Topic {i}: {top_words}")
```

## Embedding + 相似度
```python
from sentence_transformers import SentenceTransformer
from sklearn.metrics.pairwise import cosine_similarity

model = SentenceTransformer("uer/sbert-base-chinese-nli")

texts = ["我愛吃牛肉麵", "我喜歡牛肉麵", "Python 很棒"]
embeddings = model.encode(texts)

# 計算兩兩相似度
similarities = cosine_similarity(embeddings)
print(similarities)
```

## 中文 NLP 套件比較

| 套件 | 強項 |
|---|---|
| jieba | 分詞、最快 |
| CKIP | 學術、最準 |
| spaCy | 工程友善 |
| Transformers (HF) | 最新 SOTA |
| Anthropic / OpenAI API | 最強、付費 |

## 何時用哪個
- 純分詞 / 詞頻：jieba
- 情感分析：Transformers
- 複雜理解 / 摘要：LLM API
- 學術研究：CKIP
