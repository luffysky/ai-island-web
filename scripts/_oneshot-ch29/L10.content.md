**Crawlee 進階**——L5 教基本、這課教 **Proxy Rotation + Session Pool + Apify Cloud 部署 + 監控**、真正生產級爬蟲。

> 💡 **何時需要進階？** 爬量 > 10 萬頁、被封 IP、要長期穩定跑、需要 dashboard 監控。

  
### Proxy Rotation（防 IP 被封）

**1. 自己提供 proxy 清單：**

```text
import { CheerioCrawler, ProxyConfiguration } from 'crawlee';

const proxyConfiguration = new ProxyConfiguration({
  proxyUrls: [
    'http://user:pass@proxy1.com:8080',
    'http://user:pass@proxy2.com:8080',
    'http://user:pass@proxy3.com:8080',
  ],
});

const crawler = new CheerioCrawler({
  proxyConfiguration,                        // Crawlee 自動輪流用
  async requestHandler({ $, request, proxyInfo }) {
    console.log(`用 proxy: ${proxyInfo.url}`);
    // ... 爬蟲邏輯
  },
});
```

  
**2. 用 Apify Proxy（推薦、不用自己找）：**

📄 **寫進程式檔**：`.env`

```text
APIFY_PROXY_PASSWORD=你的_apify_proxy_密碼
```

```text
import { Actor } from 'apify';

await Actor.init();

// === 用 Apify Proxy（全球 IP pool）===
const proxyConfiguration = await Actor.createProxyConfiguration({
  groups: ['RESIDENTIAL'],                   // RESIDENTIAL = 住宅 IP（最強）
  countryCode: 'TW',                         // 限制台灣 IP
});

const crawler = new CheerioCrawler({
  proxyConfiguration,
  // ...
});
```

> 💡 **3 種 Apify Proxy**：(1) `GOOGLE_SERP` Google 搜尋專用 (2) `RESIDENTIAL` 住宅 IP 最不會被擋 (3) `DATACENTER` 資料中心 IP 便宜。**價格**：免費 $5 額度、付費約 $0.5/GB。

  
### Session Pool（cookie / fingerprint 自動管理）

**情境**：多帳號爬、每個 session 有自己的 cookie / UA。

```text
import { CheerioCrawler, SessionPool } from 'crawlee';

const crawler = new CheerioCrawler({
  useSessionPool: true,
  sessionPoolOptions: {
    maxPoolSize: 100,                        // 最多 100 個 session
    sessionOptions: {
      maxUsageCount: 50,                     // 每個 session 用 50 次就換
      maxErrorScore: 3,                      // 失敗 3 次就丟掉
    },
  },
  
  async requestHandler({ $, request, session }) {
    // session 有自己的 cookie / proxy
    if ($('captcha').length > 0) {
      session.markBad();                     // 遇到驗證碼、標記這個 session 壞掉
      throw new Error('Captcha');
    }
    
    // 正常處理
    await Dataset.pushData({ title: $('h1').text() });
  },
});
```

### Auto-scaling（依機器負載自動調並發）

```text
const crawler = new CheerioCrawler({
  autoscaledPoolOptions: {
    minConcurrency: 1,                       // 最少 1 個並發
    maxConcurrency: 50,                      // 最多 50 個並發
    desiredConcurrencyRatio: 0.95,           // 目標：CPU 95% 使用率
    scaleUpStepRatio: 0.05,                  // 每次 +5%
    scaleDownStepRatio: 0.05,                // 每次 -5%
    
    // === 系統壓力過大時自動降速 ===
    systemStatusOptions: {
      maxMemoryOverloadedRatio: 0.5,         // 記憶體超過 50% 算過載
      maxEventLoopOverloadedRatio: 0.3,
      maxCpuOverloadedRatio: 0.4,
    },
  },
});
```

  
### Dataset 進階（多種輸出格式）

```text
import { Dataset } from 'crawlee';

// === 1. 預設寫 JSON 到 storage/datasets/default/ ===
await Dataset.pushData({ url, title, price });

// === 2. 用名稱分類 ===
const productsDataset = await Dataset.open('products');
const reviewsDataset = await Dataset.open('reviews');

await productsDataset.pushData({ name, price });
await reviewsDataset.pushData({ rating, text });

// === 3. 匯出成 CSV / Excel ===
await productsDataset.exportToCSV('products.csv');
await productsDataset.exportToJSON('products.json');
```

  
### Key-Value Store（跨 worker 共享狀態）

```text
import { KeyValueStore } from 'crawlee';

// === 存共用的 token / cookie ===
const kv = await KeyValueStore.open('auth');
await kv.setValue('login_token', 'eyJhbGc...');

// === 在 requestHandler 內取 ===
async requestHandler({ request, sendRequest }) {
  const kv = await KeyValueStore.open('auth');
  const token = await kv.getValue('login_token');
  
  const response = await sendRequest({
    url: request.url,
    headers: { Authorization: `Bearer ${token}` },
  });
}

// === 存進度 ===
await kv.setValue('last_scrape_at', new Date().toISOString());
```

  
### 部署到 Apify Cloud

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npm install -g apify-cli
apify login
apify init
apify push
```

**這 4 行做什麼？**
- `apify-cli` —— Apify 的命令列工具
- `apify login` —— 用 Apify 帳號登入
- `apify init` —— 把現有專案轉成 Apify Actor（建 `.actor/actor.json`）
- `apify push` —— 推到 Apify 雲端

> 💡 **Apify Actor = 雲端可執行的爬蟲**——推上去後可在 web UI 跑、設 cron、看 dashboard、串 API。

### Apify Actor 收費 API（上架賣）

📄 **寫進程式檔**：`.actor/actor.json`

```text
{
  "actorSpecification": 1,
  "name": "shopee-scraper",
  "title": "Shopee Product Scraper",
  "description": "Scrape Shopee products by keyword",
  "version": "1.0",
  
  "input": {
    "keyword": { "type": "string", "title": "搜尋關鍵字" },
    "maxItems": { "type": "integer", "default": 100 }
  }
}
```

  
**上架後使用者可：**

```text
1. 在 Apify Store 找到你的 Actor
2. 輸入 keyword + maxItems
3. 按 Run、Apify 跑你的爬蟲
4. 結果自動存 dataset、可下載 / API 取
5. 按 compute time 收費（你拿 80%、Apify 抽 20%）
```
