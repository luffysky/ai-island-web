### 用法 2：SDK 整合（程式自動跑現成 Actor）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npm install apify-client
```

**這行做什麼？** 裝 Apify 官方 client SDK、用程式呼叫 Apify Cloud 跑 Actor。

```text
import { ApifyClient } from 'apify-client';

const client = new ApifyClient({
  token: process.env.APIFY_TOKEN
});

// === 跑 Apify 官方 web-scraper（萬用爬蟲）===
const run = await client.actor('apify/web-scraper').call({
  startUrls: [{ url: 'https://example.com' }],
  pageFunction: `async ({ page }) => ({
    title: await page.title(),
    h1: await page.$eval('h1', el => el.textContent),
  })`
});

// === 拿結果 ===
const { items } = await client.dataset(run.defaultDatasetId).listItems();
console.log(items);
```

> 💡 **這招的用途**：把 Apify 當「**爬蟲 API**」用、你的 SaaS 後端呼叫、不用自己維護爬蟲基礎設施。

  
### 用法 3：寫自己的 Actor（上架 Apify Store）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npm install -g apify-cli
apify create my-scraper
cd my-scraper
apify run
```

**這 4 行做什麼？**
- `apify-cli` —— Apify 命令列工具（全域裝）
- `apify create my-scraper` —— 建專案（會問你選 template）
- `cd my-scraper` —— 進入
- `apify run` —— 本機跑、測試

  
📄 **寫進程式檔**：`src/main.ts`

```text
import { Actor } from 'apify';
import { PlaywrightCrawler } from 'crawlee';

await Actor.init();

// === 取使用者輸入 ===
const input = await Actor.getInput<{
  startUrls: { url: string }[];
  maxItems: number;
}>();

const crawler = new PlaywrightCrawler({
  maxRequestsPerCrawl: input?.maxItems || 100,
  
  async requestHandler({ page, request, enqueueLinks }) {
    const title = await page.title();
    const price = await page.locator('.price').textContent();
    
    // === 存 Apify Dataset ===
    await Actor.pushData({
      url: request.url,
      title,
      price,
    });
    
    await enqueueLinks({ selector: 'a.product-link' });
  }
});

await crawler.run(input?.startUrls);
await Actor.exit();
```

  
### 上架 Apify Store（賺錢）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

```text
apify login
apify push
```

**這 2 行做什麼？**
- `apify login` —— 用 Apify 帳號登入
- `apify push` —— 把本機 Actor 推到 Apify Cloud + 上架

  
**上架後分潤：**

```text
=== Apify Store 商業模式 ===

→ 使用者跑你的 Actor（按 compute time 收費）
→ Apify 抽 20%、你拿 80%
→ 例：一次跑 $1、你拿 $0.8
→ 100 個用戶 × 每月 10 次 = $800/月被動收入
```
