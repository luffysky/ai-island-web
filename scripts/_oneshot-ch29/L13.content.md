**Cloudflare Workers = Serverless Function on Edge**——跑在 Cloudflare 全球 300+ 城市的資料中心、**免費 100k requests/day**、爬蟲部署最便宜方案。

> 💡 **限制**：免費 10ms CPU / request、不能跑 Puppeteer。**適合**：輕量爬蟲（fetch + HTMLRewriter）+ Cron 排程 + 全球分散式。

```text
✅ 免費 100k req/day
```

📄 **寫進程式檔**：`src/index.ts`

```text
export interface Env {
  KV: KVNamespace;
}

export default {
  async fetch(request: Request, env: Env): Promise<Response> {
    const res = await fetch('https://example.com');
    
    // === HTMLRewriter = Cloudflare 自家 HTML parser（替代 cheerio）===
    const products: any[] = [];
    let currentProduct: any = {};
    
    const rewriter = new HTMLRewriter()
      .on('div.product', {
        element(el) {
          currentProduct = {};
          el.onEndTag(() => products.push(currentProduct));
        }
      })
      .on('div.product .name', {
        text(text) { currentProduct.name = (currentProduct.name || '') + text.text; }
      })
      .on('div.product .price', {
        text(text) { currentProduct.price = (currentProduct.price || '') + text.text; }
      });
    
    // === 跑 rewriter ===
    await rewriter.transform(res).text();
    
    return Response.json({ products });
  }
};
```

> 💡 **HTMLRewriter** 是 Cloudflare 的 HTML 串流解析器、跟 cheerio 不一樣——**邊串流邊解析**、不用把整個 HTML 載入記憶體、超快 + 省記憶體。

  
### Cron Trigger（定時爬蟲、不用 GitHub Actions）

📄 **寫進程式檔**：`wrangler.toml`

```text
name = "my-scraper"
main = "src/index.ts"
compatibility_date = "2026-05-01"

# === 每天 UTC 03:00 跑 ===
[triggers]
crons = ["0 3 * * *"]

# === 綁定 KV 儲存 ===
kv_namespaces = [
  { binding = "KV", id = "your_kv_id_here" }
]
```

📄 **寫進程式檔**：`src/index.ts` 加 scheduled handler

```text
export default {
  // 一般 HTTP request
  async fetch(request, env) {
    // ...
  },
  
  // === Cron 觸發時跑這個 ===
  async scheduled(event, env, ctx) {
    console.log('Cron triggered:', event.cron);
    
    const data = await scrape();
    await env.KV.put('latest_data', JSON.stringify(data));
    
    console.log(`存 ${data.length} 筆到 KV`);
  }
};
```

  
### 3 種儲存（依用途選）

**1. KV（Key-Value、適合小資料）：**

```text
// === 存 ===
await env.MY_KV.put('latest_products', JSON.stringify(products), {
  expirationTtl: 3600,                       // 1 小時後過期
});

// === 取 ===
const data = await env.MY_KV.get('latest_products', { type: 'json' });
```

| 特性 | KV |
|---|---|
| 大小 | 每 value < 25MB |
| 速度 | 讀超快（edge cache）、寫慢 |
| 適合 | cache、設定、token |

  
**2. R2（S3 替代、適合大檔）：**

```text
// === 上傳 JSON 檔 ===
await env.BUCKET.put(`scrapes/${Date.now()}.json`, JSON.stringify(data));

// === 列出檔案 ===
const list = await env.BUCKET.list({ prefix: 'scrapes/' });
for (const obj of list.objects) {
  console.log(obj.key, obj.size);
}

// === 下載 ===
const obj = await env.BUCKET.get('scrapes/123.json');
const data = await obj.json();
```

| 特性 | R2 |
|---|---|
| 大小 | 無限 |
| 速度 | 中 |
| 適合 | 大資料、log、backup |
| 價格 | 10GB 免費、超出 $0.015/GB |

  
**3. D1（SQLite on Edge、適合結構化資料）：**

```text
// === 建表（先在 dashboard 或 wrangler d1 execute 跑）===
// CREATE TABLE products (id INTEGER PRIMARY KEY, name TEXT, price REAL);

// === 插入 ===
await env.DB.prepare(`
  INSERT INTO products (name, price) VALUES (?, ?)
`).bind('AirPods', 5990).run();

// === 查詢 ===
const { results } = await env.DB.prepare(`
  SELECT * FROM products WHERE price < ?
`).bind(10000).all();
```

| 特性 | D1 |
|---|---|
| 大小 | 10GB / database |
| 速度 | 快（edge 分散） |
| 適合 | SQL 查詢、關聯資料 |

  
### Browser Rendering（付費、想跑 Puppeteer）

**情境**：需要 JS 渲染、超出 fetch + HTMLRewriter 能力。

```text
import puppeteer from '@cloudflare/puppeteer';

export default {
  async fetch(request, env) {
    // === Cloudflare 提供的瀏覽器 ===
    const browser = await puppeteer.launch(env.BROWSER);
    const page = await browser.newPage();
    await page.goto('https://example.com');
    
    const title = await page.title();
    await browser.close();
    
    return Response.json({ title });
  }
};
```

> 💡 **Browser Rendering 收費**：$5/月 + 用量、適合無法用 fetch 抓的網站。**對策**：先試 fetch + HTMLRewriter、不行再上 Browser Rendering。

### 限制 / 何時不適合用

```text
❌ 不適合
- 需要長時間運算（> 10ms CPU）
- 要跑 Puppeteer / Playwright（要付費 Browser Rendering）
- 大規模 ETL（用 Apify / Railway）
- 複雜資料分析（用 Python + pandas）

✅ 適合
- 輕量 API 爬蟲（< 10ms 處理）
- Cron 排程（每天 / 每小時跑）
- 全球分散爬蟲（IP 自動換）
- Webhook 接收 + 處理
- Edge cache layer
```
