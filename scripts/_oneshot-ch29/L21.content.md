### ScraperAPI（最簡單、入門首選）

**註冊 + 拿 API key：**

```text
1. 去 scraperapi.com 註冊（免費 1000 req 試用）
2. Dashboard 拿 API key
```

  
**用法（一個 fetch 搞定）：**

```text
const KEY = process.env.SCRAPER_API_KEY;
const TARGET = 'https://target.com/protected-page';

const res = await fetch(
  `https://api.scraperapi.com?api_key=${KEY}&url=${encodeURIComponent(TARGET)}&render=true`
);
const html = await res.text();

// === ScraperAPI 已經幫你過 Cloudflare、開瀏覽器 render JS、回 HTML ===
const $ = cheerio.load(html);
const title = $('h1').text();
```

**參數：**

```text
?api_key=...        必填、你的 key
&url=...            必填、目標 URL（要 encode）
&render=true        開瀏覽器渲染 JS（要付費）
&country_code=us    指定 IP 國家
&premium=true       用住宅 IP（過更難的反爬）
&keep_headers=true  保留你送的 headers
```

  
### Scrapfly（ASP 反爬最強）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npm install scrapfly-sdk
```

**這行做什麼？** 裝 Scrapfly 官方 SDK、支援 JS / Python、API 設計現代。

```text
import { ScrapflyClient, ScrapeConfig } from 'scrapfly-sdk';

const scrapfly = new ScrapflyClient({ key: process.env.SCRAPFLY_KEY });

const result = await scrapfly.scrape(new ScrapeConfig({
  url: 'https://example.com',
  asp: true,                                 // Anti Scraping Protection
  render_js: true,                           // 渲染 JS
  country: 'TW',                             // 台灣 IP
}));

console.log(result.content);                 // HTML
console.log(result.cost);                    // 這次花了幾個 credit
```

> 💡 **Scrapfly ASP** = Anti Scraping Protection、過 Cloudflare / Akamai / DataDome 強、價格中等。

  
### Diffbot（AI 解析、不解 HTML 拿結構化資料）

**情境**：不想自己寫 CSS selector、Diffbot 用 AI 自動辨識「商品 / 文章 / 評論」。

```text
const KEY = process.env.DIFFBOT_KEY;
const target = 'https://shop.example.com/product/123';

const res = await fetch(
  `https://api.diffbot.com/v3/product?token=${KEY}&url=${encodeURIComponent(target)}`
);
const data = await res.json();

// === Diffbot 自動辨識並回結構化資料 ===
console.log(data.objects[0]);
// {
//   title: 'AirPods Pro',
//   price: 5990,
//   currency: 'TWD',
//   images: [...],
//   description: '...',
//   sku: '...',
// }
```

> 💡 **Diffbot 適合**：(1) 爬不同網站、不想寫 N 個 parser (2) 競品多源資料整合 (3) AI / ML 訓練資料收集。

  
### Browserless（雲端 Puppeteer / Playwright）

**情境**：要跑 Puppeteer、不想自己維護 Chromium 環境。

```text
import puppeteer from 'puppeteer';

const browser = await puppeteer.connect({
  browserWSEndpoint: `wss://chrome.browserless.io?token=${KEY}`
});

const page = await browser.newPage();
await page.goto('https://example.com');
const title = await page.title();

await browser.close();
```

> 💡 **Browserless 優勢**：(1) 不用本機裝 Chromium (2) 不用設 Docker (3) 自動 scale (4) Workers / Vercel 也能用（serverless 限制）。

### 自己寫 vs 買 API 決策

```text
=== 自己寫 ===
✅ 0 月費
✅ 完全控制
❌ 維護成本高（網站改版要修）
❌ 反爬要自己搞
❌ 1 個月開發 + 持續維護

時間成本：1 個月 + 維護
適合：量 < 10k/月 + 簡單目標 + Indie 練習

=== 買 API ===
✅ 開箱即用、1 天上線
✅ 反爬全包
✅ 0 維護
❌ 月費 $30-500+
❌ 量大成本爆

時間成本：1 天
適合：量 10k-1M/月 + 商業 / 接案 + 沒時間搞反爬
```

  
### 量化決策表（依爬量）

| 月爬量 | 推薦方案 |
|---|---|
| < 1000 | 自己寫 + 免費代理 |
| 1k - 10k | 自己寫 + ScraperAPI 備援 |
| 10k - 100k | ScraperAPI / Scrapfly |
| 100k - 1M | Scrapfly / Bright Data |
| > 1M | Bright Data / Zyte 企業版 |

  
### Indie SaaS 推薦路線

```text
=== Phase 1：MVP（0-100 用戶）===
- 自己寫 + Cloudflare Workers（免費）
- 簡單目標、不過 Cloudflare

=== Phase 2：成長（100-1000 用戶）===
- ScraperAPI / Scrapfly $30-100/月
- 反爬難度提升、值得花錢
- 還賺得到毛利

=== Phase 3：規模化（1000+ 用戶）===
- Bright Data 企業合約
- 或自架反爬伺服器（成本/收入比優）
```
