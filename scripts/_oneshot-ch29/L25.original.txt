## JS vs Python 爬蟲

| 面向 | JS / Node | Python |
|---|---|---|
| 框架 | Crawlee | Scrapy |
| 動態 | 內建 | 要加 |
| 生態 | npm | PyPI |
| 招募 | 少 | 多 |

兩個都會最強。

- 賣 Apify Actor 被動收入
- 接案 + SaaS 並行
- Cloudflare Workers 免費跑

## 心法
- JS 不是 Python 的替代、是補充
- 同 stack 全端寫爬蟲體驗最爽
- 商業選 Python（生態完整）
- 個人 / Indie 選 JS（已會 stack）

## 給新手建議
1. 學 Node + TS
2. 學 fetch + cheerio
3. 學 Playwright
4. 學 Crawlee
5. 做 3 個小爬蟲
6. 接第 1 個客戶

6 個月、會 JS 爬蟲。
