### 內建 SQLite（爬蟲 + DB 不用裝套件）

```text
import { Database } from 'bun:sqlite';

const db = new Database('data.db', { create: true });

db.exec(`
  CREATE TABLE IF NOT EXISTS products (
    id INTEGER PRIMARY KEY,
    name TEXT,
    price REAL,
    link TEXT UNIQUE,
    scraped_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
  )
`);

// === 用 prepared statement（防 SQL injection）===
const insert = db.prepare(`
  INSERT OR IGNORE INTO products (name, price, link)
  VALUES (?, ?, ?)
`);

for (const p of products) {
  insert.run(p.name, p.price, p.link);
}

// === 查詢 ===
const cheap = db.query('SELECT * FROM products WHERE price < ?').all(1000);
console.log(`便宜商品：${cheap.length} 個`);
```

> 💡 **vs Node 用 sqlite3**：(1) 不用 `npm install sqlite3` (2) 不用編譯 C++（sqlite3 套件常出錯） (3) 速度快 4 倍。

### Worker（並發爬蟲）

📄 **寫進程式檔**：`worker.ts`

```text
declare var self: Worker;

self.onmessage = async (e) => {
  const url = e.data.url;
  const res = await fetch(url);
  const html = await res.text();
  
  // === 把結果傳回主程序 ===
  self.postMessage({ url, length: html.length });
};
```

📄 **寫進程式檔**：`main.ts`

```text
const urls = ['https://a.com', 'https://b.com', 'https://c.com'];

const workers = urls.map(url => {
  const worker = new Worker('./worker.ts');
  worker.postMessage({ url });
  
  return new Promise((resolve) => {
    worker.onmessage = (e) => {
      resolve(e.data);
      worker.terminate();
    };
  });
});

const results = await Promise.all(workers);
console.log(results);
```

  
### Bun.serve（內建 HTTP server、包 SaaS）

```text
Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);
    
    if (url.pathname === '/scrape') {
      const keyword = url.searchParams.get('q');
      
      // 爬蟲邏輯
      const res = await fetch(`https://shopee.tw/search?keyword=${keyword}`);
      const html = await res.text();
      
      return Response.json({ html_length: html.length });
    }
    
    return new Response('404', { status: 404 });
  },
});
```

### 何時用 Bun vs Node

```text
✅ 用 Bun
- 新爬蟲專案
- 個人 / Indie 開發
- 需要極致速度
- 喜歡 TS 但不想設定
- 部署到 Cloudflare / Vercel Edge

⚠️ 用 Node
- 老 codebase 已穩定
- 需要 Node-only 套件（少數）
- 企業環境保守
- 需要 LTS 5 年支援
```
