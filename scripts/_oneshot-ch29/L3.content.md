**Puppeteer = Google 親兒子的瀏覽器控制**——2017 出、之後 Microsoft 派系出 Playwright（更強）。**Puppeteer 仍主流、Google 自家 Chrome 整合最好**。

> 💡 **vs Playwright（L4）**：Playwright 更現代、API 更好、跨瀏覽器；Puppeteer 老牌、Chrome only、社群大。**新專案推 Playwright、學爬蟲兩個都會**。

### 基本使用

```text
import puppeteer from 'puppeteer';

async function scrape(url) {
  // === 啟動瀏覽器 ===
  const browser = await puppeteer.launch({
    headless: true,                          // 開發時改 false 看畫面
    args: ['--no-sandbox']                   // Docker / Linux 內必加
  });
  
  const page = await browser.newPage();
  
  // === 設定 UA（防擋）===
  await page.setUserAgent('Mozilla/5.0 (Macintosh) Chrome/131.0');
  
  // === 設定 viewport（影響 mobile vs desktop 版本）===
  await page.setViewport({ width: 1920, height: 1080 });
  
  // === 開頁 + 等載入完 ===
  await page.goto(url, { waitUntil: 'networkidle2' });
  // networkidle2 = 網路請求剩 ≤ 2 個就算載完
  
  // === 等元素出現 ===
  await page.waitForSelector('.product');
  
  // === 取資料（page.evaluate 在瀏覽器內跑 JS）===
  const data = await page.evaluate(() => {
    return Array.from(document.querySelectorAll('.product')).map(el => ({
      name: el.querySelector('.name')?.textContent,
      price: el.querySelector('.price')?.textContent,
      image: el.querySelector('img')?.src,
    }));
  });
  
  await browser.close();
  return data;
}
```

  
### 互動：點擊 / 填表 / 等待

```text
// === 點按鈕 ===
await page.click('button.load-more');

// === 填表單 ===
await page.type('input[name="email"]', 'a@b.com');
await page.type('input[name="password"]', 'secret');
await page.click('button[type="submit"]');

// === 等跳轉 ===
await page.waitForNavigation();

// === 4 種等待方式（按推薦度）===

// ✅ 等 selector 出現（最推薦）
await page.waitForSelector('.dashboard');

// ✅ 等 function 條件為 true
await page.waitForFunction(() =>
  document.querySelectorAll('.item').length > 20
);

// ⚠️ 等固定 N 毫秒（不推薦、不穩）
await page.waitForTimeout(2000);

// ✅ 等 navigation 完成
await page.waitForNavigation({ waitUntil: 'networkidle2' });
```

  
### 無限捲動 + lazy load

```text
async function autoScroll(page) {
  await page.evaluate(async () => {
    await new Promise(resolve => {
      let totalHeight = 0;
      const distance = 500;
      const timer = setInterval(() => {
        const scrollHeight = document.body.scrollHeight;
        window.scrollBy(0, distance);
        totalHeight += distance;
        
        if (totalHeight >= scrollHeight) {
          clearInterval(timer);
          resolve();
        }
      }, 200);
    });
  });
}

// === 用 ===
await page.goto('https://infinite-scroll.com');
await autoScroll(page);
const items = await page.$$eval('.item', els =>
  els.map(el => el.textContent)
);
```

### 截攔 Request / Response（神技、跳過解 HTML）

**最強技巧**：很多 SPA 後端打 JSON API、Puppeteer 直接攔、拿到乾淨 JSON。

```text
const apiData = [];

page.on('response', async response => {
  const url = response.url();
  if (url.includes('/api/products')) {
    try {
      const data = await response.json();
      apiData.push(data);
    } catch {}
  }
});

await page.goto('https://shop.example.com');
console.log('攔到 API data:', apiData);
```

  
**阻擋資源加速（不載圖 / 廣告 / CSS、爬純資料時用）：**

```text
await page.setRequestInterception(true);
page.on('request', req => {
  // 不要的資源直接 abort
  if (['image', 'stylesheet', 'font', 'media'].includes(req.resourceType())) {
    req.abort();
  } else {
    req.continue();
  }
});
// 結果：爬速度提升 3-5 倍（不下載圖片省超多頻寬）
```

  
### 處理 Cookie / Session

```text
// === 拿 cookie ===
const cookies = await page.cookies();
// [{ name: 'sessionid', value: 'abc', domain: '.example.com', ... }]

// === 存到檔案、之後重用 ===
import fs from 'fs/promises';
await fs.writeFile('cookies.json', JSON.stringify(cookies));

// === 載入 cookie（之後不用重登入）===
const saved = JSON.parse(await fs.readFile('cookies.json', 'utf8'));
await page.setCookie(...saved);
```

```text
npm install puppeteer-extra puppeteer-extra-plugin-stealth
```

**這 2 個套件做什麼？**
- `puppeteer-extra` —— Puppeteer 的 plugin 系統
- `puppeteer-extra-plugin-stealth` —— 反偵測 plugin、隱藏 webdriver flag

  
```text
import puppeteerExtra from 'puppeteer-extra';
import StealthPlugin from 'puppeteer-extra-plugin-stealth';

puppeteerExtra.use(StealthPlugin());

const browser = await puppeteerExtra.launch({ headless: true });
// 之後用法跟 puppeteer 一樣、但反偵測強很多
```

  
### Puppeteer vs Playwright 對比（再次）

| 面向 | Puppeteer | **Playwright** ⭐ |
|---|---|---|
| **發布年** | 2017（Google） | 2020（Microsoft） |
| **瀏覽器支援** | Chromium only | Chromium + Firefox + WebKit |
| **自動 wait** | 部分 | 完整 |
| **API 設計** | 老派 | 現代 async-first |
| **反偵測** | 需 puppeteer-extra | playwright-stealth |
| **Codegen** | ❌ | ✓ 內建 |
| **Trace Viewer** | ❌ | ✓ 內建 |
| **學習資源** | 多（老） | 中（新） |
| **新專案建議** | 維護老 codebase 才用 | ⭐ 新專案首選 |

  
    **☕ 用人話講**
    
Puppeteer 是 Google 出的瀏覽器控制：(1) **Chrome only、跟 Chrome 整合最好** (2) **PDF / 截圖功能超強**、商業價值高 (3) **2026 主流逐漸轉 Playwright**（L4）、但 Puppeteer 仍有大量老 codebase。**何時用**：(a) 維護老 codebase (b) 純做 Chrome 自動化 (c) 大量 PDF 生成。**新專案** 100% 選 Playwright（更強）。
