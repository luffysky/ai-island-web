**Node.js 爬蟲 vs Python 爬蟲**——同樣是爬蟲、語言不同。**這課讓你知道什麼時候選 Node、什麼時候選 Python**、不是「哪個比較好」而是「**什麼場景選什麼**」。

> 💡 **2026 答案**：**全端 / Indie SaaS 選 Node、純資料 + AI 選 Python**。會兩個更好、各有強項。

  
### Node.js 爬蟲 5 大優勢

```text
⚡ 原生 async / Promise
   → 並發 1000 個請求超輕鬆、不用學 asyncio

🌐 同語言全棧
   → 前端 React / 爬蟲 / 後端 Express / DB 全 JS / TS
   → 不用學兩個語言

🤝 跟瀏覽器同語言
   → Puppeteer 跑 Chromium、跑的就是 JS
   → 在瀏覽器內執行 JS 超直覺

📦 npm 生態
   → 250 萬+ 套件、爬蟲相關超多
   → cheerio / puppeteer / playwright / crawlee 全 first-class

🚀 效能
   → V8 引擎、單機並發遠勝 Python
   → 處理「**大量 I/O**」場景更快
```

  
### Python 爬蟲 5 大優勢

```text
📚 套件最完整
   → Scrapy / BeautifulSoup / requests 老牌
   → 中文資源 + StackOverflow 答案最多

🎯 學習曲線平緩
   → 語法簡單、新手友善
   → 沒有 JS 那麼多坑（this / async / promise）

🤖 跟 ML / AI 整合
   → 爬完直接接 pandas / scikit-learn / PyTorch
   → 不用換語言、Notebook 內一條龍

📊 資料分析強
   → pandas / numpy / matplotlib 一條龍
   → JS 沒對應好的工具

🌍 中文社群資源
   → 台灣 / 中國社群、Python 教學最多
```

  
### 2026 選擇指南（依場景）

| 場景 | 推薦 | 理由 |
|---|---|---|
| **全端 / Indie SaaS** | **Node** | 一個語言搞定所有（前端 + 爬蟲 + 後端） |
| **純資料抓 + 分析** | **Python** | pandas / matplotlib 一條龍 |
| **大量並發** | **Node** | async 原生、V8 快 |
| **跟 AI / ML 整合** | **Python** | scikit-learn / PyTorch 全 Python |

### 實際對照範例（同需求、兩種寫法）

**抓網頁標題：**

**Python（requests + BeautifulSoup）：**

```text
import requests
from bs4 import BeautifulSoup

res = requests.get('https://example.com')
soup = BeautifulSoup(res.text, 'lxml')
title = soup.select_one('h1').text
```

  
**Node（fetch + cheerio）：**

```text
import * as cheerio from 'cheerio';

const res = await fetch('https://example.com');
const html = await res.text();
const $ = cheerio.load(html);
const title = $('h1').text();
```

> 💡 **觀察**：兩個寫法**幾乎一樣**——`fetch` 對應 `requests`、`cheerio` 對應 `BeautifulSoup`。**會 Python 爬蟲、轉 Node 1 天上手**。

  
### 並發對比（Node 強項展示）

**抓 100 個 URL 對比：**

```text
=== Python（async 寫法、L14 教過）===
import asyncio
import httpx

async def fetch(client, url):
    return await client.get(url)

async def main():
    async with httpx.AsyncClient() as client:
        tasks = [fetch(client, u) for u in urls]
        return await asyncio.gather(*tasks)

asyncio.run(main())
# 約 8-10 秒
```

  
```text
=== Node（原生 async、超直觀）===
const results = await Promise.all(
  urls.map(url => fetch(url).then(r => r.text()))
);
// 約 3-5 秒（V8 快 + 原生 async）
```

> 💡 **Node 並發語法更乾淨**——`Promise.all` 一行解決、Python 要 `asyncio.gather` + 學 asyncio 概念。

```text
node --version
npm init -y
```

**這 2 行做什麼？**
- `node --version` —— 確認你有 Node、推薦 v20+（如果沒裝去 nodejs.org 下載）
- `npm init -y` —— 建一個 `package.json`（Node 專案的設定檔、相當於 Python 的 `pyproject.toml`）

> 💡 **要支援 ESM `import`**（不是 `require`）、需要 `package.json` 加 `"type": "module"`。本章範例都用 ESM。
