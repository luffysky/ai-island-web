**TypeScript 爬蟲 = 型別安全 + IDE 提示**——爬蟲特別容易出錯（網站變動、欄位 null、格式不對）、TS 讓**錯誤在編譯期就抓出**、不會 runtime 才爆炸。

> 💡 **vs Python Pydantic**：TypeScript 內建型別系統 + Zod 做 runtime 驗證 = Pydantic 等價物、更輕量。

  
### 為什麼爬蟲要型別

```text
❌ JS 沒型別、常見錯誤：
- 抓不到 .price → undefined → parseFloat(undefined) = NaN
- 抓到 "NT$ 1,299" → 沒 parseInt → "NT$ 1,299" 存進去
- 欄位拼錯 → product.titel（少了 e）→ silent fail
- 網站改版 → .product 改成 .item → 全部 null、沒人發現

✅ TS 解決：
- 編譯期抓型別錯
- IDE 自動提示欄位
- Zod 做 runtime 驗證（網站變動立刻 throw）
```

  
### 定義型別（interface）

```text
interface Product {
  id: string;
  name: string;
  price: number;                             // 必須是 number
  link: string;
  tags: string[];
  inStock: boolean;
  scrapedAt: Date;
}
```

> 💡 **interface vs type**：兩個都行、`interface` 適合「**物件 shape**」、`type` 適合「**聯合 / 交集**」。爬蟲場景慣用 `interface`。

**這行做什麼？** 裝 Zod、TS 最流行 runtime 驗證套件、跟 Pydantic 等價。

```text
import { z } from 'zod';

// === 1. 定義 Schema ===
const ProductSchema = z.object({
  id: z.string(),
  name: z.string().min(1),                   // 至少 1 字
  price: z.number().positive(),              // 正數
  link: z.string().url(),                    // 必須是 URL
  tags: z.array(z.string()),
  inStock: z.boolean(),
});

// === 2. 從 Schema 推 type（不用重複定義）===
type Product = z.infer<typeof ProductSchema>;
// → 等同 interface Product { id, name, price, link, tags, inStock }
```

> 💡 **z.infer 是 Zod 的魔法**——Schema 寫一次、TS type 自動生成、不用重複定義。

  
### 爬 + 驗證完整範例

```text
import { z } from 'zod';
import * as cheerio from 'cheerio';

const ProductSchema = z.object({
  id: z.string(),
  name: z.string().min(1),
  price: z.number().positive(),
  link: z.string().url(),
  tags: z.array(z.string()),
});

type Product = z.infer<typeof ProductSchema>;

async function scrapeProduct(url: string): Promise<Product> {
  const res = await fetch(url);
  const html = await res.text();
  const $ = cheerio.load(html);
  
  const raw = {
    id: $("meta[name='product-id']").attr('content'),
    name: $('h1').text().trim(),
    price: parseFloat($('.price').text().replace(/[^0-9.]/g, '')),
    link: url,
    tags: $('.tag').map((i, el) => $(el).text()).get(),
  };
  
  // === Zod 驗證 + 自動 throw 如果不符合 ===
  return ProductSchema.parse(raw);
}
```

  
### safeParse：不 throw、回傳結果物件

**情境**：你不希望解析錯誤直接 crash、想優雅處理。

```text
const result = ProductSchema.safeParse(raw);

if (result.success) {
  // === 成功、用 result.data ===
  await saveToDb(result.data);
} else {
  // === 失敗、看哪欄位錯 ===
  console.error('驗證失敗:', result.error.format());
  // {
  //   price: { _errors: ['Expected number, received NaN'] },
  //   link: { _errors: ['Invalid url'] }
  // }
  
  // === 存 log、繼續下一筆 ===
  await logFailure(raw, result.error);
}
```

  
### Transform：把字串轉成數字（爬蟲常用）

**情境**：價格抓到「NT$ 1,299」字串、要轉成 number。

```text
import { z } from 'zod';

const PriceSchema = z.string()
  .transform(s => s.replace(/[^0-9.]/g, ''))   // 「NT$ 1,299」→「1299」
  .pipe(z.coerce.number().positive());         // 轉 number + 驗證 > 0

// === 用 ===
PriceSchema.parse('NT$ 1,299');               // → 1299
PriceSchema.parse('1,299 元');                // → 1299
PriceSchema.parse('NT$ 0');                   // → throw（不是 positive）
```

> 💡 **transform + pipe** = 「**先處理、再驗證**」、Zod 的鏈式 API、爬蟲場景超實用。

  
### Result Type（不 throw、回傳明確結果）

**Rust-style 錯誤處理**：

```text
type Result<T, E = Error> = 
  | { ok: true; data: T }
  | { ok: false; error: E };

async function safeScrape(url: string): Promise<Result<Product>> {
  try {
    const data = await scrapeProduct(url);
    return { ok: true, data };
  } catch (err) {
    return { ok: false, error: err as Error };
  }
}

// === 用 ===
const result = await safeScrape('https://example.com/product/1');

if (result.ok) {
  console.log('✓', result.data.name);
} else {
  console.error('✗', result.error.message);
}
```

  
### 處理多種網站（discriminated union）

**情境**：爬蝦皮 + momo + PChome、三種網站結構不一樣。

```text
type Product = 
  | { source: 'shopee'; itemid: number; name: string; price: number }
  | { source: 'momo'; productId: string; productName: string; finalPrice: number }
  | { source: 'pchome'; goodsId: string; goodsName: string; goodsPrice: number };

// === Type guard：依 source 判斷 ===
function getName(p: Product): string {
  switch (p.source) {
    case 'shopee': return p.name;            // TS 知道這裡是 ShopeeProduct
    case 'momo': return p.productName;
    case 'pchome': return p.goodsName;
  }
}
```
