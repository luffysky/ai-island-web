**OCR + Computer Vision = 從圖片爬資料**——不是所有資料都是 HTML、**很多在圖片裡**：發票 / 監視器 / 截圖 / PDF / Captcha。

> 💡 **2026 主流**：(1) **Tesseract**（傳統、免費、中等準確） (2) **PaddleOCR**（中文最強、開源） (3) **Claude Vision / GPT-4 Vision**（AI、結構化抽取最強、付費）。

```text
npm install tesseract.js
```

**這行做什麼？** 裝 tesseract.js、Google 出的 OCR 引擎、純 JS、可在瀏覽器或 Node 跑。

  
```text
import Tesseract from 'tesseract.js';

const { data: { text } } = await Tesseract.recognize(
  'image.png',
  'chi_tra+eng',                             // 繁體中文 + 英文（多語言用 + 連接）
  {
    logger: m => console.log(m),             // 進度回呼
  }
);

console.log(text);
```

**支援語言代碼：**

```text
chi_tra      → 繁體中文
chi_sim      → 簡體中文
eng          → 英文
jpn          → 日文
kor          → 韓文
chi_tra+eng  → 中英混合（常用）
```

> 💡 **Tesseract 準確度**：簡單清晰文字 90%+、手寫 / 模糊 / 旋轉 < 50%、**碰到複雜 case 換 Claude Vision**。

  
### PaddleOCR（中文最強、Python）

🖥️ **在終端機輸入**

```text
uv add paddleocr paddlepaddle
```

**這 1 行做什麼？** 裝 PaddleOCR + PaddlePaddle 引擎、百度開源、**中文準確度業界最強**。

  
📄 **寫進程式檔**：`ocr.py`

```text
from paddleocr import PaddleOCR

# === 初始化（第一次跑會下載模型約 100MB）===
ocr = PaddleOCR(use_angle_cls=True, lang='ch')

# === 辨識 ===
result = ocr.ocr('image.png', cls=True)

for line in result[0]:
    bbox = line[0]                           # 文字位置（四個角座標）
    text = line[1][0]                        # 文字內容
    confidence = line[1][1]                  # 信心度（0-1）
    print(f'{text} (confidence: {confidence:.2f})')
```

> 💡 **PaddleOCR 殺手鐧**：(1) 中文準確度 95%+（Tesseract 約 70%） (2) 支援表格 / 印章 / 手寫 (3) 開源免費 (4) 有 PP-Structure 解析整份文件結構。

  
### Claude Vision（AI、2026 最強）

**情境**：不只 OCR、要**結構化抽取**（如「從發票拿商家 + 日期 + 總金額 + 品項」）。

📄 **寫進程式檔**：`vision_scraper.ts`

```text
import Anthropic from '@anthropic-ai/sdk';
import fs from 'fs/promises';

const client = new Anthropic();

const imgBuffer = await fs.readFile('invoice.png');
const imgBase64 = imgBuffer.toString('base64');

const response = await client.messages.create({
  model: 'claude-opus-4-7',
  max_tokens: 1024,
  messages: [{
    role: 'user',
    content: [
      {
        type: 'image',
        source: {
          type: 'base64',
          media_type: 'image/png',
          data: imgBase64
        }
      },
      {
        type: 'text',
        text: `從發票上抽出以下欄位、回傳 JSON：
- merchant：商家名稱
- date：日期（YYYY-MM-DD）
- total：總金額（數字）
- items：品項陣列 [{name, qty, price}]`
      }
    ]
  }]
});

const data = JSON.parse(response.content[0].text);
console.log(data);
// {
//   merchant: '7-11',
//   date: '2026-05-27',
//   total: 158,
//   items: [
//     { name: '思樂冰', qty: 1, price: 35 },
//     { name: '茶葉蛋', qty: 2, price: 26 },
//     ...
//   ]
// }
```

> 💡 **Claude Vision 為什麼強**：(1) **理解 context**：知道「發票」「商家」「品項」是什麼 (2) 自動結構化、不用後處理 (3) 手寫 / 模糊 / 多語言全包 (4) 一個 prompt = 一個 OCR + parser。**比 Tesseract 強 100 倍**、但**付費**。

**方案 1：2Captcha / CapSolver（付費、最常用）：**

```text
- 文字 captcha：$0.5 / 1000 次
- reCAPTCHA v2：$2 / 1000 次
- reCAPTCHA v3：$2 / 1000 次
- hCaptcha：$3 / 1000 次
- Image select（選圖）：$2 / 1000 次
```

  
```text
// === 1. 傳 captcha 圖片給 2Captcha ===
const formData = new FormData();
formData.append('key', process.env.CAPTCHA_KEY);
formData.append('file', imageBlob);

const res1 = await fetch('http://2captcha.com/in.php', {
  method: 'POST',
  body: formData
});
const captchaId = (await res1.text()).split('|')[1];

// === 2. 輪詢結果（人工 / AI 解、通常 10-30 秒）===
while (true) {
  const r = await fetch(
    `http://2captcha.com/res.php?key=${KEY}&action=get&id=${captchaId}`
  );
  const text = await r.text();
  
  if (text.startsWith('OK')) {
    const solution = text.split('|')[1];     // 解出的文字
    console.log('Captcha solved:', solution);
    break;
  }
  
  if (text === 'CAPCHA_NOT_READY') {
    await new Promise(r => setTimeout(r, 2000));
    continue;
  }
  
  throw new Error('Captcha failed: ' + text);
}
```

### 心法選擇

```text
=== 場景對應工具 ===

簡單文字（如標題、價格）
  → Tesseract.js / PaddleOCR

複雜文件（發票、合約）
  → Claude Vision（結構化最強）

中文準確度要求高
  → PaddleOCR

預算 0、要免費
  → Tesseract.js / PaddleOCR

要 OCR + 解 + 入庫一條龍
  → Claude Vision

Captcha
  → 2Captcha（穩）/ Claude Vision（便宜）

PDF
  → pdf-parse + Claude Vision
```
