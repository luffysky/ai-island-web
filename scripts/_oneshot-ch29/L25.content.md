### 6 個月學習地圖


### JS vs Python 爬蟲對比（最後一次）

| 面向 | **JS / Node** | **Python** |
|---|---|---|
| **框架完整度** | Crawlee（強） | Scrapy（最強） |
| **動態網站** | 內建（Playwright） | 要裝 Playwright |
| **生態** | npm（爬蟲套件多） | PyPI（資料分析多） |
| **學習曲線** | 中（需懂 async） | 易（語法直觀） |
| **招募市場** | 少（Node 爬蟲職位） | 多（Python 標準） |
| **跟前端整合** | ⭐ Next.js 一條龍 | 要兩語言 |
| **跟資料分析** | 弱 | ⭐ pandas / matplotlib |
| **Vibe Coding** | ⭐ Claude 生 TS 強 | 也行 |

> 💡 **兩個都會最強**——8 成場景用 Node、需要資料分析時用 Python。

### Indie 路線（最推薦給已會前端的人）

```text
=== 為什麼 Indie ===
- 同 stack（Next.js + 爬蟲都 TS）
- 客戶不需要、自己跑
- 收入無上限（MRR 模式）
- 自由（無老闆 / 無 deadline）

=== Indie 黃金路線 ===
1. 學完 Crawlee（Month 1-3）
2. 找 1 個利基市場（如台灣房價 / 餐廳 / 競品）
3. 開發爬蟲 + dashboard（Month 4）
4. 上線（Vercel + Supabase + Cloudflare）
5. 行銷（Indie Hackers / 社群）
6. 6 個月達 MRR NT$ 30k = USD $1000
7. 1 年達 USD $5000
8. 全職 Indie hacker
```

### 心法總結

```text
✅ JS 不是 Python 的替代、是補充
✅ 同 stack 全端寫爬蟲體驗最爽
✅ 商業選 Python（生態完整）
✅ 個人 / Indie 選 JS（已會 stack）
✅ 兩個都會 = 接案範圍翻倍
✅ 學 Crawlee = 解鎖框架級爬蟲
✅ 學 Playwright = 解鎖動態網站
✅ 學部署 + 監控 = 業界專業水準

❌ 別宗教戰爭（Python vs JS）
❌ 別只學爬蟲、不學前後端
❌ 別爬違法內容（個資 / 仿冒）
❌ 別硬幹 Apple / Google / Meta
❌ 別忽略 ROI
```
