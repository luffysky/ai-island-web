**Playwright = Microsoft 出的瀏覽器自動化、2026 Node 爬蟲首選**。比 Puppeteer 強：跨瀏覽器（Chrome / Firefox / Safari）、auto-wait、Locator API、Codegen 自動產生 code。

> 💡 **vs Puppeteer**（L3）：Playwright 更現代、踩坑少、新專案 100% 選 Playwright。

### 基本使用

```text
import { chromium } from 'playwright';

async function scrape(url) {
  const browser = await chromium.launch();
  
  // === context = 一個獨立的 cookie / cache 空間 ===
  const context = await browser.newContext({
    userAgent: 'Mozilla/5.0 (Macintosh) Chrome/131.0',
    viewport: { width: 1920, height: 1080 },
    locale: 'zh-TW',                         // 影響 Accept-Language header
  });
  
  const page = await context.newPage();
  
  await page.goto(url);
  await page.waitForLoadState('networkidle');
  
  // === Locator API（比 Puppeteer 強）===
  const title = await page.locator('h1').innerText();
  
  // === 多個元素 ===
  const items = await page.locator('.product').all();
  const data = [];
  for (const item of items) {
    data.push({
      name: await item.locator('.name').innerText(),
      price: await item.locator('.price').innerText(),
    });
  }
  
  await browser.close();
  return data;
}
```

> 💡 **Browser vs Context vs Page**：(1) Browser = Chrome 進程 (2) Context = 獨立 cookie 空間（像無痕視窗、可同時開多個帳號） (3) Page = 一個 tab。

  
### Playwright vs Puppeteer（為什麼 2026 選 Playwright）

| 面向 | Puppeteer | **Playwright** ⭐ |
|---|---|---|
| **跨瀏覽器** | Chromium only | Chrome + Firefox + WebKit |
| **Auto-wait** | 要手動 wait | 自動等元素出現 |
| **Locator API** | 弱 | 強、自動 retry |
| **Codegen** | ❌ | ✓ 內建錄製器 |
| **Trace Viewer** | ❌ | ✓ 視覺化每步 |
| **多語言 SDK** | JS only | Py / Java / .NET |
| **網路攔截** | 中 | 強 + route() |

  
### Codegen：錄製操作生 code（Playwright 殺手鐧）

```text
npx playwright codegen https://example.com
```

**這行做什麼？** 開瀏覽器 + 錄製器、你**手動操作網頁**（點 / 填 / 捲）、右側自動生成爬蟲 code。

> 💡 **Codegen 大殺器**——複雜流程（登入 / 多步驟）原本要研究 selector 半小時、Codegen 點完直接複製貼上、5 分鐘搞定。

  
**完整工作流：**

```text
1. 開 codegen：npx playwright codegen https://shopee.tw
2. 手動：輸入帳號 / 點登入 / 點商品 / 加入購物車
3. 右邊看到自動生成的 code、複製
4. 微調（資料解析 / loop / 寫檔）
5. 完成
```

### 並發爬多頁

```text
import { chromium } from 'playwright';
import pLimit from 'p-limit';

async function scrapeAll(urls) {
  const browser = await chromium.launch();
  const limit = pLimit(5);                   // 同時 5 個 tab
  
  const results = await Promise.all(
    urls.map(url => limit(async () => {
      // === 每個 URL 開獨立 context（隔離 cookie）===
      const context = await browser.newContext();
      const page = await context.newPage();
      try {
        await page.goto(url, { timeout: 30000 });
        const title = await page.locator('h1').innerText();
        return { url, title };
      } finally {
        await context.close();               // 確保關閉、不漏記憶體
      }
    }))
  );
  
  await browser.close();
  return results;
}
```

  
### Trace Viewer（debug 神器）

```text
const browser = await chromium.launch();
const context = await browser.newContext();

// === 開始錄製 ===
await context.tracing.start({ screenshots: true, snapshots: true });

const page = await context.newPage();
await page.goto('https://example.com');
// ... 你的爬蟲操作

// === 停止 + 存 trace ===
await context.tracing.stop({ path: 'trace.zip' });

await browser.close();
```

### 路由攔截（route）—— 更強的 request 控制

```text
// === 阻擋圖片 / 廣告 ===
await page.route('**/*.{png,jpg,gif}', route => route.abort());
await page.route('**/google-analytics.com/**', route => route.abort());

// === 攔截 + 改 response ===
await page.route('**/api/products', async route => {
  const response = await route.fetch();
  const json = await response.json();
  // 改 response 內容
  json.modified = true;
  await route.fulfill({ json });
});
```
