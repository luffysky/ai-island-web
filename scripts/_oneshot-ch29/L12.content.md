**攔截 + 修改 Network**——爬蟲的「**X 光透視眼**」+ 「**動態替換器**」。可以**看所有網路請求 + 改 request / response 內容**、是 SPA 爬蟲的關鍵技巧。

> 💡 **核心觀念**：Puppeteer / Playwright 不只是「**自動點按鈕**」、它是**完整的瀏覽器 + 網路代理**——可以攔截任何 HTTP 請求、改任何回應。

  
### 4 種網路操作（依用途）

| 操作 | API | 用途 |
|---|---|---|
| **監聽 Response** | `page.on('response')` | 拿後端 JSON、跳過解 HTML |
| **攔截 Request** | `page.route('**/*')` | 阻擋圖片 / 加 headers |
| **Mock API** | `route.fulfill()` | 假裝 API 回應、測試用 |
| **修改 Response** | `route.fetch() + fulfill()` | 改網站內容、注入資料 |

  
### 1. 監聽 Response（拿 SPA 後端 JSON）

```text
page.on('response', async response => {
  const url = response.url();
  if (url.includes('/api/products')) {
    const data = await response.json();
    console.log('攔到 API response:', data);
  }
});
```

**這段做什麼？**
- `page.on('response', cb)` —— 每個網路回應都會觸發 cb
- `url.includes('/api/products')` —— 只處理我們關心的 API
- `response.json()` —— 直接拿 JSON、不用解 HTML

> 💡 **這是 SPA 爬蟲神技**——蝦皮 / momo / 任何 React 網站、直接攔 API、跳過所有 HTML 解析。

  
### 2. 攔截 Request（阻擋 / 改 headers）

```text
await page.route('**/*', route => {
  const req = route.request();
  
  // === 阻擋圖片（加速 5x）===
  if (req.resourceType() === 'image') return route.abort();
  
  // === 阻擋 Google Analytics（不發 tracking）===
  if (req.url().includes('googletagmanager')) return route.abort();
  
  // === 加自訂 header ===
  route.continue({
    headers: { ...req.headers(), 'X-Custom': 'value' }
  });
});
```

**這段做什麼？**
- `page.route('**/*')` —— 攔截所有 URL（`**/*` 是 glob 萬用）
- `route.abort()` —— 取消這個 request、不發出去
- `route.continue()` —— 讓 request 正常進行、可改 headers

  
### 3. Mock API（假裝 API 回應）

**情境**：測試前端、不想真打後端、用假資料。

```text
await page.route('**/api/users', route => {
  route.fulfill({
    status: 200,
    contentType: 'application/json',
    body: JSON.stringify([
      { id: 1, name: 'Mock User 1' },
      { id: 2, name: 'Mock User 2' },
    ])
  });
});

await page.goto('https://example.com/users');
// → 頁面以為打到後端、其實是我們塞的假資料
```

  
### 4. 修改 Response（截 + 改 + 放回去）

**情境**：網站有資料、但你想加欄位 / 過濾 / 注入。

```text
await page.route('**/api/products', async route => {
  // === 步驟 1：實際打 API ===
  const response = await route.fetch();
  
  // === 步驟 2：拿 JSON ===
  const data = await response.json();
  
  // === 步驟 3：改資料 ===
  data.products.forEach(p => {
    p.scraped_at = new Date().toISOString();
    p.discount_price = p.price * 0.9;
  });
  
  // === 步驟 4：把改過的 response 還給頁面 ===
  route.fulfill({ response, json: data });
});
```

> 💡 **這招用途**：(1) 注入測試資料 (2) 過濾廣告 (3) 改顯示語言 (4) 安裝 A/B test。

  
### 5. 抓 cookies / localStorage（拿登入狀態）

```text
// === 拿所有 cookies ===
const cookies = await page.context().cookies();
// [{ name: 'sessionid', value: 'abc', domain: '.example.com', ... }, ...]

// === 拿 localStorage（在頁面內跑 JS）===
const localStorage = await page.evaluate(() => ({ ...localStorage }));
// { token: 'eyJhbGc...', user_id: '123' }

// === 拿 sessionStorage ===
const sessionStorage = await page.evaluate(() => ({ ...sessionStorage }));
```

  
### 6. 加速：只載 HTML（省 80% 頻寬）

```text
await page.route('**/*', route => {
  const type = route.request().resourceType();
  
  // === 不要的資源直接 abort ===
  if (['stylesheet', 'image', 'font', 'media'].includes(type)) {
    route.abort();
  } else {
    route.continue();
  }
});
```

**省了什麼？**
- ❌ CSS（網站樣式、爬蟲不用）
- ❌ 圖片（占 80% 頻寬）
- ❌ 字型（30+ MB 一些網站）
- ❌ 影片 / 音訊
- ✅ HTML / JS / XHR（保留、爬資料要用）

  
### 7. 抓 GraphQL（現代 SPA 主流）

```text
page.on('response', async response => {
  if (response.url().includes('/graphql')) {
    try {
      const json = await response.json();
      
      // GraphQL response 結構：{ data: { ... }, errors: [...] }
      if (json.data?.products) {
        console.log('GraphQL products:', json.data.products);
      }
    } catch {}
  }
});
```

> 💡 **GraphQL 比 REST 好抓**——一個 endpoint、所有資料、JSON 格式統一。**Facebook / Shopify / GitHub** 都用 GraphQL。

  
### 8. WebSocket 攔截（即時資料）

```text
page.on('websocket', ws => {
  console.log(`WebSocket opened: ${ws.url()}`);
  
  ws.on('framesent', e => {
    console.log('→', e.payload);             // 送出去的訊息
  });
  
  ws.on('framereceived', e => {
    console.log('←', e.payload);             // 收到的訊息
  });
});
```

> 💡 **WS 攔截用途**：即時股價 / 聊天訊息 / live stream comment / 訂單推送。

  
### 9. 整合：爬蟲完整網路截圖

```text
import { chromium } from 'playwright';

const browser = await chromium.launch();
const page = await browser.newPage();

const allRequests = [];
const apiData = [];

// === 記錄所有請求 ===
page.on('request', req => {
  allRequests.push({ url: req.url(), method: req.method() });
});

// === 攔 API ===
page.on('response', async res => {
  if (res.url().includes('/api/')) {
    try { apiData.push(await res.json()); } catch {}
  }
});

// === 阻擋圖片 ===
await page.route('**/*', route => {
  if (route.request().resourceType() === 'image') return route.abort();
  route.continue();
});

await page.goto('https://example.com');

console.log(`總共發了 ${allRequests.length} 個請求`);
console.log(`攔到 ${apiData.length} 個 API`);
```
