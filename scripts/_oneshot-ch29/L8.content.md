**Playwright Test Runner = 內建測試框架**——L4 教爬蟲用法、這課教**測試框架用法**（同樣 API、加 test 套件）、用來測試爬蟲穩定性 / 跑大量 worker 並行。

> 💡 **為什麼爬蟲也要 Test Runner？** 爬蟲常因「網站改版」壞掉、用 Test Runner 寫 assertion、CI 每天跑、第一時間發現失效。

### 第一個爬蟲測試

📄 **寫進程式檔**：`tests/shopee.spec.ts`

```text
import { test, expect } from '@playwright/test';

test('蝦皮搜尋頁有商品列表', async ({ page }) => {
  await page.goto('https://shopee.tw/search?keyword=AirPods');
  
  // === 等商品載入 ===
  await page.waitForSelector('.shopee-search-item-result__item');
  
  // === assertion：至少有 30 個商品 ===
  const items = page.locator('.shopee-search-item-result__item');
  await expect(items).toHaveCount(30, { timeout: 10000 });
});

test('商品有價格欄位', async ({ page }) => {
  await page.goto('https://shopee.tw/search?keyword=iPhone');
  
  const firstPrice = page.locator('[data-sqe="price"]').first();
  
  // === assertion：價格存在 + 是數字格式 ===
  await expect(firstPrice).toBeVisible();
  const text = await firstPrice.innerText();
  expect(text).toMatch(/\$[\d,]+/);
});
```

### Trace Viewer Replay（debug 大殺器）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npx playwright test --trace on
npx playwright show-report
```

**這 2 行做什麼？**
- `--trace on` —— 每個測試都錄製 trace（每步 screenshot + DOM + 網路請求）
- `show-report` —— 開 HTML 報告 GUI、點失敗的測試看 trace

> 💡 **Trace Viewer = 時光機**——點任何時間點、看當時 DOM / 網路 / console、debug 爬蟲不用再 try-catch 印 log。

  
### 多 worker 並行（爬蟲加速）

📄 **寫進程式檔**：`playwright.config.ts`

```text
import { defineConfig } from '@playwright/test';

export default defineConfig({
  workers: 5,                                // 同時 5 個 worker 跑
  retries: 2,                                // 失敗 retry 2 次
  use: {
    trace: 'on-first-retry',                 // 第一次 retry 時開始錄 trace
    screenshot: 'only-on-failure',           // 只失敗才截圖（省空間）
    video: 'retain-on-failure',              // 失敗才存影片
  },
  projects: [
    { name: 'chromium', use: { ...devices['Desktop Chrome'] } },
    { name: 'firefox', use: { ...devices['Desktop Firefox'] } },
    { name: 'mobile', use: { ...devices['iPhone 13'] } },
  ],
});
```

  
### 排程爬蟲（用 Test 結構 + GitHub Actions）

📄 **寫進程式檔**：`.github/workflows/scrape-daily.yml`

```text
name: Daily Scrape

on:
  schedule:
    - cron: '0 0 * * *'                      # 每天 UTC 00:00
  workflow_dispatch:                          # 也可手動觸發

jobs:
  scrape:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with: { node-version: '20' }
      - run: npm ci
      - run: npx playwright install --with-deps chromium
      - run: npx playwright test
      
      # === 失敗時上傳 trace ===
      - uses: actions/upload-artifact@v4
        if: failure()
        with:
          name: playwright-report
          path: playwright-report/
```

> 💡 **GitHub Actions 免費** 公開 repo 無限、private repo 每月 2000 分鐘——爬蟲跑每天 5 分鐘 = 月 150 分鐘 = 完全免費。

  
### Fixture：共用 setup（多測試共用登入）

📄 **寫進程式檔**：`fixtures.ts`

```text
import { test as base } from '@playwright/test';

export const test = base.extend({
  // === 自訂 fixture：已登入的 page ===
  loggedInPage: async ({ page }, use) => {
    // 登入流程
    await page.goto('https://example.com/login');
    await page.fill('input[name="email"]', 'a@b.com');
    await page.fill('input[name="password"]', 'secret');
    await page.click('button[type="submit"]');
    await page.waitForURL('**/dashboard');
    
    // === 提供給測試用 ===
    await use(page);
    
    // 測試結束後清理（可省略）
  },
});
```

```text
// === 在測試內用 ===
import { test } from './fixtures';

test('已登入後抓會員資料', async ({ loggedInPage }) => {
  await loggedInPage.goto('https://example.com/account');
  // 已登入、直接取資料
});
```

  
### Visual Regression（檢測網站改版）

```text
test('蝦皮首頁 layout 沒變', async ({ page }) => {
  await page.goto('https://shopee.tw');
  
  // === 第一次跑會存 baseline ===
  // 之後跑會比對、不一樣就失敗
  await expect(page).toHaveScreenshot('shopee-home.png');
});
```

> 💡 **Visual Regression = 網站改版偵測器**——爬蟲壞掉前先偵測、CI 早期警告。
