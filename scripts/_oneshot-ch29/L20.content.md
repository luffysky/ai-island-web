### Vercel Functions 爬蟲範例

📄 **寫進程式檔**：`app/api/scrape/route.ts`（Next.js 15 App Router）

```text
import * as cheerio from 'cheerio';

export async function GET() {
  const res = await fetch('https://example.com');
  const html = await res.text();
  const $ = cheerio.load(html);
  
  const products = $('div.product').map((i, el) => ({
    name: $(el).find('.name').text().trim(),
    price: $(el).find('.price').text(),
  })).get();
  
  return Response.json(products);
}

// === 設定 runtime + 最大執行時間 ===
export const runtime = 'edge';                // 或 'nodejs'
export const maxDuration = 60;                // 秒（Pro 才能設 > 10）
```

  
### Vercel Cron Jobs（內建排程）

📄 **寫進程式檔**：`vercel.json`

```text
{
  "crons": [
    {
      "path": "/api/scrape",
      "schedule": "0 3 * * *"
    }
  ]
}
```

**Cron 表達式速查：**

```text
0 3 * * *     → 每天 UTC 03:00（台灣 11:00）
0 * * * *     → 每小時整點
*/15 * * * *  → 每 15 分鐘
0 0 * * 1     → 每週一 00:00
0 9 1 * *     → 每月 1 號 09:00
```

> 💡 **Vercel Hobby（免費）** 1 個 cron、**Pro $20/月** 40 個 cron。

### Netlify Functions（vs Vercel）

📄 **寫進程式檔**：`netlify/functions/scrape.ts`

```text
import { Handler } from '@netlify/functions';
import * as cheerio from 'cheerio';

export const handler: Handler = async (event) => {
  const res = await fetch('https://example.com');
  const html = await res.text();
  const $ = cheerio.load(html);
  
  const products = $('div.product').map((i, el) => ({
    name: $(el).find('.name').text(),
  })).get();
  
  return {
    statusCode: 200,
    body: JSON.stringify(products),
  };
};
```

**Netlify 限制：**

```text
- 免費：125k req/月、100 hr 計算
- Timeout 10s / 26s（背景）
- 支援 Background Functions（最長 15 分鐘）
```

  
### AWS Lambda（要 Puppeteer 必用）

**情境**：要跑 Puppeteer、Vercel / Netlify 不行、AWS Lambda 是答案。

```text
import chromium from '@sparticuz/chromium';
import puppeteer from 'puppeteer-core';

export const handler = async () => {
  const browser = await puppeteer.launch({
    args: chromium.args,
    executablePath: await chromium.executablePath(),
    headless: true,
  });
  
  const page = await browser.newPage();
  await page.goto('https://example.com');
  const title = await page.title();
  await browser.close();
  
  return { statusCode: 200, body: title };
};
```

**Lambda 限制：**

```text
- Timeout 15 分鐘
- Memory 最多 10 GB
- Puppeteer 透過 @sparticuz/chromium 可跑
- 冷啟動 2-5 秒（Puppeteer）
- 收費：1M req 免費、之後 $0.20 / 1M
```

  
### 處理長任務（Timeout 不夠用）

**問題**：要爬 10000 個 URL、Vercel 60s 跑不完。

**方案 1：分批 + 自呼叫**

```text
// app/api/scrape/route.ts
export async function GET(req: Request) {
  const url = new URL(req.url);
  const page = parseInt(url.searchParams.get('page') || '1');
  
  // === 一次爬 50 個 ===
  const data = await scrapePage(page);
  
  // === 有下一頁、非同步呼叫自己 ===
  if (data.hasNext) {
    fetch(`${url.origin}/api/scrape?page=${page + 1}`);
  }
  
  return Response.json({ scraped: data.items.length, page });
}
```

  
**方案 2：QStash（Upstash）排程 queue**

```text
import { Client } from '@upstash/qstash';

const qstash = new Client({ token: process.env.QSTASH_TOKEN });

// === 推 100 個 task 到 queue ===
for (let i = 0; i < 100; i++) {
  await qstash.publishJSON({
    url: 'https://my-app.vercel.app/api/scrape',
    body: { page: i },
    delay: i * 5,                            // 每 5 秒一個、避免被擋
  });
}
```

  
**方案 3：Inngest（事件驅動 workflow）**

```text
import { inngest } from '@/lib/inngest';

export const scrapeDaily = inngest.createFunction(
  { id: 'scrape-daily' },
  { cron: '0 3 * * *' },                     // 每天 UTC 03:00
  async ({ event, step }) => {
    // === step 1：拿 URL 清單 ===
    const urls = await step.run('get-urls', () => getUrls());
    
    // === step 2：並行爬（自動分散執行）===
    for (const url of urls) {
      await step.run(`scrape-${url}`, () => scrape(url));
    }
    
    // === step 3：通知完成 ===
    await step.run('notify', () => notifySlack(`完成 ${urls.length} 個`));
  }
);
```

> 💡 **Inngest** = 「**serverless 的 Airflow**」、自動處理超時 + retry + 狀態、Vercel 環境最佳搭配。
