## 主流服務

### ScraperAPI
- $49-$249/月
- 內建代理 + JS rendering

```javascript
const res = await fetch(
  `https://api.scraperapi.com?api_key=${KEY}&url=https://target.com&render=true`
);
```

### Bright Data Web Scraper IDE
老牌最大、企業級。

### Scrapfly
```javascript
import { ScrapflyClient, ScrapeConfig } from "scrapfly-sdk";

const scrapfly = new ScrapflyClient({ key: "..." });
const result = await scrapfly.scrape(
  new ScrapeConfig({ url: "https://example.com", asp: true })
);
```

### Diffbot
```javascript
const res = await fetch(
  `https://api.diffbot.com/v3/product?token=${KEY}&url=${url}`
);
```

### Browserless
```javascript
const browser = await puppeteer.connect({
  browserWSEndpoint: "wss://chrome.browserless.io?token=..."
});
```

## ROI
自己寫 vs 買 API：
- 自己：1 個月 + 維護
- 買：月費 NT$ 1.5k-5k、0 維護

> 10k req/月 = 買 API 划算。
