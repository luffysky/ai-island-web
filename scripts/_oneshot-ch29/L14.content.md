**反 Fingerprint 進階**——網站不只看 IP 和 UA、還看 **canvas / WebGL / 字型 / 行為**。這課教**對抗 17+ 種偵測技巧**。

> ⚠️ **法律提醒**：反偵測技巧在「**過 Cloudflare 等基礎防護**」內合法、但**繞過 Akamai / DataDome 等企業反爬可能違反 ToS**、商業使用評估法律風險。

  
### 網站偵測你的 8 大指標

```text
1. navigator.webdriver
   → headless 瀏覽器這個 = true、真人瀏覽器 = false
   → 最常見偵測

2. Canvas fingerprint
   → 用 canvas 畫一段文字 + 取像素值
   → 每台機器顯卡 / 字型不同、結果不同
   → 同一個 fingerprint = 同一個爬蟲

3. WebGL fingerprint
   → GPU vendor / renderer 資訊
   → headless 通常用 SwiftShader、真人用 NVIDIA / Intel

4. 字型清單
   → 你機器裝的字型清單
   → headless 通常字型少、真人多

5. 螢幕解析度
   → 1920x1080 太常見、4K 真實
   → 0x0 = headless

6. 時區
   → JS Date 的 timezone offset
   → 跟 IP 國家不一致 = 可疑

7. 沒滑鼠移動
   → 真人會移動滑鼠、爬蟲直接點
   → 鼠標 X/Y 全程沒變動

8. 鍵盤太規律
   → 真人打字快慢不一、有 typo
   → page.type() 預設等距、太規律
```

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
npm install puppeteer-extra puppeteer-extra-plugin-stealth
```

**這行做什麼？** 裝 puppeteer-extra + stealth plugin、**自動修 17+ 個偵測指標**（最簡單方案）。

```text
import puppeteer from 'puppeteer-extra';
import StealthPlugin from 'puppeteer-extra-plugin-stealth';

puppeteer.use(StealthPlugin());

const browser = await puppeteer.launch({ headless: true });
// 之後用法跟 puppeteer 一樣、但通過 95% 偵測
```

> 💡 **stealth plugin 修的指標**：webdriver / chrome.runtime / iframe contentWindow / media codecs / navigator.permissions / navigator.plugins / navigator.languages / navigator.vendor / navigator.platform / WebGL vendor 等 17+ 項。

  
### 解法 2：手動修改（精細控制）

```text
await page.evaluateOnNewDocument(() => {
  // === 1. 移除 webdriver flag ===
  Object.defineProperty(navigator, 'webdriver', {
    get: () => undefined
  });
  
  // === 2. 偽造 plugins（headless 預設空）===
  Object.defineProperty(navigator, 'plugins', {
    get: () => [1, 2, 3, 4, 5]
  });
  
  // === 3. 偽造 languages ===
  Object.defineProperty(navigator, 'languages', {
    get: () => ['zh-TW', 'zh', 'en']
  });
  
  // === 4. 偽造 platform ===
  Object.defineProperty(navigator, 'platform', {
    get: () => 'MacIntel'
  });
  
  // === 5. 防 iframe 偵測 ===
  Object.defineProperty(HTMLIFrameElement.prototype, 'contentWindow', {
    get: function() { return window; }
  });
});
```

> 💡 **`evaluateOnNewDocument`** = **在每個新頁面載入前注入** JS、修改 navigator 等全域物件、躲偵測。

  
### 解法 3：模擬人類行為

```text
// === 1. 人類打字（隨機延遲）===
async function humanType(page, selector, text) {
  for (const char of text) {
    await page.type(selector, char, {
      delay: Math.random() * 200 + 50           // 50-250ms 隨機
    });
  }
}

// === 2. 人類滑鼠移動（路徑 + 步驟）===
await page.mouse.move(100, 200);
await page.waitForTimeout(500);
await page.mouse.move(300, 400, { steps: 10 });  // 10 步分段移動、不瞬移
await page.waitForTimeout(300);
await page.mouse.click(300, 400);

// === 3. 隨機 scroll ===
await page.evaluate(() => {
  window.scrollBy(0, Math.random() * 500 + 200);
});

// === 4. 隨機停頓（看文章樣子）===
await page.waitForTimeout(Math.random() * 3000 + 1000);
```

### 目標網站難度分級

```text
=== 🟢 容易（fetch + UA 即可）===
- 個人部落格
- 政府網站
- 沒反爬的小網站
- WordPress / Ghost 預設

=== 🟡 中等（要 stealth）===
- Cloudflare 基礎防護
- 一般電商（蝦皮 / momo）
- 一般新聞網站
- 大部分 SaaS

=== 🟠 困難（要進階反偵測）===
- Akamai 防護
- DataDome
- PerimeterX
- 大型銀行 / 金融

=== 🔴 地獄（不建議爬）===
- Apple App Store
- Google 搜尋（用官方 API）
- Meta（FB / IG）
- 大型外送平台（UberEats / DoorDash）
```

  
### 爬蟲 ROI 心法

```text
評估前問 3 個問題：

1. 目標有官方 API 嗎？
   → 有 → 用 API（合法 + 穩定 + 不會被封）
   → 沒 → 才考慮爬

2. 反爬難度 vs 商業價值？
   → 難度高 + 價值低 → 放棄
   → 難度低 + 價值高 → 衝
   → 難度高 + 價值高 → 評估投入

3. 法律風險？
   → ToS 禁止商業爬蟲 → 個人學習 OK、商用評估
   → robots.txt 禁止 → 不爬
   → 公開資料 → 通常 OK
```
