**Puppeteer CDP 進階**——L3 教 Puppeteer 基本、這課教 **CDP（Chrome DevTools Protocol）進階**：網路節流 / 模擬 mobile / 攔截 console.log / DevTools 完整功能。

> 💡 **CDP = Chrome 的內部 API**——Puppeteer 是 CDP 的 wrapper、直接用 CDP 可以做 Puppeteer 沒包的功能（GeoLocation / Permission / Coverage / Performance）。

  
### CDP 是什麼

```text
=== Chrome DevTools Protocol（CDP）===

→ Chrome 內部用來控制瀏覽器的 API
→ DevTools（按 F12 那個工具）就是用 CDP
→ Puppeteer 是 CDP 的 JS wrapper
→ 直接用 CDP 可以做更底層的事
```

  
### 取得 CDP Session

```text
import puppeteer from 'puppeteer';

const browser = await puppeteer.launch();
const page = await browser.newPage();

// === 拿到 CDP Session、可以呼叫所有 CDP method ===
const client = await page.target().createCDPSession();
```

  
### 監聽所有網路請求（最完整的 Request 監聽）

```text
await client.send('Network.enable');           // 啟用 Network domain

client.on('Network.requestWillBeSent', e => {
  console.log('Request:', e.request.method, e.request.url);
});

client.on('Network.responseReceived', e => {
  console.log('Response:', e.response.status, e.response.url);
});

client.on('Network.loadingFailed', e => {
  console.log('Failed:', e.errorText, e.requestId);
});
```

> 💡 **vs `page.on('request')`**——CDP 提供更多細節（timing / initiator / priority / mimeType）、適合 debug 網路問題。

  
### 網路節流（模擬慢網路、3G）

```text
await client.send('Network.emulateNetworkConditions', {
  offline: false,
  downloadThroughput: 50 * 1024,             // 50 KB/s（慢 3G）
  uploadThroughput: 20 * 1024,
  latency: 500,                              // 500ms 延遲
});

await page.goto('https://example.com');
// 用慢網路載入、測試 lazy load / loading state
```

  
**預設網速範例：**

```text
=== 慢 3G ===
download: 50 KB/s, upload: 20 KB/s, latency: 500ms

=== 快 3G ===
download: 1.5 MB/s, upload: 750 KB/s, latency: 150ms

=== 4G ===
download: 4 MB/s, upload: 3 MB/s, latency: 70ms

=== Offline（測試 PWA）===
offline: true
```

  
### 模擬 mobile 裝置（不只是 viewport）

```text
// === 用 CDP 完整模擬 iPhone 13 ===
const iPhone13 = puppeteer.KnownDevices['iPhone 13'];
await page.emulate(iPhone13);

// === 或手動：完整 mobile 環境 ===
await client.send('Emulation.setDeviceMetricsOverride', {
  width: 390,
  height: 844,
  deviceScaleFactor: 3,                      // Retina
  mobile: true,
});

await client.send('Emulation.setTouchEmulationEnabled', {
  enabled: true,
});

await client.send('Emulation.setUserAgentOverride', {
  userAgent: 'Mozilla/5.0 (iPhone; CPU iPhone OS 17_2 like Mac OS X) ...',
});
```

> 💡 **為什麼要完整模擬？** 有些網站只看 UA、有些檢查 touch event、有些檢查螢幕尺寸——CDP 全模擬才不會被識破。

  
### GeoLocation 偽造（爬地區限定內容）

```text
await context.overridePermissions('https://example.com', ['geolocation']);
await page.setGeolocation({ latitude: 25.033, longitude: 121.5654 });   // 台北 101

await page.goto('https://example.com/local-news');
// → 網站以為你在台北、回傳台北的內容
```

  
**全球熱門座標：**

```text

紐約：40.7128, -74.0060

```

  
### 攔截 console.log（拿頁面內 log）

```text
page.on('console', msg => {
  const type = msg.type();                   // log / warn / error
  const text = msg.text();
  console.log(`[頁面 ${type}]:`, text);
});

await page.goto('https://example.com');
// → 頁面內所有 console.log 都會顯示在你的 Node 終端
```

  
### Coverage：抓出沒用到的 CSS / JS

```text
await page.coverage.startJSCoverage();
await page.coverage.startCSSCoverage();

await page.goto('https://example.com');

const jsCoverage = await page.coverage.stopJSCoverage();
const cssCoverage = await page.coverage.stopCSSCoverage();

// === 計算「沒用到」的比例 ===
let totalBytes = 0;
let usedBytes = 0;
for (const entry of [...jsCoverage, ...cssCoverage]) {
  totalBytes += entry.text.length;
  for (const range of entry.ranges) {
    usedBytes += range.end - range.start;
  }
}
console.log(`網頁有 ${((1 - usedBytes/totalBytes) * 100).toFixed(1)}% 的 CSS/JS 沒用到`);
```

> 💡 **Coverage 用途**：(1) 前端效能優化（找出 dead code） (2) 競品分析（看對方網站 bundle 多大） (3) 爬蟲時找關鍵 JS 檔。

### Puppeteer vs Playwright（再次對比、給選擇）

| 場景 | Puppeteer | **Playwright** |
|---|---|---|
| **CDP 進階功能** | ⭐ 第一手存取 | 也支援、API 稍包裝 |
| **跨瀏覽器** | ❌ Chromium only | ⭐ 三大瀏覽器 |
| **新專案** | 不推 | ⭐ 首選 |
| **Coverage / Performance** | ⭐ 完整 | 部分 |
| **Google Cloud Functions** | ⭐ 第一方支援 | 也行 |

> 💡 **何時用 Puppeteer？** (1) 需要最深的 CDP 功能 (2) 部署到 Google Cloud / Firebase (3) 維護老 codebase。**其他全選 Playwright**。

  
    **☕ 用人話講**
    
Puppeteer CDP 進階：(1) **CDP** 是 Chrome 內部 API、Puppeteer 是 wrapper (2) **網路節流** 模擬慢 3G、測 loading state (3) **完整 mobile 模擬**（viewport + touch + UA） (4) **GeoLocation 偽造** 爬地區內容 (5) **Coverage** 抓 dead code (6) **Performance metrics** 量效能。**心法**：CDP 是「**爬蟲深度** + **效能分析**」武器、Playwright 也可用、Puppeteer 更原生。
