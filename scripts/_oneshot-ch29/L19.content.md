**爬蟲監控 + 告警**——爬蟲不像 web app、**沒人在用就沒人發現壞了**。要主動監控：crash / 抓量異常 / 沒跑 / 結果空 → **第一時間推到 Slack / Discord / LINE / Telegram**。

> 💡 **Indie 黃金組合**：Sentry（crash）+ Healthchecks.io（liveness）+ Slack/Discord（通知）+ Pino（log）+ GitHub Actions（schedule）= **0 月費完整監控**。

  
### 4 種必監控指標

```text
1. Crash（程式炸了）
   → Sentry 捕捉 exception + stack trace

2. Liveness（有沒有按時跑）
   → Healthchecks.io 心跳監控
   → 排程沒跑、自動告警

3. 抓量異常（昨天 1000 筆、今天 10 筆）
   → 自己寫比較邏輯 + 通知

4. 結果空（成功跑、但沒抓到資料）
   → assertion：if (items.length === 0) notify
```

### Healthchecks.io（Liveness 心跳）

**情境**：爬蟲應該每天跑、結果 server 掛了 3 天沒人發現 → Healthchecks 解決。

```text
1. 去 healthchecks.io 註冊（免費 20 checks）
2. 建 check：「每天必須 ping 一次、超過 24 小時沒 ping 就告警」
3. 拿到 PING_URL（https://hc-ping.com/UUID）
```

  
📄 **寫進程式檔**：

```text
const PING_URL = process.env.HC_PING_URL;

async function main() {
  // === 1. 開始時 ping start ===
  await fetch(`${PING_URL}/start`);
  
  try {
    const items = await scrape();
    
    // === 2. 成功時 ping 一般 URL ===
    await fetch(PING_URL);
    
    console.log(`✓ 抓 ${items.length} 筆`);
  } catch (err) {
    // === 3. 失敗時 ping /fail ===
    await fetch(`${PING_URL}/fail`, {
      method: 'POST',
      body: err.message,
    });
    throw err;
  }
}
```

> 💡 **Healthchecks 邏輯**：(1) 沒按時 ping = 告警 (2) ping /fail = 告警 (3) 全成功 = 不打擾。**Liveness 監控就 1 個 URL**、超簡單。

### Discord 通知

async function notifyDiscord(text: string, items?: any[]) {
  await fetch(process.env.DISCORD_WEBHOOK!, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      content: text,
      embeds: items ? [{

        color: 0x00ff00,
        fields: [
          { name: '筆數', value: items.length.toString(), inline: true },

      }] : undefined,
    })
  });
}

async function detectAnomaly(today: number) {

### 完整監控架構範例

```text
import * as Sentry from '@sentry/node';
import pino from 'pino';

Sentry.init({ dsn: process.env.SENTRY_DSN });
const logger = pino();

const HC_PING = process.env.HC_PING_URL!;

async function main() {
  const startTime = Date.now();
  
  await fetch(`${HC_PING}/start`);
  logger.info('Scraper started');
  
  try {
    const items = await scrape();
    
    if (items.length === 0) {
      throw new Error('No items scraped');
    }
    
    await detectAnomaly(items.length);
    
    const duration = ((Date.now() - startTime) / 1000).toFixed(1);
    
    await fetch(HC_PING);
    await notifySlack(`✅ 完成：${items.length} 筆、用時 ${duration}s`);
    
    logger.info({ count: items.length, duration }, 'Scrape complete');
    
  } catch (err) {
    Sentry.captureException(err);
    logger.error({ err }, 'Scrape failed');
    await fetch(`${HC_PING}/fail`);
    await notifySlack(`❌ 失敗：${(err as Error).message}`);
    process.exit(1);
  }
}

main();
```
