**Crawlee = Apify 出的 Node 爬蟲框架**——Node 版的 Scrapy、生產級爬蟲必備。**Request Queue + Auto-scaling + Proxy Rotation 全自動**。

> 💡 **何時用 Crawlee？** 爬量 > 1 萬頁、需要長時間運行、要部署到雲端時。**小爬蟲（< 1000 頁）用 fetch + cheerio 就好**。

### 三種 Crawler（依場景選）

| Crawler | 用途 | 速度 | 何時用 |
|---|---|---|---|
| **CheerioCrawler** | 純 HTML 解析 | ⭐⭐⭐ 最快 | 靜態網站 |
| **PlaywrightCrawler** | JS 渲染 | ⭐⭐ 中 | SPA / 動態網站 |
| **PuppeteerCrawler** | JS 渲染（legacy） | ⭐⭐ 中 | 維護老 code |

  
### CheerioCrawler 範例（最快、靜態網站）

```text
import { CheerioCrawler, Dataset } from 'crawlee';

const crawler = new CheerioCrawler({
  maxConcurrency: 10,                        // 並發數（同時 10 個 request）
  maxRequestsPerCrawl: 1000,                 // 最多爬 1000 頁就停
  requestHandlerTimeoutSecs: 30,             // 每頁最多 30 秒
  
  // === 每爬一頁、Crawlee 自動呼叫這個函式 ===
  async requestHandler({ request, $, enqueueLinks, log }) {
    log.info(`Processing ${request.url}`);
    
    // $ = cheerio（跟 L2 用法一樣）
    const title = $('h1').text();
    const price = $('.price').text();
    
    // === 存進 Dataset（自動寫 JSON 到 storage/）===
    await Dataset.pushData({
      url: request.url,
      title,
      price,
    });
    
    // === 找下一頁、加進 queue ===
    await enqueueLinks({
      selector: 'a.next-page',
      label: 'LIST',                         // 分類標籤
    });
    
    // === 找商品連結 ===
    await enqueueLinks({
      selector: 'a.product',
      label: 'DETAIL',
    });
  },
  
  maxRequestRetries: 3,                      // 失敗自動重試 3 次
});

// === 起跑點（給一個或多個 URL）===
await crawler.run(['https://example.com/products']);

// 完成後：storage/datasets/default/*.json 內有所有結果
```

> 💡 **Crawlee 的核心概念**：(1) `Request Queue` = 要爬的 URL 清單、自動去重 (2) `Dataset` = 結果儲存、自動寫 JSON。

  
### PlaywrightCrawler 範例（動態網站）

```text
import { PlaywrightCrawler, Dataset } from 'crawlee';

const crawler = new PlaywrightCrawler({
  launchContext: {
    launchOptions: { headless: true },
  },
  maxConcurrency: 5,                         // 開 5 個瀏覽器 tab
  
  async requestHandler({ page, request }) {
    await page.goto(request.url);
    await page.waitForSelector('.product');
    
    // === 在頁面內跑 JS、取 DOM ===
    const products = await page.$$eval('.product', els =>
      els.map(el => ({
        name: el.querySelector('.name')?.textContent,
        price: el.querySelector('.price')?.textContent,
      }))
    );
    
    await Dataset.pushData({ url: request.url, products });
  },
});

await crawler.run(['https://shop.example.com']);
```

  
### Crawlee 7 大殺手功能（vs 自己寫）

```text
✅ Request Queue
   → 自動去重、進度持久化（程式 crash 重啟接著爬）
   → 不會重複爬同一個 URL

✅ Auto-scaling
   → 依 CPU / Memory 自動調並發數
   → 不會把機器爆掉

✅ Proxy Rotation
   → 自動換 IP、防被封
   → 接 Apify Proxy / 自己的 proxy pool

✅ Session 管理
   → cookie / fingerprint 自動處理
   → 偽裝多個使用者

✅ Retry / Error handling
   → 4xx / 5xx 自動重試
   → 加指數退避

✅ Dataset
   → 自動存 JSON / CSV / Excel
   → 不用自己寫 fs.writeFile

✅ Key-Value Store
   → 跨 request 共享狀態
   → 例：存登入 cookie 給所有 worker 用
```

  
### 完整範例：CheerioCrawler + 多層級爬蟲

```text
import { CheerioCrawler, Dataset } from 'crawlee';

const crawler = new CheerioCrawler({
  maxConcurrency: 5,
  
  async requestHandler({ request, $, enqueueLinks }) {
    // === 用 label 區分頁面類型 ===
    if (request.label === 'DETAIL') {
      // 商品詳情頁、抓資料
      await Dataset.pushData({
        url: request.url,
        name: $('h1').text().trim(),
        price: $('.price').text(),
        description: $('.desc').text(),
      });
    } else {
      // 列表頁、找商品連結
      await enqueueLinks({
        selector: 'a.product-link',
        label: 'DETAIL',                     // 標記為詳情頁
      });
      
      // 也找下一頁
      await enqueueLinks({
        selector: 'a.next',
        label: 'LIST',
      });
    }
  },
});

await crawler.run(['https://example.com/products']);
```
