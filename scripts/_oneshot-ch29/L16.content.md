**RSS + Sitemap = 網站主動給你的資料**——**100% 合法 + 友善爬蟲**、不會被擋、ToS 都允許。**爬蟲首選**——先看 RSS / Sitemap、找不到才考慮爬 HTML。

> 💡 **為什麼合法？** RSS 是網站「**主動提供**」給 RSS reader 用的、Sitemap 是「**告訴 Google 來爬**」用的——**你拿就是合理使用**。

  
### RSS 是什麼

```text
=== RSS = Really Simple Syndication ===

→ 網站主動匯出「最新文章列表」的 XML
→ 部落格 / 新聞網站 / Podcast 都有
→ RSS reader（如 Feedly、Inoreader）就是用這個

格式範例：
<rss>
  <channel>
    <title>科技新聞</title>
    <item>
      <title>iPhone 17 發表</title>
      <link>https://news.com/iphone-17</link>
      <pubDate>Mon, 27 May 2026 10:00:00 GMT</pubDate>
      <description>蘋果今天發表了 iPhone 17...</description>
    </item>
    <item>...</item>
  </channel>
</rss>
```


**這行做什麼？** 裝 `rss-parser`、Node 最常用的 RSS 解析套件、支援 RSS / Atom 兩種格式。

```text
import Parser from 'rss-parser';

const parser = new Parser();
const feed = await parser.parseURL('https://example.com/rss');

console.log(feed.title);                     // 「科技新聞」
console.log(feed.description);

for (const item of feed.items) {
  console.log(item.title);
  console.log(item.link);
  console.log(item.pubDate);
  console.log(item.contentSnippet);          // 摘要（去 HTML）
}
```

> 💡 **rss-parser 自動處理**：RSS 2.0 / Atom / RDF 三種格式、UTF-8 / Big5 / GBK 編碼、CDATA 等怪格式——**全自動**。

### Sitemap 是什麼

```text
=== Sitemap = 網站 URL 清單 ===

→ 給 Google 用、告訴它「我網站有這些頁面」
→ 通常在 https://example.com/sitemap.xml
→ 也可看 robots.txt 找：Sitemap: https://example.com/sitemap.xml

格式：
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://example.com/page/1</loc>
    <lastmod>2026-05-17</lastmod>
    <changefreq>daily</changefreq>
    <priority>0.8</priority>
  </url>
  <url>...</url>
</urlset>
```

```text
npm install fast-xml-parser
```

**這行做什麼？** 裝 fast-xml-parser、Node 最快的 XML 解析套件、適合大型 sitemap（10 萬+ URL）。

```text
import { XMLParser } from 'fast-xml-parser';

const res = await fetch('https://example.com/sitemap.xml');
const xml = await res.text();

const parser = new XMLParser();
const data = parser.parse(xml);

// === 取所有 URL ===
const urls = data.urlset.url.map(u => ({
  url: u.loc,
  lastmod: u.lastmod,
  priority: u.priority,
}));

console.log(`網站有 ${urls.length} 個頁面`);
```

  
### Sitemap Index（大網站常見）

**大網站的 sitemap 太大（> 50MB / 5 萬 URL）、會分多個檔：**

```text
https://example.com/sitemap.xml         → sitemap index
  ├── sitemap-products.xml              → 商品頁
  ├── sitemap-articles.xml              → 文章頁
  └── sitemap-categories.xml            → 分類頁
```

```text
async function getAllSitemaps(indexUrl) {
  const res = await fetch(indexUrl);
  const xml = await res.text();
  const data = new XMLParser().parse(xml);
  
  // === Sitemap Index ===
  if (data.sitemapindex) {
    const subSitemaps = data.sitemapindex.sitemap.map(s => s.loc);
    
    // === 遞迴抓所有子 sitemap ===
    const allUrls = [];
    for (const subUrl of subSitemaps) {
      const sub = await getAllSitemaps(subUrl);
      allUrls.push(...sub);
    }
    return allUrls;
  }
  
  // === 一般 Sitemap ===
  return data.urlset.url.map(u => u.loc);
}

const allUrls = await getAllSitemaps('https://example.com/sitemap.xml');
console.log(`總共 ${allUrls.length} 個 URL`);
```

  
### 為什麼 RSS / Sitemap 合法

```text
✅ 網站主動提供
   → RSS 給 RSS reader 用
   → Sitemap 給 Google 用
   → 你拿是合理使用

✅ robots.txt 通常 allow
   → 看 robots.txt 確認
   → 大多明確允許

✅ SEO 需要、不能擋
   → 擋了 Google 也爬不到
   → 影響 SEO 排名

✅ 量小、不會 DDoS
   → 一個 sitemap 一次抓
   → 不像爬 HTML 一頁頁打
```

### 完整範例：RSS → 通知 LINE / Telegram

📄 **寫進程式檔**：`monitor-rss.ts`

```text
import Parser from 'rss-parser';
import fs from 'fs/promises';

const RSS_URL = 'https://news.com/rss';
const SEEN_FILE = 'seen.json';

async function main() {
  // === 讀已看過的文章 ID ===
  let seen = new Set();
  try {
    seen = new Set(JSON.parse(await fs.readFile(SEEN_FILE, 'utf8')));
  } catch {}
  
  // === 抓 RSS ===
  const feed = await new Parser().parseURL(RSS_URL);
  
  // === 找新文章 ===
  const newArticles = feed.items.filter(item => !seen.has(item.link));
  
  if (newArticles.length === 0) {
    console.log('沒新文章');
    return;
  }
  
  // === 推 Telegram ===
  for (const article of newArticles) {
    await sendTelegram(`📰 ${article.title}\n${article.link}`);
    seen.add(article.link);
  }
  
  // === 存已看 ===
  await fs.writeFile(SEEN_FILE, JSON.stringify([...seen]));
  console.log(`推了 ${newArticles.length} 個新文章`);
}

async function sendTelegram(text) {
  await fetch(`https://api.telegram.org/bot${process.env.BOT_TOKEN}/sendMessage`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      chat_id: process.env.CHAT_ID,
      text,
    })
  });
}

main();
```
