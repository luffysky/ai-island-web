**fetch + cheerio = Node 爬蟲基本盤**——對應 Python 的 `requests + BeautifulSoup`、80% 場景夠用。

> 💡 **Node 18+ 內建 fetch**——不用裝套件、原生 API。Cheerio 提供 jQuery-like 語法解 HTML、會 jQuery 1 分鐘上手。

### 第一個爬蟲（3 步驟）

```text
import * as cheerio from 'cheerio';

async function scrape(url) {
  // === 1. fetch 抓網頁 ===
  const res = await fetch(url, {
    headers: {
      'User-Agent': 'Mozilla/5.0 (Macintosh; Intel Mac OS X) Chrome/131.0'
    }
  });
  
  // === 2. 拿 HTML 文字 ===
  const html = await res.text();
  
  // === 3. cheerio 解析（用 CSS selector、跟 jQuery 一樣）===
  const $ = cheerio.load(html);
  const title = $('h1').text();
  const links = $('a').map((_, el) => $(el).attr('href')).get();
  
  return { title, links };
}

scrape('https://example.com').then(console.log);
```

> 💡 **慣例**：cheerio 載入後**用 `$` 命名**（跟 jQuery 一致）、寫起來像在用 jQuery。

  
### 實戰：爬 PTT 八卦版

```text
import * as cheerio from 'cheerio';
import fs from 'fs/promises';

async function scrapePTT(board = 'Gossiping') {
  const url = `https://www.ptt.cc/bbs/${board}/index.html`;
  
  const res = await fetch(url, {
    headers: {
      'User-Agent': 'Mozilla/5.0',
      'Cookie': 'over18=1',                  // ⚠️ PTT 八卦版要 18 歲確認
    },
  });
  const html = await res.text();
  const $ = cheerio.load(html);
  
  const posts = [];
  $('.r-ent').each((_, el) => {
    const $el = $(el);
    const $title = $el.find('.title a');
    if ($title.length === 0) return;          // 防被刪文章（沒 a 元素）
    
    posts.push({
      title: $title.text().trim(),
      link: 'https://www.ptt.cc' + $title.attr('href'),
      author: $el.find('.author').text(),
      push: $el.find('.nrec').text() || '0',
      date: $el.find('.date').text(),
    });
  });
  
  return posts;
}

const posts = await scrapePTT('Stock');
console.log(`抓到 ${posts.length} 篇文章`);

// === 存 JSON（Node fs 操作）===
await fs.writeFile('posts.json', JSON.stringify(posts, null, 2));
```

  
### 並發抓多頁（Node 強項、超簡單）

**Promise.all 一行並發、比 Python asyncio 簡潔：**

```text
const urls = Array.from({ length: 10 }, (_, i) =>
  `https://www.ptt.cc/bbs/Gossiping/index${i + 1}.html`
);

// === 同時抓 10 頁、5 秒搞定 ===
const allPages = await Promise.all(
  urls.map(url => scrapePTT(url))
);
```

  
**控制並發數（避免被擋、用 p-limit）：**

npm install p-limit

**這行做什麼？** 裝 `p-limit`、Node 的並發控制套件、相當於 Python 的 `asyncio.Semaphore`。

```text
import pLimit from 'p-limit';

const limit = pLimit(3);                     // 最多同時 3 個 request

const results = await Promise.all(
  urls.map(url => limit(() => scrapePTT(url)))
);
// 自動排隊、永遠不超過 3 個並發
```

  
### Cheerio 進階用法（會 jQuery 直接會）

**取屬性：**

```text
$('a').attr('href')                          // 第一個 a 的 href
$('img').attr('src')

// === map + get 取多個（Cheerio 慣例）===
$('[data-id]').map((_, el) => $(el).attr('data-id')).get()
// .map() 回 cheerio 物件、.get() 變陣列
```

  
**找子 / 父 / 兄弟元素：**

```text
$('.product').find('.price')                 // 後代
$('.product').children('.title')             // 直接子
$('.product').parent()                       // 父
$('.product').siblings()                     // 兄弟
$('.product').closest('section')             // 最近的祖先 section
```

  
**過濾：**

```text
$('a').filter((_, el) => $(el).text().includes('下載'))
// 找文字含「下載」的 a
```

  
**取文字 vs HTML：**

```text
$('.content').text()                         // 純文字（含子元素）
$('.content').html()                         // HTML 字串
$('.content').get(0).tagName                 // tag 名稱
```

  
**修改 DOM（爬蟲少用、但偶爾要）：**

```text
$('.ads').remove();                          // 刪廣告
$('.product').addClass('selected');          // 加 class
const cleanHtml = $.html();                  // 取整個修改後的 HTML
```

  
### fetch 進階用法

**1. POST + JSON body：**

```text
const res = await fetch('https://api.example.com/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ email: 'a@b.com', password: 'secret' }),
});
```

  
**2. 帶 cookie（手動）：**

```text
const res = await fetch(url, {
  headers: {
    'Cookie': 'sessionid=abc123; over18=1',
  },
});
```

  
**3. Timeout（用 AbortController）：**

```text
const controller = new AbortController();
const timeout = setTimeout(() => controller.abort(), 10_000);

try {
  const res = await fetch(url, { signal: controller.signal });
  // ...
} finally {
  clearTimeout(timeout);
}
```

### 4 大常見坑

```text
1. ❌ 忘記 await
   const html = res.text();              // 沒 await、html 是 Promise
   const $ = cheerio.load(html);         // 錯
   
   ✓ const html = await res.text();

2. ❌ package.json 沒加 "type": "module"
   → import 用不了、要全用 require
   
   ✓ 加 "type": "module"

3. ❌ 沒設 User-Agent → 被擋
   ✓ 永遠帶真實 UA

4. ❌ 沒處理「刪文 / 空節點」NoneType
   ✓ if ($title.length === 0) return;
```
