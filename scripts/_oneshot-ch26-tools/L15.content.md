**編輯器 + Jupyter + Colab 工具大全**——L1 教你裝 Python 跟 uv、這課教你**用什麼工具寫 Python**。

> 💡 **新手最常問**：「我裝完 Python 之後在哪寫 code？」答案 = **編輯器**。本課給你完整選項。

  
### 寫 Python 的 4 大方式

```text
1. 📝 編輯器（Editor）
   → 寫成 .py 檔、終端機跑
   → 最主流、業界標準

2. 📓 Jupyter Notebook
   → 一格一格寫 code + 看結果
   → 資料分析 / ML 最愛

3. ☁️ Google Colab
   → 雲端 Jupyter、不用裝
   → 免費 GPU、學習友善

4. 🐢 REPL（互動模式）
   → 一行一行試
   → 學語法、快速測試
```

> 💡 **2026 推薦**：(1) 一般寫程式 → **VS Code + Cursor** (2) 資料分析 → **Jupyter** (3) 學習 / 試 GPU → **Colab**。

### 1. VS Code（最推薦新手）

**為什麼**：免費 + 全平台（Mac / Windows / Linux）+ 業界最多人用 + 擴充強。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **去網站下載 + 安裝**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
1. 去 https://code.visualstudio.com 下載
2. Mac：把 VS Code.app 拖進 Applications
3. Windows：跑 installer、選「加入 PATH」
4. Linux：apt install code（Ubuntu）
```

  
**裝完必裝 5 個擴充：**

```text
🐍 Python（Microsoft 官方）
   → 自動補完 + linter + debug
   → 必裝

📓 Jupyter（Microsoft 官方）
   → VS Code 內跑 .ipynb
   → 必裝

🎨 Pylance
   → Python 強化 type checker
   → 跟 Python 一起裝

🤖 GitHub Copilot（要訂閱 $10/月、學生免費）
   → AI 自動寫 code
   → 強烈推薦

🎯 Ruff
   → 超快 linter / formatter
   → 取代 black / flake8 / isort
```

  
**第一次設定：**

```text
1. 開 VS Code
2. 開資料夾（Cmd/Ctrl + O）
3. 新建 hello.py
4. 寫 print("hello")
5. 右下角選 Python interpreter（uv 環境）
6. 按 F5（或右上角 ▶）跑
```

### 4. Jupyter Notebook（資料分析必裝）

**Jupyter = 互動式筆記本**——一格一格寫 code、看結果、加 markdown 註解。

**範例：**

```text
========================
📝 Markdown cell
========================
# 我的第一個分析
今天分析房價資料

========================
💻 Code cell
========================
import pandas as pd
df = pd.read_csv("data.csv")
df.head()

========================
📊 結果（自動顯示）
========================
[5 rows × 8 columns 的漂亮表格]

========================
💻 Code cell
========================
df['price'].hist()

========================
📊 結果
========================
[直方圖、直接畫在 cell 下面]
```

  
**安裝（用 uv）：**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add jupyter notebook
uv run jupyter notebook
```

**這 2 行做什麼？**
- `uv add jupyter notebook` —— 裝 Jupyter（會自動加進 pyproject.toml）
- `uv run jupyter notebook` —— 啟動本機 Jupyter server、自動開瀏覽器

  
**或者用 VS Code 內 Jupyter（推薦）：**

```text
1. VS Code 裝 Jupyter 擴充
2. 新建 .ipynb 檔
3. 右上角選 Python interpreter
4. 寫 code、Shift+Enter 跑
```

> 💡 **2026 主流**：VS Code 內用 Jupyter > 純 Jupyter server。**統一 IDE、不用切換**。

### REPL（互動模式、最簡單）

**REPL = Read-Eval-Print Loop**——「**邊讀邊跑邊印**」、終端機開個 Python 直接寫。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
python
```

**這行做什麼？** 進入 Python REPL（會看到 `>>>` 提示符）。

```text
>>> 1 + 1
2
>>> name = "luffy"
>>> print(f"hello {name}")
hello luffy
>>> exit()
```

> 💡 **進階 REPL**：用 `ipython`（pip install ipython）—— 比預設 REPL 強 100x（自動補完 / 顏色 / magic commands）。

  
### IPython（升級版 REPL）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
uv add ipython
uv run ipython
```

**這 2 行做什麼？**
- `uv add ipython` —— 裝 IPython（互動式 Python shell 加強版）
- `uv run ipython` —— 啟動 IPython

**IPython 殺手特性：**

```text
✅ 自動補完（Tab）
✅ 語法高亮
✅ Magic commands

   - %timeit 跑多次取平均
   - %run script.py 跑檔案
✅ 直接 paste 多行 code
✅ 上下鍵翻歷史
```

### 4 大常見坑

```text
1. ❌ 同時裝多個 Python interpreter 搞混
   → VS Code 右下角永遠確認用哪個 interpreter
   ✅ 用 uv 管理、每專案一個 .venv

2. ❌ 在 VS Code 開新檔忘了選 interpreter
   → 跑起來 import 找不到套件
   ✅ 永遠先選 interpreter 再寫 code

3. ❌ Jupyter cell 順序錯
   → 上面 cell 改了沒重跑、下面用舊變數
   ✅ 養成「Restart and Run All」習慣

4. ❌ Colab 12 小時自動斷
   → 重要訓練不要用免費版
   ✅ 付 Colab Pro $10/月、12 小時 → 24 小時
```
