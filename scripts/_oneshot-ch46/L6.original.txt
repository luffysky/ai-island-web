LLM 沒有「最強」—**每個任務各有冠軍**。這節給你完整對照。

  
### 各家殺手鐧

  
#### OpenAI

  
- 最完整生態（Assistants、Realtime、Vision、TTS、Whisper、DALL-E、Sora）
- o3 推理模型（數學、競賽題強）
- Sora 影片生成
- 市場最大、開發者社群最多
- 價格中等偏高
- 多次大規模當機（2024 出包多）

  
#### Anthropic（Claude）

  
- 程式碼能力業界第一（SWE-bench 領先）
- Claude Code（agentic coding CLI）
- MCP（Model Context Protocol）— 跨應用整合標準
- Computer Use（操控電腦）
- 長文寫作質量高
- 誠實度 / 安全性業界最好（會說「我不確定」）
- 較貴（Opus 是最貴的旗艦）
- 2024-2026 開發者最愛

  
#### Google（Gemini）

  
- 最長 context（2M token，整套魔戒能塞）
- Gemini Flash 超便宜（業界最低 $0.075/M）
- 跟 Google 服務整合（Drive、Workspace、Search）
- Veo 影片生成（對標 Sora）
- Deep Think 推理模式
- 多模態強（圖、聲、影、PDF 都吃）
- 免費額度大

  
#### xAI（Grok）

  
- 跟 X 整合（即時推文資料）
- 較少安全限制（爭議性話題能答）
- 即時新聞訪問
- 2026 進步快但生態小

  
#### 開源王者

  
- **Llama 4 系列**：Meta 出，社群最大、文件最多
- **Qwen 3**：Alibaba，中文最強、多語言好
- **DeepSeek V3 / R1**：對標 GPT-4 但開源（2024 引爆業界）
- **Mistral / Mixtral**：歐洲、效能 vs size 比例好
- **Gemma 3**：Google 出，小但能上手機

### 多模型策略（業界主流）

  
沒有人「只用一家」。最佳策略：**依任務路由**。

```text
輸入請求
   ↓
路由判斷：
- 簡單問答（FAQ 風格）→ Gemini Flash（便宜）
- 一般對話 → GPT-4o-mini
- 程式碼 → Claude Sonnet（最準）
- 推理難題 → o3（最強）
- 長文摘要（> 100k）→ Gemini 2.5 Pro
- 視覺 OCR → GPT-4o
- 個人隱私資料 → 自架 Llama
- 兜底 → GPT-4o

效益：
- 成本省 30-70%
- 能力組合更全面
- 單家當機也不影響
```

新手常見錯誤
