2024-2026 LLM 進化成多模態—不只文字。

  
### 視覺（Vision）

  
給圖片、AI 看懂內容。所有主流 LLM 都支援。

```python
# Anthropic Vision
import base64

with open("image.jpg", "rb") as f:
    image_b64 = base64.b64encode(f.read()).decode()

res = client.messages.create(
    model="claude-sonnet-4-6",
    max_tokens=1024,
    messages=[{
        "role": "user",
        "content": [
            {
                "type": "image",
                "source": {
                    "type": "base64",
                    "media_type": "image/jpeg",
                    "data": image_b64
                }
            },
            {"type": "text", "text": "描述這張圖"}
        ]
    }]
)
```

  
### 視覺應用

  
- **OCR**：圖片轉文字（比傳統 OCR 強很多）
- **圖表分析**：看圖回答資料問題
- **產品識別**：拍照搜尋
- **醫療影像**（要 fine-tune）
- **UI 截圖 → 程式碼**（v0、Bolt、Lovable）
- **無障礙**：為盲人描述圖片
- **document understanding**：發票、合約抽資料

  
### 語音輸入（STT）

  
- **OpenAI Whisper**：開源 + API、最準
- **OpenAI gpt-4o-transcribe**：更準
- **Deepgram**：商務 / 即時
- **AssemblyAI**：易用
- **Faster-Whisper**：自架快 4 倍

```python
from openai import OpenAI
client = OpenAI()

with open("audio.mp3", "rb") as f:
    transcript = client.audio.transcriptions.create(
        model="whisper-1",
        file=f,
        language="zh"
    )
print(transcript.text)
```

  
### 語音合成（TTS）

  
- **ElevenLabs**：最自然、可 clone 聲音
- **OpenAI TTS**：便宜、好用
- **Google Cloud TTS**：多語
- **Azure Speech**：企業
- **Suno / Bark**：開源

  
### 圖片生成

  
- **Midjourney**：藝術 / 設計感最強
- **Flux**（Black Forest Labs）：開源、寫實
- **DALL-E 3**：OpenAI、prompt 理解好
- **Stable Diffusion**：完全開源、可 fine-tune
- **Ideogram**：擅長有文字的圖

  
### 影片生成

- **Veo 3**（Google）：對標 Sora

- **Luma Dream Machine**
