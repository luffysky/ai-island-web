**神經網路：一張圖看懂**——不用學數學、**用直覺理解**就好。這節懂了之後、後面 Transformer、LLM 都好懂。

> 💡 **2026 心法**：所有 AI 模型（GPT-4 / Stable Diffusion / AlphaGo）**本質都是神經網路、只是更深更廣**。

  
### 神經網路長什麼樣

```text
輸入層      隱藏層 1     隱藏層 2     輸出層

  ●────────●─────────●──────────●  (貓 0.9)
  ●────────●─────────●──────────●  (狗 0.1)
  ●────────●─────────●
  ●────────●─────────●
  ●────────●─────────●

  (圖片      (中間        (中間        (預測
   pixel)    特徵)        特徵)        結果)

每個「●」叫一個神經元（neuron）
每條線有一個權重（weight）
資料從左到右流動
```

> 💡 **把這張圖記在腦中**——所有 AI 模型本質都是這結構、只是更深更廣。

  
### 單個神經元怎麼運作？

一個神經元收到多個輸入、每個輸入有自己的權重：

```text
=== 神經元 A 計算流程 ===

  輸入1 (0.5)  ─權重 0.3─┐
  輸入2 (0.8)  ─權重 0.6─┼─┐
  輸入3 (0.2)  ─權重 0.1─┘ │
                          │
                       [加總 + bias]
                          │
                          ↓
                       0.5×0.3 + 0.8×0.6 + 0.2×0.1 + 0.1 = 0.78
                          ↓
                       [啟動函式（Activation Function）]
                          ↓
                       ReLU(0.78) = 0.78
                          ↓
                       輸出 0.78（傳給下一層）
```

> 💡 **每個神經元像「投票器」**——收到很多人投票（輸入）、每個人的票有不同份量（權重）、加總後決定要不要往下傳。**權重 = 「我相信這個輸入多少」**。訓練神經網路 = 找出每條線的最佳權重。

  
### 啟動函式（Activation Function）

**為什麼要啟動函式？** 因為沒有它、多層神經網路**等於一層**（線性的線性還是線性）。啟動函式加入「**非線性**」、讓網路能學複雜模式。

  
**常見啟動函式：**

| 名稱 | 公式 | 何時用 |
|---|---|---|
| **ReLU** | max(0, x) | 最常用、簡單快 |
| **Sigmoid** | 1/(1+e^(-x)) | 早期常用、現在較少 |
| **Tanh** | 雙曲正切 | 擠壓到 -1 到 1 |
| **Softmax** | 多分類用 | 輸出層、機率加起來 = 1 |
| **GELU** | 近 ReLU | Transformer 內常用 |

  
### 多層神經網路怎麼運作

```text
輸入：[0.5, 0.8, 0.2, 0.1, 0.9]（5 個 pixel 值）

第 1 層（10 個神經元）：
  每個都收到 5 個輸入、各算自己的輸出
  得到 [0.78, 0.32, 0.91, ...]（10 個值）

第 2 層（10 個神經元）：
  每個收到上一層的 10 個輸出
  得到 [0.45, 0.67, ...]（10 個值）

輸出層（2 個神經元 = 貓 / 狗）：
  得到 [0.9, 0.1]
  Softmax → 「貓 90%、狗 10%」
```

  
### 怎麼訓練？反向傳播（Backpropagation）

**關鍵問題**：權重從哪來？怎麼決定哪個權重是多少？

  
**步驟：**

```text
1. 初始化：所有權重隨機填一個小數字

2. Forward pass（前向傳播）
   → 給訓練資料、跑一遍網路、得到預測

3. 算誤差（Loss）
   → 「預測 0.6 是貓、正確答案是 1.0」
   → 誤差 0.4

4. Backward pass（反向傳播）
   → 把誤差從輸出層往回傳
   → 告訴每個權重「你需要調多少才能讓誤差變小」

5. 更新權重（Gradient Descent、梯度下降）
   → 依照剛剛算的調整方向
   → 往「讓誤差變小」的方向微調

6. 重複 2-5 幾百萬次

7. 最終所有權重就是「能正確分類貓狗」的權重
```

  
**背後是 Gradient Descent（梯度下降）+ Chain Rule（鏈式法則）**。不用懂數學、但要知道：

```text
🎯 Learning rate（學習率）
   → 每次調權重的「步伐大小」
   → 太大 → 跳過最佳解
   → 太小 → 訓練超慢
   → 常見：1e-3 到 1e-5

📦 Batch size（批次大小）
   → 每次看多少筆資料才更新一次
   → 常見：32-512

🔁 Epoch（迭代輪次）
   → 把整個訓練集走完一遍
   → 一般訓練 10-100 epoch

📉 Loss function（損失函數）
   → 衡量誤差的方式
   → MSE（迴歸用）/ Cross Entropy（分類用）

🚀 Optimizer（優化器）
   → 怎麼更新權重
   → SGD / Adam / AdamW（最常用）
```

  
### 為什麼要「深」？

```text
=== 單層神經網路 ===
只能學「線性」關係（直線分類）：
- 房價 vs 坪數：可能成線性
- 貓 vs 狗：絕對不是線性、單層做不到

=== 多層能學複雜模式 ===
第 1 層：學「邊」（直線、曲線）
第 2 層：學「形狀」（圓、方）
第 3 層：學「組合」（鼻子、眼睛）
第 4 層：學「物件」（臉）
第 5 層：學「概念」（這是 Luffy 的臉）

層越多 = 能學的概念越抽象
但也越容易過擬合（overfitting）
越難訓練、越貴
```

  
### 常見神經網路類型

| 類型 | 用途 |
|---|---|
| **MLP / Fully Connected** | 基本款、全連接 |
| **CNN**（Convolutional Neural Network） | 影像識別、用卷積核掃描局部 |
| **RNN / LSTM** | 序列資料、看上一步隱藏狀態 |
| **Transformer**（L4 詳教） | 2017 後主流 |
| **GAN**（Generative Adversarial Network） | 兩個網路對抗（產生 + 判別） |
| **Diffusion** | 從噪音去噪變成圖（2022+ 圖片 AI 主流） |
| **VAE**（Variational Autoencoder） | 壓縮 + 重建 |
| **Graph Neural Network** | 處理圖結構資料 |

  
### 規模感（參數量）

**參數**（parameters）= 神經元之間的權重 + bias 總數量。代表模型「腦容量」。

| 模型 / 年代 | 參數量 | 備註 |
|---|---|---|
| 1986 早期神經網路 | 幾千 | 跑在 386 電腦 |
| 2012 AlexNet | 6000 萬 | 第一個 ImageNet 大勝 |
| 2014 VGG-16 | 1.4 億 | 影像分類主力 |
| 2018 BERT-large | 3.4 億 | NLP 起飛 |
| 2019 GPT-2 | 15 億 | OpenAI 一開始不敢全公開 |
| 2020 GPT-3 | 1750 億 | 真正的「LLM」 |
| 2023 GPT-4（估） | 1-2 兆 | MoE 架構 |
| 2024-2026 前沿 | 2-10 兆 | Claude Opus / Gemini Ultra |

> 💡 **每個參數需要記憶體存**。**1 兆參數模型 ≈ 2 TB RAM（FP16）**。所以大模型需要分散式 GPU。

  
### 學 AI 不用學數學嗎？

看你的目標：

```text
✅ 用 AI 做產品（indie hacker / 工程師）
   → 完全不用懂數學
   → 本章 + L11 結構化輸出就夠
   → 學 prompt + RAG + tool use = 能做強的 AI 產品

⚠️ 調 AI / 訓練自己模型（ML 工程師）
   → 要懂線性代數
   → 機率 + 統計
   → 微積分基本

❌ AI 研究員（發 paper）
   → 數學 hardcore
   → 看 paper / 寫 math
   → 大學等級數學
```
