2024 推出，**2026 TS 生態的 LangChain 替代**。專為前後端整合設計。

  
### 為什麼紅？

  
- **跨模型抽象**：換 OpenAI / Anthropic / Gemini 一行
- **串流首選**：完整 React hook 支援
- **Type-safe**：完整 TS 支援
- **Tool calling 簡潔**
- **結構化輸出** with Zod
- **Multi-modal**：圖、音、影都支援
- **跟 Next.js 完美整合**

  
### 基本使用

```ts
import { generateText } from "ai";
import { openai } from "@ai-sdk/openai";

const { text } = await generateText({
  model: openai("gpt-4o"),
  prompt: "哈囉"
});

// 換 Anthropic 只要改一行：
import { anthropic } from "@ai-sdk/anthropic";
const { text } = await generateText({
  model: anthropic("claude-sonnet-4-6"),
  prompt: "哈囉"
});
```

  
### 串流（前端）

```ts
// app/api/chat/route.ts
import { streamText } from "ai";
import { anthropic } from "@ai-sdk/anthropic";

export async function POST(req: Request) {
  const { messages } = await req.json();
  
  const result = await streamText({
    model: anthropic("claude-sonnet-4-6"),
    messages
  });
  
  return result.toDataStreamResponse();
}
```

### 結構化輸出

```ts
import { generateObject } from "ai";
import { z } from "zod";

const { object } = await generateObject({
  model: openai("gpt-4o"),
  schema: z.object({
    recipe: z.object({
      name: z.string(),
      ingredients: z.array(z.string()),
      steps: z.array(z.string())
    })
  }),
  prompt: "生成義大利麵食譜"
});

object.recipe.name;       // 完全 type-safe
object.recipe.ingredients;
```

  
### Tool Use

```ts
import { generateText, tool } from "ai";
import { z } from "zod";

const { text, toolCalls } = await generateText({
  model: anthropic("claude-sonnet-4-6"),
  tools: {
    getWeather: tool({
      description: "取得城市天氣",
      parameters: z.object({
        city: z.string()
      }),
      execute: async ({ city }) => {
        const data = await fetch(`/weather/${city}`);
        return await data.json();
      }
    })
  },
  prompt: "台北現在幾度？",
  maxSteps: 5
});
```

### 2026 TS 開發者建議

  
用 Next.js / TS 做 AI 產品：**直接 Vercel AI SDK**，不用糾結 LangChain。

  
原因：

  
- 更輕量、更乾淨
- 完整串流支援
- 跟前端整合最好
- Vercel 投入大、更新快
- 文件清楚
