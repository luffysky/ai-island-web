### 視覺（Vision）

**給圖片、AI 看懂內容。所有主流 LLM 都支援。**

```text
import base64
from anthropic import Anthropic

client = Anthropic()

with open("image.jpg", "rb") as f:
    image_b64 = base64.b64encode(f.read()).decode()

res = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[{
        "role": "user",
        "content": [
            {
                "type": "image",
                "source": {
                    "type": "base64",
                    "media_type": "image/jpeg",
                    "data": image_b64
                }
            },
            {"type": "text", "text": "描述這張圖"}
        ]
    }]
)
```

```text

        model="whisper-1",
```

### 2026 即時對話（Realtime API）

**OpenAI Realtime API / Gemini Live**：**真正的語音對話**——不用 STT → LLM → TTS 三步、直接**語音對語音**。延遲 < 300ms、像跟真人講話。

```text
import asyncio
from openai import AsyncOpenAI

client = AsyncOpenAI()

async def realtime_chat():
    async with client.beta.realtime.connect(
        model="gpt-4o-realtime-preview"
    ) as connection:
        # === 串麥克風音訊進去 ===
        # === AI 邊聽邊想、即時用語音回 ===
        
        async for event in connection:
            if event.type == "response.audio.delta":
                play_audio(event.delta)   # 即時播放 AI 語音

asyncio.run(realtime_chat())
```

  
### 完整實戰：發票 OCR → 結構化

```text
import anthropic
import base64
from pydantic import BaseModel

class Invoice(BaseModel):
    merchant: str
    date: str
    total: float
    items: list[dict]

client = anthropic.Anthropic()

with open("invoice.jpg", "rb") as f:
    img = base64.b64encode(f.read()).decode()

tools = [{
    "name": "extract_invoice",
    "description": "從發票圖片抽出結構化資料",
    "input_schema": Invoice.model_json_schema()
}]

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=2048,
    tools=tools,
    tool_choice={"type": "tool", "name": "extract_invoice"},
    messages=[{
        "role": "user",
        "content": [
            {"type": "image", "source": {"type": "base64", "media_type": "image/jpeg", "data": img}},
            {"type": "text", "text": "請抽取這張發票"}
        ]
    }]
)

invoice = Invoice(**response.content[0].input)
print(f"商家：{invoice.merchant}")
print(f"總金額：{invoice.total}")
```
