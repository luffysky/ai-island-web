Embedding = 把任何東西變向量。RAG、搜尋、推薦、分類的基礎。

  
### 什麼是 Embedding？

```text
"咖啡"   → [0.12, -0.45, 0.78, ..., 0.23] (1536 維)
"咖啡因" → [0.15, -0.42, 0.76, ..., 0.21]
"拿鐵"   → [0.18, -0.40, 0.74, ..., 0.25]
"汽車"   → [0.89, 0.32, -0.21, ..., 0.65]

關鍵：語義相近 → 向量接近
「咖啡」「拿鐵」相似度 0.9
「咖啡」「汽車」相似度 0.2
```

  
### 怎麼用？算相似度

```python
import numpy as np

def cosine_similarity(a, b):
    return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

e1 = get_embedding("我喜歡咖啡")
e2 = get_embedding("我愛喝咖啡")
e3 = get_embedding("今天天氣很好")
e4 = get_embedding("I love coffee")

cosine_similarity(e1, e2)  # 0.92（很像）
cosine_similarity(e1, e3)  # 0.21（不像）
cosine_similarity(e1, e4)  # 0.85（跨語言也行！）
```

  
### 應用

  
- **語義搜尋**：找含義相似的內容
- **分類**：算跟各類別中心距離
- **推薦**：找相似商品
- **去重**：找重複內容（措辭不同）
- **異常檢測**：跟正常資料 embedding 差太遠
- **聚類**：自動分群
- **RAG**：找相關文件給 LLM

  
### 關鍵字 vs Embedding

  
| 查詢 | 關鍵字 | Embedding |
| --- | --- | --- |
| 「咖啡」 | 只找含「咖啡」 | 找咖啡、拿鐵、espresso |
| 「能解渴的飲料」 | 找不到水 | 找到水、果汁、茶 |
| 跨語言「coffee」 | 找不到中文 | 能找到「咖啡」 |
| 精確「2024」 | 強 | 弱（可能混進 2023） |

  
2026 主流：**兩個都用（Hybrid）**。

  
### 多模態 Embedding

  
- **CLIP**（OpenAI 2021）：圖 + 文一起，能搜「貓的照片」找到貓圖
- **SigLIP**（Google 2023）：CLIP 改進
- **Voyage Multimodal**：圖文檢索
- **Cohere Embed v3 Multilingual**：100+ 語言

  
### 維度

  
| 模型 | 維度 |
| --- | --- |
| OpenAI text-embedding-3-small | 1536 |
| OpenAI text-embedding-3-large | 3072 |
| Cohere embed v3 | 1024 |
| BGE-M3 / Voyage-3 | 1024 |

  
維度越高：精度略高、儲存大、計算慢、成本高。**1024-1536 維足夠 90% 場景**。

  
### Matryoshka Embeddings（2024 新）

  
OpenAI text-embedding-3 可截短維度：

```python
# 原 3072 維
emb = client.embeddings.create(
    input="hello",
    model="text-embedding-3-large",
    dimensions=256  # 截短到 256 維
)
# 一樣有意義、儲存 / 計算便宜 12 倍
```

  
用途：先 256 維粗搜、再完整維度精搜。

  
### 價格

```text
OpenAI text-embedding-3-small：$0.020 / 1M tokens

整本書（10 萬字 ≈ 15 萬 tokens）：$0.003
10000 個 chunks（每 500 tokens）：$0.10
100 萬 chunks：$10

→ Embedding 通常很便宜
```

  
### 2026 建議

  
- **新專案**：text-embedding-3-small（CP 最高）
- **精度重要**：voyage-3-large 或 cohere-embed-v3
- **多語場景**：cohere-embed-multilingual-v3
- **自架**：BGE-M3、jina-embeddings-v3
- **跨模態**：CLIP / SigLIP / VoyageAI Multimodal
