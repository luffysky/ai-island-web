**向量資料庫：pgvector / Pinecone / Qdrant**——有 embedding、**要存哪裡 + 怎麼快速搜尋**？這就是向量 DB 的工作。

> 💡 **2026 推薦**：indie / 新專案用 **pgvector**（免費 + Supabase 整合）、規模化用 **Pinecone / Qdrant**、邊緣用 **Cloudflare Vectorize**。

  
### 為什麼不能用 PostgreSQL（普通版）？

```text
你可以用 SQL 算 cosine、但慢

=== 純 SQL ===
10 萬筆向量、純 SQL 要 3-5 秒
百萬筆 30 秒+

=== 專門向量 DB（ANN 演算法）===
10 萬筆、10ms 內找到
百萬筆 50ms

→ 快 100-1000x
```

> 💡 **ANN（Approximate Nearest Neighbors、近似最近鄰）**——犧牲一點精準度（< 5%）換 100x 速度。

### pgvector 完整實戰

  
**Step 1：開 extension：**

```text
CREATE EXTENSION IF NOT EXISTS vector;
```

  
**Step 2：建表：**

```text
CREATE TABLE documents (
    id SERIAL PRIMARY KEY,
    content TEXT,
    metadata JSONB,
    embedding vector(1536),     -- ⭐ vector type
    user_id INT,
    created_at TIMESTAMPTZ DEFAULT NOW()
);
```

  
**Step 3：建索引（重要！沒索引慢 100 倍）：**

```text
-- HNSW 索引（2026 主流）
CREATE INDEX ON documents 
USING hnsw (embedding vector_cosine_ops);

-- 加 metadata index 加速 filter
CREATE INDEX ON documents (user_id);
```

  
**Step 4：插入：**

```text
INSERT INTO documents (content, embedding, metadata, user_id)
VALUES (
    '我喜歡咖啡',
    '[0.1, 0.2, ...]',
    '{"source": "blog"}',
    123
);
```

  
**Step 5：搜尋（cosine similarity）：**

```text
-- <=> 是 cosine distance 運算子
SELECT 
    content, 
    metadata,
    1 - (embedding <=> '[0.1, 0.2, ...]'::vector) AS score
FROM documents
WHERE user_id = 123          -- metadata filter
ORDER BY embedding <=> '[0.1, 0.2, ...]'::vector
LIMIT 5;
```

  
### Python 完整範例（pgvector）

```text
import psycopg
import json
from openai import OpenAI

client = OpenAI()
conn = psycopg.connect("postgresql://...")

def add_doc(content: str, user_id: int, metadata: dict = {}):
    # === 取 embedding ===
    emb = client.embeddings.create(
        input=content, 
        model="text-embedding-3-small"
    ).data[0].embedding
    
    # === 寫入 DB ===
    with conn.cursor() as cur:
        cur.execute(
            "INSERT INTO documents (content, embedding, metadata, user_id) "
            "VALUES (%s, %s, %s, %s)",
            (content, emb, json.dumps(metadata), user_id)
        )
    conn.commit()

def search(query: str, user_id: int, top_k: int = 5):
    # === 取 query embedding ===
    emb = client.embeddings.create(
        input=query, 
        model="text-embedding-3-small"
    ).data[0].embedding
    
    # === 搜尋 ===
    with conn.cursor() as cur:
        cur.execute(
            """SELECT content, metadata, 
                      1 - (embedding <=> %s::vector) AS score
               FROM documents 
               WHERE user_id = %s
               ORDER BY embedding <=> %s::vector 
               LIMIT %s""",
            (emb, user_id, emb, top_k)
        )
        return cur.fetchall()
```

**這行做什麼？** 裝 Pinecone 官方 client。

```text
from pinecone import Pinecone

pc = Pinecone(api_key="...")
index = pc.Index("my-index")

# === 插入 ===
index.upsert([
    ("id1", [0.1, 0.2, ...], {"text": "咖啡", "category": "drink"}),
    ("id2", [0.3, 0.4, ...], {"text": "茶", "category": "drink"})
])

# === 搜尋 + metadata filter ===
results = index.query(
    vector=[0.1, 0.2, ...],
    top_k=5,
    include_metadata=True,
    filter={"category": "drink"}
)
```

```text
✅ language = "zh-TW"

→ 效能 + 隔離兼顧
→ Pinecone / Qdrant / pgvector 都支援
```

  
### 索引演算法對比

| 演算法 | 速度 | 準確度 | 用途 |
|---|---|---|---|
| **HNSW** | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 2026 主流、推薦 |
| **IVFFlat** | ⭐⭐⭐ | ⭐⭐⭐ | 較舊、便宜 |

### Qdrant 範例（開源 + 雲端）

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
docker run -p 6333:6333 qdrant/qdrant
```

**這行做什麼？** 用 Docker 跑 Qdrant 本機 server。

```text
from qdrant_client import QdrantClient
from qdrant_client.http.models import Distance, VectorParams, PointStruct

client = QdrantClient("localhost", port=6333)

# === 建 collection ===
client.recreate_collection(
    collection_name="my_docs",
    vectors_config=VectorParams(size=1536, distance=Distance.COSINE)
)

# === 插入 ===
client.upsert(
    collection_name="my_docs",
    points=[
        PointStruct(
            id=1,
            vector=[0.1, 0.2, ...],
            payload={"text": "咖啡", "user_id": 123}
        )
    ]
)

# === 搜尋 + filter ===
results = client.search(
    collection_name="my_docs",
    query_vector=[0.1, 0.2, ...],
    query_filter={"must": [{"key": "user_id", "match": {"value": 123}}]},
    limit=5
)
```

  
### Cloudflare Vectorize（serverless 友善）

```text
// CF Workers 內
export default {
  async fetch(request, env) {
    const query = "咖啡優惠";
    
    // === 取 embedding ===
    const { data } = await env.AI.run("@cf/baai/bge-base-en-v1.5", {
      text: [query]
    });
    
    // === 搜 Vectorize ===
    const results = await env.VECTOR_INDEX.query(data[0], {
      topK: 5,
      returnMetadata: true
    });
    
    return Response.json(results);
  }
};
```

> 💡 **Cloudflare 殺手鐧**：(1) Workers + Vectorize 全 serverless (2) 邊緣節點超快 (3) 免費起步、規模化價格低。
