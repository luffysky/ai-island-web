
### 10 大省錢招

  
### 1. 多模型路由（最重要）

```text
def route_model(query: str) -> str:
    if is_simple_question(query):
        return "gemini-flash"          # $0.075/M
    if is_code_related(query):
```

> 💡 **省 50-70%**：80% 場景用便宜模型、20% 用旗艦。

  
### 2. Prompt Caching（最 ROI）

```text
# Anthropic prompt caching
client.messages.create(
    system=[{
        "type": "text",
        "text": long_system_prompt,    # 50,000 tokens
        "cache_control": {"type": "ephemeral"}
    }],
    messages=[...]
)

# → 5 分鐘內再用、cache hit = 10% 價格
# → 重複 system prompt 省 90%
```

  
### 3. Response Cache（Redis）

```text
import hashlib

def chat(query: str):
    key = hashlib.md5(query.encode()).hexdigest()
    
    # === 查 cache ===
    if cached := redis.get(f"chat:{key}"):
        return cached
    
    # === 沒 cache、call LLM ===
    response = llm.complete(query)
    
    # === 存 cache 1 小時 ===
    redis.setex(f"chat:{key}", 3600, response)
    return response

# → 重複問題 0 成本
```

  
### 4. Semantic Cache

**情境**：「請假規定？」「請假怎麼請？」意思一樣、傳統 cache miss、semantic cache hit。

```text
使用者 A：「請假規定是什麼？」
  → 算 embedding、存 cache
  → 給答案

使用者 B：「請假怎麼請？」
  → 算 embedding
  → 跟之前比、相似度 0.92
  → cache hit！直接給 A 的答案
  → 省 LLM call

工具：GPTCache / LangChain Cache
```

  
### 5. 對話歷史壓縮

```text
20 輪對話：
- 前 15 輪 → 自動摘要成「使用者問了 X / Y / Z、AI 回了 A / B / C」
- 最近 5 輪 → 原文

→ 省 70%+ token
```

  
### 6. 結構化輸出（JSON < 自然語言）

```text
=== 自然語言（多 token）===
「商品名稱：AirPods Pro
顏色：白色
價格：5990 元
庫存：有」

=== JSON（少 token）===
{"n":"AirPods Pro","c":"白","p":5990,"s":true}

→ 短 30-50%
```

  
### 7. 串流 + 中斷

```text

// → 已產的還是付錢
// → 剩下的不付
// → 省 30-50%（看 user 行為）
```

  
### 8. Batch API（2024+、5 折）

```text

    requests=[

        {"custom_id": "req-2", "params": {...}},
        # ... 數千個 request
    ]

# 24h 內處理完、價格 5 折
```

### 完整 production 範本

```text
import redis
import hashlib
from anthropic import Anthropic
from openai import OpenAI

class CostOptimizedLLM:
    def __init__(self):
        self.anthropic = Anthropic()
        self.openai = OpenAI()
        self.redis = redis.Redis()
    
    def chat(self, query: str, user_id: str) -> str:
        # === 1. Rate limit 檢查 ===
        if not self.check_rate_limit(user_id):
            raise Exception("Rate limit exceeded")
        
        # === 2. Response cache ===
        cache_key = self.get_cache_key(query)
        if cached := self.redis.get(cache_key):
            return cached.decode()
        
        # === 3. 模型路由 ===
        model = self.route_model(query)
        
        # === 4. Call LLM ===
        if "claude" in model:
            response = self.anthropic.messages.create(
                model=model,
                max_tokens=1024,
                system=[{
                    "type": "text",
                    "text": SYSTEM_PROMPT,
                    "cache_control": {"type": "ephemeral"}   # ⭐ Prompt caching
                }],
                messages=[{"role": "user", "content": query}]
            )
            text = response.content[0].text
        else:
            response = self.openai.chat.completions.create(
                model=model,
                messages=[
                    {"role": "system", "content": SYSTEM_PROMPT},
                    {"role": "user", "content": query}
                ]
            )
            text = response.choices[0].message.content
        
        # === 5. 存 cache ===
        self.redis.setex(cache_key, 3600, text)
        
        # === 6. log 成本 ===
        self.log_cost(user_id, model, len(text))
        
        return text
    
    def route_model(self, query: str) -> str:
        if len(query) < 50:
            return "gemini-flash"
        if "code" in query.lower() or "程式" in query:
            return "claude-opus-4-7"
        return "gpt-4o-mini"
```
