## 決策樹
```
你的資料是什麼？
├─ 結構化 → PostgreSQL ✓
├─ 半結構 → PG JSONB / MongoDB
├─ 快取 → Redis
├─ 搜尋 → Meilisearch / ES
├─ 向量 → pgvector / Pinecone
├─ 時序 → TimescaleDB / InfluxDB
├─ 關係 → Neo4j
└─ 不知道 → PG
```

## Indie SaaS 三階段

### 階段 1（0-1k 用戶）
Supabase 全套

### 階段 2（1k-100k）
PG + Redis + Meilisearch

### 階段 3（> 100k）
自架 PG + read replicas + Redis cluster + Elasticsearch
