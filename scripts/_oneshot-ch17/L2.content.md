**安裝 PostgreSQL + 第一次連線**—— 學 SQL 第一步不是寫 query、是「先有一個能跑 SQL 的環境」。本課把本機安裝 + 雲端 Supabase + GUI 工具三條路一次帶完。

> 💡 **這課的價值**：環境裝錯、後面 26 課全卡住。新手最常死在這一步——本課手把手把所有坑先擋掉、讓你今天就能寫第一句 SQL。


### 1. 三種「擁有 PostgreSQL」的方法

```text
1️⃣ 本機安裝
   優點：離線可用、改設定自由
   缺點：要裝、Windows 偶爾出狀況
   推薦給：想深入學底層的人

2️⃣ Supabase 雲端（推薦新手）
   優點：5 分鐘起一個、不用裝、有網頁介面
   缺點：要網路、Free tier 有限制
   推薦給：99% 的新手 / Indie / MVP

3️⃣ Docker 容器
   優點：乾淨、可以多版本並存
   缺點：要先學 Docker
   推薦給：已經會 Docker 的人
```

**林董的建議**：**先用 Supabase 練**、之後想深入再裝本機。理由——SQL 是語言、不是工具、用哪個 PG 寫起來都一樣。


### 2. 走法 A：Supabase 雲端（5 分鐘上線）

🖥️ **在瀏覽器輸入**：

```text
https://supabase.com
```

**步驟：**

```text
1. Sign up with GitHub（推薦、之後串 deploy 方便）
2. New Project
   - Name：my-first-sql
   - Database Password：自己設（記下來！）
   - Region：Northeast Asia (Tokyo) 或 Southeast Asia (Singapore)
3. 等約 2 分鐘、儀表板亮綠燈 = 起好
4. 左側選單點 SQL Editor
```

🖥️ **在 SQL Editor 輸入第一句 SQL**：

```text
SELECT 'Hello SQL' AS greeting, NOW() AS time_now;
```

按 Run（或 Ctrl+Enter）。

💬 **電腦會顯示**：

```text
greeting    | time_now
Hello SQL   | 2026-05-28 14:23:51.123+00
```

**恭喜——你已經跑了人生第一句 SQL**。


### 3. 走法 B：本機安裝 PostgreSQL（Windows）

🖥️ **在瀏覽器下載**：

```text
https://www.postgresql.org/download/windows/
→ 點 "Download the installer"
→ 選最新版（2026 年是 PostgreSQL 17）
```

**安裝精靈會問 4 件事：**

```text
1. Installation Directory
   → 預設 C:\Program Files\PostgreSQL\17 就好

2. Components
   → 4 個全勾（PostgreSQL Server / pgAdmin 4 / Stack Builder / Command Line Tools）

3. Password（postgres 帳號的密碼）
   → 自己設一個、記下來！等下要用

4. Port
   → 預設 5432、不要改
```

**裝完驗證——**

🖥️ **在「命令提示字元」（cmd）或 PowerShell 輸入**：

```text
psql --version
```

💬 **電腦會顯示**：

```text
psql (PostgreSQL) 17.0
```

看到版本號 = 裝好了。


### 4. 用 psql 連線（命令列工具）

**psql** 是 PostgreSQL 內建的命令列工具、雖然介面陽春、但**所有 PG 工程師都會用**。

🖥️ **在終端機輸入**：

```text
psql -U postgres
```

💬 **電腦會問**：

```text
Password for user postgres:
```

輸入剛剛設的密碼（**輸入時不會顯示星號、是正常的**）。

💬 **進去後會看到**：

```text
psql (17.0)
Type "help" for help.

postgres=#
```

`postgres=#` 是提示符、表示準備好接 SQL 了。

🖥️ **試第一句**：

```text
SELECT version();
```

💬 **會顯示完整 PG 版本資訊**：

```text
PostgreSQL 17.0 on x86_64-windows, ...
```

**離開 psql：**輸入 `\q` 然後 Enter。


### 5. psql 必會的 10 個 meta-command

psql 有自己的「斜線指令」（meta-command）、不是 SQL、但超好用。

```text
\l              列出所有資料庫（list databases）
\c dbname       切換到 dbname 這個 DB（connect）
\dt             列出當前 DB 所有 table
\d tablename    看 table 的結構（欄位、型別、index）
\du             列出所有使用者（users）
\dn             列出所有 schema（namespace）
\timing         開啟「顯示每句 SQL 跑多久」
\x              切換「直行 / 橫行」顯示
\?              查 meta-command 說明
\q              離開 psql
```

**實際操作：**

```text
postgres=# \l
                List of databases
   Name    |  Owner   | Encoding |  ...
-----------+----------+----------+-----
 postgres  | postgres | UTF8     | ...
 template0 | postgres | UTF8     | ...
 template1 | postgres | UTF8     | ...

postgres=# CREATE DATABASE my_first_db;
postgres=# \c my_first_db
You are now connected to database "my_first_db" as user "postgres".
my_first_db=#
```


### 6. 走法 C：GUI 工具（看圖介面寫 SQL）

純命令列很硬、新手強烈建議搭配 GUI。**3 個主流選擇：**

```text
🥇 TablePlus（推薦給 Mac 用戶）
   優點：最漂亮、操作直覺、支援 PG/MySQL/SQLite/Redis
   缺點：Free 限 2 個連線、3 個 tab
   下載：https://tableplus.com

🥈 DBeaver（推薦給 Windows / 預算為零的）
   優點：完全免費 + 開源、支援所有 DB
   缺點：介面比較老派、Java 做的較吃資源
   下載：https://dbeaver.io

🥉 pgAdmin 4（PG 安裝包附送）
   優點：免費、官方出的
   缺點：介面很 Java 風、操作較慢
   下載：跟著 PG 一起裝
```

**連線設定（任何 GUI 都一樣）：**

```text
Host:     localhost      （本機就填這個）
Port:     5432           （PG 預設）
Database: postgres       （預設 DB）
User:     postgres       （管理員）
Password: 你裝 PG 時設的密碼
```

連 Supabase 也是同樣 5 個欄位、從 Supabase Dashboard → Project Settings → Database 抄過來貼上。


### 7. 第一個 Hello World——建表 + 寫資料 + 讀資料

🖥️ **在 SQL Editor 或 psql 輸入**：

```text
-- 1. 建一張 table
CREATE TABLE friends (
  id SERIAL PRIMARY KEY,
  name TEXT NOT NULL,
  age INT
);

-- 2. 寫資料進去
INSERT INTO friends (name, age) VALUES
  ('小明', 18),
  ('小華', 22),
  ('小美', 25);

-- 3. 讀資料
SELECT * FROM friends;
```

💬 **電腦會顯示**：

```text
 id | name | age
----+------+-----
  1 | 小明 |  18
  2 | 小華 |  22
  3 | 小美 |  25
(3 rows)
```

**這 3 句 SQL 就是 90% 後端的核心**——CREATE 建表、INSERT 寫入、SELECT 讀取。後面 25 課都在深化這 3 個動作。


### 8. 5 大常見坑

```text
❌ 1. 裝 PG 時 port 改成 5433
   → 之後所有教學都假設 5432、要記得改
   → 修：除非有衝突、保留預設 5432

❌ 2. 忘記 postgres 帳號的密碼
   → 連不進去、要重設超痛
   → 修：用密碼管理器（1Password / Bitwarden）記起來

❌ 3. psql 看不到中文（亂碼）
   → Windows cmd 預設不是 UTF-8
   → 修：chcp 65001 切到 UTF-8、或用 GUI

❌ 4. Supabase 開太多 free project 被停權
   → Free tier 一個帳號最多 2 個免費 project
   → 修：練習用一個就好、刪掉舊的

❌ 5. 在 Production DB 上練習
   → 一個 DROP TABLE 全公司哭
   → 修：本機 / Supabase free project 才能玩、絕對不在線上 DB 練
```
