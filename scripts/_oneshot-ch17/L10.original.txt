## 普通子查詢（縮排深 = 痛苦）
```sql
SELECT name, total FROM (
  SELECT user_id, SUM(amount) AS total FROM orders
  WHERE created_at > NOW() - INTERVAL '30 days'
  GROUP BY user_id
) t JOIN users u ON t.user_id = u.id
WHERE total > 10000;
```

## CTE 版（讀起來像中文）
```sql
WITH recent_orders AS (
  SELECT user_id, SUM(amount) AS total
  FROM orders
  WHERE created_at > NOW() - INTERVAL '30 days'
  GROUP BY user_id
)
SELECT u.name, r.total
FROM recent_orders r
JOIN users u ON r.user_id = u.id
WHERE r.total > 10000;
```

## 多層 CTE
```sql
WITH
  active_users AS (
    SELECT id FROM users WHERE last_login > NOW() - INTERVAL '7 days'
  ),
  their_orders AS (
    SELECT o.* FROM orders o JOIN active_users a ON o.user_id = a.id
  ),
  top_categories AS (
    SELECT category, SUM(amount) AS total
    FROM their_orders
    GROUP BY category
    ORDER BY total DESC
    LIMIT 5
  )
SELECT * FROM top_categories;
```

## Recursive CTE（樹狀資料）
```sql
-- 找出某員工的所有下屬（包含下屬的下屬）
WITH RECURSIVE subordinates AS (
  SELECT id, name, manager_id
  FROM employees
  WHERE id = 1  -- 起點
  UNION ALL
  SELECT e.id, e.name, e.manager_id
  FROM employees e
  JOIN subordinates s ON e.manager_id = s.id
)
SELECT * FROM subordinates;
```
