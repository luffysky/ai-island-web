學完前 21 課是**起點、不是終點**。React 生態系大、業界要的能力一直變、這課給你 3 階段持續精進的具體路徑 + 業界推薦的學習資源。

> 💡 **這課的價值**：很多人學完基礎卡在「然後呢？」。這課告訴你下一步該學什麼、看什麼書、追蹤誰、做什麼專案。**從新手到資深的完整 roadmap**。

  
### 1. 3 階段學習路徑

```text
🌱 階段 1：能寫（第 1-2 個月）

必會：
✓ JSX / Component（元件）/ Props（屬性）
✓ useState（狀態 hook）/ useEffect（副作用 hook）
✓ Event handling（事件處理）
✓ Forms（表單）— React Hook Form + Zod
✓ Data fetching（資料抓取）— React Query
✓ Basic Next.js（基礎全端框架）

目標：ship 1 個 production SaaS（線上產品）

🚀 階段 2：能精準（第 2-6 個月）

必會：
✓ Custom Hooks（自訂 hook）— 抽出可重用邏輯
✓ Context（全域狀態）/ useReducer（複雜 state）
✓ Performance（效能）— memo / virtualization（虛擬列表）
✓ Testing（測試）— Vitest + Testing Library + Playwright
✓ Server Components（伺服器元件）/ Server Actions
✓ TypeScript 進階 — generics（泛型）/ utility types

目標：ship 1 個有 1000+ 使用者的產品

🏆 階段 3：能設計（6 個月後）

必會：
✓ Architecture（架構）— Monorepo / Microfrontend
✓ Design system（設計系統）— Storybook / 元件庫
✓ State management（狀態管理）— Zustand / Jotai / Redux Toolkit
✓ A11y（無障礙）/ i18n（國際化）/ SEO
✓ Performance budget — bundle size / Core Web Vitals
✓ Mentoring（指導他人）— code review / tech writing

目標：lead 1 個團隊 / open source 專案 / blog
```

### 7. 5 大新手卡點 + 解法

```text
❌ 1. 「教學看完了、自己寫不出來」
   → 沒實作、純看教學 ROI 極低
   → 修：每週 ship 一個小專案、不求大

❌ 2. 「不知道哪個 stack 才對」
   → 選擇障礙、永遠在挑 framework
   → 修：選 Next.js + Tailwind 就對、別再比

❌ 3. 「TypeScript 太難」
   → 看到泛型 / utility types 就投降
   → 修：先寫 JS、慢慢加 type、不必一次到位

❌ 4. 「不會看官方文件」
   → 太長、看到一半放棄
   → 修：搜尋具體問題、用 Ctrl+F 跳重點

❌ 5. 「英文文件看不懂」
   → 想等中文教學
   → 修：用 ChatGPT / Claude 邊看邊翻、強迫自己讀英文
```
