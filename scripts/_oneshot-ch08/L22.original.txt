```text

2. ship > perfect
   - 上線 + iterate 才能進步

   - shadcn / Cal.com / Resend

4. 寫 type-safe code
   - TypeScript strict
   - Zod validate

   - Twitter / Threads 分享
   - 拿 feedback
```
