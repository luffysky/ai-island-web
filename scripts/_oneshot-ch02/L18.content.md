**Bootstrap** = 2011 由 Twitter 開源的 CSS 框架——**老牌但實用**。Bootstrap 5（2021+）是當代版本、去除 jQuery 依賴。**後台、行政系統、企業內部工具的常勝軍**——學會幾天內可以拼出完整介面。

> 💡 **這課的價值**：學會 Bootstrap = 接案 / 內部工具 / 後台案件接到飛。**Tailwind 是新潮、Bootstrap 是穩定**。台灣中小企業、政府案、銀行後台還大量在用。

  
### 1. Bootstrap vs Tailwind 哲學差別

```text
Bootstrap：
  → 「成品元件」優先
  → 用 .btn / .card / .alert / .modal 等預製元件
  → 改主題色就能用
  → 適合：快速做出「能用就好」的介面

Tailwind：
  → 「utility 原料」優先
  → 沒元件、自己組合
  → 適合：高度客製化的品牌設計

選擇心法：
  → 後台 / 原型 / 內部工具 → Bootstrap（快）
  → SaaS 產品 / 品牌網站 → Tailwind（精）
```

  
### 2. 安裝 Bootstrap

**方法 A：CDN（最快、不用 build）**：

📄 **寫進程式檔**：`index.html`

```text
<head>
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css"
    rel="stylesheet"
  >
</head>
<body>
  <!-- 你的 HTML -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js"></script>
</body>
```

**方法 B：npm 安裝**：

🖥️ **在終端機輸入**

```text
pnpm add bootstrap
```

📄 **寫進程式檔**：`main.js`

```text
import 'bootstrap/dist/css/bootstrap.min.css';
import 'bootstrap';  // JS 元件（modal / dropdown / 等）
```

  
### 3. Grid 系統（12 欄）

Bootstrap 的招牌：**12 欄響應式網格**。

📄 **寫進程式檔**：`index.html`

```text
<div class="container">
  <div class="row">
    <!-- 平板以上 8 欄、手機全寬 -->
    <div class="col-12 col-md-8">主內容</div>
    <!-- 平板以上 4 欄、手機全寬 -->
    <div class="col-12 col-md-4">側邊欄</div>
  </div>
  
  <div class="row">
    <!-- 三欄等分 -->
    <div class="col-md-4">A</div>
    <div class="col-md-4">B</div>
    <div class="col-md-4">C</div>
  </div>
</div>
```

**斷點對照**：

| Class | min-width | 對應裝置 |
|---|---|---|
| `col-` | < 576px | 手機 |
| `col-sm-` | ≥ 576px | 大手機 |
| `col-md-` | ≥ 768px | 平板 |
| `col-lg-` | ≥ 992px | 小筆電 |
| `col-xl-` | ≥ 1200px | 桌機 |
| `col-xxl-` | ≥ 1400px | 大螢幕 |

> 💡 **12 欄系統的好處**：12 可被 1/2/3/4/6 整除——所以「半半」「三三分」「四四分」都能用同一套網格。

  
### 4. 常用元件

<!-- 尺寸 -->
<button class="btn btn-primary btn-lg">大</button>
<button class="btn btn-primary btn-sm">小</button>

**警告框（Alert）**：

```text

  操作成功！

**導覽列（Navbar）**：

```

**Modal（對話框）**：

📄 **寫進程式檔**：`index.html`

```text

  打開對話框
</button>

  <div class="modal-dialog">

      <div class="modal-footer">

        <button class="btn btn-primary">確認</button>
      </div>
    </div>
  </div>
</div>
```

  
### 5. Utility Classes（類似 Tailwind）

Bootstrap 5 也加了大量 utility（但不像 Tailwind 那麼全）：

```text
間距：
  m-{0-5}    margin
  p-{0-5}    padding
  mx-3       margin 左右
  py-2       padding 上下

排版：
  d-flex     display: flex
  d-none     display: none
  d-md-block 平板以上 display: block

對齊：
  text-center / text-start / text-end
  justify-content-center
  align-items-center

顏色：
  text-primary / text-success / text-muted
  bg-primary / bg-light / bg-dark
```

  
### 6. 客製化主題（Sass）

**Bootstrap 用 Sass 寫的、可以改變數**：

📄 **寫進程式檔**：`custom.scss`

```text
// 1. 覆寫變數（必須在 import 之前）
$primary: #5e35b1;
$border-radius: 12px;
$font-family-sans-serif: "Inter", sans-serif;
$enable-shadows: true;

// 2. 引入 Bootstrap
@import "bootstrap/scss/bootstrap";
```

🖥️ **在終端機輸入**

```text
pnpm sass custom.scss bootstrap-custom.css
```

> 💡 **Bootstrap 主題客製是接案常見需求**——「我要 Bootstrap 風格、但顏色換成我們品牌色」——改幾個 Sass 變數搞定、客戶 NT$ 10-30K 願意付。

### 8. Tailwind vs Bootstrap 怎麼選

| 場景 | 選 |
|---|---|
| 客製化、品牌、設計重 | Tailwind + shadcn/ui |
| 後台、原型、企業內部 | Bootstrap |
| 快速 demo | Bootstrap |
| SaaS 產品 | Tailwind |
| 政府案、銀行案 | Bootstrap（穩定保守）|
| Next.js / Vercel 生態 | Tailwind |
| 2026 新專案 | 多半選 Tailwind |

  
### 9. Bootstrap 在 2026 還活著嗎

```text
✅ 仍主流：
   → 台灣中小企業內部工具
   → 政府案、銀行後台、學校系統
   → 快速原型、能用就好
   → Laravel / Django / Rails 預設配套

❌ 已被取代：
   → 設計感重的 SaaS（Tailwind 勝）
   → React / Vue 元件庫（shadcn / Element Plus 勝）

→ 不會消失、但市占持續被 Tailwind 蠶食
→ 接案賺錢用 Bootstrap 仍是好選擇
```

  
### 5 大常見坑

```text
❌ 1. 用 Bootstrap 又寫一堆自訂 CSS
   → 失去 Bootstrap 意義（套裝速度）
   → 修：要客製就走 Tailwind 路線

❌ 2. 沒引入 JS bundle、Modal / Dropdown 不動
   → 修：bootstrap.bundle.min.js 必引

❌ 3. 用了 Bootstrap 但版面還是醜
   → 沒用 .container 包、grid 失效
   → 修：所有頁面用 <div class="container">

❌ 4. 想做高度客製設計、硬塞 Bootstrap
   → Bootstrap 風格固定、改起來累
   → 修：客製化重的案子改 Tailwind

❌ 5. 用 Bootstrap 4 的習慣寫 Bootstrap 5
   → ml-* 改成 ms-*、float 系統大改
   → 修：看官方 migration guide
```

  
### 🎯 重點回顧

- **Bootstrap 5 是「成品元件」優先**——快速拼介面
- **12 欄 Grid 系統**——`col-12 col-md-8` 響應式網格
- **常用元件**：btn / card / alert / navbar / modal
- **Utility class 也有**（不像 Tailwind 那麼全）
- **Sass 變數客製主題**——改色、改圓角、改字型
- **後台 / 內部工具首選**——Tailwind 不會比較快
- **Bootstrap 5 去 jQuery**——但 JS 元件還是要引 bundle
