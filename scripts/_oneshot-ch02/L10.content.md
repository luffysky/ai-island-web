字型是閱讀體驗的核心——**80% 的「好看」來自字型設計**。這課教完整的字型系統：**font-family / font-weight / font-size / line-height / @font-face / variable font / 截斷文字**。

> 💡 **這課的價值**：學會字型 = 你的網頁從「能看」升級「好看」。**Medium / Substack / Apple 為什麼讓人想一直讀下去**——80% 是字型功勞。

  
### 1. font-family（字型家族）

**font-family** = 用什麼字。**永遠寫一串「備用字型」**：

📄 **寫進程式檔**：`style.css`

```text
body {
  font-family:
    'Inter',                 /* 第一選擇 */
    -apple-system,           /* macOS / iOS 系統字 */
    BlinkMacSystemFont,      /* macOS Chrome */
    'Segoe UI',              /* Windows */
    'Roboto',                /* Android */
    'Helvetica Neue',
    sans-serif;              /* 最後保底（無襯線通用）*/
}
```

> 💡 **為什麼寫一串**：使用者電腦不一定有 Inter——找不到就跳到下一個、最後保底用系統字。**永遠以 `sans-serif` 或 `serif` 結尾**。

  
### 2. 系統字型堆疊（System Font Stack）

**完全不下載字型、用每個系統的「最佳原生字」**——速度最快：

📄 **寫進程式檔**：`style.css`

```text
body {
  font-family:
    -apple-system,
    BlinkMacSystemFont,
    'Segoe UI',
    Roboto,
    'Helvetica Neue',
    Arial,
    sans-serif,
    'Apple Color Emoji',
    'Segoe UI Emoji';
}
```

| 系統 | 用到的字 |
|---|---|
| macOS / iOS | `-apple-system` → SF Pro |
| Windows | `Segoe UI` |
| Android | `Roboto` |
| Linux | `sans-serif` 保底 |

> 🚀 **GitHub / Notion / Linear 都用系統字**——零下載時間、每個系統都看起來原生。**效能至上選這個**。

  
### 3. 載入 Google Fonts

📄 **寫進程式檔**：`index.html`

```text
<head>
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link
    href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;700&display=swap"
    rel="stylesheet"
  >
</head>
```

📄 **寫進程式檔**：`style.css`

```text
body {
  font-family: 'Inter', sans-serif;
}
```

> 💡 **`display=swap` 一定要加**：載入過程先顯示備用字、Inter 下載完才換——避免「字型載入時整片空白」（FOIT、Flash of Invisible Text）。

  
### 4. 自架字型（@font-face）

效能比 Google Fonts 更好（少一個 DNS、HTTP/2 受惠）：

📄 **寫進程式檔**：`style.css`

```text
@font-face {
  font-family: 'MyFont';
  src: url('fonts/myfont.woff2') format('woff2');
  font-weight: 100 900;  /* variable font 範圍 */
  font-display: swap;     /* 載入時顯示備用字 */
}

body {
  font-family: 'MyFont', -apple-system, system-ui, sans-serif;
}
```

> 💡 **永遠用 `.woff2`**——比 `.woff` 小 30%、比 `.ttf` 小 50%。**`.woff2` 是 2026 唯一推薦格式**。

  
### 5. font-weight（字重）

**font-weight** = 字的粗細、100（最細）到 900（最粗）：

| 數值 | 名稱 | 用在 |
|---|---|---|
| 100 | Thin | 極細裝飾標題 |
| 300 | Light | 輕量標題 |
| **400** | **Regular** | **正文標配** |
| 500 | Medium | 強調文字、小標 |
| 600 | Semibold | 卡片標題 |
| **700** | **Bold** | **粗體** |
| 800 | Extrabold | 強調大標 |
| 900 | Black | 極粗大標 |

📄 **寫進程式檔**：`style.css`

```text
body { font-weight: 400; }
h1 { font-weight: 700; }
.subtitle { font-weight: 500; }
```

  
### 6. font-size + line-height（最重要的兩個）

📄 **寫進程式檔**：`style.css`

```text
body {
  font-size: 16px;
  line-height: 1.6;
  /* → 行高 1.6 倍 = 16 × 1.6 = 25.6px */
}

h1 {
  font-size: 48px;
  line-height: 1.2;
  /* → 標題行高小（緊湊）*/
}

p {
  font-size: 16px;
  line-height: 1.75;
  /* → 內文行高大（好讀）*/
}
```

> 💡 **`line-height` 寫純數字不加單位**——`1.6` 而非 `1.6em` 或 `25px`。寫數字是「**倍數**」、子元素會用「自己字級 × 倍數」算；寫單位會被強制繼承固定值、子元素字級變了行高沒跟著變。

  
### 7. 完整字型屬性速查

📄 **寫進程式檔**：`style.css`

```text
.text {
  font-family: 'Inter', sans-serif;
  font-size: 16px;
  font-weight: 400;
  font-style: italic;          /* italic / oblique */
  font-variant: small-caps;     /* 小大寫字 */
  
  line-height: 1.6;
  letter-spacing: 0.02em;       /* 字距 */
  word-spacing: 0.1em;          /* 詞距 */
  
  text-align: left;             /* left / right / center / justify */
  
  /* 裝飾 */
  text-decoration: underline red wavy 2px;
  text-underline-offset: 4px;
  
  /* 大小寫 */
  text-transform: uppercase;     /* lowercase / capitalize */
  
  /* 縮排 */
  text-indent: 2em;
  
  /* 換行控制 */
  word-break: break-word;
  overflow-wrap: anywhere;
  hyphens: auto;                 /* 自動斷字 */
  
  /* 字型平滑（macOS）*/
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}
```

  
### 8. 截斷文字（業界必備）

**單行截斷加 `...`**：

📄 **寫進程式檔**：`style.css`

```text
.truncate {
  white-space: nowrap;       /* 不換行 */
  overflow: hidden;          /* 超出隱藏 */
  text-overflow: ellipsis;   /* 用 ... 表示 */
}
```

**多行截斷**（3 行後 `...`）：

📄 **寫進程式檔**：`style.css`

```text
.line-clamp-3 {
  display: -webkit-box;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;
  overflow: hidden;
}
```

> 💡 **這 2 個寫法 99% 卡片標題、商品摘要都會用**——背起來。

  
### 9. Variable Font（可變字型）—— 2026 新標配

**Variable Font（可變字型）** = 一個字型檔包所有粗細、不用下載 7 個檔：

📄 **寫進程式檔**：`style.css`

```text
@font-face {
  font-family: 'Inter Variable';
  src: url('Inter-Variable.woff2') format('woff2');
  font-weight: 100 900;  /* 範圍：100 到 900 全包 */
}

.text {
  font-family: 'Inter Variable';
  font-variation-settings:
    "wght" 450,   /* 字重（可任意值、不限定 100/200/...）*/
    "slnt" -5;    /* 傾斜 */
}
```

> 🚀 **Variable Font 的優勢**：(1) **一個檔搞定所有粗細**（省下 7 個 HTTP request）(2) **可以用 380、450 這種「非標準」字重** (3) **可以動畫過渡字重**（hover 時字慢慢變粗）。**Inter、Roboto Flex、Recursive 都是 variable font**。

  
### 10. 實戰：完整文章排版

📄 **寫進程式檔**：`style.css`

```text
:root {
  --font-sans: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
  --font-serif: 'Georgia', 'Noto Serif TC', serif;
}

body {
  font-family: var(--font-sans);
  font-size: 16px;
  line-height: 1.6;
  color: #1a1a1a;
}

article {
  max-width: 70ch;  /* ★ 最佳閱讀寬度 50-80 字元 */
  margin: 0 auto;
  padding: 24px;
}

article h1 {
  font-size: clamp(2rem, 5vw, 3rem);  /* 響應式字級 */
  font-weight: 700;
  line-height: 1.2;
  letter-spacing: -0.02em;            /* 大標微縮字距 */
}

article p {
  font-size: 1.125rem;  /* 18px */
  line-height: 1.75;
  margin: 1.2em 0;
}

article .lead {
  font-size: 1.25rem;
  color: #555;
  font-weight: 500;
}
```

> 🚀 **進階技巧**：字型最佳寬度是**每行 50-80 字元**——用 `max-width: 70ch` 限制文章寬度、閱讀體驗立刻提升。**Medium / Substack 都這樣設**。

  
### 5 大常見坑

```text
❌ 1. font-family 只寫一個（沒備用）
   → 字型載入失敗整頁變奇怪預設字
   → 修：永遠寫一串、結尾 sans-serif

❌ 2. line-height 加單位（1.6em）
   → 子元素字級變大、行高沒跟著變
   → 修：line-height 永遠寫純數字（1.6）

❌ 3. 載 Google Fonts 沒加 display=swap
   → 字型載入時整片空白（FOIT）
   → 修：URL 加 &display=swap

❌ 4. 文章寬度撐滿（沒限制）
   → 每行 100+ 字元、讀起來累
   → 修：max-width: 70ch

❌ 5. 直接設 font-size: 12px 期待手機放大
   → 手機字級需要 16px 以上才不需要放大
   → 修：內文最小 16px、小字限制在輔助資訊
```

  
### 🎯 重點回顧

- **font-family 永遠寫一串**、結尾 `sans-serif`
- **系統字型堆疊**——速度最快、不下載
- **`@font-face` + `.woff2` + `display: swap`**——自架字型黃金組合
- **font-weight 400 正文、700 粗體**——數值寫法
- **`line-height: 1.6`**——純數字不加單位
- **`max-width: 70ch`**——閱讀最佳寬度
- **Variable Font**——2026 新標配、一檔包所有粗細

> 💡 **下一課（L11）**：**背景 / 邊框 / 陰影 / 濾鏡**——把元素變立體、有質感。**Apple / Linear / Vercel 的高級感都來自陰影設計**——這課讓你升級設計品味。
