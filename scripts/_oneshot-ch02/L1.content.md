**CSS（Cascading Style Sheets、層疊樣式表）** = 把網頁變漂亮的化妝術。HTML 給你「文字、圖片、按鈕」、但長相全部都是瀏覽器預設的「Word 1995 年那種樣子」。**CSS 就是把這堆灰灰白白的東西變成 Apple 官網質感的工具**。

> 💡 **這課的價值**：HTML 蓋骨架、CSS 化妝。**沒 CSS 的網頁能用、但沒人想看**。學會 CSS = 你寫的東西終於「看起來像個產品」。

  
### 1. 為什麼需要 CSS？

HTML 像剛蓋好的房子骨架（牆、地板、門框）。CSS 是裝潢（刷漆、家具、燈光、地毯）。沒裝潢的房子能住嗎？能。**但你想住嗎？**

```text
🏗️ HTML
   → 這是標題、這是段落、這是按鈕

🎨 CSS
   → 標題紅色 48px、段落灰色 16px、按鈕藍底白字圓角

⚡ JavaScript
   → 按按鈕做事
```

  
### 2. CSS 三大威力

**CSS 不只是「畫漂亮」**、它做三件大事：

```text
1. 視覺呈現
   → 顏色、字型、間距、陰影、漸層、圓角

2. 排版佈局
   → 誰在左、誰在右、誰置中
   → 響應不同螢幕（手機 / 平板 / 桌機）

3. 互動與動畫
   → hover 變色、按鈕彈跳、頁面切換動畫
```

> 💡 **重點**：很多人以為 CSS 只是「換顏色」、其實**整個版面怎麼排都是 CSS 的事**。這是 CSS 最值錢的部分。

  
### 3. CSS 的三種寫法

CSS 可以寫在三個地方、各有用途：

📄 **寫進程式檔**：`index.html`

```text
<!-- 寫法一：inline（行內樣式） -->
<h1 style="color: red; font-size: 48px;">標題</h1>

<!-- 寫法二：internal（內部樣式表）寫在 <head> 裡 -->
<head>
  <style>
    h1 { color: red; font-size: 48px; }
  </style>
</head>

<!-- 寫法三：external（外部樣式表）連到另一個 .css 檔 -->
<head>
  <link rel="stylesheet" href="style.css">
</head>
```

> 💡 **三種寫法的差別**：(1) **inline（行內）**—— 直接寫在標籤的 `style="..."` 屬性裡 (2) **internal（內部）**—— 寫在 `<head>` 的 `<style>` 標籤裡 (3) **external（外部）**—— 寫在獨立 `.css` 檔、用 `<link>` 連進來。

  
### 4. 三種寫法優缺點

| 寫法 | 優點 | 缺點 | 用在哪 |
|---|---|---|---|
| inline | 直接、不用切檔 | 無法重用、難維護、優先級最高 | 臨時 debug、Email 模板 |
| internal | 不用多載入、首屏快 | 跨頁無法共用 | 單頁應用、首屏關鍵樣式 |
| external | 可重用、可快取、可壓縮 | 多一個 HTTP request | **99% 場景都用這個** |

> ⚠️ **新手最常犯的錯**：把所有樣式寫成 inline（`<div style="color: red; padding: 10px; ...">`）。看起來「直接」但 5 個頁面就維護不動。**永遠用 external CSS**、這是進入專業領域的第一道門檻。

  
### 5. external CSS 該放哪？

📄 **寫進程式檔**：`index.html`

```text
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="style.css">
  <title>我的網頁</title>
</head>
<body>
  <h1>哈囉</h1>
</body>
</html>
```

📄 **寫進程式檔**：`style.css`

```text
h1 {
  color: crimson;
  font-size: 48px;
  text-align: center;
}
```

> 💡 **`<link>` 永遠放 `<head>` 裡**：瀏覽器會先讀完 CSS 才開始畫畫面、避免「先看到沒樣式的醜版、再瞬間變漂亮」（叫 FOUC、無樣式內容閃爍）。

  
### 6. 第一個 CSS 規則（CSS 一輩子都這樣寫）

```text
selector { property: value; }
   ↑          ↑         ↑
  選誰      改什麼     改成啥

範例：
h1 { color: crimson; font-size: 48px; }
↑    ↑          ↑           ↑       ↑
選 h1 屬性=顏色 值=紅      屬性=字級 值=48px
```

**三個關鍵詞**：
- **selector（選擇器）**—— 要對誰下指令（例：`h1` 選所有 h1 標籤）
- **property（屬性）**—— 要改什麼（例：`color` 是顏色、`font-size` 是字級）
- **value（值）**—— 改成什麼（例：`crimson` 是深紅、`48px` 是 48 像素）

> 💡 **記住這個結構、CSS 一輩子都這樣寫**。後面 24 課都是在學「有哪些 selector / property / value 可以用」。

  
### 7. 載多個 CSS 檔

實際專案會把 CSS 切成多個檔（reset、基礎、元件、主題）：

📄 **寫進程式檔**：`index.html`

```text
<head>
  <link rel="stylesheet" href="reset.css">    <!-- 清除瀏覽器預設 -->
  <link rel="stylesheet" href="base.css">     <!-- 全域字型、顏色 -->
  <link rel="stylesheet" href="layout.css">   <!-- 排版 -->
  <link rel="stylesheet" href="theme.css">    <!-- 主題色 -->
</head>
```

> 💡 **載入順序很重要**：後面寫的會「**蓋掉**」前面寫的（這就是 Cascading 串接、下一課詳教）。

  
### 8. 第一次寫 CSS（5 分鐘上手）

🖥️ **在終端機輸入**

```text
mkdir my-css-test
cd my-css-test
```

📄 **寫進程式檔**：`my-css-test/index.html`

```text
<!DOCTYPE html>
<html lang="zh-Hant">
<head>
  <meta charset="UTF-8">
  <link rel="stylesheet" href="style.css">
  <title>我的第一個 CSS</title>
</head>
<body>
  <h1>哈囉、我學會 CSS 了</h1>
  <p>這段是普通文字、會被 CSS 變漂亮。</p>
</body>
</html>
```

📄 **寫進程式檔**：`my-css-test/style.css`

```text
body {
  font-family: -apple-system, system-ui, sans-serif;
  background: #f5f5f5;
  padding: 40px;
}

h1 {
  color: crimson;
  font-size: 48px;
  text-align: center;
}

p {
  color: #555;
  line-height: 1.8;
  max-width: 600px;
  margin: 20px auto;
}
```

**用瀏覽器開檔案：** 雙擊 `index.html`、或 VS Code 裝 Live Server 擴充。**這是你寫前端的第二個 wow moment**（第一個是 HTML 那課）。

  
### 5 大常見坑

```text
❌ 1. 把所有樣式寫成 inline
   → 5 個頁面就維護爆炸
   → 修：永遠用 external（單獨 .css 檔）

❌ 2. 忘記寫 <link rel="stylesheet" href="...">
   → CSS 沒被載入、樣式完全沒套用
   → 修：F12 看 Network 確認 CSS 有下載

❌ 3. selector 拼錯（例：.crad 應該是 .card）
   → 沒人選到、樣式安靜失效
   → 修：DevTools 看右邊 Styles 面板有沒有那條規則

❌ 4. 結尾忘記分號（;）
   → 下一條規則整個壞掉
   → 修：每行結尾都加 ;（Prettier 會自動補）

❌ 5. 改完 CSS 沒重整瀏覽器
   → 看到舊版以為「沒效果」
   → 修：Ctrl + F5 強制重整、或裝 Live Server 自動重整
```

  
### 🎯 重點回顧

- **HTML 蓋骨架、CSS 化妝、JS 動起來**——三者分工合作
- **三種寫法**：inline / internal / **external（99% 用這個）**
- **CSS 一輩子的結構**：`selector { property: value; }`
- **`<link>` 永遠放 `<head>` 裡**——避免無樣式閃爍
- **後面的規則會蓋掉前面的**——這是 Cascading（下課詳教）

> 💡 **下一課（L2）**：CSS 的核心是「**選誰、做什麼**」。選誰 = selector（選擇器）。下一課把所有選擇器一次學完——標籤、class、id、屬性、組合、偽類、偽元素。**這課學會 = 任何元素都選得到**。
