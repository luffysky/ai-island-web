10–20% 使用者有某種無障礙需求（視障、色盲、運動障礙、認知障礙）。寫無障礙的 CSS 不難，但要記住。

  
### 對比度（Contrast）

```css
/* 文字對背景的對比 */
.good { color: #333; background: #fff; }   /* 12.6:1 ✓ */
.bad  { color: #999; background: #fff; }   /* 2.8:1 ✗ */
```

  
WCAG 標準：

  
- **AA 級**：正常文字 4.5:1、大文字 3:1
- **AAA 級**：正常文字 7:1、大文字 4.5:1

  
檢查工具：Chrome DevTools 內建、`https://webaim.org/resources/contrastchecker/`。

  
### focus 樣式（鍵盤使用者必備）

```css
/* 永遠不要關掉 focus */
/* ✗ 大忌 */
*:focus { outline: none; }

/* ✓ 要客製就客製清楚 */
button:focus-visible {
  outline: 3px solid #0070f3;
  outline-offset: 2px;
}
```

```css
.sr-only {

  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip-path: inset(50%);
  white-space: nowrap;
  border: 0;
}
```
