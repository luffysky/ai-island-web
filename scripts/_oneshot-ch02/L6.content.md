**Flexbox（彈性盒子）** = Flexible Box、**主要解一維問題**——一排東西怎麼排、怎麼對齊、怎麼分配空間。**2026 年所有前端工程師必精通的東西**。在 flex 出現前、垂直水平置中是「千古難題」、現在 3 行解決。

> 💡 **這課的價值**：學會 flex = **80% 的排版需求都搞定**（導覽列、卡片組、置中、footer 永遠在底）。後面的 grid 是補刀、flex 才是主力。

  
### 1. flex 的兩個世界：容器 vs 子項目

flex 屬性分兩類：寫在**容器**上（控制怎麼排）、寫在**子項目**上（控制自己怎麼長）：

```text
容器（flex container）= 父元素
  → display: flex
  → flex-direction（方向）
  → justify-content（主軸對齊）
  → align-items（交叉軸對齊）
  → gap（間距）
  → flex-wrap（換行）

子項目（flex item）= 子元素
  → flex-grow / shrink / basis
  → align-self（自己的對齊）
  → order（順序）
```

  
### 2. 主軸 vs 交叉軸（最重要的觀念）

flex 排東西、分成兩個方向：

```text
main axis（主軸）= 元素排列的方向
cross axis（交叉軸）= 跟主軸垂直的方向

預設 flex-direction: row
  → 主軸：水平（左 → 右）
  → 交叉軸：垂直（上 → 下）

如果 flex-direction: column
  → 主軸：垂直（上 → 下）
  → 交叉軸：水平（左 → 右）
```

> 💡 **記住口訣**：「**justify 管主軸、align 管交叉軸**」——這條記住、所有對齊都不會錯。

  
### 3. 容器屬性（寫在父元素）

📄 **寫進程式檔**：`style.css`

```text
.container {
  display: flex;
  
  /* 主軸方向：row（預設）/ column / row-reverse / column-reverse */
  flex-direction: row;
  
  /* 換行：nowrap（預設不換）/ wrap / wrap-reverse */
  flex-wrap: wrap;
  
  /* 縮寫：direction + wrap */
  flex-flow: row wrap;
  
  /* 主軸對齊（justify-content）*/
  justify-content: space-between;
  /* 可選值：
     flex-start（靠主軸起點）
     center（主軸置中）
     flex-end（靠主軸終點）
     space-between（兩端對齊、中間平均）
     space-around（每個元素左右等距）
     space-evenly（每個間距都相等）
  */
  
  /* 交叉軸對齊（align-items）*/
  align-items: center;
  /* 可選值：
     stretch（預設、撐滿）
     flex-start / center / flex-end
     baseline（文字基線對齊）
  */
  
  /* 多行時的交叉軸對齊 */
  align-content: space-between;
  
  /* 元素間距（神器！）*/
  gap: 16px;
  row-gap: 16px;
  column-gap: 24px;
}
```

> 💡 **`gap` 是 flex 後期才加的神器**：以前用 margin 處理間距、會多出邊緣 margin 要扣掉。**`gap` 只在元素之間生效、不影響容器邊緣**。永遠用 gap。

  
### 4. 子項目屬性（寫在子元素）

📄 **寫進程式檔**：`style.css`

```text
.item {
  /* 縮放三兄弟 */
  flex-grow: 1;       /* 能變多大（佔剩餘空間的比例）*/
  flex-shrink: 1;     /* 能變多小（空間不夠時的縮小比例）*/
  flex-basis: 200px;  /* 起始大小 */
  
  /* 縮寫 */
  flex: 1;           /* = 1 1 0%、平均分配剩餘空間 */
  flex: 0 0 auto;    /* = 不縮放、用 content 大小 */
  flex: 2;           /* 比其他 flex:1 的多兩倍 */
  
  /* 單獨對齊（覆蓋容器的 align-items）*/
  align-self: flex-end;
  
  /* 順序（不改 HTML、改視覺順序）*/
  order: -1;
  /* → 預設都是 0、負值往前、正值往後 */
}
```

> 💡 **`flex: 1` 是最常用的**——「**平均分配剩餘空間**」。例：5 個 `flex: 1` 的子元素、容器寬 1000px、每個都 200px。

  
### 5. 實戰例 1：經典導覽列

📄 **寫進程式檔**：`style.css`

```text
nav {
  display: flex;
  justify-content: space-between;  /* logo 在左、選單在右 */
  align-items: center;              /* 垂直置中 */
  padding: 16px 32px;
  background: white;
  border-bottom: 1px solid #eee;
}

.nav-links {
  display: flex;
  gap: 24px;
  list-style: none;
  margin: 0;
  padding: 0;
}
```

📄 **寫進程式檔**：`index.html`

```text
<nav>
  <div class="logo">AI 島</div>
  <ul class="nav-links">
    <li><a href="/">首頁</a></li>
    <li><a href="/courses">課程</a></li>
    <li><a href="/about">關於</a></li>
  </ul>
</nav>
```

  
### 6. 實戰例 2：垂直水平置中（千古難題終結）

```text
過去要 absolute + translate + 一堆運算：
  .center {
    position: absolute;
    top: 50%; left: 50%;
    transform: translate(-50%, -50%);
  }

現在 3 行解決：
```

📄 **寫進程式檔**：`style.css`

```text
.center {
  display: flex;
  justify-content: center;  /* 水平置中 */
  align-items: center;      /* 垂直置中 */
  height: 100vh;            /* 父容器要有高度 */
}
```

> 💡 **這 3 行解決了 CSS 史上最常被問的問題**——「怎麼把元素垂直水平置中」。面試常考、現在答案超簡單。

  
### 7. 實戰例 3：永遠在最下面的 footer

```text
需求：頁面內容少時 footer 也要貼底、不要浮在中間
```

📄 **寫進程式檔**：`style.css`

```text
body {
  min-height: 100dvh;
  display: flex;
  flex-direction: column;
  margin: 0;
}

main {
  flex: 1;  /* ← main 撐開剩餘空間 */
}

/* footer 自然會被推到底、不用設定 */
```

  
### 8. 實戰例 4：彈性卡片組（響應式）

```text
需求：螢幕寬時一排 4 個、窄時 2 個、超窄 1 個
```

📄 **寫進程式檔**：`style.css`

```text
.cards {
  display: flex;
  flex-wrap: wrap;
  gap: 16px;
}

.card {
  flex: 1 1 300px;
  /* 至少 300px、能撐多大撐多大 */
  /* 換算：寬 1200px → 一排 4 個（每個 300）*/
  /*       寬 700px → 一排 2 個 */
  /*       寬 400px → 一排 1 個 */
}
```

> 💡 **`flex: 1 1 300px` 是神奇咒語**——`flex-grow: 1`（能變大）+ `flex-shrink: 1`（能變小）+ `flex-basis: 300px`（最小寬度）。完全不用寫 media query 就響應式。

  
### 9. justify-content 6 種對齊視覺化

```text
flex-start：     [A][B][C].............
center：         .....[A][B][C].........
flex-end：       .............[A][B][C]
space-between：  [A].......[B].......[C]
space-around：   ..[A]....[B]....[C]..
space-evenly：   ...[A]...[B]...[C]...
```

  
### 5 大常見坑

```text
❌ 1. 父元素沒 display: flex、justify-content 完全沒效
   → 修：對齊屬性必須寫在 flex 容器上

❌ 2. 想垂直置中、結果 align-items 不動
   → 容器沒高度、根本沒空間可置中
   → 修：給父元素 height 或 min-height: 100dvh

❌ 3. 用 margin 處理間距、容器邊緣多餘
   → 例：每個 li margin-right: 16px、最後一個也多 16px
   → 修：永遠用 gap

❌ 4. flex: 1 不夠平均（某個子元素特別大）
   → 內容過大撐爆 basis
   → 修：加 min-width: 0、讓 flex 能真的縮

❌ 5. flex-direction: column 後 justify / align 搞錯
   → 主軸變垂直、justify 變垂直對齊、align 變水平
   → 修：永遠記「justify 主軸、align 交叉軸」
```

  
### 🎯 重點回顧

- **flex 解一維問題**（一排排對齊）
- **主軸 vs 交叉軸**——`justify` 管主軸、`align` 管交叉軸
- **`gap` 永遠優於 margin**——只在元素間、不影響邊緣
- **`flex: 1` = 平均分配剩餘空間**
- **`flex: 1 1 300px` = 響應式咒語**——免 media query
- **垂直水平置中 3 行解決**——千古難題終結

> 💡 **下一課（L7）**：**Grid（網格）** = 二維排版引擎（行 × 列）。儀表板、雜誌排版、複雜佈局都靠它。**flex + grid 一起用 = 任何設計都做得出來**。
