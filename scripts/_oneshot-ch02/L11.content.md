**背景（background）/ 邊框（border）/ 陰影（shadow）/ 濾鏡（filter）** —— 這四個屬性決定元素的「**質感**」。**Apple / Linear / Vercel 的高級感全來自這裡**——這課讓你的設計從「能看」升級「精緻」。

> 💡 **這課的價值**：陰影是專業度的差別。新手寫 `box-shadow: 0 0 10px black` 粗糙陰影；老手疊三層微妙陰影模擬真實光線。**這課學完 = 設計力 +1 階**。

  
### 1. background（背景）

**background** 不只是顏色、可以放圖片、漸層、甚至多層：

📄 **寫進程式檔**：`style.css`

```text
.box {
  background-color: #fff;
  background-image: url('bg.jpg');
  background-repeat: no-repeat;
  background-position: center;
  background-size: cover;
  /* cover = 填滿（可能裁切）
     contain = 完整顯示（可能留白）
     100% 100% = 強制拉伸 */
  background-attachment: fixed;
  /* fixed = 滾動時背景不動（視差效果）*/
}
```

**速寫（一行搞定）**：

📄 **寫進程式檔**：`style.css`

```text
.hero {
  background: url('bg.jpg') center / cover no-repeat;
  /* image position / size repeat */
}
```

  
### 2. 多層背景（漸層 + 圖片）

**最常見技巧**：在圖片上加半透明漸層、讓文字看得清楚：

📄 **寫進程式檔**：`style.css`

```text
.hero {
  background:
    linear-gradient(rgba(0,0,0,.5), rgba(0,0,0,.3)),
    url('hero.jpg') center / cover;
  color: white;
}
/* 順序：前面的疊在上面、最後是底圖 */
```

> 💡 **這就是電商 hero 圖的標配技巧**——黑色半透明漸層蓋在圖上、白色文字立刻清楚可讀。

  
### 3. border（邊框）—— 完整用法

📄 **寫進程式檔**：`style.css`

```text
.box {
  /* 完整 */
  border: 2px solid red;
  /* 寬 樣式 顏色 */
  
  /* 各邊獨立 */
  border-top: 1px solid #eee;
  border-bottom: 2px solid blue;
  
  /* 樣式選項 */
  border-style: solid;
  /* solid（實線）/ dashed（虛線）/ dotted（點線）
     double（雙線）/ groove / ridge / inset / outset / none */
}
```

**border-radius（圓角）—— 最常用**：

📄 **寫進程式檔**：`style.css`

```text
.box {
  border-radius: 8px;
  
  /* 各角獨立 */
  border-top-left-radius: 20px;
  border-bottom-right-radius: 20px;
  
  /* 圓形 */
  border-radius: 50%;
  
  /* 膠囊 */
  border-radius: 999px;
  
  /* 不規則花瓣形 */
  border-radius: 30% 70% 70% 30% / 30% 30% 70% 70%;
  /* → 各角的 x / y 半徑分開設 */
}
```

> 💡 **`border-radius: 999px` 做膠囊形**——只要值大於高度的一半、自動變成兩端圓的膠囊。**所有 chip / tag / 圓形按鈕都這樣寫**。

  
### 4. outline（外框）—— 跟 border 像但不占空間

```text
border：
  → 占空間、影響盒模型計算

outline：
  → 不占空間、像「漂浮」在元素外
  → 主要用於 :focus 樣式（無障礙）
```

📄 **寫進程式檔**：`style.css`

```text
button:focus-visible {
  outline: 3px solid blue;
  outline-offset: 2px;
  /* offset = 跟元素的距離 */
}
```

> ⚠️ **永遠不要 `outline: none`**——這會讓鍵盤使用者完全看不到焦點、**無障礙重大缺陷**。要客製就客製清楚（見上例）。

  
### 5. box-shadow（盒子陰影）—— 高級感的關鍵

**box-shadow** 語法：`offsetX offsetY blur spread color`

📄 **寫進程式檔**：`style.css`

```text
.card {
  box-shadow: 0 4px 8px 0 rgba(0, 0, 0, 0.1);
  /* 
     0   水平偏移（正值=右、負值=左）
     4px 垂直偏移（正值=下、負值=上）
     8px 模糊距離（越大越柔）
     0   擴散距離（正值=放大、負值=縮小）
     rgba 顏色（通常半透明黑）
  */
}
```

**多層陰影（質感大師寫法）**：

📄 **寫進程式檔**：`style.css`

```text
.card-premium {
  box-shadow:
    0 1px 2px rgba(0, 0, 0, 0.04),
    0 4px 8px rgba(0, 0, 0, 0.06),
    0 16px 32px rgba(0, 0, 0, 0.08);
  /* 三層疊加：
     近層淡而清楚（模擬接觸面）
     中層柔（主光源）
     遠層大而淡（環境光）*/
}
```

> ✨ **接案小技巧**：不要寫 `0 0 10px black` 這種粗糙陰影。**看 Apple / Linear / Vercel 的網頁**——陰影都是「層疊微妙陰影」、接近真實光線。學會這個你的設計分量會升一階。

**內陰影**（內凹質感）：

```text
.input {
  box-shadow: inset 0 2px 4px rgba(0, 0, 0, 0.2);
  /* → inset 關鍵字、陰影在內部 */
}
```

  
### 6. text-shadow（文字陰影）

```text
.title {
  text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);
}

/* 發光效果 */
.glow {
  text-shadow: 0 0 10px cyan, 0 0 20px cyan;
}
```

> 💡 **text-shadow 慎用**——大部分時候會讓文字變糊、影響可讀性。除非要做「霓虹發光」這種特殊效果、否則少用。

  
### 7. filter（CSS 濾鏡）—— 像 Instagram

**filter** 直接對元素套濾鏡、像修圖 APP：

📄 **寫進程式檔**：`style.css`

```text
.img {
  filter: blur(5px);              /* 模糊 */
  filter: brightness(1.2);         /* 亮度 1.2 倍 */
  filter: contrast(1.5);           /* 對比 1.5 倍 */
  filter: grayscale(100%);         /* 灰階 */
  filter: hue-rotate(90deg);       /* 色相旋轉 */
  filter: invert(100%);            /* 反轉顏色 */
  filter: saturate(2);             /* 飽和度 2 倍 */
  filter: sepia(100%);             /* 復古褐 */
  
  /* 組合多個 */
  filter: brightness(1.1) contrast(1.05) saturate(1.2);
}

/* hover 時恢復原樣 */
.gallery img {
  filter: grayscale(100%);
  transition: filter 0.3s;
}
.gallery img:hover {
  filter: grayscale(0%);
}
```

  
### 8. backdrop-filter（毛玻璃）—— 2026 最潮

**backdrop-filter** = 對「**元素背後的東西**」套濾鏡——做毛玻璃效果：

📄 **寫進程式檔**：`style.css`

```text
.glass {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(20px) saturate(180%);
  -webkit-backdrop-filter: blur(20px) saturate(180%); /* Safari */
  border: 1px solid rgba(255, 255, 255, 0.2);
}
```

> 🚀 **毛玻璃 = 2026 最常見視覺**：iOS 控制中心、macOS Big Sur、Apple 官網、Linear、Vercel 全都用。**背景要有東西（圖片 / 內容）才看得出效果**。

  
### 9. 實戰：質感卡片

📄 **寫進程式檔**：`style.css`

```text
.card {
  background: white;
  border-radius: 12px;
  padding: 24px;
  box-shadow:
    0 1px 2px rgba(0, 0, 0, 0.04),
    0 4px 12px rgba(0, 0, 0, 0.06);
  border: 1px solid rgba(0, 0, 0, 0.04);
  transition: all 0.3s ease;
}

.card:hover {
  transform: translateY(-2px);
  box-shadow:
    0 4px 8px rgba(0, 0, 0, 0.06),
    0 12px 24px rgba(0, 0, 0, 0.1);
}
```

> 💡 **這就是 Linear / Stripe 卡片的「**浮起來**」效果**——hover 時往上 2px + 陰影變深、製造「靠近你」的錯覺。

  
### 10. 實戰：毛玻璃 modal

📄 **寫進程式檔**：`style.css`

```text
.modal-backdrop {
  position: fixed;
  inset: 0;
  background: rgba(0, 0, 0, 0.3);
  backdrop-filter: blur(8px);
  display: flex;
  justify-content: center;
  align-items: center;
}

.modal {
  background: rgba(255, 255, 255, 0.85);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.3);
  border-radius: 16px;
  padding: 32px;
  box-shadow: 0 24px 48px rgba(0, 0, 0, 0.2);
}
```

  
### 5 大常見坑

```text
❌ 1. 陰影寫死 black（不透明）
   → 看起來像 Word 95、超粗糙
   → 修：用 rgba 半透明黑（0.1-0.2 透明度）

❌ 2. 用 outline: none 關掉 focus 樣式
   → 鍵盤使用者完全找不到焦點
   → 修：永遠保留、用 :focus-visible 客製

❌ 3. backdrop-filter 沒效
   → Safari 需要 -webkit- 前綴、或父元素沒 background
   → 修：兩個 prefix 都寫、確認背景

❌ 4. border-radius 跟 overflow hidden 沒配合
   → 圖片溢出圓角範圍
   → 修：父元素加 overflow: hidden

❌ 5. 多層陰影沒層次（都同樣 blur）
   → 看起來像「整片黑黑」、沒立體感
   → 修：3 層 blur 從小到大（2 → 8 → 32）
```

  
### 🎯 重點回顧

- **background 可以多層**——漸層疊圖片做 hero
- **border-radius: 999px**——膠囊神咒
- **outline 不占空間**——專給 :focus 用
- **box-shadow 多層疊加**——這是高級感的秘密
- **backdrop-filter**——2026 最潮毛玻璃
- **filter 可組合**——`brightness(1.1) contrast(1.05)`
