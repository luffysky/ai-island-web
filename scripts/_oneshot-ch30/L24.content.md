### 完整技能樹（4 層）


### 第 4 層：商業（Month 9-12）

```text
✅ 接案 / 報價（NT$ 5K-500K 範圍）
```

### 5 大心法（過來人建議）

```text
1. ⭐ 領域知識 > 技術深度
   → 懂金融的爬蟲 > 純技術爬蟲
   → 找你熟悉的領域深耕

2. ⭐ 法律意識
   → 商業案必諮詢律師
   → 不確定 → 不做
   → 別碰個資 / 詐騙

3. ⭐ 永遠先 API
   → 官方 API > 爬蟲
   → 80% 場景 API 滿足

4. ⭐ 客戶導向
   → Dashboard > Raw data
   → 自動更新 > 手動寄
   → 月費 > 一次性

5. ⭐ 持續學新
   → AI 爬蟲在進化（browser-use 等）
   → 反偵測在進化（fingerprint）
   → 每月學 1 個新技術
```

### Indie 路線（最推薦給已會前端的人）

```text
=== 為什麼 Indie 推薦 ===
- 收入無上限（MRR 模式）
- 自由（無老闆 / 無 deadline）
- 同 stack 全端發展
- 學習 + 賺錢同時做

=== Indie 黃金 12 個月 ===
1. 找 1 個利基市場（如台灣房價 / 餐廳 / 競品）
2. 開發爬蟲 + Dashboard（Month 1-3）
3. 免費版上線、累積使用者（Month 4-6）
4. 付費版上線（NT$ 200-1000/月）
5. 行銷（Indie Hackers / 社群、Month 7-9）
6. 達 MRR NT$ 30K（Month 10-12）

→ 12 個月達 MRR USD $1000 可能
```
