**資料清理 + ETL Pipeline**——爬到的 raw data 要經過 **Extract → Transform → Load**、變成乾淨可用的資料庫 records。**這是爬蟲的「最後一哩」、決定資料能不能用**。

> 💡 **ETL 心法**：「**抓的時候 raw、不要邊抓邊清**」——萬一爬錯可重新處理、不用重爬。

  
### ETL 三階段

```text
=== Extract（抓）===
從來源拿 raw data
→ 不清理、原汁原味
→ 存到「raw layer」

=== Transform（清理 + 標準化）===
標準化欄位
清理格式
過濾無效
→ 存到「clean layer」

=== Load（寫 DB）===
batch insert 到 DB
建 index
→ 給 app / dashboard 用
```

  
### Extract（最簡單階段）

```text
def extract():
    """爬資料、用 streaming generator"""
    yield from scrape_all()
```

> 💡 **重點**：(1) **不要清理** (2) **原始格式存起來**（S3 / R2） (3) **失敗可重做**。

  
### Transform（清理 + 標準化）

```text
from datetime import datetime
import logging

logger = logging.getLogger(__name__)

def transform(raw_items):
    """清理 + 標準化"""
    cleaned = []
    
    for item in raw_items:
        # === 1. 清理價格 ===
        price_str = item['price'].replace('NT$', '').replace(',', '').strip()
        try:
            price = float(price_str)
        except ValueError:
            logger.warning(f"無法解析價格: {item['price']}")
            continue
        
        # === 2. 清理名稱 ===
        name = item['name'].strip()
        if not name:
            continue
        
        # === 3. 標準化分類 ===
        category = standardize_category(item.get('category'))
        
        # === 4. 加 timestamp ===
        cleaned.append({
            'name': name,
            'price': price,
            'category': category,
            'scraped_at': datetime.now(),
        })
    
    return cleaned

def standardize_category(raw):
    """分類標準化"""
    if not raw: return 'other'
    raw = raw.lower().strip()
    mapping = {
        '手機': 'phone', '電腦': 'computer',
        '配件': 'accessory', '保護貼': 'accessory',
    }
    return mapping.get(raw, 'other')
```

  
### Load（寫 DB）

```text
def load(items):
    """batch insert"""
    if not items:
        return
    
    # === bulk insert 100-1000x 快 ===
    db.bulk_insert(items)
    logger.info(f"載入 {len(items)} 筆")
```

  
### Orchestrate（串起來）

```text
def daily_etl():
    """每日 ETL pipeline"""
    logger.info("=== ETL 開始 ===")
    
    # E
    raw = list(extract())
    logger.info(f"Extract：{len(raw)} 筆")
    
    # T
    cleaned = transform(raw)
    logger.info(f"Transform：{len(cleaned)} 筆有效（過濾 {len(raw)-len(cleaned)}）")
    
    # L
    load(cleaned)
    logger.info(f"Load：完成")
    
    # === 通知 ===
    notify(f"今日 ETL：處理 {len(cleaned)} 筆")
```

### 增量 ETL（只處理新資料）

**情境**：每天爬、只要處理「**今天新的**」、不重做歷史。

```text
def incremental_etl():
    """增量 ETL"""
    # === 拿上次跑的時間 ===
    last_run = get_last_run_time()
    
    # === 只爬新的 ===
    new_items = scrape_since(last_run)
    
    if not new_items:
        logger.info("沒新資料")
        return
    
    # === T + L ===
    cleaned = transform(new_items)
    load(cleaned)
    
    # === 更新時間戳 ===
    update_last_run_time(datetime.now())
    
    logger.info(f"增量處理 {len(cleaned)} 筆")
```

### 處理規則（必遵守）

```text
✅ 缺欄位 → 跳過或填預設值
✅ 格式錯 → 嘗試解析、不行就跳過 + log
✅ 異常值 → 標記、不刪（之後人工檢查）
❌ 不要 silently drop（沒記 log）
✅ 永遠記下「哪幾筆失敗、為什麼」
```

  
### 監控（生產級必加）

```text
def daily_etl_with_monitoring():
    metrics = {
        'extracted': 0,
        'valid': 0,
        'invalid': 0,
        'duplicates': 0,
        'loaded': 0,
    }
    
    # E
    raw = list(extract())
    metrics['extracted'] = len(raw)
    
    # T
    cleaned = []
    seen = set()
    for item in raw:
        try:
            p = Product(**item)
        except ValidationError:
            metrics['invalid'] += 1
            continue
        
        if p.link in seen:
            metrics['duplicates'] += 1
            continue
        seen.add(p.link)
        
        cleaned.append(p)
        metrics['valid'] += 1
    
    # L
    load(cleaned)
    metrics['loaded'] = len(cleaned)
    
    # === 推 dashboard ===
    send_to_grafana(metrics)
    
    # === 異常告警 ===
    if metrics['valid'] / metrics['extracted'] < 0.8:
        send_alert(f"⚠️ 有效率低：{metrics['valid']/metrics['extracted']*100:.1f}%")
    
    return metrics
```

  
### Data Lake 架構（大規模 ETL）

```text
=== Bronze / Silver / Gold 三層架構 ===

[Bronze（raw）]
   ← 爬蟲直接寫
   → 不清理、保留原始
   → Parquet / JSON 在 S3

[Silver（cleaned）]
   ← 從 Bronze 讀 + 清理
   → Pydantic / dbt 驗證
   → Parquet 在 S3

[Gold（business-ready）]
   ← 從 Silver 讀 + 業務邏輯
   → 聚合、join、加 derived field
   → Postgres / ClickHouse、給 dashboard
```

> 💡 **Databricks 推這套**——分層好處：壞了從 Bronze 重做、不用重爬。

  
### 完整 production ETL 範本

```text
from datetime import datetime
import pandas as pd
from sqlalchemy import create_engine
import logging

logger = logging.getLogger(__name__)
engine = create_engine('postgresql://...')

def daily_etl():
    start = datetime.now()
    
    # === E：爬 + 存 raw 到 S3 ===
    raw = list(scrape_all())
    raw_df = pd.DataFrame(raw)
    raw_df.to_parquet(f's3://bronze/{start:%Y%m%d}/raw.parquet')
    logger.info(f"Bronze 層：存 {len(raw_df)} 筆")
    
    # === T：清理 ===
    clean_df = (raw_df
        .assign(price=lambda x: x['price'].str.replace(r'[^\d.]', '', regex=True).astype(float))
        .assign(name=lambda x: x['name'].str.strip())
        .dropna(subset=['name', 'price'])
        .drop_duplicates(subset=['link'])
        .assign(scraped_at=start)
    )
    clean_df.to_parquet(f's3://silver/{start:%Y%m%d}/clean.parquet')
    logger.info(f"Silver 層：清出 {len(clean_df)} 筆")
    
    # === L：寫 Postgres ===
    clean_df.to_sql('products', engine, if_exists='append', index=False)
    logger.info(f"Gold 層：寫 {len(clean_df)} 筆到 DB")
    
    # === 通知 ===
    duration = (datetime.now() - start).total_seconds()
    notify(f"✅ ETL 完成：{len(clean_df)} 筆、用時 {duration:.0f}s")

if __name__ == '__main__':
    daily_etl()
```

  
### 4 大心法

```text
1. ⭐ 抓的時候 raw、不要邊抓邊清
   → 萬一爬錯可重新處理
   → Bronze 層永遠存原始

2. ⭐ T 寫成 idempotent
   → 重跑結果一樣
   → 用 UPSERT 不用 INSERT

3. ⭐ 永遠保留 raw（debug 用）
   → S3 / R2 便宜
   → 之後查問題不用重爬

4. ⭐ 失敗的記下、不要 silently drop
   → log 哪幾筆失敗
   → 累積告警
```

  
    **☕ 用人話講**
    
ETL Pipeline 5 件事：(1) **Extract / Transform / Load** 三階段 (2) **pandas 鏈式** 寫起來最簡潔 (3) **增量 ETL** 只處理新資料 (4) **Bronze / Silver / Gold** 分層架構 (5) **工具**：Prefect / Inngest 推薦、Airflow 太重。**心法**：抓 raw → 清 → load、永遠保留 raw、idempotent、失敗要記 log。
