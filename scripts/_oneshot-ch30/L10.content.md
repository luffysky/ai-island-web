### jsoup（Java、最簡單）

📄 **寫進程式檔**：`Scraper.java`

```text
import org.jsoup.Jsoup;
import org.jsoup.nodes.Document;
import org.jsoup.nodes.Element;
import org.jsoup.select.Elements;

public class Scraper {
    public static void main(String[] args) throws Exception {
        // === 1. 抓 + 解析 一行搞定 ===
        Document doc = Jsoup.connect("https://example.com")
            .userAgent("Mozilla/5.0 (Macintosh) Chrome/131.0")
            .timeout(10_000)
            .get();
        
        // === 2. CSS selector 抓商品 ===
        Elements products = doc.select("div.product");
        
        for (Element p : products) {
            String name = p.select(".name").text();
            String price = p.select(".price").text();
            System.out.println(name + " - " + price);
        }
    }
}
```

> 💡 **jsoup 設計哲學**：HTTP + 解析合體、CSS selector 語法跟 BS4 一樣。**Java 爬蟲首選**。

### Kotlin（更簡潔、像 Python）

📄 **寫進程式檔**：`Scraper.kt`

```text
import org.jsoup.Jsoup

fun main() {
    val doc = Jsoup.connect("https://example.com")
        .userAgent("Mozilla/5.0 (Macintosh) Chrome/131.0")
        .get()
    
    // === Kotlin lambda 寫法、超像 Python ===
    doc.select("div.product").forEach { product ->
        val name = product.select(".name").text()
        val price = product.select(".price").text()
        println("$name - $price")
    }
}
```

> 💡 **Kotlin vs Java**：(1) Kotlin 簡潔（少 30% code） (2) 跟 Java 100% 互通 (3) Android 官方推薦 (4) Spring 全支援。**新專案直接上 Kotlin**。

  
### Kotlin Coroutines（並發）

📄 **寫進程式檔**：`ParallelScraper.kt`

```text
import kotlinx.coroutines.*
import org.jsoup.Jsoup

suspend fun fetch(url: String): String {
    return withContext(Dispatchers.IO) {
        Jsoup.connect(url).get().html()
    }
}

fun main() = runBlocking {
    val urls = listOf(
        "https://example.com/page/1",
        "https://example.com/page/2",
        "https://example.com/page/3",
    )
    
    // === 並行 fetch、async 模式 ===
    val results = urls.map { url ->
        async(Dispatchers.IO) { fetch(url) }
    }.awaitAll()
    
    println("Got ${results.size} pages")
}
```

> 💡 **Kotlin Coroutines** = Python asyncio 等價物、語法更直觀、效能更強。

  
### Spring Boot 排程爬蟲（企業常見）

📄 **寫進程式檔**：`ScraperService.java`

```text
import org.springframework.scheduling.annotation.Scheduled;
import org.springframework.stereotype.Service;
import org.jsoup.Jsoup;

@Service
public class ScraperService {
    
    // === @Scheduled cron 語法（每天 03:00）===
    @Scheduled(cron = "0 0 3 * * *")
    public void dailyScrape() throws Exception {
        var doc = Jsoup.connect("https://example.com").get();
        var products = doc.select("div.product");
        
        for (var p : products) {
            // 寫入 DB（用 JPA / Spring Data）
            productRepository.save(new Product(
                p.select(".name").text(),
                p.select(".price").text()
            ));
        }
    }
}
```

> 💡 **Spring `@Scheduled`** = Java 版 cron、不用外接系統、enterprise 標準。

  
### Android 內爬蟲（Kotlin）

**情境**：app 內抓資料、不過後端 API。

```text
import kotlinx.coroutines.*
import org.jsoup.Jsoup

class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        
        // === 在 background coroutine 跑（不阻塞 UI）===
        lifecycleScope.launch(Dispatchers.IO) {
            val doc = Jsoup.connect("https://example.com").get()
            val products = doc.select("div.product").map {
                it.select(".name").text()
            }
            
            // === 回 UI thread 更新畫面 ===
            withContext(Dispatchers.Main) {
                adapter.submitList(products)
            }
        }
    }
}
```

⚠️ **Android 限制**：需在 `AndroidManifest.xml` 加 `INTERNET` permission、不能在 main thread 跑 network call。

import com.microsoft.playwright.*;

try (Playwright playwright = Playwright.create()) {
    Browser browser = playwright.chromium().launch();
    Page page = browser.newPage();
    
    page.navigate("https://example.com");
    page.waitForSelector(".product");
    
    String title = page.locator("h1").innerText();

}
