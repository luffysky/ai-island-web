**Headless Browser Cluster**——單一 Playwright 跑 1 頁要 5 秒、10 個瀏覽器並行跑 = 0.5 秒 / 頁、**效率 10x**。

> 💡 **何時用**：(1) 動態網站 + 大量爬蟲（> 1000 頁） (2) 單 browser 不夠快 (3) 要榨乾機器資源。**核心套件**：puppeteer-cluster / 自製 Browser Pool。


**這 2 個套件做什麼？**
- `puppeteer` —— 瀏覽器控制
- `puppeteer-cluster` —— 包裝 puppeteer 成 cluster、管理多 browser instance

  
📄 **寫進程式檔**：`cluster-scrape.ts`

```text
import { Cluster } from 'puppeteer-cluster';

const cluster = await Cluster.launch({
  // === 3 種並發模式 ===
  concurrency: Cluster.CONCURRENCY_CONTEXT,    // 共用 browser、每 task 新 context（推薦）
  // CONCURRENCY_PAGE     - 共用 browser + context、每 task 新 page（記憶體最省）
  // CONCURRENCY_BROWSER  - 每 task 新 browser（最隔離、記憶體最多）
  
  maxConcurrency: 10,                          // 同時 10 個 task
  puppeteerOptions: { headless: true }
});

// === 定義 task ===
await cluster.task(async ({ page, data: url }) => {
  await page.goto(url);
  const title = await page.title();
  return { url, title };
});

// === 加 100 個 task ===
const urls = [...]; // 100 個 URL
for (const url of urls) {
  cluster.queue(url);
}

// === 等全部完成 ===
await cluster.idle();
await cluster.close();
```

> 💡 **3 種並發模式選擇**：(1) `CONTEXT` 平衡（推薦）(2) `PAGE` 最省（同 context、cookie 共用） (3) `BROWSER` 最隔離（每 task 全新環境）。

  
### Playwright + p-limit（手動 Cluster）

**情境**：要 Playwright（不要 Puppeteer）、用 p-limit 控制並發。

```text
import { chromium } from 'playwright';
import pLimit from 'p-limit';

const browser = await chromium.launch();
const limit = pLimit(10);                      // 同時 10 個

const results = await Promise.all(
  urls.map(url => limit(async () => {
    // === 每個 task 新 context（隔離 cookie）===
    const context = await browser.newContext();
    const page = await context.newPage();
    
    try {
      await page.goto(url, { timeout: 30000 });
      const data = await page.locator('h1').textContent();
      return { url, data };
    } finally {
      await context.close();                   // 確保關閉
    }
  }))
);

await browser.close();
```

  
### 自製 Browser Pool（最進階）

**情境**：要極致效能、自己管 pool。

```text
import { chromium, Browser } from 'playwright';

class BrowserPool {
  private browsers: Browser[] = [];
  private queue: ((b: Browser) => void)[] = [];
  
  constructor(private size = 5) {}
  
  // === 初始化、預先開 N 個 browser ===
  async init() {
    for (let i = 0; i < this.size; i++) {
      this.browsers.push(await chromium.launch());
    }
  }
  
  // === 取一個 browser（沒有就排隊等）===
  async acquire(): Promise<Browser> {
    if (this.browsers.length > 0) {
      return this.browsers.pop()!;
    }
    return new Promise(resolve => this.queue.push(resolve));
  }
  
  // === 還回去 ===
  release(browser: Browser) {
    if (this.queue.length > 0) {
      // 有人在排隊、直接給他
      this.queue.shift()!(browser);
    } else {
      this.browsers.push(browser);
    }
  }
  
  async close() {
    for (const b of this.browsers) await b.close();
  }
}

// === 用 ===
const pool = new BrowserPool(5);
await pool.init();

async function scrape(url: string) {
  const browser = await pool.acquire();
  try {
    const page = await browser.newPage();
    await page.goto(url);
    const title = await page.title();
    await page.close();
    return title;
  } finally {
    pool.release(browser);
  }
}

const results = await Promise.all(urls.map(scrape));
await pool.close();
```

  
### 記憶體控制（cluster 殺手）

**問題**：每個 browser 約 200MB、10 個 = 2GB、跑久了會 leak 到 4GB+。

**解法 1：定期重啟 browser**

```text
let browser = await chromium.launch();
let count = 0;

async function scrape(url: string) {
  // === 每 100 個 task 重啟 browser ===
  if (count++ % 100 === 0 && count > 0) {
    await browser.close();
    browser = await chromium.launch();
  }
  
  const page = await browser.newPage();
  // ...
}
```

**解法 2：limit context 數量**

```text
// CONCURRENCY_CONTEXT 模式自動回收
// 你只要設 maxConcurrency 即可
```

**解法 3：cluster 內建記憶體上限**

```text
const cluster = await Cluster.launch({
  concurrency: Cluster.CONCURRENCY_CONTEXT,
  maxConcurrency: 10,
  retryLimit: 3,                               // 失敗 retry 3 次
  retryDelay: 5000,                            // retry 間隔 5 秒
  timeout: 60_000,                             // 單 task 60 秒超時
  monitor: true,                               // 印即時統計
});
```

### Browser Cluster 6 大坑

```text
1. ❌ 記憶體 leak
   ✓ 定期重啟 browser

2. ❌ Context 沒關
   ✓ try / finally 確保 close

3. ❌ Page 沒關
   ✓ 每個 task 結束關 page

4. ❌ Headful 模式跑大量
   ✓ 用 headless（除非要看畫面 debug）

5. ❌ 並發太高機器爆
   ✓ 監控 CPU / RAM、調 maxConcurrency

6. ❌ 一個 task 卡住全卡
   ✓ 設 timeout
```

  
### 完整 production 範本

```text
import { Cluster } from 'puppeteer-cluster';
import puppeteer from 'puppeteer-extra';
import StealthPlugin from 'puppeteer-extra-plugin-stealth';

puppeteer.use(StealthPlugin());

const cluster = await Cluster.launch({
  puppeteer,
  concurrency: Cluster.CONCURRENCY_CONTEXT,
  maxConcurrency: 10,
  retryLimit: 3,
  retryDelay: 5000,
  timeout: 60_000,
  puppeteerOptions: {
    headless: true,
    args: ['--no-sandbox', '--disable-setuid-sandbox'],
  },
  monitor: true,
});

await cluster.task(async ({ page, data: url }) => {
  // === 反偵測 ===
  await page.setUserAgent('Mozilla/5.0 (Macintosh) Chrome/131.0');
  
  // === 阻擋圖片加速 ===
  await page.setRequestInterception(true);
  page.on('request', req => {
    if (['image', 'stylesheet', 'font'].includes(req.resourceType())) req.abort();
    else req.continue();
  });
  
  // === 爬 ===
  await page.goto(url, { waitUntil: 'networkidle2' });
  const data = await page.$$eval('.product', els => 
    els.map(el => ({
      name: el.querySelector('.name')?.textContent,
      price: el.querySelector('.price')?.textContent,
    }))
  );
  
  return { url, data };
});

// === 加 task ===
for (const url of urls) {
  cluster.queue(url);
}

await cluster.idle();
await cluster.close();
```
