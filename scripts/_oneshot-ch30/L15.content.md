
**這行做什麼？** 裝 PaddleOCR + PaddlePaddle 引擎（百度開源）、**中文準確度業界最強**。

```text
from paddleocr import PaddleOCR

# === 初始化（第一次跑會下載模型約 100MB）===
ocr = PaddleOCR(use_angle_cls=True, lang="ch")   # use_angle_cls=自動偵測旋轉

# === 辨識 ===
result = ocr.ocr("image.png", cls=True)

for line in result[0]:
    bbox = line[0]                                # 文字位置（4 個角座標）
    text = line[1][0]                             # 文字內容
    confidence = line[1][1]                       # 信心度（0-1）
    print(f"{text} (信心: {confidence:.2f})")
```

> 💡 **PaddleOCR 殺手鐧**：(1) 中文準確度 95%+ (2) 支援表格 / 印章 / 手寫 (3) 開源免費 (4) PP-Structure 解整份文件。

  
### Claude Vision（AI、2026 最強）

**情境**：要**結構化抽取**（如「**從發票拿商家 + 日期 + 總金額 + 品項**」）。

```text
import anthropic
import base64

client = anthropic.Anthropic()

# === 讀圖片 + base64 編碼 ===
with open("invoice.png", "rb") as f:
    img = base64.b64encode(f.read()).decode()

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=1024,
    messages=[{
        "role": "user",
        "content": [
            {
                "type": "image",
                "source": {
                    "type": "base64",
                    "media_type": "image/png",
                    "data": img
                }
            },
            {
                "type": "text",
                "text": """從發票上抽出以下欄位、回傳 JSON：
- merchant：商家名稱
- date：日期（YYYY-MM-DD）
- total：總金額（數字）
- items：品項陣列 [{name, qty, price}]"""
            }
        ]
    }]
)

import json
data = json.loads(response.content[0].text)
print(data)
# {
#   "merchant": "7-11",
#   "date": "2026-05-27",
#   "total": 158,
#   "items": [
#     {"name": "思樂冰", "qty": 1, "price": 35},
#     {"name": "茶葉蛋", "qty": 2, "price": 26}
#   ]
# }
```

> 💡 **Claude Vision 強在哪**：(1) **理解 context**——知道「發票」「商家」是什麼 (2) 自動結構化、不用後處理 (3) 手寫 / 模糊 / 多語言全包 (4) 一個 prompt 完成 OCR + parser。**比 Tesseract 強 100 倍**。

  
### 解 Captcha（驗證碼）

**情境**：爬蟲遇到 captcha、需要解。

**方案 1：2Captcha（付費、人工 + AI 解）：**

```text
import requests
import time

API_KEY = "你的_2captcha_key"

# === 1. 傳 captcha 圖片 ===
with open("captcha.png", "rb") as f:
    res = requests.post("http://2captcha.com/in.php", files={
        "file": f
    }, data={
        "key": API_KEY,
        "method": "post"
    })

captcha_id = res.text.split("|")[1]

# === 2. 輪詢結果（通常 10-30 秒）===
while True:
    res = requests.get(
        f"http://2captcha.com/res.php?key={API_KEY}&action=get&id={captcha_id}"
    )
    
    if res.text.startswith("OK"):
        solution = res.text.split("|")[1]
        print(f"Captcha solved: {solution}")
        break
    
    if res.text == "CAPCHA_NOT_READY":
        time.sleep(2)
        continue
    
    raise Exception(f"Failed: {res.text}")
```

  
**方案 2：用 Claude Vision 自己解（便宜 100x）：**

```text
# === 把 captcha 圖片給 Claude ===
prompt = "讀出這張驗證碼圖片上的文字、只回答文字本身"

response = client.messages.create(
    model="claude-opus-4-7",
    max_tokens=100,
    messages=[{
        "role": "user",
        "content": [
            {"type": "image", "source": {...}},
            {"type": "text", "text": prompt}
        ]
    }]
)

captcha_text = response.content[0].text.strip()
# 成本約 $0.003 / 張、比 2Captcha 便宜 100x
```

**這行做什麼？** 裝 ultralytics（YOLO v8 官方套件）、物件偵測 SOTA。

```text
from ultralytics import YOLO

# === 載入預訓練模型（80 個常見物件）===
model = YOLO("yolov8n.pt")                        # n = nano（最小最快）

# === 偵測 ===
results = model("image.jpg")

for r in results:
    # 取每個偵測到的物件
    for box in r.boxes:
        cls = r.names[int(box.cls)]               # 物件類別（如 person, car）
        conf = float(box.conf)                    # 信心度
        x1, y1, x2, y2 = box.xyxy[0]              # 框的座標
        
        print(f"{cls}: 信心 {conf:.2f}, 位置 ({x1:.0f}, {y1:.0f}, {x2:.0f}, {y2:.0f})")

# === 存帶框圖 ===
results[0].save("output.jpg")
```

> 💡 **YOLO v8 內建 80 個類別**：person / car / dog / chair / cup ... 等。**自訓練可加自家類別**（如「商品標籤」「特定 logo」）。

### 完整範例：發票記帳 SaaS

📄 **寫進程式檔**：`invoice_scanner.py`

```text
import anthropic
import base64
import json
from sqlalchemy import create_engine
from sqlalchemy.orm import Session
from models import Expense

client = anthropic.Anthropic()
engine = create_engine("postgresql://...")

def scan_invoice(image_path: str) -> dict:
    """掃發票、回結構化資料"""
    with open(image_path, "rb") as f:
        img = base64.b64encode(f.read()).decode()
    
    response = client.messages.create(
        model="claude-opus-4-7",
        max_tokens=2048,
        messages=[{
            "role": "user",
            "content": [
                {
                    "type": "image",
                    "source": {
                        "type": "base64",
                        "media_type": "image/jpeg",
                        "data": img
                    }
                },
                {
                    "type": "text",
                    "text": """分析發票、回 JSON：
{
  "type": "<food/electronics/clothing/other>",
  "merchant": "商家",
  "date": "YYYY-MM-DD",
  "total": 數字,
  "items": [{"name": "", "qty": 1, "price": 0}]
}"""
                }
            ]
        }]
    )
    
    return json.loads(response.content[0].text)

def save_to_db(data: dict):
    """存進 DB"""
    with Session(engine) as session:
        expense = Expense(
            type=data["type"],
            merchant=data["merchant"],
            date=data["date"],
            total=data["total"],
            items=data["items"]
        )
        session.add(expense)
        session.commit()

# === 用 ===
data = scan_invoice("photo.jpg")
save_to_db(data)
print(f"✓ 記帳：{data['merchant']} NT$ {data['total']}")
```

  
### 心法選擇

```text

簡單文字（標題、價格）
  → Tesseract / PaddleOCR

複雜文件（發票、合約、表格）
  → Claude Vision（結構化最強）

中文準確度要求高
  → PaddleOCR

預算 0、要免費
  → Tesseract / PaddleOCR

要 OCR + 解析 + 入庫一條龍
  → Claude Vision

Captcha
  → 2Captcha（穩）/ Claude Vision（便宜 100x）

物件偵測（人 / 車 / 商品）
  → YOLO

PDF
  → PyMuPDF + PaddleOCR + Claude Vision
```
