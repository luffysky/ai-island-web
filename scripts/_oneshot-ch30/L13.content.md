**Excel / Google Sheets 爬蟲 = 不寫程式的爬蟲**——給**完全不會 code 的人**（行政 / 業務 / 老闆 / 行銷）用、3 個內建公式就能抓網頁資料。

> 💡 **適合**：(1) 每天小量更新（< 100 筆） (2) 簡單表格 / RSS / CSV (3) 給非工程師交付結果。**不適合**：JS 渲染 / Cloudflare / 大量資料。

  
### Excel Power Query（最強內建爬蟲）

**3 步驟抓網頁表格：**

```text
1. 開 Excel → 「資料」分頁 → 「從網路」
2. 貼 URL（如 https://en.wikipedia.org/wiki/List_of_countries_by_population）
3. Excel 自動偵測 <table>、選你要的、按「載入」

→ 表格直接變 Excel sheet
→ 設「自動更新」、每小時抓新資料
```

> 💡 **Power Query 殺手鐧**：(1) 視覺化、不寫 code (2) 自動偵測表格 (3) 支援轉換 / 清理 / 合併多源 (4) 設定定時更新。

  
### Power Query M 語言（進階）

📄 **在 Power Query 編輯器寫**：

```text
let
    // === 抓網頁 ===
    Source = Web.Page(Web.Contents("https://example.com")),
    
    // === 取第 1 個表格 ===
    Data = Source{0}[Data],
    
    // === 篩選 ===
    Filtered = Table.SelectRows(Data, each [Price] < 1000)
in
    Filtered
```

> 💡 **M 語言** = Power Query 內建程式語言、像 SQL + functional、適合進階使用者。

  
### Google Sheets IMPORTHTML

**情境**：抓網頁的 `<table>` 或 `<ul>`。

**在 Sheets 任一儲存格輸入：**

```text
=IMPORTHTML("https://en.wikipedia.org/wiki/List_of_countries_by_population", "table", 1)
```

**參數解析：**
- 第 1 個 —— URL
- 第 2 個 —— 「table」或「list」
- 第 3 個 —— 第幾個（1 開始）

> 💡 **IMPORTHTML 限制**：(1) 只抓 `<table>` / `<ul>` / `<ol>` (2) 不執行 JS (3) 重新整理頻率 1 小時。

  
### Google Sheets IMPORTXML（XPath 抓任何元素）

```text
# === 抓所有 price class 的元素 ===
=IMPORTXML("https://example.com", "//div[@class='price']")

# === 抓所有連結 ===
=IMPORTXML("https://example.com", "//a/@href")

# === 抓第一個 h1 ===
=IMPORTXML("https://example.com", "//h1[1]")

# === 抓 meta description ===
=IMPORTXML("https://example.com", "//meta[@name='description']/@content")
```

> 💡 **XPath 基本語法**：(1) `//tag` 找所有 tag (2) `[@attr='val']` 屬性過濾 (3) `/@attr` 取屬性值 (4) `[n]` 第 n 個。

  
### Google Sheets IMPORTDATA（CSV）

```text
=IMPORTDATA("https://example.com/data.csv")
```

**這做什麼？** 抓 URL 的 CSV、自動分欄、填到 sheet。

  
### Google Sheets IMPORTFEED（RSS）

```text
# === 抓 RSS、預設 5 個項目 ===
=IMPORTFEED("https://example.com/rss")

# === 指定欄位 + 數量 ===
=IMPORTFEED("https://example.com/rss", "items", FALSE, 10)
```

  
### Google Apps Script（最強、可寫 JS）

**情境**：上述函數不夠用、要寫 logic 抓 API。

📄 **在 Google Sheets 點「擴充功能」→「Apps Script」**：

```text
function scrapeAndStore() {
  // === 1. 打 API ===
  const response = UrlFetchApp.fetch("https://api.example.com/products");
  const data = JSON.parse(response.getContentText());
  
  // === 2. 拿到目前 sheet ===
  const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // === 3. 加 header（第一次）===
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['Name', 'Price', 'Scraped At']);
  }
  
  // === 4. 加每筆資料 ===
  data.products.forEach(p => {
    sheet.appendRow([p.name, p.price, new Date()]);
  });
}
```

  
**設 trigger（自動跑）：**

```text
1. 點「觸發條件」（左側時鐘 icon）
2. 「新增觸發條件」
3. 選函數：scrapeAndStore
4. 時間驅動 → 每天 09:00
5. 儲存
```

  
### Apps Script 進階：發 email 報表

```text
function dailyReport() {
  const sheet = SpreadsheetApp.getActiveSheet();
  const data = sheet.getDataRange().getValues();
  
  // === 統計 ===
  const today = new Date().toISOString().split('T')[0];
  const todayData = data.filter(row => 
    row[2] && row[2].toISOString().startsWith(today)
  );
  
  // === 發 email ===
  MailApp.sendEmail({
    to: 'boss@company.com',
    subject: `今日爬蟲報表 - ${today}`,
    body: `今天抓了 ${todayData.length} 筆\n平均價格：${avg(todayData.map(r => r[1]))}`,
  });
}
```

  
### 限制（必看）

```text
❌ 不能執行 JS
   → SPA / React 網站抓不到
   → 解：用 Selenium / Playwright

❌ 大量資料慢
   → Sheets 上限 1000 萬格
   → IMPORTHTML 同時 ≤ 50 個

❌ 複雜邏輯難
   → 多步流程 / 登入 / 反爬 不行
   → 解：寫 Apps Script 或外部爬蟲存進來

❌ 不能解 Cloudflare
   → 有反爬就抓不到

⚠️ Apps Script 每天 6 小時上限
   → 個人帳號免費 6h、Workspace 帳號 6h
   → 超過要等隔天
```
