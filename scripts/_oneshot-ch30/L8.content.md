### Hello World 爬蟲

📄 **寫進程式檔**：`main.go`

```text
package main

import (
    "fmt"
    "github.com/gocolly/colly/v2"
)

func main() {
    // === 建 Collector ===
    c := colly.NewCollector(
        colly.AllowedDomains("example.com"),
        colly.UserAgent("Mozilla/5.0 (Macintosh) Chrome/131.0"),
    )
    
    // === 註冊 callback：抓商品 ===
    c.OnHTML("div.product", func(e *colly.HTMLElement) {
        fmt.Println(e.ChildText(".name"), e.ChildText(".price"))
    })
    
    // === 起跑 ===
    c.Visit("https://example.com/products")
}
```

### 自動跟連結（深度爬蟲）

```text
// === 找所有 a[href]、自動進入下一頁 ===
c.OnHTML("a[href]", func(e *colly.HTMLElement) {
    c.Visit(e.Request.AbsoluteURL(e.Attr("href")))
})
```

> 💡 **AbsoluteURL** = 把相對路徑（`/page/2`）轉成絕對路徑（`https://example.com/page/2`）、爬蟲必用。

  
### 儲存結果（JSON）

```text
type Product struct {
    Name  string `json:"name"`
    Link  string `json:"link"`
    Price float64 `json:"price"`
}

var products []Product

c.OnHTML("div.product", func(e *colly.HTMLElement) {
    products = append(products, Product{
        Name: e.ChildText(".name"),
        Link: e.ChildAttr("a", "href"),
    })
})

// === OnScraped：爬完一頁觸發 ===
c.OnScraped(func(r *colly.Response) {
    file, _ := os.Create("products.json")
    defer file.Close()
    json.NewEncoder(file).Encode(products)
})

c.Visit("https://example.com")
```

> 💡 **`json` tag** = struct 序列化成 JSON 時的欄位名稱、Go 慣例。

  
### 動態網站（chromedp、Go 版 Puppeteer）

**情境**：JS 渲染、colly 抓不到、用 chromedp 開真瀏覽器。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ **在終端機輸入**

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

```text
go get -u github.com/chromedp/chromedp
```

**這行做什麼？** 裝 chromedp、Go 控 Chrome 的套件、用 CDP 協議直連（不用 WebDriver）。

  
📄 **寫進程式檔**：`dynamic.go`

```text
package main

import (
    "context"
    "fmt"
    "log"
    "time"
    "github.com/chromedp/chromedp"
)

func main() {
    // === 1. 建 context（自動管理瀏覽器生命週期）===
    ctx, cancel := chromedp.NewContext(context.Background())
    defer cancel()
    
    // === 2. 加 timeout（30 秒）===
    ctx, cancel = context.WithTimeout(ctx, 30*time.Second)
    defer cancel()
    
    // === 3. 開頁 + 等元素 + 取 HTML ===
    var html string
    err := chromedp.Run(ctx,
        chromedp.Navigate("https://example.com"),
        chromedp.WaitVisible("h1"),           // 等 h1 出現
        chromedp.OuterHTML("body", &html),    // 取整個 body 的 HTML
    )
    
    if err != nil {
        log.Fatal(err)
    }
    
    fmt.Println(html)
}
```

> 💡 **chromedp 設計哲學**：(1) 用 `Action` slice 描述操作 (2) `chromedp.Run` 依序執行 (3) `context.WithTimeout` 控制超時。

  
### chromedp + colly 整合（最強）

**情境**：先用 chromedp 拿渲染後 HTML、再給 colly 解析。

```text
// === Step 1：chromedp 取 HTML ===
var html string
chromedp.Run(ctx,
    chromedp.Navigate(url),
    chromedp.WaitVisible(".product"),
    chromedp.OuterHTML("body", &html),
)

// === Step 2：colly 解析 ===
c := colly.NewCollector()
c.OnHTML("body", func(e *colly.HTMLElement) {
    e.ForEach(".product", func(_ int, p *colly.HTMLElement) {
        fmt.Println(p.ChildText(".name"))
    })
})

// 從 HTML 字串解析（不用 visit URL）
c.OnRequest(func(r *colly.Request) {
    r.Abort()                                 // 阻止實際 fetch
})

// hack：用 NewRequest + ParseHTML
// 或更簡單：直接用 goquery 解析
```

### 何時用 Go 爬蟲

```text

- 規模 100 萬+/天（colly 並發 5 萬）

- ML / 資料分析（Python 接 pandas）
- 接案（客戶找不到 Go 工程師維護）
- MVP / 1 天 demo
- 需要超完整框架（Scrapy 強很多）
```
