## 為什麼
傳統：累積 list → 記憶體爆
Streaming：邊抓邊存 → 記憶體小

## Python generator
```python
def scrape_all():
    for page in range(1, 1000):
        for item in scrape_page(page):
            yield item

for item in scrape_all():
    save_to_db(item)
```

## JSONL streaming
```python
import json

with open("data.jsonl", "w") as f:
    for item in scrape_all():
        f.write(json.dumps(item, ensure_ascii=False) + "\n")
```

## Async generator
```python
async def scrape_all():
    async with httpx.AsyncClient() as client:
        for page in range(1, 1000):
            res = await client.get(f"https://example.com/page/{page}")
            for item in parse(res.text):
                yield item

async for item in scrape_all():
    await save_async(item)
```

## Scrapy Pipeline（自動 streaming）
```python
class MongoPipeline:
    def process_item(self, item, spider):
        self.db.items.insert_one(dict(item))
        return item
```

## Node Streams
```javascript
import { pipeline } from "node:stream/promises";
import { createWriteStream } from "node:fs";

async function* scrapeAll() {
  for (let page = 1; page <= 1000; page++) {
    const items = await scrapePage(page);
    for (const item of items) {
      yield JSON.stringify(item) + "\n";
    }
  }
}

await pipeline(scrapeAll(), createWriteStream("data.jsonl"));
```

## 批量寫 DB
```python
BATCH_SIZE = 1000
batch = []

for item in scrape_all():
    batch.append(item)
    if len(batch) >= BATCH_SIZE:
        db.bulk_insert(batch)
        batch = []

if batch:
    db.bulk_insert(batch)
```

## 記憶體監控
```python
import psutil, os

process = psutil.Process(os.getpid())
print(f"Memory: {process.memory_info().rss / 1024 / 1024:.1f} MB")
```

百萬筆用 streaming：< 100 MB。
不用：可能 10 GB OOM。
