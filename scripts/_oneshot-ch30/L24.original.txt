## 全能技能樹

### 基礎（必備）
- HTTP / HTML / CSS Selector
- 至少 1 個語言（Python / Node）
- requests / fetch
- 解析（BS4 / cheerio）

### 中階
- 動態網站（Playwright）
- 反爬蟲（UA / 代理 / Cookie）
- Async / 並發
- Database
- 排程（cron / Airflow）

### 進階
- 反偵測（stealth / fingerprint）
- TLS impersonate
- Distributed（Celery）
- AI 爬蟲（browser-use）
- OCR / Vision

### 商業
- 接案 / 報價
- 法律 / 道德
- Dashboard / 交付
- SaaS 化

## 跨語言能力

主語言精通（Python or Node）+ 略懂其他：
- Go：知道 colly、巨量場景用
- Rust：知道 reqwest、極致場景
- Java：知道 jsoup、企業環境
- PHP / Ruby：知道、不用主用
- cURL：CLI 必懂

## 心法
- **領域知識 > 技術深度**：懂金融的爬蟲 > 純技術
- **法律意識**：商業案必諮詢律師
- **永遠先 API**：減少不必要爬
- **客戶導向**：dashboard > raw data
- **持續學新**：AI 爬蟲、反偵測在進化

## 給新手最佳路徑
1. 學 Python（最大生態）
2. 跑通第一個爬蟲（任何網站）
3. 學 Scrapy + Playwright
4. 接第一個付費案
5. 累積 5 個 case
6. 開始 SaaS / 顧問

1 年後、你會是專業爬蟲工程師。
