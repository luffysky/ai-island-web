### 網站偵測你的 8 大指紋

```text
1. TLS / JA3 fingerprint
   → TLS Client Hello 順序、加密套件清單
   → Python requests / Node fetch ≠ Chrome
   → 解：用 curl-impersonate / got-scraping

2. HTTP/2 fingerprint
   → H2 frame 順序、settings 值
   → Python requests 不支援 H2
   → 解：用 httpx[http2] / undici H2

3. Canvas fingerprint
   → 用 canvas 畫文字 + 取像素 hash
   → 每台機器 GPU / 字型不同 = 唯一指紋
   → 解：每次微擾像素

4. WebGL fingerprint
   → GPU vendor / renderer 字串
   → headless Chrome 預設 SwiftShader
   → 解：偽造成 Intel / NVIDIA

5. AudioContext fingerprint
   → 用 AudioContext 處理音訊 + 取 hash
   → 較難偽造
   → 解：stealth plugin 自動修

6. 字型列表
   → 機器裝的字型清單（測 measureText）
   → headless 字型少
   → 解：注入常見字型清單

7. 時區 / 語言
   → Date.getTimezoneOffset / navigator.language
   → 跟 IP 不一致可疑
   → 解：對齊 IP 國家設定

8. 螢幕解析度
   → screen.width / height
   → 1920x1080 太常見
   → 解：用真實 4K / 多元解析度
```

### 修補 2：Canvas fingerprint（每次微擾像素）

```text
await page.evaluateOnNewDocument(() => {
  const orig = HTMLCanvasElement.prototype.toDataURL;
  
  HTMLCanvasElement.prototype.toDataURL = function(...args) {
    const ctx = this.getContext('2d');
    if (ctx) {
      // === 抓 image data ===
      const imageData = ctx.getImageData(0, 0, this.width, this.height);
      
      // === 每個像素隨機 ±1 ===
      for (let i = 0; i < imageData.data.length; i += 4) {
        imageData.data[i] += Math.random() < 0.5 ? -1 : 1;
      }
      
      ctx.putImageData(imageData, 0, 0);
    }
    return orig.apply(this, args);
  };
});
```

> 💡 **為什麼這樣修**：(1) 對方畫 canvas + hash 像素 (2) 我們微擾像素 ±1（肉眼看不出來） (3) hash 完全不同、每次 hash 都不一樣 = 騙過 Canvas fingerprint。

  
### 修補 3：WebGL fingerprint

```text
await page.evaluateOnNewDocument(() => {
  const orig = WebGLRenderingContext.prototype.getParameter;
  
  WebGLRenderingContext.prototype.getParameter = function(param) {
    // === 37445 = UNMASKED_VENDOR_WEBGL ===
    if (param === 37445) return 'Intel Inc.';
    
    // === 37446 = UNMASKED_RENDERER_WEBGL ===
    if (param === 37446) return 'Intel Iris OpenGL Engine';
    
    return orig.apply(this, [param]);
  };
});
```

  
### 修補 4：navigator 全套（plugins / languages / platform）

```text
await page.evaluateOnNewDocument(() => {
  // === 1. webdriver flag（最常見偵測）===
  Object.defineProperty(navigator, 'webdriver', { get: () => undefined });
  
  // === 2. plugins（headless 預設空 array）===
  Object.defineProperty(navigator, 'plugins', { 
    get: () => [
      { name: 'PDF Viewer', filename: 'internal-pdf-viewer' },
      { name: 'Chrome PDF Viewer', filename: 'internal-pdf-viewer' },
      { name: 'Native Client', filename: 'internal-nacl-plugin' },
    ]
  });
  
  // === 3. languages（headless 預設 en-US 一個）===
  Object.defineProperty(navigator, 'languages', { 
    get: () => ['zh-TW', 'zh', 'en-US', 'en']
  });
  
  // === 4. platform ===
  Object.defineProperty(navigator, 'platform', { get: () => 'MacIntel' });
  
  // === 5. hardwareConcurrency（CPU 核數）===
  Object.defineProperty(navigator, 'hardwareConcurrency', { get: () => 8 });
  
  // === 6. deviceMemory ===
  Object.defineProperty(navigator, 'deviceMemory', { get: () => 8 });
});
```

### ML 偵測（最難、看「行為」）

**Cloudflare / DataDome / PerimeterX 2026 主流用 ML**：

```text
=== ML 偵測 5 大指標 ===

1. 滑鼠軌跡
   → 真人軌跡曲線 / 速度變化
   → 爬蟲直接點（瞬移）

2. 鍵盤節奏
   → 真人打字有停頓 / 錯字 / 速度變化
   → 爬蟲等距打字（page.type 預設）

3. 點擊位置
   → 真人點按鈕「中間」常見
   → 爬蟲點「精確中心」太完美

4. 捲動模式
   → 真人不規律捲、停下看
   → 爬蟲固定速度捲

5. 頁面停留時間
   → 真人讀 10-60 秒
   → 爬蟲 1 秒就跳
```

  
### 對策：模擬真人行為

```text
// === 1. 真人打字 ===
async function humanType(page, selector, text) {
  for (const char of text) {
    await page.type(selector, char, {
      delay: Math.random() * 200 + 50  // 50-250ms 隨機
    });
    
    // === 5% 機率「想一下」===
    if (Math.random() < 0.05) {
      await page.waitForTimeout(Math.random() * 1000 + 500);
    }
  }
}

// === 2. 真人滑鼠（路徑、不瞬移）===
await page.mouse.move(100, 200);
await page.waitForTimeout(300);
await page.mouse.move(300, 400, { steps: 10 });   // 10 步分段
await page.waitForTimeout(200);
await page.mouse.click(300, 400);

// === 3. 真人捲動 ===
async function humanScroll(page) {
  const steps = 5 + Math.floor(Math.random() * 5);  // 5-10 次
  for (let i = 0; i < steps; i++) {
    await page.evaluate((dy) => window.scrollBy(0, dy), 200 + Math.random() * 400);
    await page.waitForTimeout(500 + Math.random() * 2000);
  }
}

// === 4. 真人停留 ===
await page.waitForTimeout(Math.random() * 5000 + 3000);  // 3-8 秒
```
