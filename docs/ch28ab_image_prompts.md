# Ch28a / Ch28b 教學圖 — 26 張完整出圖 Prompt（貼了就能生）

> 用法：每張圖**跑兩次**——先生「深色版」（用下方 prompt）、再生「淺色版」（把背景那句換掉、見下）。
> 生好存成對應檔名、放進資料夾、網站會自動顯示（已佈線、只寫 dark、light 自動推導）。

**共通風格（每段 prompt 已內含、不用另外加）**：教學用資訊圖、繁體中文（台灣用語）、直式 3:4 適合手機、深色科技風（背景近黑 `#0a0e14`、用綠 `#50fa7b` + 青 `#8be9fd` 當點綴 / 箭頭 / 外框）、左上角粗體白字標題、底部一句「☕ 一句話總結」、概念用流程圖 / 對照表 / 箭頭 / 配色區塊呈現（不是截圖）、字大留白足、新手一眼看懂、不要浮水印或亂碼英文。

**淺色版做法**：同一段 prompt、把「背景近黑 #0a0e14、綠 #50fa7b + 青 #8be9fd 點綴」整句換成 **「背景淺灰白 #f6f8fa、文字深色、點綴用深綠 #1f883d + 藍 #218bff」**、其餘不變。

> 💡 一致性：生完第一張後、之後每張 prompt 結尾都加「沿用前一張的配色、字體、版面比例」、整套圖才像同一系列。
